# 审查-实用性-概述与审查原则

> **来源：** 《专利审查指南》第二部分第五章 §1 引言 + §2 实用性的概念 + §3.1-3.2 审查原则与基准
> **对应法条：** 《专利法》第22条第1款、第4款
> **最新修订：** 无修改
> **关联页面：** [[实用性-法条-实用性的概念]]、[[实用性-判断标准与积极效果]]、[[审查-实用性-无实用性的情形]]

## 核心要点

实用性是授予专利权的三个必要条件之一，要求发明或者实用新型的主题必须**能够在产业上制造或者使用**，并且能够**产生积极效果**。实用性审查在新颖性和创造性审查**之前**进行，以申请日提交的整体技术内容为依据，与发明创造的产生方式或是否已经实施无关。

---

## 审查标准

### 一、引言（§1）

根据《专利法》第22条第1款的规定，授予专利权的发明和实用新型应当具备新颖性、创造性和实用性。因此，申请专利的发明和实用新型具备实用性是授予其专利权的必要条件之一。

### 二、实用性的概念（§2）

根据《专利法》第22条第4款，实用性是指发明或者实用新型申请的主题必须能够在**产业上制造或者使用**，并且能够产生**积极效果**。

#### 2.1 能够在产业上制造或者使用

授予专利权的发明或者实用新型，必须是能够解决技术问题，并且能够应用的发明或者实用新型。

| 申请类型 | 要求 |
|---------|------|
| 产品发明和实用新型 | 该产品必须在产业中能够制造，并且能够解决技术问题 |
| 方法发明（仅限发明） | 该方法必须在产业中能够使用，并且能够解决技术问题 |

"能够制造或者使用"是指发明或者实用新型的技术方案具有在产业中被制造或者使用的**可能性**，并不要求已经实际制造或使用。只要根据申请文件公开的内容，所属技术领域的技术人员能够确认该技术方案可以在产业中实现即可。

#### 2.2 产业的范围

所谓"产业"，包括：
- 工业
- 农业
- 林业
- 水产业
- 畜牧业
- 交通运输业
- 文化体育、生活用品和医疗器械等行业

**产业的界定标准**：产业的范围应当从广义上理解，不限于传统的制造业，凡是有组织、有规模地实施技术方案的行业均属产业范畴。即使某些技术方案仅在特定场景下使用（如航天、深海探测），只要其具有在对应产业中应用的可能性，即满足产业上的可制造或使用要求。

#### 2.3 技术方案的可实施性

在产业上能够制造或者使用的技术方案，是指**符合自然规律**、**具有技术特征**的任何可实施的技术方案。这些方案并不一定意味着使用机器设备，或者制造一种物品，还可以包括例如：
- 驱雾的方法；
- 将能量由一种形式转换成另一种形式的方法。

可实施性的判断基准是：所属技术领域的技术人员根据说明书和权利要求书记载的内容，能否在产业中重复实施该技术方案。若技术方案依赖于不可重现的随机因素，或者需要违背自然规律的条件才能实现，则不具备可实施性。

#### 2.4 积极效果的要求

能够产生积极效果，是指发明或者实用新型专利申请在提出申请之日，其产生的**经济、技术和社会的效果**是所属技术领域的技术人员**可以预料到的**。这些效果应当是积极的和有益的。

积极效果的判断要点：
- **可预料性**：积极效果不需要在说明书中通过实验数据严格证明，只要所属技术人员根据现有技术和申请文件可以合理预期即可；
- **有益性**：效果应当是对社会、经济或技术发展有利的，明显无益或脱离社会需要的技术方案不具备实用性；
- **与创造性的区别**：实用性所要求的积极效果侧重于"可预料的有益性"，而创造性要求的"显著的进步"则侧重于与现有技术相比的改进程度。

### 三、实用性的审查（§3）

#### 3.1 审查顺序

发明或者实用新型专利申请是否具备实用性，应当在新颖性和创造性审查**之前**首先进行判断。

```
审查顺序：实用性 → 新颖性 → 创造性
```

审查顺序的逻辑基础：实用性是专利权授予的最基本条件。若一项申请不具备实用性（如违背自然规律），则无论其是否新颖、是否具有创造性，均不应授予专利权。先进行实用性审查可以避免后续不必要的检索和审查工作，提高审查效率。

#### 3.2 审查原则（§3.1）

审查发明或者实用新型专利申请的实用性时，应当遵循下列原则：

**原则一：整体技术内容依据**

以申请日提交的说明书（包括附图）和权利要求书所公开的**整体技术内容**为依据，而不仅仅局限于权利要求所记载的内容。

这意味着，即使权利要求的文字描述存在瑕疵，只要说明书和附图中公开了完整的技术方案，且该技术方案具备实用性，就不能仅因权利要求的文字表述而否定其实用性。

**原则二：与创造方式和实施无关**

实用性与所申请的发明或者实用新型是**怎样创造出来的**或者**是否已经实施**无关。

| 无关因素 | 说明 |
|---------|------|
| 创造方式 | 无论通过灵感、反复试验还是系统研究获得，不影响实用性判断 |
| 是否已实施 | 即使尚未实际制造或使用，只要技术方案具有在产业中被制造或使用的可能性即可 |
| 成品率高低 | 产品的成品率低与不具备实用性有本质区别（详见下文） |

#### 3.3 审查基准概述（§3.2）

专利法第22条第4款所说的"能够制造或者使用"，是指发明或者实用新型的技术方案具有在产业中被制造或者使用的**可能性**。

满足实用性要求的技术方案须满足：
- **不能违背自然规律**；
- **应当具有再现性**。

因不能制造或者使用而不具备实用性是由**技术方案本身固有的缺陷**引起的，与说明书公开的程度无关。

### 四、不具备实用性的具体情形（§3.2.1-3.2.6）

审查实践中，以下六种情形属于不具备实用性的典型情况：

#### 4.1 无再现性（§3.2.1）

具有实用性的发明或者实用新型专利申请主题，应当具有再现性。反之，无再现性的发明或者实用新型专利申请主题不具备实用性。

**再现性**，是指所属技术领域的技术人员，根据公开的技术内容，能够重复实施专利申请中为解决技术问题所采用的技术方案。这种重复实施不得依赖任何随机的因素，并且实施结果应该是相同的。

**成品率低与不具有再现性的本质区别**：

| 情形 | 特征 | 是否具备实用性 |
|------|------|--------------|
| 成品率低 | 能够重复实施，但由于实施过程中未能确保某些技术条件（如环境洁净度、温度等）导致成品率低 | **具备** |
| 不具有再现性 | 在确保发明或者实用新型专利申请所需全部技术条件下，所属技术人员仍不可能重复实现该技术方案所要求达到的结果 | **不具备** |

#### 4.2 违背自然规律（§3.2.2）

具有实用性的发明或者实用新型专利申请应当符合自然规律。违背自然规律的发明或者实用新型专利申请是不能实施的，因此，不具备实用性。

审查员应当特别注意，那些违背能量守恒定律的发明或者实用新型专利申请的主题，例如**永动机**，必然是不具备实用性的。

其他常见违背自然规律的情形包括：
- 声称不消耗能量而持续对外做功的装置；
- 声称违背热力学定律的制冷/制热方法；
- 声称实现超光速通信的技术方案。

#### 4.3 利用独一无二的自然条件的产品（§3.2.3）

具备实用性的发明或者实用新型专利申请不得是由自然条件限定的独一无二的产品。利用特定的自然条件建造的自始至终都是不可移动的唯一产品不具备实用性。

例如，利用某处独一无二的自然地形建造的特定建筑物，因无法在其他地方复制，不具备实用性。

应当注意的是，不能因为上述利用独一无二的自然条件的产品不具备实用性，而认为其构件本身也不具备实用性。构成该产品的零部件如果具有通用性，仍可单独获得专利权。

#### 4.4 人体或者动物体的非治疗目的的外科手术方法（§3.2.4）

外科手术方法包括治疗目的和非治疗目的的手术方法。以治疗为目的的外科手术方法属于不授予专利权的客体；**非治疗目的的外科手术方法**，由于是以有生命的人或者动物为实施对象，无法在产业上使用，因此不具备实用性。

例如：
- 为美容而实施的外科手术方法；
- 采用外科手术从活牛身体上摘取牛黄的方法；
- 为辅助诊断而采用的外科手术方法，例如实施冠状造影之前采用的外科手术方法等。

#### 4.5 测量人体或者动物体在极限情况下的生理参数的方法（§3.2.5）

测量人体或者动物体在极限情况下的生理参数需要将被测对象置于极限环境中，这会对人或者动物的生命构成威胁，不同的人或者动物个体可以耐受的极限条件是不同的，需要有经验的测试人员根据被测对象的情况来确定其耐受的极限条件，因此这类方法无法在产业上使用，不具备实用性。

以下测量方法属于不具备实用性的情况：
- 通过逐渐降低人或者动物的体温，以测量人或者动物对寒冷耐受程度的测量方法；
- 利用降低吸入气体中氧气分压的方法逐级增加冠状动脉的负荷，并通过动脉血压的动态变化观察冠状动脉的代偿反应，以测量冠状动脉代谢机能的非侵入性的检查方法。

#### 4.6 无积极效果（§3.2.6）

具备实用性的发明或者实用新型专利申请的技术方案应当能够产生预期的积极效果。明显无益、脱离社会需要的发明或者实用新型专利申请的技术方案不具备实用性。

例如，一种方法的发明目的仅仅是为了产生污染环境的废弃物，该方法即属于无积极效果的情形。

### 五、实用性与说明书"能够实现"条款的衔接

实用性判断与专利法第26条第3款"能够实现"条款既有联系又有区别：

| 比较维度 | 实用性（法第22条第4款） | 能够实现（法第26条第3款） |
|---------|----------------------|------------------------|
| 关注焦点 | 技术方案本身是否具有在产业中应用的可能性 | 说明书是否充分公开，使技术人员能够实现 |
| 缺陷性质 | 技术方案固有的、不可克服的缺陷 | 公开不充分导致的可实施性问题 |
| 克服可能 | 一般无法通过修改克服 | 通常可以通过补充实验数据或修改说明书克服 |
| 判断依据 | 整体技术内容 | 说明书公开程度 |

**实务要点**：如果一项申请因说明书公开不充分而被认定不符合法第26条第3款，但该技术方案本身具备实用性，则申请人可以通过补充实验数据或修改说明书来克服该缺陷。反之，如果技术方案本身因违背自然规律等原因不具备实用性，则无论说明书如何修改，均无法获得专利权。

## 实务应对

### 1. 说明书的充分公开

代理人在撰写说明书时，应确保：
- 技术方案符合自然规律，不涉及永动机等违背基本物理定律的发明；
- 提供足够的实施例和具体实施方式，使本领域技术人员能够实现该发明；
- 在说明书中明确记载预期的技术效果和经济效果；
- 对于依赖特定工艺条件的方案，应详细记载工艺参数的范围和优选值。

### 2. 再现性的保障

- 对于涉及生物材料、特殊工艺条件等技术方案，应确保其可重复实施；
- 避免技术方案依赖于不可控制的随机因素；
- 如技术方案涉及特殊的原材料或设备，应在说明书中明确其来源或替代方案；
- 提供多个实施例以证明技术方案的可再现性。

### 3. 成品率与再现性的区分应对

在审查意见中，审查员有时会以"成品率低"为由质疑实用性。代理人应注意区分：
- **成品率低**：属于可以通过优化工艺条件解决的问题，不影响实用性。应通过补充实验数据或技术说明，证明成品率低是由可控因素导致，而非技术方案本身不可再现；
- **无再现性**：属于技术方案本身的固有缺陷，通常难以克服。若审查员的理由实质上是成品率问题，应据理力争。

### 4. 面对实用性质疑的应对

- 核实审查员否定实用性的理由是否属于法定的几种不具备实用性的情形；
- 补充实验数据或技术资料证明技术方案的可实施性；
- 强调实用性判断以整体技术内容为依据，而非仅看权利要求的文字；
- 若审查员将"公开不充分"与"不具备实用性"混淆，应明确指出二者的区别。

### 5. 跨领域发明的注意

- 对于医疗领域的发明，需注意区分治疗方法（不授予专利权）和非治疗目的的外科手术方法（不具备实用性）；
- 对于计算机软件相关的发明，需证明其在产业中的应用可能性；
- 对于涉及人体或动物体的发明，应避免将其归入极限测量或非治疗手术方法的范畴。

### 6. 积极效果的撰写策略

- 在说明书中预先记载技术方案可带来的经济、技术和社会效果；
- 效果描述应具体、可预期，避免空泛的宣称；
- 对于涉及环保、节能等领域的发明，应特别强调其社会有益性。

## 审查要点速查

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 技术方案违背自然规律（如永动机） | 不具备实用性 | §3.2.2 违背自然规律 |
| 技术方案符合自然规律且可实施 | 具备实用性 | §2 产业制造/使用可能性 |
| 产品尚未实际制造 | 不影响实用性 | §3.1(2) 是否已实施无关 |
| 发明通过偶然发现获得 | 不影响实用性 | §3.1(2) 创造方式无关 |
| 仅权利要求文字描述不完整但说明书充分 | 以整体技术内容判断 | §3.1(1) 整体技术内容依据 |
| 方法发明可在产业中使用 | 具备实用性 | §2 产业使用要求 |
| 成品率低但可重复实施 | 具备实用性 | §3.2.1 成品率与再现性的区别 |
| 技术方案依赖不可控制的随机因素 | 不具备实用性 | §3.2.1 无再现性 |
| 利用独一无二自然条件建造的不可移动产品 | 不具备实用性 | §3.2.3 独一无二的自然条件 |
| 非治疗目的的外科手术方法 | 不具备实用性 | §3.2.4 非治疗目的手术方法 |
| 测量人体/动物体极限生理参数的方法 | 不具备实用性 | §3.2.5 极限情况测量 |
| 明显无益、脱离社会需要的技术方案 | 不具备实用性 | §3.2.6 无积极效果 |
