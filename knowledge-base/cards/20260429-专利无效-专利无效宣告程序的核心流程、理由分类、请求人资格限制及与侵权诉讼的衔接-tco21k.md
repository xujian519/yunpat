# 专利无效宣告程序的核心流程、理由分类、请求人资格限制及与侵权诉讼的衔接机制是什么？

## 卡片内容

### 核心回答

专利无效宣告程序是一种行政纠正机制，旨在纠正专利授权过程中可能存在的错误。任何认为专利权授予不符合专利法规定的单位或个人，均有权请求专利复审委员会宣告该专利权无效。该程序不仅关注专利的实体性缺陷，也涵盖了程序上的义务。

### 1. 无效宣告制度概述

**制度目的：** 专利审查是一个复杂的技术活动，为确保审查结果的准确性，无效宣告程序设立是为了提供一种纠正机制，以期纠正错误的专利授权。

**启动方式：** 根据《专利法》第45条规定，自国务院专利行政部门公告授予专利权之日起，任何单位或个人均可请求专利复审委员会宣告该专利权无效。

### 2. 无效宣告的理由

无效宣告的理由主要分为实体性理由和程序性理由。

**实体性理由：** 主要针对专利不符合授权的实质要件，包括：
*   **保护客体缺陷：** 不属于专利法意义上的发明（如违反第2条）、违法或违反公德（第5条第1款）、或构成法定排除客体（如第25条）。
*   **三性缺陷：** 不具备实用性、新颖性或创造性（涉及第22条和第23条）。
*   **公开不充分：** 存在未充分公开的情况（如违反第26条第3款和第4款）。
*   **修改超范围：** 不符合专利法第33条的规定。

**程序性理由：** 包括违反保密审查规定（如第20条第1款）、违反遗传资源披露义务（如第5条第2款和第26条第5款），以及重复授权（如违反第9条）。

### 3. 请求人资格限制

**一般规则：** 原则上，任何人都可以对专利提出无效宣告请求，这旨在鼓励对不应获得授权的专利提出挑战，以清除技术进步的障碍。

**例外情形：** 《专利审查指南》对请求人资格设置了限制，包括但不限于：
1.  请求人不具备民事诉讼主体资格。
2.  以外观设计与他人在先取得的合法权利相冲突为由请求无效，但请求人无法证明其为在先权利人或利害关系人。
3.  专利权人针对其专利权提出无效宣告请求且请求宣告专利权全部无效，且提交的证据并非公开出版物或请求人不是共有专利权的所有专利权人。
4.  多个请求人共同提出无效宣告请求（除非是所有专利权人针对其共有专利权提出的除外）。

**专利权人自己提出无效请求：** 《专利审查指南》并未直接禁止专利权人自己提出无效请求，但禁止专利权人请求宣告专利权全部无效。许可权利人自己提出无效请求的目的是为专利权人提供机会，以便在无效程序中修改其权利要求。

### 4. 无效程序的中止

**中止情形：** 在专利复审委员会的无效程序中，如果涉及专利权属争议，根据原中国专利局第31号公告，应当先中止无效程序。

**滥用问题：** 实践中存在权利人滥用中止程序拖延专利被宣告无效时间的情况。例如，权利人可能通过引入第三方（“稻草人”）来提出专利权属争议，从而导致专利复审委中止无效宣告程序。如何有效防止权利人滥用此机制，是当前面临的难题。

### 5. 专利无效诉讼

**诉讼当事人：** 专利复审委员会对无效宣告请求进行审查后，将决定专利的有效性。如果请求人或专利权人对复审委员会的决定不服，可在收到通知后3个月内向法院提起诉讼。法院受理案件后，应当通知无效宣告请求程序的对方当事人作为第三人参加诉讼。

**诉讼结果处理：** 在法院最终撤销专利复审委员会的决定时，专利复审委员会需要重新作出决定。法院不能直接替代复审委员会的审查决定，这可能导致一个“马拉松式”的无效宣告诉讼循环（请求-决定-起诉-一审-二审-复审委重新决定-再起诉）。

### 6. 无效宣告与侵权程序的衔接

**侵权诉讼的中止：** 实践中，无效宣告请求人常是侵权诉讼中的被控侵权者。法院对侵权案件的审理是否应中止，取决于专利类型和审查结果：
*   **发明专利或经复审委维持的实用新型、外观设计专利：** 法院可以不中止诉讼。
*   **普通的实用新型与外观设计专利：** 法院应当中止诉讼程序。

**程序分离的背景：** 根据中国专利法，专利一经授权，只能通过无效宣告程序来消灭专利权，法院不能在侵权诉讼中直接宣告专利权无效。被告只能提出现有技术抗辩，不能提出专利权无效抗辩。这种制度安排基于行政行为公定力原理，但存在效率低下问题，可能导致侵权判决后专利被宣告无效，或侵权人拖延无效宣告程序，使专利权人无法及时获得救济。
