# 审查-化学-化学方法发明与实用新型

> **来源：** 《专利审查指南》第二部分第十章 §9
> **对应法条：** 《专利法》第二十五条第一款、第二十二条、第二十六条
> **最新修订：** 2023年修订（动植物个体及组成部分定义更新，植物品种定义调整）
> **关联页面：** [[审查-化学-不授予专利权的化学发明]]、[[审查-化学-化学产品的确认与制备]]、[[审查-不授权-动物和植物品种]]、[[创造性-化学领域发明]]、[[新颖性-化学领域发明的新颖性]]

---

## 核心要点

生物技术领域发明专利申请的审查涉及微生物、基因/DNA片段、蛋白质、载体、转化体、融合细胞、单克隆抗体等多种保护主题。审查中的特殊问题包括：生物材料的保藏要求、核苷酸/氨基酸序列表的提交、权利要求的特殊表征方式、遗传资源来源的披露等。2023年修订对"动物品种"和"植物品种"的定义进行了更新，明确了动物体细胞/组织器官、植物细胞/组织器官是否属于排除范围。

---

## 审查标准

### 1. 基本概念

**"生物材料"**：是指任何带有遗传信息并能够自我复制或者能够在生物系统中被复制的材料，如基因、质粒、微生物、动物和植物等。

**"动物""植物"的定义**：适用本部分第一章第4.4节的规定。其中所述的动物和植物可以是动物和植物的各级分类单位，如界、门、纲、目、科、属和种等。

### 2. 对要求保护的客体的审查

#### 2.1 依据专利法第五条的审查

**处于各形成和发育阶段的人体**：包括人的生殖细胞、受精卵、胚胎及个体，均不能被授予专利权。人类胚胎干细胞不属于此范围。

**违法获取或利用遗传资源**：违反法律、行政法规的规定获取或者利用遗传资源，并依赖该遗传资源完成的发明创造，不能被授予专利权。

#### 2.2 根据专利法第二十五条的审查

**微生物**：微生物（细菌、放线菌、真菌、病毒、原生动物、藻类等）不属于专利法第二十五条第一款第（四）项所列的情况。但未经人类技术处理而存在于自然界的微生物属于科学发现。只有当微生物经过分离成为纯培养物，并且具有特定的工业用途时，微生物本身才属于可给予专利保护的客体。

**基因或者DNA片段**：从自然界找到以天然形态存在的基因或DNA片段仅是发现，不能被授予专利权。但首次从自然界分离或提取出来的基因或DNA片段，其碱基序列是现有技术中不曾记载的，并能被确切地表征，且在产业上有利用价值，则该基因或DNA片段本身及其得到方法均属于可给予专利保护的客体。

**动物和植物个体及其组成部分（2023年修订）**：

- **动物方面**：
  - 动物的胚胎干细胞、动物个体及其各个形成和发育阶段（生殖细胞、受精卵、胚胎等）属于"动物品种"范畴，不能被授予专利权。
  - 动物的体细胞以及动物组织和器官（除胚胎以外）**不符合**"动物"的定义，因此**不属于**第二十五条第一款第（四）项规定的范畴。

- **植物方面（2023年修订后）**：
  - 可以借助光合作用合成碳水化合物、蛋白质来维系生存的植物的单个植株及其繁殖材料（如种子等），属于"植物品种"范畴，不能被授予专利权。
  - 植物的细胞、组织和器官如果不具有上述特性，则其不能被认为是"植物品种"，因此不属于第二十五条第一款第（四）项规定的范畴。

**转基因动物和植物**：转基因动物或植物本身仍然属于"动物品种"或"植物品种"的范畴，不能被授予专利权。

### 3. 说明书的充分公开

#### 3.1 生物材料的保藏

如果申请涉及的完成发明必须使用的生物材料是公众不能得到的，而申请人没有按规定进行保藏，或者未在申请日或最迟自申请日起四个月内提交保藏证明和存活证明的，审查员应当以申请不符合专利法第二十六条第三款的规定驳回该申请。

**需要在说明书记载的信息**：生物材料的分类命名、拉丁文学名、保藏单位名称及简称、地址、保藏日期和保藏编号。

**公众可得到、不要求保藏的情况**：
- 公众能从商业渠道买到的生物材料；
- 在各国专利局承认的保藏机构保藏的、在申请日前已公布或已授权的生物材料；
- 申请日前已在非专利文献中公开的，且申请人提供了保证从申请日起二十年内向公众发放生物材料的证明。

**国家知识产权局认可的保藏单位**：包括CGMCC（北京）、CCTCC（武汉）和GDMCC（广州）。

#### 3.2 涉及遗传工程的发明

对于涉及基因、载体、重组载体、转化体、多肽/蛋白质、融合细胞、单克隆抗体等本身的发明，说明书应包括：产品的确认、产品的制备、产品的用途和/或效果。

**单克隆抗体的特殊保藏要求**：当发明涉及满足特定条件的单克隆抗体时，即使记载了制备杂交瘤的方法，由于获得某一特定结果是随机的、不能重复再现，所述杂交瘤应当按规定进行保藏（申请人能提供足够证据证明可重复制备的除外）。

#### 3.3 核苷酸或氨基酸序列表

当发明涉及由10个或更多核苷酸组成的核苷酸序列，或者由4个或更多L-氨基酸组成的蛋白质或肽的氨基酸序列时，应当递交符合规定的序列表电子文件。序列表应作为说明书的一个单独部分。

### 4. 权利要求书

#### 4.1 基因权利要求

可按以下方式描述：
（1）直接限定其碱基序列；
（2）限定由所述基因编码的多肽或蛋白质的氨基酸序列；
（3）参见序列表或附图；
（4）"取代、缺失或添加"与功能相结合；
（5）在严格条件下"杂交"与功能相结合；
（6）通过功能、理化特性、起源或来源等描述。

#### 4.2 单克隆抗体权利要求

可以用结构特征限定（如CDR序列），也可以用产生它的杂交瘤来限定。

### 5. 新颖性、创造性和实用性

#### 5.1 新颖性

- **基因**：如果某蛋白质本身具有新颖性，则编码该蛋白质的基因也具有新颖性。
- **重组蛋白**：如果以单一物质形式被分离和纯化的蛋白质是已知的，由不同制备方法定义的具有同样氨基酸序列的重组蛋白不具有新颖性。
- **单克隆抗体**：如果抗原A具有与已知抗原A'相同的表位，则推定已知抗原A'的单克隆抗体就能与A结合，A的单克隆抗体不具有新颖性（除非申请人证明的确不同）。

#### 5.2 创造性

- **基因**：如果结构基因编码的蛋白质与已知蛋白质相比具有不同类型的或改善的性能，且现有技术没有给出技术启示，则具有创造性。已知蛋白质的编码基因通常不具有创造性，除非该基因具有特定碱基序列并具有预料不到的技术效果。
- **融合细胞**：已知亲代细胞融合得到的融合细胞通常不具有创造性，除非具有预料不到的技术效果。
- **单克隆抗体**：已知抗原、采用结构特征表征的单克隆抗体，如果与已知单克隆抗体在关键序列上明显不同，且现有技术没有给出技术启示，则具有创造性。
- **微生物**：与已知种分类学特征明显不同的微生物（新的种）具有创造性。

#### 5.3 实用性

**不能授予专利权的情况**：
- 由自然界筛选特定微生物的方法（大多不能重现）；
- 通过物理、化学方法进行人工诱变生产新微生物的方法（突变是随机的）。

除非申请人能给出充足证据证明方法可以重复实施或必然得到所需特性微生物。

### 6. 遗传资源来源的披露

**直接来源**：获取遗传资源的直接渠道（时间、地点、方式、提供者等）。

**原始来源**：遗传资源所属的生物体在原生环境中的采集地。

**审查要求**：审查员应确定发明是否依赖于遗传资源。如果依赖，审查是否提交了遗传资源来源披露登记表。未提交或不合规定的，应通知补正。

**重要注意**：登记表中的内容不属于原说明书和权利要求书记载的内容，不能作为判断说明书是否充分公开的依据，也不得作为修改的基础。

## 实务应对

**代理人视角：**

1. **动物体细胞/组织器官的保护**：2023年修订明确了动物体细胞以及动物组织和器官（除胚胎外）不属于"动物品种"范畴，可以申请专利。这对于再生医学、组织工程等领域的发明是重大利好，撰写时应充分利用这一规定。

2. **植物细胞/组织的保护**：植物细胞、组织和器官如果不具有光合作用等植物品种特性，则不属于排除范围。撰写涉及植物细胞培养、植物组织工程的发明时，应强调其不具有完整植株的特性。

3. **生物材料保藏的时效性**：必须在申请日或最迟自申请日起四个月内提交保藏证明和存活证明。建议在申请前完成保藏，避免因时间问题导致驳回。

4. **序列表的电子文件**：涉及10个以上核苷酸或4个以上氨基酸序列的发明必须提交序列表电子文件。注意书面提交的序列表与电子文件不一致时，以书面为准。

5. **遗传资源来源披露**：提前准备遗传资源来源披露登记表。如果原始来源无法提供（如种子库未记载），应陈述理由并提供相关书面证明。注意登记表内容不能用于修改说明书和权利要求书。

6. **单克隆抗体权利要求的撰写**：可采用结构特征（如CDR序列）限定，也可用杂交瘤保藏号限定。建议两种方式同时使用，以获得更全面的保护。

## 审查要点速查

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 自然界存在的微生物（未经技术处理） | 不授予专利权 | 属于科学发现 |
| 分离纯培养物+特定工业用途的微生物 | 可授予专利权 | 非科学发现 |
| 首次从自然界分离的基因/DNA（能表征+有产业价值） | 可授予专利权 | 非简单发现 |
| 天然形态存在的基因/DNA | 不授予专利权 | 属于"科学发现" |
| 动物胚胎干细胞、个体及各发育阶段 | 不授予专利权 | 属于"动物品种" |
| 动物体细胞、组织和器官（除胚胎外） | 可授予专利权 | 不符合"动物"定义 |
| 植物单个植株及繁殖材料 | 不授予专利权 | 属于"植物品种" |
| 不具光合作用等特性的植物细胞/组织 | 可授予专利权 | 不属于"植物品种" |
| 转基因动物或植物个体 | 不授予专利权 | 仍属"动物品种"/"植物品种" |
| 人类胚胎干细胞 | 可授予专利权 | 不属于"人体" |
| 未按规定保藏公众不能得到的生物材料 | 驳回 | 公开不充分 |
| 自然界筛选特定微生物的方法 | 不具备实用性 | 不能重现 |
| 人工诱变生产新微生物的方法 | 不具备实用性 | 随机突变 |
| 已知蛋白质的编码基因 | 通常无创造性 | 除非具有特定碱基序列+预料不到效果 |
| 已知种分类学特征明显不同的微生物（新种） | 具备创造性 | 结构差异显著 |
