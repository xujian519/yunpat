# 创造性-原理-辅助判断因素

> **来源：** 《专利法：原理与案例（第二版）》第5章 §4.1-4.4，页460-475
> **核心法条：** 《专利法》第22条第3款；《专利审查指南》第二部分第四章第5节
> **关联页面：** [[创造性-原理-基本原理]]、[[创造性-原理-技术启示的认定]]、[[创造性-辅助判断因素]]、[[审查-创造性-辅助判断因素]]

---

## 核心要点

创造性审查以"三步法"为主要工具，辅助证据用于反驳创造性不足的指控。辅助证据包括：商业上的成功、超出预期效果、克服技术偏见、长期渴望解决的技术难题等。它们与创造性之间不具有逻辑必然性，既非充分条件也非必要条件，仅供审查员或法官综合权衡。商业上的成功需证明由技术特征直接导致；超出预期效果在选择发明中尤为重要；克服技术偏见须是偏离客观事实的权威性认识。

---

## 1. 《专利审查指南》规定的其他因素

### 四类辅助因素

《专利审查指南》第二部分第四章第5节规定，当申请属于以下情形时，审查员应当予以考虑，不应轻易作出不具备创造性的结论：

1. **解决了长期渴望但始终未能成功的技术难题**
   - 例：农场牲畜无痛永久性标记——基于冷冻能使牲畜表皮着色这一发现而发明的冷冻"烙印"方法

2. **克服了技术偏见**
   - 技术偏见：某段时间内某技术领域中，技术人员对某技术问题普遍存在的、偏离客观事实的认识
   - 例：电动机换向器表面制出一定粗糙度的细纹，电流损耗反而更小——克服了"越光滑越好"的偏见

3. **取得了预料不到的技术效果**
   - 技术效果产生"质"的变化（新性能）或"量"的变化（超出预期）
   - 对所属技术领域的技术人员来说，事先无法预测或推理出来

4. **在商业上获得成功**
   - 商业成功必须是由于发明的技术特征直接导致的
   - 如因销售技术改进或广告宣传造成，不能作为判断创造性的依据

## 2. 商业上的成功

### 逻辑链条

商业上的成功与创造性之间的联系需要多层假设推导：
1. 一项技术能够取得商业上的成功 → 吸引很多人努力研发
2. 在发明人之前没有人成功 → 发明该技术方案本身不容易
3. 市场对产品作出积极反应 → 产品可能具有迎合消费者需要的创新成分

前提条件：技术可能的商业成功能够为潜在发明人所感知，且商业上的成功不是非技术因素（如广告）造成的。一旦前提不存在，商业上的成功就不再具有说服力。

### 案例：胡颖v.专利复审委员会（现专利复审和无效审理部）

**审理法院：** 最高人民法院
**案号：** （2012）行提字第8号

- **争议焦点：** "女性计划生育手术B型超声监测仪"实用新型专利的创造性，以及本专利产品是否获得商业上的成功
- **决定要点：** 商业成功是创造性判断的辅助性因素，应当持相对严格的标准。主张商业上成功需审查：（1）技术方案是否真正取得商业上的成功；（2）商业上的成功是否源于技术方案相比现有技术改进的技术特征。本案中，湖北、河南、黑龙江三省仅采购116台产品，从销售量来看不足以证明达到商业上成功的标准。二审判决认定商业成功证据不足。
- **启示：** 商业成功的证明标准较高，单纯的产品销售不能代表商业上的成功。必须对导致商业成功的原因进行详细分析，排除技术特征之外的其他因素。

### 举证责任分配

主流观点认为：
1. 权利人首先证明存在商业上的成功
2. 证明成功的商品是权利要求所指向的发明
3. 证明商业成功与发明特点有联系
4. 权利人**无须**排除其他可能因素——排除所有非发明因素过于苛刻
5. 相对人反驳时，由其自行举证证明成功是由外部因素（广告、加工技术等）造成

中国司法实践对商业上成功持较为谨慎甚至排斥的态度。

### 相关案例一览

- **韦国举v.专利复审委员会（现专利复审和无效审理部）**（北京一中院（2006）一中行初字第653号）：未举证商业业绩，也未证明业绩由技术特征直接导致
- **东莞清溪三中万宝表业厂v.专利复审委员会（现专利复审和无效审理部）**（北京一中院（2004）一中行初字第951号）：媒体报道真实性和客观性难以确定，且不能证明市场业绩唯一由技术方案带来
- **卡比斯特制药公司v.专利复审委员会（现专利复审和无效审理部）（II）**（最高人民法院（2012）知行字第75号）：证据仅能证明药品商业成功，不能证明成功由区别于现有技术的技术特征直接导致

## 3. 超出预期效果

### 基本规则

《专利审查指南》规定，发明取得了预料不到的技术效果，是指发明同现有技术相比，其技术效果产生"质"的变化（具有新的性能）或"量"的变化（超出人们预期）。这种变化对所属技术领域的技术人员来说，事先无法预测或推理出来。

### 超出预期效果的性质

超出预期效果本身并不一定能够证明发明存在创造性。如果现有技术对于发明方案给出了明确的启示，沿着该启示公众经过有限的选择甚至没有选择就得到了发明方案，则无论该方案的效果是否超出预期，都没有必要承认该发明的创造性。

因此，超出预期效果依然只是反驳发明不存在创造性的**辅助性证据**。只是在选择发明（从属发明）的案例中，它显得更为重要。

### 案例：新日铁住金不锈钢株式会社v.专利复审委员会（现专利复审和无效审理部）

**审理法院：** 北京市高级人民法院
**案号：** （2013）高行终字第1754号

- **争议焦点：** "耐间隙腐蚀性优良的铁素体系不锈钢"的创造性判断
- **决定要点：** 当本领域技术人员难以预测技术方案中组分及其含量的变化所带来的效果时，不能机械地适用三步法，应当根据技术方案是否取得预料不到的技术效果来判断创造性。本专利权利要求7的技术方案落入附件4的范围之内，属于选择发明。本专利实施例的最大侵蚀深度比对比例C16（落入附件4而未落入权利要求7）提高了44%，取得了预料不到的技术效果，具备创造性。
- **启示：** 在化学混合物/组合物领域，当效果不可预测时，预料不到的技术效果成为判断创造性的主要方法，而非仅是辅助因素。

### 案例：北京亚东生物制药有限公司v.专利复审委员会（现专利复审和无效审理部）（II）

**审理法院：** 最高人民法院
**案号：** （2013）知行字第77号

- **争议焦点：** 改变剂型后的药物组合物是否具有预料不到的技术效果
- **决定要点：** 在现有技术整体上存在明确技术启示的情况下，由制备方法所必然产生的技术效果并未超出本领域技术人员的合理预期。常规颗粒剂制法有利于保持药物活性、产品易于崩解等效果是在合理预期之内的。临床疗效的提高是制备方法本身特点导致的，且专利权人未举证证明超出了合理预期。
- **启示：** 现有技术中给出的技术启示越明确，技术效果的可预见性就越高。可预料到的效果不能证明创造性。

## 4. 克服技术偏见

### 基本规则

如果相同领域的专家事先存在某些偏见，使得熟练技术人员远离发明人所努力的方向，则发明人可以用这一事实来佐证其发明存在创造性。克服偏见是创造力的最好表现。

### 技术偏见的认定要件

1. 偏见必须是**具有指导意义的认识**，例如在教科书中肯定过的认识
2. 仅仅以人们以往没有采用某种技术方案为由说明采用这种方案克服了技术偏见是**不充分**的
3. 偏见是指偏离**客观事实**的认识——如果认识本身并未偏离客观事实，则不构成技术偏见

### 案例：专利复审委员会（现专利复审和无效审理部）第510号无效宣告请求审查决定

- **争议焦点：** 在盆景背后设置电器室的方案是否克服了技术偏见
- **决定要点：** 现有证据不足以说明在盆景制造业中已形成"任何情况下都不能在山石侧面设置电器室"的权威性认识。在山石一侧设置电器室会影响该侧观赏性的认识并未偏离客观事实——设置后该侧美观性确实受到影响。本方案与现有技术的差别仅仅是基于相同的技术认识、针对不同的使用要求而提出的常规设计方案，不能认为是克服了技术偏见。
- **启示：** "想到但因效果不理想而未采用"不等于"因技术偏见而未想到"。技术偏见须是对客观事实的偏离性认识，而非对效果的主观取舍。

## 判断流程

```
主张辅助证据证明创造性
├── 商业上的成功
│   ├── 是否真正取得商业成功？（市场份额而非单纯销售）
│   ├── 成功是否由区别技术特征导致？
│   └── 是否排除非技术因素？
├── 超出预期效果
│   ├── 现有技术是否给出明确技术启示？
│   │   ├── 是 → 效果可预见，不具创造性
│   │   └── 否 → 效果不可预见 → 可能具创造性
│   └── 选择发明中尤为重要
└── 克服技术偏见
    ├── 是否存在具有指导意义的权威性认识？
    ├── 该认识是否偏离客观事实？
    └── 两个条件均满足 → 可证明创造性
```
