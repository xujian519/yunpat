# 专利创造性判断中的辅助判断因素及其认定标准是什么？

## 卡片内容

### 核心概念
辅助判断因素是用于保障专利创造性评判结论准确性的辅助性判断依据，它主要围绕三个方面展开：一是解决人们长期以来渴望但未能攻克的（或未能成功解决的）技术难题；二是克服技术偏见；三是取得预料不到的技术效果。

### 一、人们一直渴望解决但始终未能成功的技术难题

*   **基本规则：** 发明若解决了人们长期渴望解决但未能成功的技术难题，则具有突出的实质性特点和显著的进步。然而，如果现有技术中已存在有效解决该问题的技术手段，则不构成此种情形。
*   **案例启示：** 第52281号复审决定中，对比文件已公开了基本相同的制备方法，解决了相同技术问题并达到了基本相同的效果，因此不属于“一直渴望解决”的情形。

### 二、技术偏见

*   **基本规则：** 技术偏见是指技术人员在特定领域内对某个技术问题存在普遍性的、偏离客观事实的认识，从而引导人们不去考虑某些可能性。发明克服技术偏见则具备创造性。
*   **认定标准：** 认定技术偏见至少需要满足以下三个条件：
    1.  在所属领域形成**普遍共识**。
    2.  该认识**偏离客观事实**。
    3.  该认识**阻碍了研究开发**。
*   **案例启示：** 仅因同类药物中某一种药物存在毒副作用，不能推定该类药物都具有相同毒副作用的偏见。技术偏见需要普遍共识、偏离事实和阻碍研究开发这三者缺一不可。

### 三、预料不到的技术效果

*   **基本规则：** 判断发明取得的技术效果是否超出了所属领域技术人员基于现有知识和能力所预期的效果。

    *   **结构改造中的预料不到效果：** 即使基于已知理论（如生物电子等排）进行结构改造，如果实际效果远超预期，仍可能构成预料不到的效果。
        *   *案例：* 嘌呤化合物的结构改造，其抑制癌细胞生长的活性比对比文件化合物强约40倍，且这种活性提高并非运用电子等排理论时就能预料到的。

    *   **组合的预料不到效果：** 组合的实际效果数量上的显著差异，可能构成预料不到的效果。
        *   *案例：* 特定配比的混合赋形剂，其抗吸湿效果（如吸湿率大幅下降）可能超出可预期范围。

    *   **常规变化不构成预料不到：** 常规手段的变化带来的可预期效果，不构成预料不到的技术效果。

    *   **新效果与已知效果的关联：** 如果新效果与已知效果之间存在关联，技术人员能够从已知效果预期新效果，则不属于预料不到。

    *   **补充实验数据的限制：** 补充实验数据证明的技术效果，必须能从原始申请文件公开的内容中得到，否则不能作为依据。

### 判断流程总结

| 辅助判断因素 | 判断标准 | 结论倾向 | 
| :--- | :--- | :--- |
| 长期渴望的技术难题 | 现有技术是否有有效手段 | 有效手段存在 → 不构成 | 
| 技术偏见 | 普遍共识 + 偏离事实 + 阻碍研究 | 三者缺一不可 | 
| 预料不到的技术效果 | 效果超越预期 / 常规变化 / 关联 / 补充数据限制 | 依据判断是否超预期 |
