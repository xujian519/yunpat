## 指导性案例

### 案例11：（2005）民三提字第2号【指导性案例】

- **审理法院：** 最高人民法院
- **裁判日期：** 2005-09-29
- **争议焦点：** 权利要求是否得到说明书支持；无效审查是否违反听证原则
- **裁判要点：** 最高法提审撤销了专利复审委员会无效决定。关于权利要求的说明书支持，最高法指出："权利要求通常由在说明书中公开的一个或多个实施例概括而成。权利要求的概括应当适当，使其保护范围正好适应说明书所公开的内容。"判断时应"从本领域技术人员角度出发"，不仅包括说明书直接描述的内容，"还包括所属技术领域的技术人员能够从说明书中公开的内容概括得出或者识别出来两种情况"。同时认定专利复审委员会在审查过程中违反了听证原则。
- **启示：** 最高法提审撤销无效决定的标志性案例，确立了权利要求说明书支持的"本领域技术人员"判断标准，明确无效审查必须遵守听证原则。

### 案例12：（2007）行提字第4号【指导性案例】

- **审理法院：** 最高人民法院
- **裁判日期：** 2008-07-14
- **争议焦点：** 禁止重复授权原则——同一申请人就同样的发明创造先后获得实用新型和发明专利是否构成重复授权
- **裁判要点：** 最高法作出权威阐释："'同样的发明创造只能被授予一项专利'，可以理解为是指同样的发明创造不能有两项或者两项以上的处于有效状态的专利权同时存在；在现行的制度安排下，同一申请人就同样的发明创造既申请实用新型专利又申请发明专利的，只要两项专利权不同时存在，就不违反禁止重复授权原则。"同时指出"双报"做法存在"有待完善的问题"，"应当通过修改有关规定和进一步明确有关规则加以解决"。
- **启示：** 确立了禁止重复授权"不同时存在即不违反"标准，确认发明与实用新型"双报"策略的合法性。

### 案例13：（2002）一中行初字第138号【指导性案例】

- **审理法院：** 北京市第一中级人民法院
- **裁判日期：** 2002-09-20
- **争议焦点：** 外观设计专利无效中"同样的发明创造"的认定标准
- **裁判要点：** 本案涉及摩托车外观设计专利无效纠纷。法院认定二者的不同点"在整体形状中均属于局部的变化，在视觉上均处于非醒目明显的部位"，两者在大部分主要组成部件"均采用了相同或者相近似的设计，使二者的整体外观形状产生了同样的视觉效果，尤其是在异时异地、间接对比的情况下，足以导致一般购买者从单纯外观设计的角度上对二者产生混淆和误认"，构成相近似的外观设计。
- **启示：** 外观设计相近似性判断采用"整体观察、综合判断"方法，以"异时异地、间接对比"条件下一般购买者是否产生混淆误认为标准。

## 实务要点速查

| 情形 | 裁判倾向 | 关键依据 | 代表案例 |
|------|---------|---------|---------|
| 销售定制化产品是否构成公开销售 | 一般认定构成公开，除非有充分证据证明技术内容未为公众得知 | 专利法第22条（现有技术） | （2024）最高法知行终794号 |
| 无效程序中修改权利要求引入新的术语 | 若该术语不能从原文件中直接地、毫无疑义地确定，认定修改超范围 | 专利法第33条 | （2019）最高法知行终145号 |
| 参数范围远超实施例实际数据（单侧限定） | 若说明书未描述参数与结构关系，认定得不到支持 | 专利法第26条第4款 | （2024）最高法知行终501号 |
| 含计算机程序的实用新型权利要求 | 若改进在于程序而非硬件构造，不属于实用新型保护客体 | 专利法第2条第3款 | （2023）最高法知行终576号 |
| 常规传感器类型的替换 | 认定为公知常识，不具备创造性 | 专利法第22条第3款 | （2024）最高法知行终444号 |
| 现有技术客观存在但未记载的技术问题 | 不阻碍改进动机的认定，本领域技术人员能认识即可 | 专利法第22条第3款 | （2023）最高法知行终825号 |
| 外观设计特征组合的"组合启示" | 视觉上可自然区分、拼合后外观可预期的，认定存在组合启示 | 专利法第23条第2款 | （2023）最高法知行终937号 |
| 实用新型与不同领域证据的创造性对比 | 功能和用途相同或相近即属相近领域，可选用 | 专利法第22条第3款 | （2023）最高法知行终1199号 |
| 实审阶段审查通过是否豁免无效审查 | 不豁免，无效程序是纠错机制 | 专利法第33条、第45条 | （2019）最高法知行终145号 |
| 隐含公开的认定 | 以本领域技术人员普通知识为基础，从整体内容合乎逻辑推导 | 专利法第22条第3款 | （2022）最高法知行终189号 |
| 口头审理中已审理的证据组合方式 | 当事人已陈述意见的，不构成程序违法 | 行政诉讼法、司法解释 | （2023）最高法知行终1199号 |
| 说明书未记载的技术效果主张 | 不作为确定实际解决技术问题的依据 | 专利法第22条第3款 | （2024）最高法知行终444号 |
