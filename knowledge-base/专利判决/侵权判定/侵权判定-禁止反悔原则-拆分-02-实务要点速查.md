# 实务要点速查

> **拆分自：** [[侵权判定-禁止反悔原则]]

---

## 核心要点

禁止反悔原则是专利侵权诉讼中限制等同原则适用的一项重要规则。专利申请人、专利权人在专利授权或者无效宣告程序中，通过对权利要求、说明书的修改或者意见陈述而放弃的技术方案，在侵权纠纷案件中不得又将其纳入专利权保护范围。该原则以诚实信用为基础，旨在防止专利权人"两头得利"——在授权确权程序中限缩保护范围以获得或维持专利权，在侵权诉讼中又试图将已放弃的技术方案重新纳入保护范围。

## 实务要点速查

| 情形 | 裁判倾向 | 关键依据 | 代表案例 |
|------|---------|---------|---------|
| 审查阶段三次陈述强调"仅选取"某类技术特征，诉讼中主张相同侵权包含其他方案 | 适用禁止反悔，不支持 | 专利权纠纷解释第六条；专利权纠纷解释二第十三条 | （2024）最高法知民终245号 |
| 为克服审查意见将"部分或全部显示"修改为"分页显示"并解释含义 | 构成放弃"部分显示"方案 | 专利权纠纷解释第六条 | （2021）最高法知民终1787号 |
| 无效程序中陈述与权利要求文义一致，未放弃任何方案 | 不适用禁止反悔 | 专利权纠纷解释第六条 | （2023）最高法知民终2170号 |
| 无效决定对技术特征的概括性描述，非专利权人限缩性陈述 | 不构成禁止反悔 | 专利权纠纷解释第六条 | （2022）最高法知民终977号 |
| 对技术特征测量方法的说明 | 不构成对特征含义的限缩 | 专利权纠纷解释第六条 | （2021）最高法知民终2426号 |
| 同日发明专利审查中的陈述用于限制同日实用新型 | 原则上不能直接适用 | 专利法第五十九条第一款 | （2022）最高法知民终154号 |
| 审查阶段为区别现有技术明确否定某技术方案 | 构成放弃，适用禁止反悔 | 专利权纠纷解释第六条 | （2019）最高法知民终684号 |
| 无效程序中对技术特征的解释被采纳并维持有效 | 构成限缩，适用禁止反悔 | 专利权纠纷解释第六条 | （2023）最高法知民终465号 |
| 区别技术特征系授权基础，被诉方案缺少该特征 | 不落入保护范围 | 专利权纠纷解释第七条 | （2023）最高法知民终1906号 |
| 权利要求的目的解释导致被诉方案与发明目的不同 | 不构成侵权 | 专利法第五十九条第一款 | （2020）最高法知民终1856号 |
| 限缩性陈述被审查机关明确否定 | 不导致技术方案的放弃 | 专利权纠纷解释二第十三条 | （2024）最高法知民终245号（引述规则） |
| 主张相同侵权但实质恢复限缩前保护范围 | 仍适用禁止反悔 | 诚实信用原则 | （2024）最高法知民终245号 |
