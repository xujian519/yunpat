## 🚀 核心功能

### 1. PatentWriterAgentWithMemory 类

**继承自**: `Agent<PatentWritingInput, PatentWritingOutput>`

**核心特性**:
- 🧠 自动初始化记忆层组件
- 🔍 RAG 增强检索（规划阶段）
- 💾 自动保存专利（执行阶段）
- 🗜️ Token 窗口压缩
- 📊 统计信息追踪

**初始化**:
```typescript
import { createPatentWriterAgentWithMemory } from './PatentWriterAgentWithMemory.js';
import { createDeepSeekModel } from '@yunpat/core';

const agent = await createPatentWriterAgentWithMemory({
  llm: createDeepSeekModel(process.env.DEEPSEEK_API_KEY),
  memoryConfig: {
    bgeApiKey: 'xj781102@',
    databaseUrl: 'postgres://yunpat:yunpat123@localhost:5432/yunpat',
    enableRAG: true,
    enableTokenWindow: true,
  },
});
```

**使用方法**:
```typescript
const result = await agent.run(patentInput, {});

// 语义搜索
const patents = await agent.searchPatents('深度学习图像识别', 5);

// 获取统计信息
const stats = await agent.getStats();
```

### 2. RAG 增强检索流程

```
用户输入专利信息
    ↓
构建查询（标题 + 领域 + 内容摘要）
    ↓
BGE-M3 向量化（1024维，~50ms）
    ↓
PostgreSQL 向量检索（HNSW索引，<50ms）
    ↓
返回 Top-5 相关专利
    ↓
构建 RAG 增强提示词
    ↓
LLM 生成专利（参考历史案例风格）
    ↓
自动保存到记忆库
```

### 3. Token 窗口压缩

**效果**: 降低 64% Token 使用

**工作原理**:
- 滑动窗口算法：保留最近 N 条重要消息
- Token 估算：中英文混合计数
- 重要性评分：智能过滤关键消息
- 语义摘要：压缩历史对话

**配置**:
```typescript
const tokenWindow = createTokenWindowManager({
  maxTokens: 4000,      // DeepSeek 上下文窗口
  reservedTokens: 500,  // 预留响应空间
  enableSummary: true,   // 启用语义摘要
});
```

## 💡 使用示例

### 示例 1：基础使用

```typescript
import { createPatentWriterAgentWithMemory } from './PatentWriterAgentWithMemory.js';
import { createDeepSeekModel } from '@yunpat/core';

// 1. 创建 Agent
const agent = await createPatentWriterAgentWithMemory({
  llm: createDeepSeekModel(process.env.DEEPSEEK_API_KEY),
});

// 2. 撰写专利
const result = await agent.run({
  title: '基于注意力机制的医学图像分析方法',
  field: '医学影像分析',
  applicant: '某科技公司',
  inventors: ['张三', '李四'],
  technicalDisclosure: '本发明提供了一种基于注意力机制的医学图像分析方法...',
  drawings: ['图1：流程图', '图2：结构图'],
}, {});

console.log(result.patentApplication.description);
```

### 示例 2：语义搜索历史专利

```typescript
// 搜索相似专利
const patents = await agent.searchPatents('深度学习图像识别', 5);

for (const patent of patents) {
  console.log(`标题: ${patent.metadata.title}`);
  console.log(`相似度: ${(patent.similarity * 100).toFixed(2)}%`);
  console.log(`内容: ${patent.content.slice(0, 100)}...`);
}
```

### 示例 3：获取统计信息

```typescript
const stats = await agent.getStats();

console.log('向量存储:', stats.vector.totalMemories, '条专利');
console.log('缓存命中率:', (stats.bge.cacheHitRate * 100).toFixed(2), '%');
console.log('Token 窗口:', stats.tokenWindow.availableTokens, '可用');
```
