# 创造性-化学领域发明

> **来源：** 《以案说法——专利复审、无效典型案例指引》第四章 §6，页205-215
> **核心法条：** 《专利法》第22条第3款
> **关联页面：** [[创造性-概述与三步法框架]]、[[创造性-辅助判断因素]]、[[创造性-不同类型发明的创造性]]、[[创造性-综合-用途发明]]、[[审查-化学-化学产品的确认与制备]]、[[审查-中药-中药领域创造性判断]]、[[审查-化学-化学领域创造性判断]]

---

## 核心要点

化学领域发明的创造性判断有其领域特点。化合物发明的关键在于结构与效果的关系；组合物发明需分析组分联用的指引和障碍；制备方法发明需围绕技术问题梳理区别特征；制药用途发明的核心是发现新适应症。晶体化合物与化学结构相同的已知产品通常被认为结构接近，需产生预料不到效果才具备创造性。

---

## 1. 化合物发明的创造性（§6.1）

### 基本规则

应选择用途或性能相同或相似、且化学结构尽可能相近的现有技术化合物作为最接近现有技术。化合物的结构与效果之间的关系是判断创造性的关键。

**结构接近 + 无预料不到效果 → 显而易见**

### 案例：第22284号无效决定（97197460.8）

- **争议焦点：** bis(POC)PMPA相对于bis(pom)PMPA是否具备创造性
- **决定要点：** 两者均为PMPA前药，具有相同的PMPA结构单元（决定活性），区别仅在于修饰基团为异丙基氧基vs叔丁基。碳酸酯和羧酸酯均是前药修饰的常用酯类基团，结构类似，可预期替换达到相似效果。实际解决的问题仅是"提供另一种适于口服的PMPA前药"。
- **启示：** 相同活性结构单元 + 常用修饰基团替换 + 无特殊效果 → 不具备创造性。

### 晶体化合物

化学结构相同或核心结构相同的情况下，晶体发明与已知化合物在结构上接近。晶体要具备创造性，应当具有预料不到的技术效果。

### 案例：第21950号无效决定（00802360.3）

- **争议焦点：** 6-羟基-2-萘甲酸的柱状晶体是否具备创造性
- **决定要点：** 晶体发明与已知化合物的化学结构相同。柱状是晶体常见外观形态，技术人员根据形态即可预期表观比重和流动性方面的改善。在无法确定产生了预料不到效果的情况下，不具备创造性。
- **启示：** 常见晶型 + 可预期的改善 → 不具备创造性。

### 基因和抗体

- 已知结构基因可自然获得的突变基因，若来源相同、性质功能相同 → 不具备创造性
- 已知抗原的单克隆抗体 → 通常不具备创造性
- 但若抗体产生了预料不到的效果（如对突变体更广泛的抗性）→ 具备创造性

### 案例：第92402号复审决定（200880010301.5）

- **争议焦点：** 针对已知表位的单克隆抗体是否具备创造性
- **决定要点：** 虽然针对已知抗原表位制备单抗是惯用手段，但涉案申请抗体对HCV天然突变体的Asn415突变较不敏感，具有更广泛的抗性，这是无法预期的效果。
- **启示：** 即使是已知抗原的抗体，若产生了不可预期的优异效果，仍具备创造性。

## 2. 组合物发明的创造性（§6.2）

### 基本规则

需分析现有技术是否给出将药物组分联用的指引，以及是否存在妨碍联用的技术障碍或偏见。联用后是否取得预料不到效果也是重要考量。

### 案例：第104654号复审决定（201120012109.1）

- **争议焦点：** 拉唑莫德+甲氨蝶呤联用是否具备创造性
- **决定要点：** 两者均为治疗类风湿性关节炎的已知药物，对比文件1已提示可以含有其他药物活性物质，给出了联用指引。将作用机制不同的同类药物联合追求更好疗效是常规选择。联用效果虽好于单独使用但未超出预期。
- **启示：** 同类疾病不同机制药物联用 + 已有联用指引 + 效果可预期 → 不具备创造性。

### 案例：第102739号复审决定（201010285313.8）

- **争议焦点：** 阿折地平+非马沙坦组合是否具备创造性
- **决定要点：** 对比文件1公开了阿折地平+奥美沙坦的联用效果，非马沙坦与奥美沙坦母核结构相同、作用机制相同（均为AT1受体拮抗剂）。技术人员能合理预期非马沙坦替代奥美沙坦同样能产生协同效果。
- **启示：** 同类机制药物替换 + 可预期效果 → 有动机替换，不具备创造性。

### 中药组合物

药味及药量变化若仅遵循中药方剂一般组方规律（随证加减、数方加减、相互代用、药对配伍等），且未使功能、疗效、治法治则产生实质变化 → 不具备创造性。

### 非药物组合物

判断时关注区别特征的组分在现有技术和发明中所起作用是否相同。若作用不同则不构成技术启示。

### 案例：第50230号复审决定（200780049586.9）

- **争议焦点：** 陶瓷珠粒作为消光剂是否可从对比文件2获得启示
- **决定要点：** 对比文件2中陶瓷微球是作为吸收辐射能量的材料，包含其的组合物是"透明或半透明"的。与涉案申请中利用折射率差异降低透明度的消光作用完全不同。
- **启示：** 同一物质在不同组合物中作用不同，不构成技术启示。

## 3. 制备方法发明的创造性（§6.3）

### 基本规则

制备方法权利要求通常包含产物、原料、步骤、工艺、条件等多个特征，需围绕技术问题梳理区别特征，分析各特征在解决问题中的角色。

### 案例：第27275号无效决定（200610021311.1）

- **争议焦点：** 柴黄制剂中黄芩清膏加碱调pH的步骤是否具备创造性
- **决定要点：** 附件7公开了黄芩提取物在碱性条件下溶解性较好。为改善附件3冲剂中黄芩提取物溶解性，技术人员有动机将附件7中的增溶方法应用于附件3。碱的种类、浓度、加水量等属于常规选择。
- **启示：** 跨剂型借鉴已知增溶方法 + 常规参数调整 → 不具备创造性。

## 4. 制药用途发明的创造性（§6.4）

### 基本规则

核心在于发现已知化合物可治疗新适应症。需分析新适应症与现有技术公开的适应症之间的关系，重点考察从现有技术能否推知该化合物可用于治疗新适应症。

### 案例：第78285号复审决定（200780047062.6）

- **争议焦点：** 副干酪乳杆菌用于防治远缘链球菌等引起的龋是否具备创造性
- **决定要点：** 对比文件1公开该菌用于防治变形链球菌引起的龋。对比文件2公开变形链球菌群包括远缘链球菌、仓鼠链球菌等，其表面蛋白抗原Ag I/II具有高度保守性。技术人员有理由将同样通过Ag I/II特异性结合的副干酪乳杆菌用于防治群内其他致龋菌。
- **启示：** 新适应症与已知适应症的致病机理高度关联 → 有动机扩展用途。

---

## 化学领域创造性判断要点

| 类型 | 判断关键 | 创造性肯定条件 |
|------|---------|--------------|
| 化合物 | 结构与效果关系 | 结构改变带来预料不到效果 |
| 晶体 | 与已知产品结构比较 | 晶型改变产生预料不到效果 |
| 基因/抗体 | 与已知序列/抗原比较 | 产生不可预期的优异效果 |
| 组合物 | 联用指引+效果预期 | 无指引/有障碍/预料不到效果 |
| 中药组合物 | 组方规律 | 超出常规加减的实质变化 |
| 制备方法 | 区别特征的角色 | 非常规的工艺创新 |
| 制药用途 | 新旧适应症关系 | 无法从现有技术推知新用途 |
