# 审查-外观-外观设计相同与实质相同的判断

> **来源：** 《专利审查指南》第四部分第五章 §1-5
> **对应法条：** 《专利法》第23条第1款、第64条第2款 / 《专利法实施细则》第2条第4款
> **最新修订：** 无重大修改
> **关联页面：** [[审查-外观-外观设计明显区别与现有设计组合]]、[[审查-外观-外观设计权利冲突与优先权]]、[[程序-无效宣告请求的审查]]

---

## 核心要点

外观设计专利的无效审查中，根据专利法第二十三条第一款判断新颖性时，需判断涉案专利与对比设计是否相同或实质相同。判断主体是涉案专利产品的一般消费者。判断采用单独对比、直接观察、整体观察综合判断等方式。涉案专利与对比设计的区别仅属于六种特定情形的，构成实质相同。

---

## 审查标准

### 一、现有设计（§2）

根据专利法第二十三条第四款的规定，现有设计是指申请日（有优先权的，指优先权日）以前在国内外为公众所知的设计。现有设计包括申请日以前在国内外出版物上公开发表过、公开使用过或者以其他方式为公众所知的设计。

**惯常设计**：现有设计中一般消费者所熟知的、只要提到产品名称就能想到的相应设计。例如，提到包装盒就能想到其有长方体、正方体形状的设计。

### 二、判断客体（§3）

在对外观设计专利进行审查时，被请求宣告无效的外观设计专利简称涉案专利，与涉案专利进行比较的判断客体简称对比设计。

在确定判断客体时，对于涉案专利，除应当根据外观设计专利的图片或者照片进行确定外，还应当根据简要说明中是否写明请求保护色彩、"不限定边界"等内容加以确定。

**涉案专利的六种类型**：

1. **单纯形状的外观设计**：无图案且未请求保护色彩的产品的形状设计。
2. **单纯图案的外观设计**：未请求保护色彩并且不限定边界的平面产品的图案设计。
3. **形状和图案结合的外观设计**：未请求保护色彩的产品的形状和图案设计。
4. **形状和色彩结合的外观设计**：请求保护色彩的无图案产品的形状和色彩设计。
5. **图案和色彩结合的外观设计**：请求保护色彩的并且不限定边界的平面产品的图案和色彩设计。
6. **形状、图案和色彩结合的外观设计**：请求保护色彩的产品的形状、图案和色彩设计。

### 三、判断主体（§4）

在判断外观设计是否符合专利法第二十三条第一款、第二款规定时，应当基于涉案专利产品的一般消费者的知识水平和认知能力进行评价。

作为某种类外观设计产品的一般消费者应当具备下列特点：

1. 对涉案专利申请日之前相同种类或者相近种类产品的外观设计及其常用设计手法具有常识性的了解。常用设计手法包括设计的转用、拼合、替换等类型。
2. 对外观设计产品之间在形状、图案以及色彩上的区别具有一定的分辨力，但不会注意到产品的形状、图案以及色彩的微小变化。

### 四、根据专利法第二十三条第一款的审查（§5）

#### 4.1 判断基准

**外观设计相同（§5.1.1）**

外观设计相同，是指涉案专利与对比设计是相同种类产品的外观设计，并且涉案专利的全部外观设计要素（形状、图案、色彩）与对比设计的相应设计要素相同。

如果涉案专利与对比设计仅属于常用材料的替换，或者仅存在产品功能、内部结构、技术性能或者尺寸的不同，而未导致产品外观设计的变化，二者仍属于相同的外观设计。

**相同种类产品**是指用途完全相同的产品。对于局部外观设计，相同种类产品是指产品的用途和该局部的用途均相同的产品。在确定产品的种类时，可以参考产品的名称、国际外观设计分类以及产品销售时的货架分类位置，但是应当以产品的用途是否相同为准。

**外观设计实质相同（§5.1.2）**

外观设计实质相同的判断仅限于相同或者相近种类的产品外观设计。对于产品种类不相同也不相近的外观设计，不进行实质相同的比较和判断。

**相近种类的产品**是指用途相近的产品。例如，玩具和小摆设的用途是相近的。当产品具有多种用途时，如果其中部分用途相同，而其他用途不同，则二者应属于相近种类的产品。对于局部外观设计，应综合考虑产品的用途和该局部的用途。

如果一般消费者经过对涉案专利与对比设计的整体观察可以看出，二者的区别**仅属于**下列情形，则涉案专利与对比设计实质相同：

1. 其区别在于施以一般注意力不易察觉到的局部的细微差异（如百叶窗仅有具体叶片数不同）。
2. 其区别在于使用时不容易看到或者看不到的部位，但有证据表明在不容易看到部位的特定设计对于一般消费者能够产生引人瞩目的视觉效果的情况除外。
3. 其区别在于将某一设计要素整体置换为该类产品的惯常设计的相应设计要素（如将饼干桶的形状由正方体置换为长方体）。
4. 其区别在于将对比设计作为设计单元按照该种类产品的常规排列方式作重复排列或者将其排列的数量作增减变化（如将影院座椅成排重复排列）。
5. 其区别在于互为镜像对称。
6. 其区别在于局部外观设计要求保护部分在产品整体中的位置和/或比例关系的常规变化。

#### 4.2 判断方式

**单独对比（§5.2.1）**

一般应当用一项对比设计与涉案专利进行单独对比，而不能将两项或者两项以上对比设计结合起来与涉案专利进行对比。

例外情形：
- 涉案专利包含有若干项具有独立使用价值的产品的外观设计的，可以用不同的对比设计与其所对应的各项外观设计分别进行单独对比。
- 涉案专利是由组装在一起使用的至少两个构件构成的产品外观设计的，可以将与其构件数量相对应的明显具有组装关系的构件结合起来作为一项对比设计与涉案专利进行对比。

**直接观察（§5.2.2）**

在对比时应当通过视觉进行直接观察，不能借助放大镜、显微镜、化学分析等其他工具或者手段进行比较。

**仅以产品的外观作为判断的对象（§5.2.3）**

- 在对比时应当仅以产品的外观作为判断的对象，考虑产品的形状、图案、色彩这三个要素产生的视觉效果。
- 在涉案专利仅以部分要素限定其保护范围的情况下，其余要素在与对比设计比较时不予考虑。
- 在涉案专利为产品零部件的情况下，仅将对比设计中与涉案专利相对应的零部件部分作为判断对象。
- 对于外表使用透明材料的产品而言，通过人的视觉能观察到的其透明部分以内的形状、图案和色彩，应当视为该产品的外观设计的一部分。

**整体观察、综合判断（§5.2.4）**

对比时应当采用整体观察、综合判断的方式。以一般消费者为判断主体，整体观察涉案专利与对比设计，确定两者的相同点和区别点，判断其对整体视觉效果的影响，综合得出结论。

- **确定对比设计公开的信息**（§5.2.4.1）：依据一般消费者的认知能力来确定。根据已公开内容即可推定出产品其他部分或其他变化状态的外观设计的，也视为已经公开。例如在对称情况下，公开了一个对称面，其余对称面也视为公开。
- **确定涉案专利**（§5.2.4.2）：应当以授权文本中的图片或者照片表示的外观设计为准。简要说明可以用于解释图片或者照片。对于局部外观设计，应以要求保护部分为准，并考虑该部分在所示产品中的位置和比例关系。

**组件产品和变化状态产品的判断（§5.2.5）**

- **组装关系唯一的组件产品**：以组合状态下的整体外观设计为对象进行判断。
- **组装关系不唯一的组件产品**：以所有单个构件的外观为对象进行判断。
- **各构件之间无组装关系的组件产品**：以所有单个构件的外观为对象进行判断。
- **变化状态产品**：对比设计中产品在不同状态下的外观设计均可用作比较对象；涉案专利应当以其使用状态所示的外观设计作为比较对象，判断结论取决于对各种使用状态的综合考虑。

**设计要素的判断（§5.2.6）**

- **形状的判断**：圆形和三角形、四边形相比，形状有较大差异，通常不能认定为实质相同（惯常设计除外）。包装盒类产品应以使用状态下的形状为判断依据。
- **图案的判断**：图案不同包括题材、构图方法、表现方式及设计纹样等因素的不同。题材相同但构图方法、表现方式、设计纹样不相同的，通常不构成实质相同。产品外表出现的包括产品名称在内的文字和数字应当作为图案予以考虑，而不应当考虑字音、字义。
- **色彩的判断**：根据颜色的色相、纯度和明度三个属性以及两种以上颜色的组合、搭配进行综合判断。单一色彩的外观设计仅作色彩改变，两者仍属于实质相同。

**抵触申请**：

在涉案专利申请日以前任何单位或者个人向专利局提出并且在申请日以后（含申请日）公告的同样的外观设计专利申请，称为抵触申请。判断对比设计是否构成抵触申请时，应当以对比设计所公告的专利文件全部内容为判断依据。

## 实务应对

**代理人视角：**

1. **一般消费者的确定**：在无效宣告程序中，准确定义"一般消费者"至关重要。不同产品的消费者群体不同，其知识水平和认知能力也不同。应结合产品的实际消费群体来确定一般消费者的特征。

2. **对比设计的选取**：应选择与涉案专利相同种类或相近种类的产品外观设计作为对比设计。注意单独对比原则，不能将多项对比设计结合起来对比（除非是组装关系唯一的组件产品）。

3. **整体观察综合判断的运用**：在分析区别点时，应从一般消费者的角度判断区别点对整体视觉效果的影响。使用时容易看到的部位的变化通常比不容易看到的部位的变化更具有显著影响。

4. **惯常设计的论证**：如果涉案专利的区别点在于惯常设计，应积极举证证明该设计为惯常设计，以论证构成实质相同。

5. **透明材料的处理**：对于使用透明材料的产品，透明部分以内的形状、图案和色彩应作为外观设计的一部分考虑，这在实务中容易被忽略。

## 审查要点速查

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 产品用途不同 | 不属于相同或相近种类，不进行实质相同比较 | §5.1.2 |
| 区别仅为使用时不易看到的部位 | 实质相同（引人瞩目的除外） | §5.1.2(2) |
| 区别为镜像对称 | 实质相同 | §5.1.2(5) |
| 区别为惯常设计的替换 | 实质相同 | §5.1.2(3) |
| 仅色彩改变的单一色彩设计 | 实质相同 | §5.2.6.3 |
| 将两项以上对比设计结合对比 | 不允许（单独对比原则） | §5.2.1 |
| 透明产品内部通过视觉可观察到 | 作为外观设计的一部分考虑 | §5.2.3 |
