# 客体-原理-治疗方法

> **来源：** 《专利法：原理与案例（第二版）》第2章 §5，崔国斌著
> **核心法条：** 《专利法》第25条第1款第（3）项
> **关联页面：** [[客体-原理-基本原理]]

---

## 核心要点

疾病的诊断和治疗方法不授予专利权，但为实施诊断和治疗而使用的器械、仪器、试剂等可获得专利保护。

---

## 1. 排除的范围

### 1.1 法定排除

《专利法》第25条第1款第（3）项明确规定，疾病的诊断和治疗方法不授予专利权。

### 1.2 排除的理由

**人道主义考虑：**
- 疾病的诊断和治疗方法直接关系到人的生命健康
- 如果对这些方法授予专利权，可能导致治疗方法的高昂价格
- 影响公众获得基本的医疗服务

**公共利益：**
- 疾病的诊断和治疗方法是人类的基本需求
- 专利保护可能导致对医疗方法的垄断
- 不符合社会公共利益

**职业性质：**
- 医生在诊断和治疗过程中需要自由地选择和使用各种方法
- 专利保护可能限制医生的职业自主权
- 不利于医学的发展和创新

---

## 2. 诊断方法的排除

### 2.1 诊断方法的定义

疾病的诊断方法是指为确定疾病的原因、性质或状态，或者为监测疾病的发展、治疗效果等而对人体或动物体实施的检查、检测、分析等方法。

### 2.2 典型类型

以下属于疾病的诊断方法：
1. **体格检查方法**：如触诊、听诊、叩诊
2. **影像学诊断方法**：如X光、CT、MRI的诊断方法
3. **实验室诊断方法**：如血液检测、尿液检测的诊断方法
4. **病理诊断方法**：如组织病理检查的诊断方法

### 2.3 判断要素

**直接目的：**
- 是否以诊断疾病为目的？
- 是否确定疾病的原因、性质或状态？
- 是否监测疾病的发展或治疗效果？

**实施对象：**
- 是否针对人体或动物体？
- 是否直接作用于人体或动物体？

**方法性质：**
- 是否为检查、检测、分析等方法？
- 是否涉及医疗判断？

---

## 3. 治疗方法的排除

### 3.1 治疗方法的定义

疾病的治疗方法是指为消除疾病、缓解症状、预防疾病、恢复健康等而对人体或动物体实施的方法。

### 3.2 典型类型

以下属于疾病的治疗方法：
1. **药物治疗方法**：如药物的使用方法、给药方案
2. **手术治疗方法**：如外科手术方法
3. **物理治疗方法**：如理疗、按摩
4. **心理治疗方法**：如心理咨询方法

### 3.3 判断要素

**直接目的：**
- 是否以治疗疾病为目的？
- 是否消除疾病、缓解症状、预防疾病？
- 是否恢复健康？

**实施对象：**
- 是否针对人体或动物体？
- 是否直接作用于人体或动物体？

**方法性质：**
- 是否涉及治疗操作？
- 是否涉及医疗干预？

---

## 4. 例外情形

### 4.1 医疗器械和仪器

为实施诊断和治疗而使用的医疗器械和仪器可以获得专利保护。

**可专利的情形：**
1. 医疗器械：如手术器械、诊断设备
2. 医疗仪器：如医疗成像设备、监护设备
3. 医疗试剂：如诊断试剂、检测试剂盒

### 4.2 药物和化合物

用于诊断和治疗的药物和化合物可以获得专利保护。

**可专利的情形：**
1. 药物化合物：如新的化学药物
2. 药物组合物：如复方制剂
3. 药物制剂：如新的药物剂型

### 4.3 非医疗目的的应用

以非医疗目的对人体或动物体实施的方法，可以获得专利保护。

**可专利的情形：**
1. 美容方法：如美容护肤方法
2. 健身方法：如体育锻炼方法
3. 保健方法：如预防性保健方法

## 5. 判断流程

### 5.1 诊断方法判断

```
诊断方法审查
├── 是否以诊断疾病为目的？
│   ├── 否 → 不属于诊断方法 ✓
│   └── 是 → 继续
├── 是否针对人体或动物体？
│   ├── 否 → 不属于诊断方法 ✓
│   └── 是 → 继续
├── 是否为检查、检测、分析等方法？
│   ├── 否 → 不属于诊断方法 ✓
│   └── 是 → 继续
├── 是否涉及医疗判断？
│   ├── 否 → 不属于诊断方法 ✓
│   └── 是 → 属于诊断方法 ✗
└── 结论
    ├── 属于诊断方法 → 不授予专利权 ✗
    └── 不属于诊断方法 → 可授予专利权 ✓
```

### 5.2 治疗方法判断

```
治疗方法审查
├── 是否以治疗疾病为目的？
│   ├── 否 → 不属于治疗方法 ✓
│   └── 是 → 继续
├── 是否针对人体或动物体？
│   ├── 否 → 不属于治疗方法 ✓
│   └── 是 → 继续
├── 是否涉及治疗操作？
│   ├── 否 → 不属于治疗方法 ✓
│   └── 是 → 继续
├── 是否涉及医疗干预？
│   ├── 否 → 不属于治疗方法 ✓
│   └── 是 → 属于治疗方法 ✗
└── 结论
    ├── 属于治疗方法 → 不授予专利权 ✗
    └── 不属于治疗方法 → 可授予专利权 ✓
```

## 6. 审查实务中的注意事项

### 6.1 目的分析

重点分析发明的直接目的：
- 不能仅仅因为应用于医疗领域就认定为诊断或治疗方法
- 需要分析发明的直接目的是诊断、治疗，还是其他目的

### 6.2 对象分析

重点分析发明的实施对象：
- 是否直接作用于人体或动物体？
- 是否通过中间媒介间接作用于人体或动物体？
- 是否涉及对人体或动物体的直接干预？

### 6.3 方法分析

重点分析方法的性质：
- 是否涉及医疗判断和医疗干预？
- 是否属于医疗专业活动？
- 是否涉及医疗技术和医疗决策？

---

## 7. 边界问题的处理

### 7.1 诊断与检测的区分

**诊断方法：**
- 以诊断疾病为目的
- 涉及医疗判断
- 直接作用于人体或动物体

**检测方法：**
- 以获取数据为目的
- 不涉及医疗判断
- 可能不直接作用于人体或动物体

### 7.2 治疗与保健的区分

**治疗方法：**
- 以治疗疾病为目的
- 涉及医疗干预
- 直接作用于人体或动物体

**保健方法：**
- 以预防疾病或促进健康为目的
- 不涉及医疗干预
- 通常不属于医疗专业活动

---

## 判断流程

```
诊断与治疗方法综合审查
├── 第一步：分析发明目的
│   ├── 诊断疾病 → 可能属于诊断方法 ✗
│   ├── 治疗疾病 → 可能属于治疗方法 ✗
│   ├── 获取数据 → 可能属于检测方法 ✓
│   └── 其他目的 → 可能可授予专利权 ✓
├── 第二步：分析实施对象
│   ├── 人体或动物体 → 可能属于诊断或治疗方法 ✗
│   ├── 医疗器械、试剂 → 可授予专利权 ✓
│   └── 其他对象 → 可能可授予专利权 ✓
├── 第三步：分析方法性质
│   ├── 涉及医疗判断 → 可能属于诊断或治疗方法 ✗
│   ├── 涉及医疗干预 → 可能属于治疗方法 ✗
│   ├── 纯技术方法 → 可授予专利权 ✓
│   └── 其他方法 → 可能可授予专利权 ✓
└── 结论
    ├── 诊断或治疗方法 → 不授予专利权 ✗
    └── 其他情形 → 可授予专利权 ✓
```
