# 如何界定专利新颖性判断中“实质上相同”的标准及其在不同技术特征替换（如惯用手段的直接置换或等同替代）中的适用性？

## 卡片内容

### 1. “实质上相同”的标准

新颖性判断中，要求对比文件与申请文件相比，不仅技术方案要相同，还必须综合考量以下四个要素才能认定为“实质上相同”：

1. **技术领域**相同。
2. **所解决的技术问题**相同。
3. **技术方案**实质上相同。
4. **预期效果**相同。

**关键理解：** 审查标准并非要求对比文件中明确撰写了与权利要求完全一致的技术问题和预期效果，而是要求本领域技术人员判断，二者在技术领域、技术问题、技术方案和预期效果上是相同的。

**简化原则：** 如果技术方案在文字表述上完全一致，则实际上无需再深入审查其他三个要素。

### 2. 惯用手段的直接置换

如果保护的发明与对比文件的区别仅仅在于**所属技术领域的惯用手段的直接置换**，则该发明不具备新颖性。

**实践要点：** 这种置换的判断标准依赖于本领域技术人员的知识水平。例如，如果对比文件使用“螺钉”固定，而申请仅改用“螺栓”固定，则可能被认定为惯用手段的直接置换，不构成新颖性障碍。

**相关案例：**
*   **刘信中v.专利复审委员会：** 争议焦点在于磁钢与手柄的螺接方法，区别仅在于使用“螺钉”或“螺丝”，被认定为惯用手段的直接置换，二者技术方案实质相同。

### 3. 模糊新颖性与创造性审查的界限

在实践中，审查标准容易与创造性审查的界限产生模糊：

*   **等同替代：** 如果后申请的发明中的技术特征只是在先申请中相对应的技术特征的**等同替代**，则该申请发明不具备新颖性。
    *   **相关案例：** 东泰工业v.专利复审委员会，涉及皮带传动与链条传动，两者属于惯用手段的直接替换，不具备新颖性。

*   **公知技术常识：** 某些技术手段的替换属于公知技术常识，即使区别存在，也可能被认定为惯用的直接置换。
    *   **相关案例：** 黄孟柱v.专利复审委员会，涉及管道密封连接，不同密封方式（如轴向和径向密封）属于公知技术常识，可以相互替换，不具备新颖性。

### 4. 审查基准概要

*   **完全相同或简单文字变换：** 不具备新颖性，应理解为可以从对比文件中直接确定的技术内容。
*   **概念层级：** 下位概念破坏上位概念的新颖性（例如，用铜制成破坏用金属制成的新颖性），但上位概念不破坏下位概念的新颖性。
*   **数值与范围：** 数值范围的重叠或共同端点会破坏新颖性；数值范围的端点破坏会破坏离散数值的新颖性。
*   **特征类型：**
    *   **性能/参数特征：** 如果隐含了特定结构/组成，则有新颖性；否则可推定无新颖性。
    *   **用途特征：** 如果用途由产品固有特性决定且未改变结构/组成，则无新颖性。
    *   **制备方法特征：** 如果方法导致产品结构/组成不同，则有新颖性。
