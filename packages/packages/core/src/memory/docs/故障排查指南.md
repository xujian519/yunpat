## 常见问题

### 1. 数据库连接问题

#### 问题：无法连接到 PostgreSQL

**错误信息**:

```
Error: connect ECONNREFUSED 127.0.0.1:5432
```

**可能原因**:

1. PostgreSQL 服务未启动
2. 数据库地址或端口配置错误
3. 防火墙阻止连接

**解决方案**:

```bash
# 1. 检查服务状态
sudo systemctl status postgresql

# 2. 启动服务
sudo systemctl start postgresql

# 3. 验证连接
psql -U yunpat -d yunpat -c "SELECT 1;"

# 4. 检查配置
echo $DATABASE_URL
```

---

### 2. pgvector 扩展问题

#### 问题：pgvector 扩展不可用

**错误信息**:

```
Error: type "vector" does not exist
```

**可能原因**:

1. pgvector 扩展未安装
2. 扩展未启用
3. 版本不兼容

**解决方案**:

```bash
# 1. 检查扩展是否安装
psql -U yunpat -d yunpat -c "SELECT * FROM pg_available_extensions WHERE name = 'vector';"

# 2. 安装扩展
CREATE EXTENSION IF NOT EXISTS vector;

# 3. 验证
SELECT * FROM pg_extension WHERE extname = 'vector';
```

### 3. BGE-M3 服务问题

#### 问题：BGE-M3 服务无响应

**错误信息**:

```
Error: fetch failed
```

**可能原因**:

1. BGE-M3 服务未启动
2. 端口被占用
3. API Key 无效

**解决方案**:

```bash
# 1. 检查服务状态
curl http://localhost:8009/health

# 2. 重启服务
pkill -f bge-m3
python -m bge_m3.server --port 8009

# 3. 检查端口占用
lsof -i :8009

# 4. 测试 API
curl -X POST http://localhost:8009/embed \
  -H "Content-Type: application/json" \
  -d '{"texts": ["test"]}'
```

### 4. 向量维度不匹配

#### 问题：向量维度不匹配

**错误信息**:

```
Error: 向量维度不匹配：期望 1024，实际 768
```

**可能原因**:

1. 使用了错误的嵌入模型
2. 配置不一致

**解决方案**:

```typescript
// 检查向量维度
const bge = createBGEM3Client()
const embedding = await bge.embed('test')
console.log('向量维度:', embedding.length) // 应该是 1024

// 配置正确的维度
const vectorStore = new PostgresVectorStore({
  databaseUrl: process.env.DATABASE_URL,
  vectorDimension: 1024, // 必须匹配
})
```

### 5. 内存溢出

#### 问题：内存占用过高

**错误信息**:

```
Error: JavaScript heap out of memory
```

**可能原因**:

1. 缓存无限制增长
2. 向量数据过多
3. 内存泄漏

**解决方案**:

```typescript
// 1. 配置缓存大小
const bge = createBGEM3Client({
  cacheMaxSize: 1000, // 限制缓存大小
})

// 2. 定期清理
setInterval(async () => {
  const deleted = await memory.compress(1000)
  console.log(`清理了 ${deleted} 条记忆`)
}, 60000) // 每分钟清理

// 3. 归档旧数据
await vectorStore.archiveOldMemories(30) // 归档 30 天前的数据
```

## 调试技巧

### 1. 启用详细日志

```typescript
// 设置环境变量
process.env.LOG_LEVEL = 'debug'

// 使用调试模式
const DEBUG = true
if (DEBUG) {
  console.log('记忆存储:', await memory.getAll())
  console.log('历史记录:', memory.getHistory())
}
```

### 2. 性能分析

```typescript
// 测量执行时间
console.time('向量搜索')
const results = await vectorStore.search(embedding, 10)
console.timeEnd('向量搜索')

// 输出: 向量搜索: 45.123ms
```

### 3. 数据验证

```typescript
// 验证向量格式
function validateEmbedding(embedding: number[]): boolean {
  if (!Array.isArray(embedding)) return false
  if (embedding.length !== 1024) return false
  if (embedding.some((n) => typeof n !== 'number' || isNaN(n))) return false
  return true
}

// 使用
if (!validateEmbedding(embedding)) {
  throw new Error('向量格式无效')
}
```
