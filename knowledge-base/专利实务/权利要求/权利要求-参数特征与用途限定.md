# 权利要求 — 参数特征与用途限定

> **来源：** 《以案说法》第七章 §1.2.2-1.2.4，页262-264
> **核心法条：** 《专利法》第59条第1款 / 《专利审查指南》第二部分第二章第3.2.2节
> **关联页面：** [[权利要求-保护范围的确定]]、[[权利要求-功能性限定]]、[[权利要求-清楚的要求]]、[[权利要求-以说明书为依据]]、[[专利实务/侵权/侵权-原理-权利要求解释原理]]、[[审查-权利要求-保护范围]]

---

## 核心要点

除结构特征外，权利要求还可以采用方法特征、参数特征、用途限定等方式界定保护范围。方法特征对产品权利要求的限定作用取决于其是否影响产品的结构或组成；参数特征的保护范围限定作用取决于参数差异是否导致产品物质结构不同；用途限定则取决于其对产品本身是否带来结构、功能或性质上的改变。判断这些非结构性特征的限定作用，应当站位所属领域技术人员的视角，结合技术特征与保护主题之间的关系综合判断。

---

## 1. 方法特征对产品权利要求的影响

### 1.1 基本判断原则

对于含有方法特征的产品权利要求，其特征的实际作用取决于其对产品本身的形状、结构等是否具有实质性影响。如果所述方法特征影响着产品的结构或组成，则具有限定作用；反之，如果方法特征仅描述产品的使用方式或效果，而不改变产品结构，则限定作用有限。

**判断步骤**：
1. 分析方法特征所限定的具体工艺、步骤或条件
2. 判断该方法特征是否导致产品在结构、组成或性能上产生可区分的变化
3. 如果导致可区分变化，则对保护范围具有限定作用

### 案例：第86605号复审决定（200780039612.X）— 喷雾干燥方法特征的限定作用

- **争议焦点：** 产品权利要求中限定"所述组合物经喷雾干燥"，该方法特征是否具有限定作用？
- **决定要点：** "喷雾干燥是利用雾化器将溶液、乳浊液等分裂成细小雾状液滴，然后使其在下落过程中与热气体接触进行热传质，瞬间将大部分水分除去而形成粉末或颗粒状产品。与此相比，对比文件1公开的制备组合物的方法是普通热风干燥，该干燥过程仅是利用热空气对流传热作用蒸发水分。由于喷雾干燥包括了雾化形成液滴的过程，所以所获得的物料在结构上与普通热风干燥不同。与采用热风干燥相比，采用喷雾干燥获得的粒子均一性更好，粒径更小，比表面积更大，孔隙间更加疏松。对比数据确也证明，喷雾干燥和热风干燥制备得到的组合物具有不同的结构和性质。因此，权利要求1限定的方法特征'所述组合物经喷雾干燥'对产品的结构产生了影响，对保护范围具有限定作用。"
- **启示：** 方法特征是否对产品权利要求具有限定作用，关键在于该方法是否导致产品在结构上产生可区分的变化。喷雾干燥与热风干燥虽然都是干燥方法，但由于前者包含雾化过程，导致产品粒子结构不同，因此具有限定作用。

### 案例：第63284号复审决定（200680025990.8）— 方法特征隐含结构限定

- **争议焦点：** 权利要求限定"在用气体进行反冲洗的过程中，当液体存在于水道中时，将水道中的气/液界面维持于高于所述液体开口"，该方法特征是否具有限定作用？
- **决定要点：** "所述反冲洗装置包括过滤器、水道和隔离装置，根据权利要求1的技术方案可知，水道中的液体存在于水道、第二隔离装置所形成的空间，液体经由第二隔离装置进入第二辅助水道，气/液界面高于液体开口，必然要求第二隔离装置具备使水道中的液体满足一定存量的结构，因此所述方法特征对权利要求1的反冲洗分配装置具有限定作用。"
- **启示：** 某些方法特征虽然在字面上描述的是操作步骤或工艺条件，但其隐含了对产品结构的要求。当方法特征的实施必然要求产品具备某种特定结构时，该方法特征对产品权利要求具有限定作用。

## 2. 物理、化学参数对产品权利要求保护范围的影响

### 2.1 参数特征的解释规则

对于以物理、化学参数表征的产品权利要求，如果因所述物理、化学参数不同导致产品物质结构不同，则该物理、化学参数对权利要求的保护范围具有限定作用。

**参数特征的解释要点**：
- **参数与结构的关联性**：参数必须能够反映产品的结构或组成差异，而非仅仅描述使用效果
- **参数的可测量性**：权利要求中的参数应当是可以通过申请日前已有的测试方法测定的
- **参数对产品性能的影响**：参数差异应当导致产品性能或结构的实质性变化

**常见具有限定作用的参数特征**：
- 晶体学参数（如X射线衍射半值宽度）— 反映晶体结构状态
- 热学参数（如加热失重率）— 反映表面状态或组成
- 微观结构参数（如粒径、比表面积、孔隙率）— 反映物理结构
- 光谱参数（如红外光谱特征峰）— 反映化学组成或结构

### 案例：第97832号复审决定（201080038124.9）— SCR催化剂的参数特征

- **争议焦点：** 权利要求中限定的X射线晶体衍射半值宽度（FWHM）和加热失重率等参数特征是否具有限定作用？
- **决定要点：** "权利要求1的参数半值宽度（FWHM）限定了催化剂的晶体状态，水合处理后900°C的加热失重率限定了催化剂的表面状态，而特定的晶体状态及表面状态可以提高沸石与过渡金属的相互作用，能够显著提高低温、特别是在200°C以下的NOx还原性能。由此可见，权利要求1所限定的半值宽度（FWHM）范围以及水合处理后的900°C加热失重范围等参数特征意味着催化剂结构的改变，也影响了催化剂的性能，因此对权利要求1具有限定作用。"
- **启示：** 参数特征对保护范围具有限定作用的前提是：该参数能够表征产品的结构状态，且参数差异导致产品结构或性能发生实质性变化。仅描述使用效果而无法反映结构差异的参数，通常不具有实质性限定作用。

## 3. 给药特征与用途限定对权利要求的影响

### 3.1 给药特征的限定作用

在制药用途发明中，若权利要求含有给药对象、给药方式、给药途径、药物用量及给药时间间隔等给药特征的限定，但这类特征对制药过程不产生影响，同时也不会影响发明涉及的适应症，则它们对权利要求的保护范围不具有限定作用。

**判断标准**：给药特征的实际限定作用取决于其对产品本身的影响：
- **对产品结构无影响**：如果给药特征既不改变产品的结构，也不会为产品带来功能或性质上的改变，则对保护范围不具有限定作用
- **对产品组成有影响**：如果给药特征意味着所制备的药物组合物中必须包括相应的组分（如皮下施用所需的缓冲剂、稳定剂），则对保护范围具有限定作用

### 案例：第84987号复审决定（200780019257.X）— 给药对象特征无限定作用

- **争议焦点：** 权利要求限定动物为猪且具有特定抗体滴定度，该给药对象特征是否具有限定作用？
- **决定要点：** "无论施用对象体内是否存在母源性抗体、所存在的抗体滴度高低以及是否影响疫苗的免疫效果，所述疫苗预防的疾病是相同的，都是预防动物感染胞内劳森菌而发生的回肠炎；预防的机制也是相同的，都是通过使用相同的疫苗来诱导动物体内产生相同的中和性抗体。因此，涉案发明的制药用途已经被已知用途的作用机理、药理作用所揭示。虽然药物和疫苗的使用效果在不同个体中会存在差异，但这些差异通常不会使药物或疫苗所治疗或预防的疾病发生改变。而且，具体使用对象的限定也没有导致涉案发明的疫苗与对比文件3的疫苗在减毒菌含量、剂型等方面出现不同，进而影响到药剂的制备过程。基于此，权利要求1中的给药对象特征对其保护范围不具有限定作用。"
- **启示：** 给药对象、给药剂量等特征如果仅与医生对治疗方案的选择相关，与制药方法没有直接、必然的关联性，属于仅体现于用药行为中的特征，对制药用途权利要求不具有限定作用。

### 案例：第118708号复审决定（201180048732.2）— 给药方式的限定作用

- **争议焦点：** 权利要求中限定的给药剂量、给药时间、频次、给药途径等特征是否具有限定作用？
- **决定要点：** "权利要求1中所限定的具体给药剂量、给药时间、频次等通常与医生对治疗方案的选择密切相关，与制药方法没有直接、必然的关联性，属于仅体现于用药行为中的特征，对制药用途不具有限定作用。但权利要求1中限定的给药方式意味着所制备的药物组合物中包括了相应用于皮下施用的缓冲剂、稳定剂和增溶剂，即对药物组合物的组成产生了影响，因此，其中限定的给药方式对制药用途权利要求具有限定作用。"
- **启示：** 给药特征的限定作用需区分层次：与用药行为相关的特征（剂量、频次、时间）通常不具有限定作用；但与产品组成相关的特征（给药方式决定了制剂成分）具有限定作用。

### 案例：第87170号复审决定（200680005096.4）— 使用对象限定对产品权利要求无影响

- **争议焦点：** 产品权利要求中对试剂盒使用对象的限定是否具有限定作用？
- **决定要点：** "权利要求1对试剂盒使用对象的限定并没有对试剂盒的组成、结构或含量等产生影响，也没有为试剂盒带来任何功能和性质上的改变，没有给产品本身带来任何实质性影响，因此对产品权利要求不具有限定作用。"
- **启示：** 在产品权利要求中，用途或使用对象的限定如果不改变产品本身的结构、组成或性能，则属于"标签式"限定，对保护范围不产生实质性影响。

---

### 3.2 治疗机理特征的限定作用

治疗机理特征通常不会直接影响药物的制备过程。对于以治疗机理特征限定制药用途的权利要求，需要重点考虑机理是否对制药用途所涉及的适应症产生实质性影响。

**判断标准**：如果治疗机理的限定并没有将所治疗的疾病限定至某种特定类型，而是涵盖了同一疾病的所有亚型或成因，则该机理限定对保护范围不会产生实质性影响。

### 案例：第56322号复审决定（200810128453.7）— 治疗机理限定无实质影响

- **争议焦点：** 权利要求限定"所说的血压降低不是通过治疗动脉粥样硬化达到的"，该治疗机理限定是否具有限定作用？
- **决定要点：** "权利要求1中'所说的血压降低不是通过治疗动脉粥样硬化达到的'属于用治疗机理对用途进行的限定。而在所属领域中，无论高血压是否由动脉粥样硬化引起，均可以通过番茄红素降低血液中LDL浓度以增加血管横切面积予以治疗，该治疗机理的限定并没有将所治疗的高血压限定至某种特定类型的高血压，因此对权利要求的保护范围不会产生实质性影响，不具有限定作用。"
- **启示：** 治疗机理限定是否影响保护范围，取决于该限定是否将适应症限定至特定亚型。如果无论疾病成因如何，均可用相同药物治疗，则机理限定不改变保护范围。

---

### 3.3 用途限定在侵权判定中的作用

用途限定在侵权判定中的作用，取决于该限定是否隐含了产品的特定结构或组成：

- **隐含结构/组成限定**：如果用途限定要求产品必须具备某种特定结构或组成才能实现该用途，则该用途限定对保护范围有实质性限定作用
- **仅描述使用场景**：如果用途限定仅描述产品的使用场景或目标人群，而不改变产品结构，则对保护范围的限定作用有限

**实务判断要点**：
1. 分析用途限定与产品结构之间的因果关系
2. 判断实现该用途是否必须依赖特定结构
3. 考察用途限定是否仅为产品的"标签"或"说明"

## 4. 非结构性特征限定作用的综合判断

### 4.1 判断框架

```
非结构性特征限定作用判断
│
├── 方法特征
│   ├── 是否影响产品结构/组成？
│   │   ├── 是 → 具有限定作用
│   │   └── 否 → 限定作用有限
│   └── 是否隐含结构要求？
│       ├── 是 → 具有限定作用
│       └── 否 → 需进一步判断
│
├── 参数特征
│   ├── 是否反映产品结构差异？
│   │   ├── 是 → 具有限定作用
│   │   └── 否 → 限定作用有限
│   └── 是否导致性能实质性变化？
│       ├── 是 → 具有限定作用
│       └── 否 → 限定作用有限
│
└── 用途/给药特征
    ├── 是否改变产品组成/结构？
    │   ├── 是 → 具有限定作用
    │   └── 否 → 限定作用有限
    └── 是否仅为用药行为特征？
        ├── 是 → 通常无限定作用
        └── 否 → 需结合具体判断
```

### 4.2 实务速查表

| 特征类型 | 典型示例 | 限定作用判断 | 关键因素 |
|---------|---------|------------|---------|
| 方法特征 | "经喷雾干燥" | 有（改变粒子结构） | 方法是否导致结构差异 |
| 方法特征 | "在用气体进行反冲洗时..." | 有（隐含结构要求） | 方法实施是否要求特定结构 |
| 参数特征 | "X射线半值宽度0.28-0.34" | 有（反映晶体状态） | 参数是否表征结构 |
| 参数特征 | "加热失重15.0-18.0%" | 有（反映表面状态） | 参数差异是否导致性能变化 |
| 给药对象 | "动物为猪" | 无 | 不改变产品组成 |
| 给药方式 | "皮下施用" | 有（决定制剂组成） | 是否影响产品组成 |
| 给药剂量 | "每毫升至少1:4滴定度" | 无 | 仅与用药行为相关 |
| 治疗机理 | "非通过治疗动脉粥样硬化" | 无 | 未限定至特定适应症亚型 |
| 使用对象 | "已用破伤风类毒素预先免疫" | 无 | 不改变试剂盒结构 |
