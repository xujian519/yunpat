# 专利审查制度与初步审查

> **来源：** 崔国斌《专利法：原理与案例（第二版）》第7章 §3.1.1、§3.1.2
> **核心法条：** 《专利法》第34条、第40条
> **关联页面：** [[程序-原理-发明专利实质审查]]、[[程序-原理-专利复审程序]]、[[程序-原理-专利申请文件与申请日]]

---

## 核心要点

中国对发明专利申请实行审查制，对实用新型和外观设计专利申请实行登记制。审查制下，专利局在授予专利权之前对专利申请进行实质审查；登记制下，专利局仅进行初步审查即授予专利权。初步审查是专利局收到申请后的第一道审查程序，既包含形式审查，也包含部分明显的实质性缺陷审查。

---

## 1. 审查制与登记制的对比

### 1.1 审查制

审查制是指专利局在授予专利权之前，对专利申请进行实质审查，确认申请的技术方案满足专利法要求后才授予专利权。中国对发明专利申请实行审查制。

**优点**：
- 经过实质审查产生的专利权可靠性较大
- 权利人和社会公众能够建立合理的预期

**缺点**：
- 费时费力，延缓发明获得保护的时间
- 耗费申请人和专利局的大量资源
- 发明专利商业化比例低，审查制造成资源浪费

审查制适用于可替代性较低、相对重要的发明专利申请。

### 1.2 登记制

登记制是指专利局事先不对专利申请进行实质审查，只要申请符合形式要求，就直接登记并授予专利权。中国对实用新型和外观设计实行登记制。

**优点**：
- 申请人和专利局无须花费过多资源用于事先审查
- 申请人能在较短时间内获得专利授权
- 节省成本（出现交易或纠纷的专利仅占申请量的很小比例）

**缺点**：
- 专利权效力极度不可靠
- 权利人和社会公众难以建立合理预期

登记制适合可替代性较大、数量较多而价值相对较小的专利申请。

## 2. 初步审查

### 2.1 初步审查的内容

依据《专利法实施细则》第44条，发明专利的初步审查主要包括：

1. **申请文件形式审查**：申请文件是否齐备、是否符合格式要求
2. **明显实质性缺陷审查**：
   - 是否明显属于违法或违反公德的发明（专利法第5条）
   - 是否明显属于法定排除的客体（专利法第25条）
   - 是否明显不属于专利法意义上的发明（专利法第2条第2款）
3. **主体资格审查**：外国申请人的主体资格或其委托代理人的资格
4. **保密审查**：申请人向境外申请专利是否履行了保密审查手续（专利法第20条第1款）
5. **遗传资源披露**：涉及遗传资源的专利申请是否履行了披露手续（专利法第26条第5款）
6. **单一性审查**：是否违反单一性原则（专利法第31条第1款）
7. **修改超范围审查**：修改申请文件是否超出原申请文件公布的范围（专利法第33条）

### 2.2 初步审查的特点

- 绝大部分为形式方面的要求
- 包含部分实体方面的要求（如明显实质性缺陷审查）
- 对于实体内容的审查仅限于"明显"不符合法律规定的情形
- 审查员在定义何为"明显"时有一定的裁量权

### 2.3 初步审查后的处理

**发明专利**：
- 符合要求的 → 自申请日起满18个月公布申请
- 不符合要求的 → 要求申请人陈述意见或补正；仍未达到要求的 → 驳回申请

**实用新型和外观设计**：
- 符合要求的 → 直接作出授予专利权的决定，专利权自授权公告之日生效

## 3. 申请的公布与临时保护

### 3.1 申请公布

发明专利申请通过初步审查后，自申请日起满18个月即行公布。国务院专利行政部门可以根据申请人的请求早日公布其申请。

申请公开之后，发明内容不可逆转地进入公开状态。在此之前，申请人可以申请撤回其申请，该申请将不被公开。

### 3.2 临时保护

《专利法》第13条规定："发明专利申请公布后，申请人可以要求实施其发明的单位或者个人支付适当的费用。"

临时保护的特点：
- 仅适用于发明专利申请（实用新型和外观设计无临时保护）
- 起算时间为申请公布之日
- 如果实施者拒绝支付适当费用，申请人须等到专利授权后依据专利权起诉
- 临时保护期的"适当费用"通常低于专利授权后的侵权赔偿

### 3.3 保密义务

《专利法》第21条第3款规定："在专利申请公布或者公告前，国务院专利行政部门的工作人员及有关人员对其内容负有保密责任。"

## 4. 不同专利类型的审查流程对比

| 项目 | 发明专利 | 实用新型 | 外观设计 |
|------|----------|----------|----------|
| 审查制度 | 审查制 | 登记制 | 登记制 |
| 初步审查 | 有 | 有 | 有 |
| 实质审查 | 有（依请求） | 无 | 无 |
| 申请公布 | 18个月 | 无 | 无 |
| 临时保护 | 有 | 无 | 无 |
| 授权时间 | 较长（2-4年或更长） | 较短（约6-12个月） | 较短 |
| 权利稳定性 | 较高 | 较低 | 较低 |
