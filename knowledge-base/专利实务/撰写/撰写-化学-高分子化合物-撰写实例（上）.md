# 高分子化合物-撰写实例（上）

## 三、撰写实例

### 摘 要

本发明提供了一种由至少一种(甲基)丙烯酸正(C₈~C₃₀烷基)酯和马来酸酐构成的共聚物，该共聚物含有交替的(甲基)丙烯酸酯和马来酸酐单体单元，其特征在于该共聚物的交替度至少为85%。还提供了所述共聚物制备方法；以及该共聚物作为油品流动改进添加剂的应用。

### 权利要求书

1. 一种由至少一种(甲基)丙烯酸正C₈~C₃₀烷基酯和马来酸酐构成的共聚物，它含有交替的(甲基)丙烯酸酯和马来酸酐单体单元，其特征在于共聚物的交替度至少为85%。

2. 根据权利要求1的共聚物，其中所述至少一种(甲基)丙烯酸酯选自(甲基)丙烯酸正C₈~C₃₀烷基酯及其混合物。

3. 根据权利要求2的共聚物，其中所述至少一种(甲基)丙烯酸酯选自(甲基)丙烯酸正C₁₂~C₃₀烷基酯及其混合物。

4. 根据权利要求3的共聚物，其中所述至少一种(甲基)丙烯酸酯选自丙烯酸正C₁₂烷基酯或选自丙烯酸正(C₄~C₈烷基)酯的混合物。

5. 根据前面任何一项权利要求中的共聚物，其中交替度至少为86%。

6. 根据权利要求1~4之一的共聚物，其中交替度至少为88%。

7. 根据权利要求1~4之一的共聚物，其中交替度至少为90%。

8. 一种权利要求1的共聚物的制备方法，该方法包括至少一种(甲基)丙烯酸正C₈~C₃₀烷基酯与马来酸酐在能生成自由基的引发剂存在下反应，其特征在于预定摩尔数量的(甲基)丙烯酸酯以每小时20%~100%(摩尔)(甲基)丙烯酸酯的加料速度加到预定摩尔数量的马来酸酐中，而且预定摩尔数量的(甲基)丙烯酸酯与预定摩尔量的马来酸酐的比在1:1~1:5范围内。

9. 权利要求1~7中任何一项所述的共聚物作为油品流动改进添加剂的应用。

**权利要求书布局分析：**

上述权利要求书采用了典型的"产品-方法-用途"三层保护结构。权利要求1为独立产品权利要求，采用[[结构特征]]（交替的单体单元）与[[参数特征]]（交替度≥85%）相结合的方式进行表征，这对于表征高分子化合物的微观结构是必要且有效的。权利要求2~7为从属权利要求，逐步限定了(甲基)丙烯酸酯的烷基链长范围（C₈~C₃₀→C₁₂~C₃₀→C₁₂或C₄~C₈混合物）以及交替度的优选范围（85%→86%→88%→90%），形成了合理的保护层次。权利要求8为方法独立权利要求，权利要求9为用途独立权利要求，三者相互配合，形成了对产品、制备方法及应用领域的全面保护。

### 说明书

**丙烯酸酯共聚物及其制备方法和应用**

#### 技术领域

本发明涉及(甲基)丙烯酸正烷基酯与马来酸酐的共聚物、其制备方法及其作为油品流动改进添加剂的应用。

#### 背景技术

烃油如瓦斯油、柴油、燃料油、润滑油和原油都含有不同数量的烷烃。特别是长链正构烷烃的多少决定了油品的低温流动性能。冷却时，正构烷烃作为片状结晶分离出来，它们相互作用形成其中仍封有液体油品的三维网状结构，使得油品的粘度增加且流动性下降。在应用瓦斯油和柴油时，这种现象导致过滤器堵塞，而在使用原油时，它引起原油胶凝以及在管道中和在贮罐中生成沉淀，造成产量和生产能力上相当大的损失。

人们已熟知，这些问题可通过将所谓的流动改进剂加入到这些油品中得到缓解。通常有两类流动改进剂：成核改性剂和生长改性剂。成核改性剂促进许多微小烷烃结晶的生成，而生长改性剂则促进更大的结晶簇团的生成。但在这两种情况下，都不生成上述的片状结晶。成核改性剂在瓦斯油和柴油中找到应用，因为生成的烷烃晶体小到足以通过过滤器。相反，生长改性剂用于无过滤器的应用场合，例如用于原油。

美国专利US-A-4663491公开了马来酸酐和有以下通式的丙烯酸正烷基酯的共聚物在原油中用作结晶抑制剂：

```
    O
   //
CH₂=C-COOR
     |
    R₁
```

式中R₁为H或甲基，R₂为至少有16（优选16~30）个碳原子的正构烷基。

用以下方法制备这些共聚物：首先将全部用量的丙烯酸正烷基酯与全部用量的马来酸酐混合，丙烯酸酯与酸酐的摩尔比为20:1~1:10，优选8:1~1:5，特别优选的是4:1~1:1；在有机溶剂中生成自由基的催化剂存在下，在50℃~150℃的温度范围下，优选80℃~120℃，加热该混合物。从该专利的实施例可清楚地看到，最优选的温度为85℃~95℃。在这样的温度下，该反应约在4小时内完成。

在该专利中未提供用上述方法得到的共聚物的组成数据（例如，丙烯酸正烷基酯和马来酸酐单体的摩尔百分数）或结构数据。但是，已知丙烯酸正烷酯单体比马来酸酐单体有更高的反应性以及大得多的均聚倾向，又由于全部用量的丙烯酸正烷基酯和全部用量的马来酸酐从一开始就混合，其结果，生成的共聚物中含有的丙烯酸正烷基酯比马来酸酐多，且结构不均匀。

欧洲专利公开EP-A-485773涉及(甲基)丙烯酸烷基酯与马来酸酐共聚物的制备，及其在石油馏分中作为流动改性剂的应用。在实施例1~5的每一个实施例中都使用间歇式溶液聚合法，制得的聚合物含有摩尔比为80:20~40:60的丙烯酸酯与马来酸酐。未给出有关聚合物结构的数据。但是，假设生成有交替单体单元的聚合物，根据上述的摩尔比计算，在聚合物中的最大交替度在40%和80%之间变化。

#### 发明内容

为了解决现有技术在应用瓦斯油和柴油等烃油时，因油品粘度增加和流动性下降导致过滤器堵塞的技术问题，发明人预料不到地发现，基本上由交替的(甲基)丙烯酸酯和马来酸酐单体单元构成的交替共聚物作为烃油流动改性剂，能很好地改进各种烃油的流动性，基于上述发现，从而完成了本发明。

因此，本发明首先提供一种基本上由交替的(甲基)丙烯酸酯和马来酸酐单体单元构成的交替共聚物。

本发明另外提供一种本发明的(甲基)丙烯酸酯和马来酸酐交替共聚物的制备方法。

本发明还将提供本发明的共聚物作为油品流动改性添加剂的应用。

具体地，本发明提供了一种由至少一种(甲基)丙烯酸正(C₈~C₃₀烷基)酯与马来酸酐的共聚物，它含有交替的(甲基)丙烯酸酯和马来酸酐单体单元，其特征是该共聚物的交替度至少为85%。

优选的共聚物是由一种或多种(甲基)丙烯酸正(C₁₀~C₃₀烷基)酯制备的，最优选的是由一种或多种(甲基)丙烯酸正(C₁₂~C₂₈烷基)酯制备的。丙烯酸酯是最优选的。

优选至少一种(甲基)丙烯酸酯选自丙烯酸正C₁₂烷基酯或选自丙烯酸正(C₄~C₈烷基)酯的混合物。

优选地，所述共聚物的交替度至少为88%，更优选交替度至少为90%。
