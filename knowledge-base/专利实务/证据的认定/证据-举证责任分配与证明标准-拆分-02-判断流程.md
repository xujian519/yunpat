---

## 判断流程

```
举证责任分配
├── 使用公开主张 → 请求人举证
│   ├── 发票+实物型号一致 → 初步成立
│   └── 专利权人抗辩
│       ├── 主张售后修改 → 专利权人举证
│       │   ├── 证明结构变化时间线 → 抗辩成立
│       │   └── 仅有来源不明图纸 → 抗辩不成立
│       └── 主张保密义务 → 专利权人举证
│           ├── 证明保密协议存在 → 举证转回请求人
│           └── 未证明 → 使用公开成立
└── 证明标准判断
    ├── 多份证据相互印证 + 符合常识 → 达到标准 ✓
    ├── 生效判决认定事实 → 达到标准 ✓
    └── 孤证 / 矛盾 / 违反常理 / 用户可修改平台 → 未达标准 ✗
```
