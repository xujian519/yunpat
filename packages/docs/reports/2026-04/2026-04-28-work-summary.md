## 🚀 如何运行测试

### 前置条件

1. 配置 `.env` 文件：

```bash
DEEPSEEK_API_KEY=sk-your-api-key-here
KNOWLEDGE_BASE_PATH=/path/to/baochen-knowledge-base
```

2. 安装依赖：

```bash
pnpm install
pnpm build
```

### 运行测试

```bash
# 方式1: 使用 ts-node
npx ts-node examples/integration-test.ts

# 方式2: 编译后运行
pnpm build
node examples/dist/integration-test.js
```
