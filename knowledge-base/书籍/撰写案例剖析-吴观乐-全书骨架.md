# 《发明和实用新型专利申请文件撰写案例剖析（第2版）》全书骨架

> **来源：** 吴观乐 主编，知识产权出版社，2004年5月第2版
> **ISBN：** 978-7-80011-970-5
> **页数：** 460页
> **丛书：** 专利申请文件撰写指导丛书
> **特点：** 结合大量实例详细分析如何撰写发明与实用新型专利申请文件，是专利代理人资格考试备考经典读物

---

## 核心要点

本书通过"理论讲解 + 撰写练习 + 错误剖析"三段式结构，系统训练专利申请文件中权利要求书和说明书的撰写能力，案例全部改编自全国专利代理人资格考试机械类试题。

---

## 第一部分 发明和实用新型专利申请文件的撰写

### 第一章 发明和实用新型专利申请文件简介

- 发明和实用新型专利申请文件的组成与作用
- 权利要求书与说明书的关系
- 撰写前的准备工作

### 第二章 权利要求书和说明书的撰写要求

- [[说明书-法条-申请文件要求]]：说明书应当满足的要求（清楚、完整、能够实现）
- [[权利要求-法条-权利要求书]]：权利要求书应当满足的要求（以说明书为依据、清楚、简要）
- 独立权利要求与从属权利要求的撰写规定
- 形式要求与实质要求的区分

### 第三章 权利要求书和说明书的撰写

- 权利要求书的撰写步骤与方法
- 说明书的撰写方式与顺序
- 技术交底书的处理
- 充分公开与技术秘密的平衡

## 第三部分 撰写存在问题剖析

> 第2版含11个案例，选自实际案例及审查员培训考试试题

| 案例 | 主题 | 典型问题 |
|------|------|---------|
| 案例一 | 磁化防垢除垢器 | 功能性限定过度 |
| 案例二 | 防爆板的夹紧措施 | 缺少必要技术特征 |
| 案例三 | 面片层叠设备 | 权利要求层次混乱 |
| 案例四 | 柱挂式广告板 | 说明书公开不充分 |
| 案例五 | 用于沸腾液体的传热壁 | 权利要求得不到说明书支持 |
| 案例六 | 水龙头 | 技术方案描述不清 |
| 案例七 | 用于车辆的车身装置 | 单一性问题 |
| 案例八 | 密封装置 | 现有技术引用不当 |
| 案例九 | 带吸墨水腔室的墨水瓶 | 实用新型与发明界限模糊 |
| 案例十 | 直流煤粉燃烧器 | 效果与方案不对应 |
| 案例十一 | （备用试题） | 撰写格式不规范 |
