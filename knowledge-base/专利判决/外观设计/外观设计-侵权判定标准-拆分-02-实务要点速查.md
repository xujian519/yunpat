# 实务要点速查

> **拆分自：** [[外观设计-侵权判定标准]]

---

## 核心要点

外观设计侵权判定的核心方法是"整体观察、综合判断"，判断主体为外观设计专利产品的一般消费者。人民法院在认定被诉侵权设计与授权外观设计是否构成相同或近似时，以外观设计的整体视觉效果为标准进行综合判断。对于主要由技术功能决定的设计特征、产品正常使用时不容易被直接观察到的部位以及局部细微差别，应当排除或降低其对整体视觉效果的影响。

## 实务要点速查

| 情形 | 裁判倾向 | 关键依据 | 代表案例 |
|------|---------|---------|---------|
| 被诉侵权设计与授权外观设计整体视觉效果无实质性差异 | 认定近似，落入保护范围 | 专利法司法解释一第十一条 | （2019）最高法民再370号 |
| 区别点属于产品正常使用时不易观察的部位（安装处、底部等） | 不影响近似的认定 | 专利法司法解释一第十一条第一款第（一）项 | （2019）最高法民再278号 |
| 设计特征主要由技术功能决定（如内部结构） | 不予考虑 | 专利法司法解释一第十一条第一款 | （2018）最高法民再59号 |
| 部件具有技术功能，但其外观形状和位置并非主要由功能决定 | 仍应纳入比对 | 专利法司法解释一第十一条 | （2018）最高法民再59号 |
| 设计空间较大，区别为局部细微差异 | 不构成明显区别/实质性差异 | 专利法第二十三条第二款 | （2024）最高法知行终242号 |
| 设计空间较小，区别虽细微但可被一般消费者注意到 | 可能构成明显区别 | 专利法第二十三条第二款 | （2023）最高法知行终433号 |
| 被诉侵权设计与一个现有设计相同或无实质性差异 | 现有设计抗辩成立，不构成侵权 | 专利法第六十二条、司法解释一第十四条第二款 | （2019）最高法民再278号 |
| 设计单元数量的均匀增减未改变整体结构布局 | 不构成实质性差异 | 整体观察、综合判断原则 | （2019）最高法民再278号 |
| 某项设计属于该类产品公认的惯常设计 | 对整体视觉效果影响较小 | 整体观察、综合判断原则 | （2020）最高法知行终415号 |
| 标贴为外观设计产品的组成部分且面积较大 | 应纳入比对范围 | 专利法第二条第四款 | （2018）最高法民再59号 |
| 标贴可轻易分离且仅起标识来源作用 | 不纳入比对范围 | 专利法第二十五条第一款第六项 | （2018）最高法民再59号 |
| 被诉侵权人以拥有在后专利权为由抗辩不侵权 | 不予支持 | 专利法司法解释二第二十三条 | （2019）最高法民再370号 |
| 涉案专利被宣告无效，侵权判决尚未执行完毕 | 无效决定具有追溯力 | 专利法第四十七条 | （2017）最高法民再98号 |
| 一般消费者认定为专业采购商的主张 | 不予支持，应以一般消费者为准 | 专利法第二条第四款 | （2023）最高法知行终433号 |
