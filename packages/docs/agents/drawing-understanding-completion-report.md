## 使用建议

### 适用场景 ✅

- 结构清晰的爆炸图、原理图
- 标注清晰的框图、流程图
- 高质量、高分辨率的图像
- 需要批量处理多个附图

### 不适用场景 ❌

- 复杂的照片图像
- 手绘草图（除非非常清晰）
- 过于复杂的装配图
- 纯文字的附图

### 最佳实践

1. **提供充分的上下文**: 技术领域、技术方案
2. **使用高质量图像**: 至少 1024x768 分辨率
3. **启用缓存**: 对于重复处理的图像
4. **批量处理**: 对于大量附图使用批量处理
5. **验证结果**: 检查置信度，低置信度时人工复核
