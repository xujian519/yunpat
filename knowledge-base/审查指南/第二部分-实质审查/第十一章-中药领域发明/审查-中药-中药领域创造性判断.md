# 审查-中药-中药领域创造性判断

> **最新修订：** 2010年版（沿用至今）
> **关联页面：** [[审查-中药-中药发明专利审查]]、[[审查-化学-化学产品的确认与制备]]

## 核心要点

中药组合物发明是以中医药理论为指导形成的，通常具有一定的组方结构（如君臣佐使），各药味之间存在主次关系，在功能上相互关联、相互配伍发挥作用。创造性判断时需要考量"理、法、方、药"，从组方结构出发分析区别特征。中药组合物发明分为加减方发明（包括药味变更和合方）和自组方发明两类，判断标准各有侧重。

---

## 审查标准

### 1. 中药组合物创造性的判断原则

中药组合物是以中医药理论为指导形成的，通常具有一定的组方结构，各中药原料或药味之间存在主次关系（例如君臣佐使），在功能上相互关联、相互配伍而发挥作用。

中药组合物发明，包括加减方发明和自组方发明，其中加减方包括中药原料变更的组方和合方。

在进行中药组合物发明的创造性判断时需要注意以下几点：

**（1）确定最接近的现有技术**：需要考量发明与现有技术中组合物的"理、法、方、药"，并从发明实质出发，分析组方结构，选择所属技术领域、所要解决的技术问题、技术效果或用途最接近，和/或起主要作用的中药原料（主要药味、主药或君药）相同或相近的现有技术。

**（2）确定区别特征**：通常可以将区别药味按其在组方中发挥作用的主次地位进行分层，例如针对主病或主证的为主要药味，治疗兼证或次要症状的为次要药味。如果发明的组方结构不明晰，或者同一层级的中药原料较多，可将它们按功效或作用进行分类。

**（3）判断是否显而易见**：需要站位本领域技术人员，以最接近的现有技术和发明实际解决的技术问题为出发点，从整体上判断现有技术中是否存在该区别特征以及将该区别特征用于最接近的现有技术解决该技术问题的技术启示。

**常见的技术启示来源**：最接近的现有技术的其他部分、教科书、工具书或综述性文献等现有技术中所公开的相关技术信息，例如药味的加减信息、药味的功效/用量用法/药理作用、发明所述疾病的病因病机/治法治则/常见病程变化和兼证等信息。

### 2. 加减方发明

加减方发明，包括中药原料变更的组方发明和合方发明。

#### 2.1 中药原料变更的组方发明

中药原料变更的组方发明，是指发明以现有技术某一已知方为基础方，在不改变已知方主要药味的基础上，对次要药味和/或其药量进行调整而形成的组方发明，包括药味的增减、药味的替换和药量的加减等。

对于中药原料变更的组方发明，尽管现有技术中已经公开了与其主证和主药相同或相似的基础方，但如果现有技术没有给出将药味或药量变化等区别特征应用到基础方中以解决其存在的技术问题的技术启示，且发明产生了有益的技术效果，则发明具备创造性。反之，发明不具备创造性。

**（1）药味增减的发明**

如果现有技术不存在对已知方进行药味增减以解决发明实际解决的技术问题的技术启示，且发明产生了有益的技术效果，则发明具备创造性。反之，不具备创造性。

**【具备创造性的示例】**

发明和对比文件1均涉及治疗癫痫的中药组合物。发明在对比文件1（11味药）的基础上删除了臣药白附子、当归以及佐使药柴胡、郁金，并增加了佐使药丹参。由于被删除药味和新增药味的功效存在差异，本领域技术人员根据现有技术无法得到删除并增加药味后可产生相当疗效的技术启示，发明具备创造性。

**【不具备创造性的示例】**

发明和对比文件1均涉及治疗骨折的中药组合物，区别仅在于发明还添加了没药。由于乳香和没药均为活血化瘀药，常作为药对配伍使用属于公知常识，本领域技术人员有动机增加功效相同或相近的药味，发明不具备创造性。

**（2）药味替换的发明**

如果发明的药味替换属于现有技术已知的相同功效的药味替代，且没有产生预料不到的技术效果，则发明不具备创造性。

**【示例】**

发明用土木香替代了青木香治疗冠心病。现有技术公开了青木香具有严重肾毒性、临床可用土木香代替。该替代无需付出创造性劳动，且无预料不到的技术效果，不具备创造性。

**（3）药量加减的发明**

如果发明的药量加减属于不改变基础方的组方结构即主药不变的常规药量加减，与基础方相比没有取得预料不到的技术效果，则发明不具备创造性。

**【示例】**

发明和对比文件1药味组成和组方结构相同，发明仅将臣药麦冬由9份减为6份、佐药白芍由3份增加为5份。这种常规用量加减未影响原方的君臣佐使配伍关系，没有改变组方的功用和主治，不具备创造性。

#### 2.2 合方发明

合方发明，是指将两个及以上的已知方合并使用或合方化裁而形成的组方发明。

判断合方发明的创造性，通常需要考虑现有技术中是否存在组合的技术启示、组合的难易程度以及组合后的技术效果。

如果现有技术没有给出合方组合以解决发明实际解决的技术问题的技术启示，且发明产生了有益的技术效果，则发明具备创造性。

**【具备创造性的示例】**

发明将银翘散与平胃散合方治疗病毒性肺炎。银翘散清热解毒可用于治疗病毒性肺炎，但平胃散燥湿运脾、行气和胃，现有技术中没有其可用于治疗病毒性肺炎的记载。本领域技术人员没有动机在银翘散基础上结合平胃散，发明具备创造性。

**【不具备创造性的示例】**

发明将加味六味地黄汤和甘麦大枣汤合方治疗更年期综合征。两者分别治疗肝肾阴虚型和心阴不足型更年期综合征，本领域技术人员为获得标本兼治的方案，有动机将两者合用，且效果仅为加和，不具备创造性。

### 3. 自组方发明

自组方发明，是指未以已知方为基础，而是依据中医药理论和用药经验进行直接遣药组方或者改变了已知方的主要药味形成的组方发明。

对于自组方发明，由于没有以已知方为基础，故说明书需要记载发明的组方原则、组方结构或方解以及足以证明其技术效果的实验数据，以体现发明对现有技术作出的贡献。

在判断自组方发明的创造性时，通常需要在对组方原则和组方结构或方解进行分析的基础上，考量现有技术中是否存在将组方中各药味进行配伍以解决发明存在的技术问题的技术启示。如果无法从现有技术中得到这种技术启示，且发明产生了有益的技术效果，则发明具备创造性。否则，不具备创造性。

**【具备创造性的示例】**

发明涉及治疗子宫肌瘤的组合物（桃仁、当归、蒲黄、没药、牡丹皮、牛膝6味药）。对比文件1公开了治疗子宫肌瘤的常用方法和单味药（当归、没药、蒲黄），但未给出将6味药按主次配伍组合的组方。没有将各药味配伍以获得治疗子宫肌瘤组合物的技术启示，发明具备创造性。

## 实务应对

**代理人视角：**

1. **组方结构的充分描述**：对于中药组合物发明，说明书应当记载组方原则和组方结构（方解），说明各药味的君臣佐使地位及其在组方中的作用。这是创造性答辩的基础——如果无法说明组方结构，审查员可能将区别药味简单归类，从而更容易认定存在技术启示。

2. **加减方发明的答辩策略**：
   - **药味增减**：重点论证被增减药味的功效与现有技术中其他药味存在差异，本领域技术人员无法预期增减后的效果。
   - **药味替换**：如果替换属于公知的功效相同药味替代，应提供对比数据证明产生了预料不到的效果。
   - **药量加减**：如果药量改变导致主药变更或组方结构改变，可以据此论证创造性。

3. **合方发明的答辩策略**：重点论证现有技术中不存在合方的技术启示。如果两个已知方分别治疗不同的证型，且合方后产生了协同效果（而非简单加和），则更容易被认定为具备创造性。

4. **自组方发明的证据要求**：自组方发明由于没有已知方基础，需要更充分的实验数据支持。建议在说明书中提供药效学实验数据、与已知方的对比数据等，以证明技术效果。

5. **技术启示来源的排查**：撰写前应充分检索教科书、工具书、综述性文献中的药味加减信息和治法治则信息，避免权利要求的方案落入常见技术启示的范围内。

6. **"理、法、方、药"四位一体的论证**：在创造性答辩中，应从"理"（病因病机）、"法"（治法治则）、"方"（组方结构）、"药"（药味选择）四个层面进行系统性论证，说明发明的整体构思不是现有技术的简单组合。

## 审查要点速查

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 药味增减（无技术启示+有益效果） | 具备创造性 | 现有技术无增减启示 |
| 药味增减（公知药对配伍） | 不具备创造性 | 本领域有动机增加 |
| 药味替换（公知功效相同替代+无预料不到效果） | 不具备创造性 | 无需创造性劳动 |
| 药量加减（不改变组方结构+常规加减） | 不具备创造性 | 未改变功用和主治 |
| 药量加减（导致主药变更或组方结构改变） | 可能具备创造性 | 组方实质改变 |
| 合方（无组合启示+有益效果） | 具备创造性 | 现有技术无组合动机 |
| 合方（简单叠加+效果仅为加和） | 不具备创造性 | 简单叠加无创造性 |
| 合方（有动机合方+治法相关） | 不具备创造性 | 本领域有动机合方 |
| 自组方（无配伍启示+有益效果） | 具备创造性 | 无法从现有技术得到启示 |
| 自组方（单味药已知但无组方公开） | 具备创造性 | 公开单味药不等于公开组方 |
