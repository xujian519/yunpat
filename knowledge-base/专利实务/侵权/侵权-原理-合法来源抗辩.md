# 侵权-原理-合法来源抗辩

> **来源：** 崔国斌《专利法：原理与案例（第二版）》第10章 §4.4.1
> **核心法条：** 《专利法》（2008）第70条
> **关联页面：** [[侵权-原理-主观过错与过错推定]]、[[侵权-法条-善意侵权]]、[[侵权-原理-侵权基本原理]]

---

## 核心要点

合法来源抗辩是指销售者、使用者能够证明其销售或使用的专利侵权产品具有合法来源，且不知道该产品是未经专利权人许可而制造的，可以不承担赔偿责任。该抗辩仅免除赔偿责任，不免除停止侵权的责任。

---

## 1. 法律依据

《专利法》（2008）第70条规定：

> 为生产经营目的使用、许诺销售或者销售不知道是未经专利权人许可而制造并售出的专利侵权产品，能证明该产品合法来源的，不承担赔偿责任。

### 1.1 适用主体

合法来源抗辩的适用主体限于：
- **使用者**
- **许诺销售者**
- **销售者**

**制造者不适用**本条规定。制造专利产品的行为人即使不知道自己的制造行为侵权，依然要承担损害赔偿责任。

### 1.2 主观要件：不知道

行为人必须"不知道是未经专利权人许可而制造并售出的专利侵权产品"。这一主观状态通常为善意，即行为人确实不知道且没有合理理由应当知道该产品是侵权产品。

### 1.3 客观要件：合法来源

行为人必须"能证明该产品合法来源"。合法来源通常指通过正当的商业渠道获得产品，如从正规供应商处采购、有合法的进货凭证等。

### 1.4 法律效果

满足上述条件的，"不承担赔偿责任"。但需要注意：
- **不免除停止侵权的责任**：善意销售者或使用者仍需停止销售或使用侵权产品
- **不免除其他侵权责任**：如消除影响等（如适用）

---

## 2. 司法实践中的认定标准

### 2.1 合法来源的举证责任

主张合法来源抗辩的销售者或使用者，应当就以下事项承担举证责任：

1. 产品的来源渠道合法
2. 有相应的交易凭证（如合同、发票、付款记录等）
3. 交易行为真实发生

### 2.2 "不知道"的认定

法院在认定行为人是否"不知道"时，通常会考虑以下因素：

1. **产品的来源渠道**：是否从正规渠道采购
2. **价格是否合理**：是否明显低于市场价格
3. **专利权人的通知**：是否在收到侵权通知后仍继续销售
4. **行为人的注意义务**：作为专业经营者，是否尽到了合理的审查义务

### 2.3 加盟店与普通零售商的区别

在司法实践中，法院可能会区分不同类型的销售者：

- **特许加盟店**：由于与品牌方存在紧密的合作关系，法院可能对其注意义务要求更高。如果加盟店销售的侵权产品实际上来源于特许方，其能否主张合法来源抗辩存在争议
- **普通零售商**：通常只需要证明其从上游供应商处合法采购即可

## 3. 典型案例

### 案例：广东联邦家私集团有限公司 v. 陈军等（最高人民法院（2009）民申字第1495号）

**审理法院：** 最高人民法院

**案情概要：**
- 联邦公司为"沙发（J2557）"外观设计专利的专利权人
- 万事达厂为"庭院世家"注册商标的所有人
- 陈军承包经营"庭院世家"展销厅，销售被控侵权沙发
- 陈军主张被控侵权产品具有合法来源，来源于佛山市奥彪家具厂

**法院认定：**

广东省高级人民法院二审认为，陈军经营的"庭院世家"展销厅是万事达厂在赣州的特许加盟店。陈军与赣州市深港商业运营有限公司签订的承包合同进一步证明陈军展销厅经营品牌仅为庭院世家、艾丽格。

最高人民法院认为：

> 法律赋予不知道而销售、使用，且提供证据证明被控侵权产品具有合法来源的销售者或使用者不承担赔偿责任的抗辩权，该抗辩权的成立以销售者或使用者主张并提供证据予以证明为前提。本案中，陈军主张该抗辩权所提供的证据欲证明被控侵权产品是由案外人奥彪家具厂所生产、销售，二审法院在认定陈军所提交的抗辩证据不足以证明被控侵权产品由奥彪家具厂生产、销售的情形下，认定陈军所提出的被控侵权产品具有合法来源的抗辩主张不能成立，符合专利法第六十三条第二款的规定。

**启示：**
1. 合法来源抗辩的举证责任由主张者承担
2. 举证需要达到"证明"的标准，而不仅仅是"主张"
3. 特许加盟店的身份可能影响法院对合法来源证据的审查严格程度

## 4. 实务要点

### 4.1 对销售者的建议

1. **保留进货凭证**：合同、发票、付款记录等应妥善保存
2. **审查供应商资质**：选择正规渠道进货
3. **关注价格异常**：对于明显低于市场价格的产品应保持警惕
4. **及时停止销售**：在收到侵权通知后应立即停止销售涉嫌侵权的产品

### 4.2 对专利权人的建议

1. 在发现侵权产品后，可以向销售者发出侵权通知
2. 如果销售者在收到通知后仍继续销售，则其主观状态可能从"不知道"转变为"知道"
3. 在诉讼中可以主张销售者与制造者承担连带责任（如存在共同侵权情形）

## 判断流程

```
合法来源抗辩判断流程
├── 主体审查
│   ├── 制造者 → 不适用合法来源抗辩
│   └── 销售者/使用者/许诺销售者 → 继续判断
├── 主观要件审查
│   ├── 是否知道是侵权产品？
│   │   ├── 是 → 抗辩不成立
│   │   └── 否 → 继续判断
│   └── 是否应当知道？（合理注意义务）
│       ├── 是 → 抗辩可能不成立
│       └── 否 → 继续判断
├── 客观要件审查
│   └── 能否证明合法来源？
│       ├── 否 → 抗辩不成立
│       └── 是 → 抗辩成立
└── 法律效果
    ├── 不承担赔偿责任
    └── 但仍需停止侵权
```
