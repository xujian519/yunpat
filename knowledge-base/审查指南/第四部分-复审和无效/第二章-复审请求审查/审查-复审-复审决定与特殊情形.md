# 审查-复审-复审决定与特殊情形

> **来源：** 《专利审查指南》第四部分第二章 §5-9 + 第四部分第一章 §6-7
> **对应法条：** 《专利法》第41条 / 《专利法实施细则》第65条、第105条
> **最新修订：** 无重大修改
> **关联页面：** [[审查-复审-前置审查与合议审查]]、[[程序-复审请求的审查]]、[[程序-原理-专利复审]]

## 核心要点

复审决定分为驳回复审请求、撤销驳回决定和在修改文本基础上撤销驳回决定三种类型。撤销驳回决定后，审查部门应当执行复审决定，不得以同样的事实、理由和证据作出相反决定。复审程序可因请求人撤回、期满未答复或被驳回请求而终止。复审程序的中止适用本指南第五部分第七章第7节的规定。

---

## 审查标准

### 一、复审请求审查决定的类型（§5）

复审请求审查决定（简称复审决定）分为下列三种类型：

**(1) 复审请求不成立，驳回复审请求。**

**(2) 复审请求成立，撤销驳回决定。**

上述第(2)种类型包括下列情形：
- (i) 驳回决定适用法律错误的；
- (ii) 驳回理由缺少必要的证据支持的；
- (iii) 审查违反法定程序的，例如：
  - 驳回决定以申请人放弃的申请文本或者不要求保护的技术方案为依据；
  - 在审查程序中没有给予申请人针对驳回决定所依据的事实、理由和证据陈述意见的机会；
  - 驳回决定没有评价申请人提交的与驳回理由有关的证据，以至可能影响公正审理的；
- (iv) 驳回理由不成立的其他情形。

**(3) 专利申请文件经复审请求人修改，克服了驳回决定所指出的缺陷，在修改文本的基础上撤销驳回决定。**

#### 1.1 三种决定类型的适用情形对比

| 决定类型 | 适用情形 | 后续程序 |
|---------|---------|---------|
| 驳回复审请求 | 复审请求人陈述的意见和提交的修改文本不足以撤销驳回决定 | 请求人可在3个月内向人民法院起诉 |
| 撤销驳回决定 | 驳回决定存在法律错误、证据不足、程序违法或理由不成立 | 案卷返回审查部门继续审查 |
| 在修改文本基础上撤销驳回决定 | 复审请求人提交的修改文本克服了驳回指出的缺陷 | 案卷返回审查部门，以修改文本为基础继续审查 |

#### 1.2 复审决定的审批

合议组应当对审查决定的事实认定、法律适用、结论以及决定文件的形式和文字负全面责任。合议组作出的审查决定，属下列情形的，须经部门负责人审核批准：
- 组成五人合议组审查的案件；
- 合议组的表决意见不一致的案件；
- 审查决定被法院的判决撤销后，重新作出决定的案件。

负责审批合议组决定的部门负责人不同意合议组作出的审查决定时，可以提出意见并指示合议组重新合议；合议组重新合议后，与部门负责人的意见仍不一致的，部门负责人认为有必要在较大范围内进行研究的，应当召开部门会议进行讨论，合议组和负责审批的部门负责人应当按照与会人员二分之一以上的多数意见处理。

### 二、复审决定的送达（§6）

根据专利法第四十一条第一款的规定，复审决定应当送达复审请求人。

送达方式包括邮寄、直接送交、电子形式送达等。复审决定一经送达，即产生法律效力（对于可起诉的决定，起诉期限自收到决定之日起计算）。

### 三、复审决定的约束力（§7）

复审决定撤销原驳回决定的，复审和无效审理部应当将有关的案卷返回审查部门，继续进行审查程序。

**审查部门应当执行复审决定，不得以同样的事实、理由和证据作出与该复审决定意见相反的决定。**

#### 3.1 约束力的范围与限制

复审决定的约束力具有以下特点：

| 方面 | 内容 |
|------|------|
| 约束对象 | 原审查部门（即作出驳回决定的审查部门） |
| 约束内容 | 不得以同样的事实、理由和证据作出与复审决定意见相反的决定 |
| 不约束的情形 | 审查部门可以以不同的事实、理由或证据再次提出审查意见 |
| 修改文本的情形 | 复审决定在修改文本基础上撤销驳回决定的，审查部门应当以修改后的文本为基础继续审查 |

#### 3.2 审查部门后续审查的注意事项

- 审查部门受复审决定约束，不能以同样的事实、理由和证据再次驳回；
- 但审查部门可能以其他理由提出新的审查意见，继续进行审查；
- 例如，复审决定因创造性理由不成立而撤销驳回，审查部门仍可以新颖性、公开不充分等其他理由继续审查；
- 若审查部门认为修改文本仍存在其他缺陷，可以发出新的审查意见通知书。

### 四、复审程序的中止（§8）

适用本指南第五部分第七章第7节的规定。

#### 4.1 中止的适用情形

复审程序的中止通常因以下原因发生：

| 中止原因 | 说明 |
|---------|------|
| 权属纠纷 | 当事人因专利申请权权属纠纷向专利局提出中止请求 |
| 财产保全 | 人民法院要求专利局协助执行财产保全措施 |
| 其他法定事由 | 其他依法应当中止的情形 |

#### 4.2 中止的请求手续

- **权属纠纷当事人请求中止**：应当提交中止程序请求书，以及权属纠纷已被人民法院或者地方知识产权管理部门受理的证明文件；
- **财产保全中止**：人民法院应当出具协助执行通知书，载明要求保全的专利申请的申请号、发明创造名称、申请人姓名或名称等事项。

#### 4.3 中止的期限

| 中止类型 | 中止期限 |
|---------|---------|
| 权属纠纷当事人请求的中止 | 一般不超过一年，期满前可请求延长 |
| 因协助执行财产保全的中止 | 至人民法院出具的协助执行通知书中载明的期限届满或人民法院通知解除保全 |
| 涉及无效宣告程序的中止 | 参照无效宣告程序的中止期限处理 |

#### 4.4 中止的恢复

中止原因消除后，复审和无效审理部应当恢复复审程序，并通知当事人。

### 五、复审程序的终止（§9）

复审程序因以下情形终止：

#### 5.1 期满未答复

复审请求因期满未答复而被视为撤回的，复审程序终止。

此处的"期满未答复"主要指：针对合议组发出的复审通知书（包括复审请求口头审理通知书），复审请求人未在指定期限内进行书面答复，且未参加口头审理的，其复审请求视为撤回。

#### 5.2 主动撤回

在作出复审决定前，复审请求人撤回其复审请求的，复审程序终止。

**撤回的时机考量**：
- 在合议审查过程中，如果判断复审成功的可能性较低，或者申请人在此期间对申请文件进行了重大修改希望通过重新申请来保护，可以考虑主动撤回复审请求；
- 复审请求人提交无具体答复内容的意见陈述书的，视为对复审通知书中的审查意见无反对意见；
- 请求人在审查决定的结论已宣布或者书面决定已经发出之后撤回请求的，不影响审查决定的有效性。

#### 5.3 驳回请求

已受理的复审请求因不符合受理条件而被驳回请求的，复审程序终止。

驳回请求的情形通常包括：复审请求不是针对驳回决定的、复审请求人不是被驳回申请的申请人的、提出复审请求的期限不符合规定且未恢复权利的等。

#### 5.4 程序终止后的后续救济

| 终止情形 | 后续救济 |
|---------|---------|
| 视为撤回 | 可在规定期限内请求恢复权利 |
| 主动撤回 | 不可恢复，但可重新提出专利申请 |
| 驳回请求 | 如对驳回请求的决定不服，可在规定期限内起诉 |

### 六、复审决定被法院生效判决撤销后的审查程序

根据第四部分第一章第8节的规定：

1. 复审请求审查决定被人民法院的生效判决撤销后，应当重新作出审查决定；
2. 因主要证据不足或者法律适用错误导致审查决定被撤销的，不得以相同的理由和证据作出与原决定相同的决定；
3. 因违反法定程序导致审查决定被撤销的，根据人民法院的判决，在纠正程序错误的基础上，重新作出审查决定。

## 实务应对

**代理人视角：**

1. **撤销驳回决定后的应对**：复审决定撤销驳回决定后，案件返回审查部门继续审查。审查部门受复审决定约束，不能以同样的事实、理由和证据再次驳回，但可能以其他理由提出新的审查意见。代理人应密切关注后续审查动态，做好应对新审查意见的准备。

2. **复审请求的撤回时机**：在合议审查过程中，如果判断复审成功的可能性较低，或者申请人在此期间对申请文件进行了重大修改希望通过重新申请来保护，可以考虑主动撤回复审请求。但需注意，撤回后复审程序即终止，不得恢复。

3. **程序违法的主张**：若原审查程序存在法定程序瑕疵（如未给予陈述意见的机会、以放弃的文本为依据驳回等），应在复审请求中明确指出，这是撤销驳回决定的有力理由。程序违法的认定通常比实体争议的认定更为明确，成功率相对较高。

4. **后续程序的衔接**：收到驳回复审请求的决定后，应注意三个月内向人民法院起诉的期限，及时决定是否提起行政诉讼。起诉期限自收到复审决定之日起计算，逾期未起诉的，复审决定生效。

5. **修改文本的策略**：在复审程序中，复审请求人可以对申请文件进行修改，但修改应当仅限于消除驳回决定或者合议组指出的缺陷。建议代理人在提交复审请求时，同步提交经过修改的申请文件，以争取在修改文本基础上撤销驳回决定。

6. **中止程序的运用**：若复审请求涉及权属纠纷或财产保全，代理人应及时了解中止程序的规定，合理利用中止期限，为当事人争取时间。同时注意中止期限届满后及时申请恢复或延长。

## 审查要点速查

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 驳回决定适用法律错误 | 撤销驳回决定 | §5(2)(i) |
| 驳回理由缺少必要证据支持 | 撤销驳回决定 | §5(2)(ii) |
| 审查程序中未给予申请人陈述意见的机会 | 撤销驳回决定（违反法定程序） | §5(2)(iii) |
| 修改文本克服了驳回指出的缺陷 | 在修改文本基础上撤销驳回 | §5(3) |
| 审查部门以同样事实理由作出相反决定 | 违反复审决定约束力 | §7 |
| 复审请求人期满未答复复审通知书 | 复审请求视为撤回，程序终止 | §9(1) |
| 复审请求人在决定前撤回请求 | 复审程序终止 | §9(2) |
| 复审请求因不符合受理条件被驳回 | 复审程序终止 | §9(3) |
| 权属纠纷当事人请求中止 | 适用第五部分第七章第7节 | §8 |
| 法院生效判决撤销复审决定 | 重新作出审查决定 | 第一部分§8 |
| 因证据不足或法律适用错误被撤销 | 不得以相同理由证据作相同决定 | 第一部分§8(2) |
| 因违反法定程序被撤销 | 纠正程序错误后重新作出决定 | 第一部分§8(3) |
