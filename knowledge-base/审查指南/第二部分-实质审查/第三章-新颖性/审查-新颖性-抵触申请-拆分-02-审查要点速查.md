## 审查要点速查

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 申请日前提交、申请日后公布的中国专利申请 | 构成抵触申请，破坏新颖性 | §2.2 抵触申请定义 |
| 申请日当天提交的同主题专利申请 | 不构成抵触申请 | §2.2 排除申请日当天 |
| 进入中国国家阶段的PCT申请（满足时间条件） | 构成抵触申请 | §2.2 国际申请纳入 |
| 申请人本人的在先申请（满足条件） | 可构成抵触申请 | §2.2 "任何单位或者个人" |
| 对比文件附图中推测的技术特征 | 不属于公开内容 | §2.3 附图引用规则 |
| 对比文件附图中直接确定的技术特征 | 属于公开内容 | §2.3 附图引用规则 |
| 对比文件中隐含且可直接确定的内容 | 属于公开内容 | §2.3 公开内容认定 |
| 对比文件中无文字说明的测量尺寸 | 不属于公开内容 | §2.3 附图引用规则 |
| 优先权日以前提出、优先权日以后公布的申请 | 构成抵触申请（以优先权日为准） | 优先权效力 |
| 本国优先权基础申请自在后申请提出日起视为撤回 | 不再构成抵触申请（未公布前提下） | 本国优先权特殊规则 |
| 同日提交的相同主题申请 | 适用专利法第9条，不构成抵触申请 | 专利法第9条 |
