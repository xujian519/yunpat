# 相同侵权

> **来源：** 《中国专利法详解》第7章 第59条 三(一)，尹新天著
> **对应法条：** 《专利法》第59条

## 核心要点

相同侵权（literal infringement）是指在被控侵权的产品或者方法中能够找出与权利要求中记载的每一个技术特征相同的对应技术特征。判断相同侵权可以借用新颖性的概念——将被控侵权行为客体看成一份对比文献，如果权利要求不具备新颖性，则相同侵权成立。相同侵权与等同侵权之间存在递进逻辑关系：相同侵权不成立时才需进一步判断等同侵权。

---

## 立法沿革

1984年《专利法》制定时即规定了第59条（现为第59条），确立了"专利权的保护范围以权利要求的内容为准"的基本原则。2009年《最高人民法院关于审理侵犯专利权纠纷案件应用法律若干问题的解释》第七条确立了"全部技术特征"原则，为相同侵权的判断提供了明确标准。

---

## 条文解析

### 一、相同侵权的定义

相同侵权又称为文字含义上的侵权（literal infringement），是指在被控侵权的产品或者方法中能够找出与权利要求中记载的每一个技术特征相同的对应技术特征。

相同侵权的判断与新颖性的判断有良好的可比性。下表展示了两者之间的对应关系：

| 新颖性的判断 | | | 相同侵权的判断 | | |
|---|---|---|---|---|---|
| 权利要求 | 对比文献 | 结论 | 权利要求 | 被控侵权行为 | 结论 |
| A, B, C | A, B, C | 无新颖性 | A, B, C | A, B, C | 成立 |
| A, B, C, D | A, B, C | 有新颖性 | A, B, C, D | A, B, C | 不成立 |
| A, B, C | A, B, C, D | 无新颖性 | A, B, C | A, B, C, D | 成立 |
| A, B, C, D | a, b, c, d | 无新颖性 | A, B, C, D | a, b, c, d | 成立 |
| a, b, c, d | A, B, C, D | 有新颖性 | a, b, c, d | A, B, C, D | (不存在此情况) |

注：大写字母表示的技术特征分别是小写字母所表示的技术特征的上位概念。

### 二、判断规则详解

#### （一）技术特征完全对应

被控侵权行为客体具有的技术特征不多不少，恰好与权利要求中记载的技术特征一一对应。此时当然得出相同侵权成立的结论。现实中，这种完全相同的侵权情况极少发生。

#### （二）缺少技术特征

如果被控侵权行为客体没有包含权利要求中的某一个技术特征，则应当得出相同侵权不成立的结论。此时还不能得出侵权指控不成立的最终结论，因为还需要继续进行是否构成等同侵权的判断。

#### （三）增加技术特征

如果被控侵权行为客体除了包含与权利要求中记载的全部技术特征相同的对应技术特征之外，又增加了其他技术特征，则不论增加的技术特征本身或者与其他技术特征相结合产生何种功能和效果，均应当得出相同侵权成立的结论。

从这一结论来看，《专利法》第十一条所述的"其专利产品""其专利方法"并不是要求被控侵权行为客体与专利产品或者专利方法相同才构成专利侵权行为，而是指被控侵权行为客体落入权利要求保护范围。

#### （四）上位概念与具体实施方式

当权利要求中记载的是用上位概念表述的技术特征时，被控侵权行为人采用的具体实施方式与专利说明书中用于支持权利要求中该上位概念的具体实施方式可能不完全相同，但是依照"以权利要求的内容为准"的原则，只要被控侵权人采用的具体实施方式落入权利要求记载的该上位概念的含义范围，就应当得出相同侵权成立的结论。现实中，绝大多数相同侵权行为都包含这种情况。

#### （五）不存在"反向"情况

在侵权判断中不存在被控侵权行为客体采用上位概念而权利要求中记载的是具体实施方式的情况，因为被控侵权行为涉及的都是具体的产品或者方法，其零部件或者步骤都是具体的物品或者具体的操作方式。这是相同侵权的判断与新颖性判断不同的地方。

### 三、与新颖性判断的区别

相同侵权的判断与新颖性判断还存在另外一个区别。根据《审查指南2001》的规定，不具备新颖性的情况还包括惯用手段的直接替换。例如权利要求中包括A、B、C三个技术特征，对比文献公开了A、B、C'三个技术特征，如果C与C'的可替换性是一种公知常识，则构成惯用手段的直接替换，仍然不具备新颖性。在专利侵权判断中，最好不要将这种情况列入相同侵权的范畴，用等同原则进行判断更为合适。

### 四、"全部技术特征"原则

《最高人民法院关于审理侵犯专利权纠纷案件应用法律若干问题的解释》第七条确立了"全部技术特征"原则：

> 人民法院判定被诉侵权技术方案是否落入专利权的保护范围，应当审查权利人主张的权利要求所记载的全部技术特征。
>
> 被诉侵权技术方案包含与权利要求记载的全部技术特征相同或者等同的技术特征的，人民法院应当认定其落入专利权的保护范围；被诉侵权技术方案的技术特征与权利要求记载的全部技术特征相比，缺少权利要求记载的一个以上的技术特征，或者有一个以上技术特征不相同也不等同的，人民法院应当认定其没有落入专利权的保护范围。

该原则最为主要的作用在于对"等同侵权"的认定施加必要的限制，同时也为相同侵权的判断提供了明确标准——被控侵权技术方案必须包含权利要求记载的全部技术特征，缺少任何一个都不能认定落入保护范围。

### 五、相同侵权与等同侵权的递进关系

相同侵权与等同侵权之间存在类似新颖性与创造性之间的递进逻辑关系：

- 相同侵权不成立，并不意味着侵权判断就此结束，还可能需要进一步判断等同侵权是否成立
- 当相同侵权成立时，则无须再判断等同侵权是否成立
- 当法官讨论等同侵权问题时，就表明他已经承认相同侵权不成立

但需注意，判断等同侵权时不能采用判断创造性的标准，两者有本质区别。

相同侵权的判断是一种客观的判断，其结论基本上完全取决于事实，公众和当事人都有足够的把握预见判断结果。相比之下，等同侵权的判断灵活得多，包含一定程度的主观判断。

## 司法实践要点

1. **判断客体是具体的产品或方法**，不存在抽象的上位概念，因此无需考虑被控侵权行为客体采用上位概念的情况。

2. **增加技术特征不影响侵权认定**，在采用了权利要求记载的全部技术特征的基础上再增加其他特征，仍然构成相同侵权。

3. **相同侵权成立时无需适用等同原则**，也无需考虑禁止反悔原则，因为落入权利要求文字所表达的保护范围时，可以直接得出侵权成立的结论。

4. **相同侵权判断具有高度客观性**，判断结论基本上完全取决于事实，不同于等同侵权包含主观判断因素。
