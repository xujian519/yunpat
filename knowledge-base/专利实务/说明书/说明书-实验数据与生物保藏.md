# 实验数据与生物材料保藏

> **来源：** 《以案说法》第六章 §2.3-2.4，页239-248
> **核心法条：** 《专利法》第26条第3款、第26条第1款（涉及保藏）
> **关联页面：** [[说明书-充分公开概述]]、[[说明书-能够实现的判断]]、[[审查-说明书-附图与实施例]]、[[审查-说明书-核苷酸和氨基酸序列]]

---

## 核心要点

并非所有专利都需要实验数据——只有当技术人员**无法预期**技术方案效果时，实验证据才是必需的。补充实验数据只能**印证**原始公开内容，不能改变公开事实。涉及生物材料时，"公众不能得到"的材料必须保藏。

---

## 1. 需要实验证据的情形（§2.3.1）

### 基本判断规则

```
是否需要实验数据？
│
├── 技术人员根据技术方案能否预期效果？
│   ├── 能预期 → 不需要实验数据
│   └── 不能预期 → 必须提供实验数据
│
├── 典型需要实验数据的情形：
│   ├── 结构全新的化合物（无已知类似结构）
│   ├── 已知化合物的全新用途
│   └── 晶体化合物（需表征组成和结构）
│
└── 典型不需要实验数据的情形：
    ├── 公知原理可推知效果
    ├── 已知类似结构具有相同活性
    └── 现有技术已有相关证据
```

### 案例：第73780号复审决定（200780042615.9）— 全新化合物无实验数据

- **案件：** 要求保护通式化合物，声称可治疗癌症等多种疾病，未公开任何实验数据
- **决定要点：** 化合物属结构全新，无已知类似结构化合物。仅泛泛提到"具有多种机理的活性"，实际只是"结果未定的宽泛研究方向"
- **启示：** 全新化合物**必须**提供实验数据，泛泛声称效果不够

### 案例：第66546号复审决定（200680050481.0）— 可预期效果无需实验

- **案件：** 补体旁路途径抑制剂用于治疗补体活化相关眼部疾病，未提供实验数据
- **决定要点：** 已知补体抑制剂可通过阻断MAC治疗该类疾病，旁路途径是公知的补体活化途径之一，技术人员能预见旁路途径抑制剂同样有效
- **启示：** 即使是新的具体应用，只要技术人员根据已知机制能合理预期效果，就无需实验数据

### 案例：第13582号无效决定（96195564.3）— 立普妥晶体案（重要）

- **案件：** 权利要求1保护含1~8摩尔水的I型结晶阿托伐他汀水合物
- **争议焦点：** 三重缺陷——(1)水是否占位影响XPRD未证明；(2)未提供数据证明确实含1~8摩尔水；(3)无法受控制备得到特定含水量的晶体
- **决定要点：** 综合三项缺陷认定公开不充分。仅声称"水不占位"无证据支撑；制备方法无法确保得到目标产物
- **启示：** 晶体化合物专利需要：①物质组成和晶体结构的表征数据 ②至少一种可重复的制备方法

### 案例：第63757号复审决定（200680053252.X）— 半导体聚合物

- **案件：** 半导体芳胺聚合物，未记载光电性能数据
- **决定要点：** 实施例使用的化合物都是公知的电致发光材料，技术人员基于公知常识能预期其光电性能
- **启示：** 基于公知常识能预期效果时，无需额外实验数据

### 案例：第74341号复审决定（200880023224.X）— 前药

- **案件：** 儿茶酚胺前药，未提供实验数据
- **决定要点：** 母体化合物药理活性已知，前药在体内代谢为母体药物可预期
- **启示：** 前药专利中，如果母体药物活性已知，代谢途径可预期，则无需额外实验

## 2. 对实验证据的要求（§2.3.2）

### 四要素要求

实验证据必须包含以下四个不可或缺的组成部分：

| 要素 | 说明 |
|------|------|
| ①实验所采用的化合物/样品 | 必须明确具体是什么物质 |
| ②实验方法 | 测试条件、参数 |
| ③实验结果 | 具体数据 |
| ④实验结果与声称用途/效果的对应关系 | 数据如何证明效果 |

### 案例：第53577号复审决定（200680050758.X）— 实验数据笼统

- **案件：** 1272个制备实施例，但实验结果仅记载"在小于100µM浓度下抑制CETP 30%以上"，未说明哪些样品获得
- **决定要点：** 略去全部实验用化合物名称和结构，代之以"本发明化合物"，技术人员无法得知活性数据对应何种化学结构
- **启示：** **笼统的实验结论不是有效的实验证据**，必须建立数据与具体化合物之间的对应关系

### 重要区分：专利审查标准 ≠ 药品上市标准

### 案例：第66176号复审决定（200880114256.8）— 肝癌诊断标志物

- **决定要点：** 专利审查只要求证明可行性（标志物与肝癌有合理相关性），不要求达到医药管理部门的可靠性/安全性/有效性标准。标志物灵敏度偏低不是判断充分公开的因素
- **启示：** 不要用药品审批标准来要求专利申请中的实验数据

## 3. 补充实验数据（§2.3.3）

### 基本规则

> 申请日后提交的补充实验数据**只能用于印证**原始公开内容，**不能改变**原始公开的事实。

### 案例：第69977号复审决定（200780033059.9）— 可接受的补充数据

- **案件：** 火蚁毒液提取物治疗阿尔茨海默病，申请日后提交动物模型实验数据
- **决定要点：** 原始说明书的实施例1和2已能建立火蚁毒液提取物与治疗阿尔茨海默病的合理关联；补充数据使用相同模型，是进一步验证而非新发现
- **启示：** 补充数据可以佐证 → 前提是原始公开已建立合理关联

### 案例：第73780号复审决定（200780042615.9）— 不可接受的补充数据

- **案件：** 原始说明书无任何效果数据，审查中补充32个化合物的抑制TNFα实验
- **决定要点：** 虽然补充化合物在原始58个化合物范围内，但不能证明申请日时已完成效果验证
- **启示：** 补充数据**不能弥补原始公开的缺失**——如果原始说明书什么都没证明，后来的数据不能"无中生有"

## 4. 生物材料保藏（§2.4）

### 保藏条件（缺一不可）

```
是否需要保藏？
├── 条件1: 生物材料是实现发明必不可少的要素
│   └── 如果能通过其他途径（如DNA序列）重复实施 → 不需要
└── 条件2: 生物材料是"公众不能得到"的
    ├── 无法根据说明书信息制备
    └── 无法通过商业购买等途径获得
```

### 4.1 "公众不能得到"的含义

### 案例：第20304号复审决定（99807588.4）— 临床分离物

- **案件：** 粘膜炎布兰汉氏球菌K65菌株，来自澳大利亚某医院痰液的临床分离物
- **决定要点：** 临床分离物因样品中特定微生物的偶然性，无法重复获得，且非可商业购买的已知菌株
- **启示：** 临床分离物通常是"公众不能得到"的

### 案例：第44765号复审决定（200910238154.3）— 可商业购买

- **案件：** 鸡新城疫病毒ZM10株
- **决定要点：** 根据兽药注册证书和销售发票，该病毒株可通过商业渠道购买
- **启示：** 已商业化供应的生物材料属于"公众可以得到"

### 4.2 "实现发明必不可少"的含义

### 案例：第5350号复审决定（93121424.6）— 杂交瘤非必需

- **案件：** 免疫活性嵌合抗CD20抗体，最初通过杂交瘤2B8获得，但说明书已公开编码抗体的DNA序列
- **决定要点：** 技术人员可依据DNA序列通过常规技术合成抗体，不必从杂交瘤开始
- **启示：** 如果有替代路径可实现发明，特定生物材料就不再是"必不可少"的

### 4.3 说明书中的保藏信息

### 案例：第18816号无效决定（95117436.3）— 保藏日期缺失

- **案件：** 说明书记载了保藏单位（CGMCC）和保藏编号（0239），但未记载保藏日期
- **决定要点：** 保藏单位和编号确定 → 保藏日期唯一 → 技术人员可据此获知保藏日期
- **启示：** 部分保藏信息缺失，如果不影响技术人员获得生物材料，不导致公开不充分
