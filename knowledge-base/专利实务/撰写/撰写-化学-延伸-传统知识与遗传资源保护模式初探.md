# 传统知识、民间文艺及遗传资源保护模式初探

近年来，传统知识、民间文艺和遗传资源的保护问题已成为WIPO、CBD和WTO等国际论坛的重要议题，也是发展中国家与发达国家在知识产权保护问题上争论的焦点。

## 一、范围和性质

### 定义

- **传统知识**：基于传统产生的文学、艺术或科学作品，表演、发明、科学发现、外观设计，标志、名称及符号，未披露信息，以及基于传统的革新与创造
- **民间文学艺术**：由一国国民或种族群落创作、代代相传并构成其传统文化遗产之基本组成部分的全部文学、艺术与科学作品
- **遗传资源**：具有现实或潜在价值的遗传物质，即植物、动物、微生物或其他来源具备遗传机能单位的任何物质

### 背景与属性

传统知识曾长期被视为公有领域。随着经济全球化和生物技术的进步，其价值日益凸显，同时文化及生物多样性的消失引发了复杂的伦理、法律及公共政策问题。

传统知识在食品安全、农业和医疗中起重要作用（如马来西亚传统医药人均消耗是现代药品的两倍以上），是保持生物多样性和文化多样性的重要资源，也是现代文艺作品创作的源泉（如《花木兰》）。

传统知识、民间文艺和遗传资源符合知识产权的客体属性，应当给予保护。否则会造成"生物海盗"行为，甚至使已公知的传统知识在他国被不当授予专利权。

## 二、种类和保护方式

### 类别

1. **保密信息**：祖传秘方（云南白药配方）、景泰蓝工艺、川剧变脸绝技等商业秘密
2. **创新成果**：基于中药古方开发的青蒿素、基于民间故事创作的《花木兰》等
3. **社区知识**：印度Neem树油、墨西哥传统黄豆品种、姜黄、Basmati香米等
4. **遗传资源**：中国野生大豆遗传信息、家族血样遗传信息等

### 保护方式

#### 1. 反不正当竞争法保护保密信息

《反不正当竞争法》第10条适用于第1类传统知识。权利归属：个人祖传秘方归个人或家庭；企业购买的归企业；国家保密范围的归国家。

#### 2. 现行知识产权制度保护创新成果

**专利保护**：发明专利保护基于传统知识的产品、方法或其改进。传统医药可通过性能（适应症）和制备方法定义来申请产品专利，不必因结构不确定而放弃专利保护。对于中药复方，可通过权利要求的合理概括和侵权诉讼中的等同原则防止变相侵权。

**商标保护**：如"同仁堂""云南白药""全聚德""茅台"等老字号。也可注册集体商标和证明商标，要求商品来源于特定社区或按该社区方法生产。

**著作权保护**：基于传统知识创作的书籍、动画片、歌曲等。

**植物品种权保护**：具有新颖性、特异性、一致性和稳定性的植物新品种。

#### 3. 追溯性行政法规保护社区知识

第3类社区知识因使用时间长，难以满足专利的新颖性和创造性要求。解决方法有两种：

- **积极性保护**：制定专门法律，如泰国传统泰药保护法、中国《传统工艺美术保护条例》
- **防御性保护**：建立数据库保护制度，实现传统知识文献化，防止在他国被不当授权

**建议**：优选积极性保护，但只能分阶段实行——先在国内甚至各省市立法（如云南省《民族民间传统文化保护条例》），再签订双边或多边协议，最后争取上升为国际协议。同时采取防御性保护作为补充，如建立民族医药数据库。

**关于《中药品种保护条例》**：具有追溯弥补性行政保护的性质，但存在保护主体错位（是使用者而非创造者）、限制国内厂家引发"内斗"、可能与专利冲突、保护期限过长不利于创新、只适用于境内品种可能被指歧视等问题，建议修改。

#### 4. 《生物多样性公约》保护遗传资源

1992年《生物多样性公约》旨在保护遗传资源、促进合理开发利用、保证知情同意和利益共享。1998年我国制定了《人类遗传资源管理暂行办法》。

建议：提高自我保护意识；积极开发遗传资源并转化为自主知识产权；加强生物技术知识产权保护。我国专利法规已增加遗传资源来源披露规定，可在一定程度上减少"生物海盗"行为。

## 三、保护的力度和期限

### 保护力度

- **最强**：现有知识产权制度（专利、商标、著作权、植物新品种权），有成熟司法救济程序
- **较弱**：反不正当竞争法对商业秘密的保护，适用范围有限，取证难
- **较弱**：《生物多样性公约》需通过国内法落实，管理困难
- **最弱且路途遥远**：追溯性行政法规保护社区知识，会遭发达国家反对

### 保护期限

应当适中：过长造成垄断，过短则显失公平。

| 保护类型 | 保护期限 |
|---------|---------|
| 发明专利 | 20年 |
| 实用新型 | 约10年 |
| 外观设计 | 7-25年 |
| 商标 | 10年，可续展（可无限期） |
| 著作权 | 作者有生之年加死后50年 |
| 植物新品种 | 15-25年 |
| 商业秘密 | 不泄密则不受限 |
| 遗传资源（信息） | 公开后大大缩短 |

追溯性行政法规的保护期应参考现有知识产权制度，适中为宜。不保护不合理，无限期保护也不可能实现。
