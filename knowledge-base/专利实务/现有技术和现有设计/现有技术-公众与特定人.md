# 现有技术 — 公众与特定人

> **来源：** 《以案说法》第二章 §1.1，页39-49
> **核心法条：** 《专利法》第22条第5款、第23条第4款
> **关联页面：** [[现有技术和现有设计-审查基准与为公众所知]]、[[现有技术-公开时间的判断]]、[[现有技术-出版物公开]]、[[现有技术-使用公开]]、[[审查-新颖性-引言与现有技术]]

---

## 核心要点

"为公众所知"是现有技术认定的核心概念，包括"公众"的范围界定和"能够为公众获得的状态"两个维度。负有保密义务的人属于特定人，不属于专利法意义上的公众；而行业背景、地域限制、购买手续等因素通常不构成对公众的限制。

---

## 1. "为公众所知"的基本概念

### 1.1 三要素框架

《专利审查指南》明确，现有技术或现有设计应当在申请日以前处于**能够为公众获得的状态**，并包含有能够使公众从中得知实质性技术或设计的内容。正确理解"为公众所知"需把握三个关键因素：

1. **公众** — 为公众所知的主体
2. **公众能够获知的状态** — 信息传播的方式与途径
3. **获知的实质性内容** — 公开的技术信息量

### 1.2 "公众"的法律定义

专利法意义上的公众，一般指**不受特定条件限制的人**。判断时需注意：

- **人的数量和地域范围不足以构成限制**：即使只有少数人实际接触到信息，只要接触者不负有保密义务，该信息仍可构成现有技术。
- **行业背景不构成限制**：某些技术手册的使用者通常具有相关行业或专业背景，但一旦公开出版发行，使用者的行业背景不能构成对公众的限制。
- **购买手续不构成限制**：需要注册、购买等手续才能获得的信息，只要任何公众都可以通过正常渠道完成这些手续，仍然属于向公众公开。

### 1.3 特定人的界定

对相应技术或设计内容负有保密义务的人**非专利法意义上的公众**，属于特定人。处于保密状态的技术或设计内容不属于现有技术。

特定人一般分为两类：

1. **基于法律规定或合同约定负有保密责任的人**
   - 例如：根据《科学技术成果鉴定办法》参加技术成果鉴定的有关人员
   - 科技行政管理机关、鉴定单位、同行专家以及其他参加鉴定的人员

2. **基于默示保密义务的人**
   - 产品用户通常应被认为是公众的一部分
   - 但如存在法定、约定或默示的保密义务，该用户从公众中特定化出来

## 2. 公众认定的典型案例

### 案例：第28737号无效决定（201320424967.3）— 技术手册的公开发行

- **争议焦点：** 行业内部出版的操作手册是否构成向公众公开？
- **决定要点：** 操作手册虽记载"操作或维修人员应该熟悉本手册"，但该措辞"仅仅表明上述人员应该熟悉本操作手册，并未限定该手册仅对操作或维修人员这一部分人公开，操作或维修人员也不负有保密义务"。该手册的发行对象应为公众，处于公开发行的状态。且从手册的编译发行者及序言可见，其发行目的不仅面向现有用户提供技术信息，还具有面向全行业宣传推广的目的。
- **启示：** 判断出版物是否面向公众，应考察其实际发行范围、面向受众、出版目的以及内容是否涉及保密义务，不能仅凭"内部资料"等字样或面向特定行业的表述即认定未向公众公开。

### 案例：第32123号无效决定（201420491206.4）— 公开招标文件的公开性

- **争议焦点：** 对投标人有资质要求且需要购买的招标文件是否属于向特定人公开？
- **决定要点：** 公开招标属于向社会公开发售招标文件的招标形式。招标人对投标人资质等的要求，只是"对投标人的入选范围作出的符合商业习惯的非特定性限制"；需要注册购买才能获得招标文件，并不妨碍"社会公众通过正常渠道能够获知招标文件的内容"。因此不属于向特定人发售招标文件的情形。
- **启示：** 商业活动中常见的资质审核、注册购买等条件，只要不导致信息实质上无法被不特定公众获取，通常不构成对公众的限制。

### 案例：第12955号无效决定（00219748.0）— "限国内发行"刊物的公开性

- **争议焦点：** 印有"限国内发行"字样的季刊杂志是否构成向公众公开？
- **决定要点：** "限国内发行"并不说明其为保密、公众不可得知的材料。该杂志封面印有"本杂志愿成为：窥探磷肥工业的窗口，交流各种肥料信息的桥梁，联系矿、工、商的纽带"，可见其目的在于交流信息，一经发行就已处于公众想得知就能够得知的状态。
- **启示：** 地域限制（如"限国内发行"）不等于保密限制，不能将其混为一谈。只要国内范围内的公众可以无保密义务地获取，即构成向公众公开。

### 案例：第881号无效决定（93228043.9）— 产品用户是否属于公众

- **争议焦点：** 公开销售给特定企业的产品用户是否属于公众？
- **决定要点：** "一般而言，产品用户应当被认为是公众的一部分，除非存在法定的、约定的或默示的保密义务使该用户从公众中特定化出来。假如产品用户被普遍地排除于公众范畴之外，则《专利法》第22条第2款中国内公开使用的规定将会形同虚设。"本案中鞍钢第一发电厂作为用户已得到该设备且能获知其实质性技术内容，尚无证据表明该用户是特定人。
- **启示：** 产品用户原则上属于公众，主张用户为特定人的一方需举证证明存在法定、约定或默示的保密义务。

## 3. 特定人认定的典型案例

### 案例：第335号无效决定（88200179）— 科技成果鉴定的保密义务

- **争议焦点：** 参加科技成果鉴定的人员是否属于特定人？
- **决定要点：** 根据原国家科委《科学技术成果鉴定办法》规定，参加鉴定的有关人员应当承担保密义务。基于该保密义务，科技行政管理机关、鉴定单位、同行专家以及其他参加技术成果鉴定的人员均属于特定人范畴。因此，通过科技成果鉴定程序获知的技术内容不构成现有技术。
- **启示：** 判断特定人时，应审查是否存在法律明文规定或合同明确约定的保密义务。科技成果鉴定、技术评审等程序中参与者通常负有法定保密义务。

---

## 4. 实务要点速查

| 情形 | 是否属于公众 | 关键判断因素 |
|------|------------|------------|
| 公开出版的技术手册/行业刊物 | ✅ 是 | 是否公开发行、读者是否有保密义务 |
| "内部资料""内部发行"刊物 | 需具体判断 | 实际发行范围、受众、出版目的、内容是否涉密 |
| 限国内/限地区发行 | ✅ 是 | 地域限制≠保密限制 |
| 公开招标文件 | ✅ 是 | 资质要求属于非特定性商业限制 |
| 产品用户 | 原则上✅是 | 是否存在法定/约定/默示保密义务 |
| 科技成果鉴定参与者 | ❌ 否 | 依法负有保密义务，属于特定人 |
| 合同约定的保密接收方 | ❌ 否 | 合同约定构成特定化 |
