## 实务要点速查

| 情形 | 裁判倾向 | 关键依据 | 代表案例 |
|------|---------|---------|---------|
| 离职后一年内作出与原单位本职工作技术领域相同的发明创造 | 倾向认定职务发明 | 专利法实施细则第12条第1款第3项 | （2024）最高法知民终825号 |
| 涉案专利与原单位工作在技术构思上有延续性和传承性 | 推定具有相关性 | 专利法第六条第一款 | （2024）最高法知民终825号 |
| 在职期间以其他公司名义申请与本职工作相关的专利 | 认定职务发明 | 专利法第六条第一款 | （2024）最高法知民终1055号 |
| 员工能证明技术方案在入职前已完成 | 不认定职务发明 | 以技术方案完成时间为判断界限 | （2023）最高法知民终1276号 |
| 中小企业管理人员的技术改进发明 | 可认定职务发明 | 不应将本职工作局限于研发工作 | （2014）民提字第90号 |
| 机械工程师转型管理岗位后的发明创造 | 可认定职务发明 | 部门名称不能排除研发功能 | （2022）最高法知民终804号 |
| 离职后短期内在新单位作出发明创造 | 倾向认定原单位职务发明 | 不符合技术开发的一般规律 | （2022）最高法知民终125号 |
| 销售人员因工作职责接触物料设备 | 不构成利用物质技术条件 | 须以研发为目的且有实质性影响 | （2020）最高法知民终931号 |
| 带技术入股股东利用公司资源完成研发 | 可认定职务发明 | "本单位"包括合作形成的临时单位 | （2021）最高法知民终535号 |
| 双方已就专利权归属达成协议 | 从其约定 | 专利法第六条第三款 | （2023）最高法知民终1049号 |
| 新单位无法证明自主研发过程 | 倾向认定职务发明 | 举证责任分配规则 | （2024）最高法知民终1038号 |
| 相关性认定不要求技术方案实质相同 | 支持相关性认定 | 延续性标准 | （2023）最高法知民终956号 |
| 离职后的免责条款 | 严格解释，不泛化放弃 | 意思表示解释规则 | （2024）最高法知民终1033号 |
