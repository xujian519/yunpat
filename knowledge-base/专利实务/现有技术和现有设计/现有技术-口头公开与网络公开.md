# 现有技术 — 口头公开与网络公开

> **来源：** 《以案说法》第二章 §4（口头公开及其他方式公开），页84；综合《专利审查指南》第二部分第三章第2.1.2.3节

> **关联页面：** [[现有技术-互联网公开]]、[[现有技术-使用公开]]、[[现有技术-出版物公开]]、[[现有技术-公众与特定人]]、[[新颖性-常见判断情形与抵触申请]]

## 核心要点

除出版物公开、使用公开和互联网公开外，现有技术还可以通过口头公开等其他方式为公众所知。口头公开包括讲座、报告、广播、电视等形式，其核心判断标准同样是相关信息是否已使公众中的任何人处于想得知就能够得知的状态。与互联网公开侧重证据时间认定不同，口头公开的关键在于公开行为是否面向不特定公众、公众能否从中获知实质性技术内容。

---

## 1. 口头公开的基本认定

### 1.1 口头公开作为"其他方式公开"

《专利审查指南》规定，现有技术与设计的公开方式包括出版物公开、使用公开和以其他方式公开。口头公开属于"以其他方式公开"的重要形式，是指通过口头语言向公众传播技术或设计内容，使相关信息处于公众想得知就能够得知的状态。

**口头公开的常见形式**：
- **学术讲座与技术报告**：在公开场合面向不特定听众进行的技术讲解
- **广播与电视节目**：通过无线电波或电视信号向公众传播的技术信息
- **会议发言与讨论**：在学术会议、技术会议上发表的口头演讲
- **课堂授课与培训**：面向公众或不特定学员进行的技术传授

### 1.2 口头公开的认定标准

判断口头公开是否构成现有技术，应当围绕以下要素：

1. **受众的公众性**：听讲者是否属于不特定的公众，还是负有保密义务的特定人
2. **内容的实质性**：口头公开的内容是否包含能够使公众从中得知实质性技术知识的内容
3. **公开的可获取性**：相关信息是否已实际处于公众想得知就能够得知的状态，而不仅仅是一种可能性
4. **时间的确定性**：能否确定口头公开行为发生在涉案专利申请日之前

**与使用公开的关联**：口头公开有时与使用公开相伴发生。例如，在产品展示会上，参展商既展示了产品（使用公开），又进行了口头讲解（口头公开）。此时，口头公开的内容与展示行为应分别考察。

## 2. 学位论文答辩中的口头公开

### 基本规则

学位论文在答辩前印送给相关学科专家进行评议，答辩过程通常被认为是在特定范围内的活动。答辩时，论文通常仅提供给答辩委员会成员，并不向旁听人员发放。旁听人员能够得知的技术内容一般仅限于论文答辩过程中**口头公开的技术内容**，其并不等同于论文的书面内容。

**关键区分**：
- **论文书面内容**：以书面形式记载在论文中的技术方案，其公开时间取决于图书馆上架日期或数据库收录日期
- **口头公开内容**：答辩过程中演讲者口头陈述的技术内容，其公开时间为答辩日期（如果答辩允许公众旁听）
- 图书馆或网络数据库平台最终收录的论文内容可能不同于论文答辩稿的内容，更不等同于论文答辩过程中口头公开的技术内容

### 案例：第28990号无效决定（201020553487.3）— 答辩日期不能等同于论文公开日

- **争议焦点：** 学位论文上载明的答辩日期能否作为该论文的公开日？
- **决定要点：** "请求人提交的证据3本身并未显示其出版日期或者印刷日期，仅凭证据3显示内容不能确定其作为公开出版物的公开时间。请求人提交的文献复制证明，仅能够表明在国家图书馆科技检索中心出具文献复制证明的检索日（该日期晚于涉案专利申请日）证据3处于公开状态，但是上述文献复制证明未表明，证据3何时被国家图书馆或其科技检索中心的数据库收录，或者证据3何时向公众公开出版。请求人也未提交其他证据证明证据3是何时公开出版或者被公共信息数据库收录"。"论文答辩时所公开的内容与国家图书馆收录的论文内容之间不存在直接的对应关系，证据3中答辩日期通常不能作为该论文已在论文答辩日期公开的依据。在请求人未提供其他证据证明证据3论文中的内容在答辩程序中已向社会公众公开的基础上，不能确定证据3所示内容已在答辩日公开"。
- **启示：** 学位论文的答辩日期通常不能等同于论文公开日。即使答辩允许旁听，旁听人员获知的技术内容仅限于口头公开部分，且与书面论文内容可能存在差异。主张论文在答辩日公开的一方，需举证证明答辩程序中论文内容已向社会公众公开。

## 3. 口头公开的时间认定与证据固定

### 3.1 口头公开的时间认定难点

与出版物公开可以适用"印刷日推定"规则不同，口头公开的时间认定面临更多不确定性：

- **缺乏固定载体**：口头公开以声音为媒介，不像出版物那样有固定的物理载体可供追溯
- **内容难以完整还原**：听讲者对口头内容的理解和记忆可能存在偏差
- **受众范围难以确定**：难以准确判断听讲者是否属于公众还是特定人

### 3.2 口头公开的证明方法

在专利确权程序中，主张口头公开构成现有技术的一方，通常需要提供以下证据：

1. **公开活动的证明**：会议通知、讲座海报、节目单等，证明公开活动确实发生
2. **时间的证明**：活动举办日期、节目播出日期等，证明公开行为发生在申请日前
3. **内容的证明**：会议记录、听众笔记、录音录像、后续报道等，证明公开的技术内容
4. **受众性质的证明**：参会人员名单、入场规则等，证明听讲者属于公众而非特定人

## 4. 网络视听内容的特别认定

随着互联网技术的发展，广播、电视节目等内容日益通过网络平台传播。网络视听内容的公开性认定，需结合其传播方式：

- **传统媒体的网络版**：如央视网发布的新闻视频，其可信度较高，发布时间通常可由服务器时间戳确定
- **网络直播平台**：直播内容的公开时间通常为直播开始时间，但需注意回放功能是否开启
- **社交媒体短视频**：用户上传的短视频，其公开性需考察账号的公开设置及平台的传播机制

**与互联网公开的衔接**：网络视听内容本质上属于互联网公开的一种特殊形式。对于通过知名媒体网站（如央视网）发布的视频内容，其公开时间的认定可参照互联网公开的一般规则，即网页显示的时间为服务器自动生成的时间，可视为公开日。
