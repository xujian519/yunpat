# 专利领域特定知识参考

> 用途：为专利撰写和分析提供各技术领域的专业知识支撑
> 引用方式：在相关 skill 模板中引用本文件

---

## 化学领域

### 创造性判断特点

- **化合物发明**：核心在于结构与效果的关系。选择用途/性能相同且化学结构尽可能相近的现有技术化合物作为最接近现有技术。结构接近+无预料不到效果 → 显而易见
- **晶体化合物**：与已知化合物化学结构相同，需产生预料不到效果才具备创造性。常见晶型+可预期改善 → 不具备创造性
- **组合物发明**：分析现有技术是否给出联用指引，是否存在妨碍联用的技术障碍或偏见
- **制药用途发明**：核心在于发现新适应症。新旧适应症的致病机理高度关联 → 有动机扩展用途
- **参数限定发明**：权利要求的数值范围大于说明书公开范围时，需端点值附近实施方式支持
- **马库什结构**：各取代基定义需清楚，概括范围需与实施例匹配

### 撰写要点

1. **充分公开要求高**：必须记载具体的实施方式、关键物质配比、用量范围等核心参数，仅笼统的原理性描述通常不够
2. **效果数据必不可少**：说明书中需记载证明化学效果的定性或定量实验数据和对比试验
3. **数值范围需端点支撑**：涉及参数限定的权利要求，应公开端点值及附近的实施例
4. **取代基定义要清楚**：马库什权利要求中每个取代基的定义须明确，避免概括过宽被认定不支持
5. **多来源一致性**：同一物质在不同组合物中作用不同时，不构成技术启示（可用于答复创造性审查意见）

## 软件领域

### 专利适格性

- **可专利的三类方案**：自动化技术处理过程、计算机内部运行性能改进、测量或测试过程
- **不可专利的情形**：单独的计算机程序、与硬件结合但属于智力活动范畴、需人的智力活动参与的非完整方案、无技术效果的纯数学运算
- **商业方法**：若仅涉及商业规则和方法的改进，不产生技术效果，不授予专利权；但若与技术手段结合解决技术问题，可能获得保护
- **算法专利**：纯算法属于智力活动规则，不授予专利权；但与具体技术领域结合、产生技术效果的算法方案可获保护

### 撰写要点

1. **用技术特征描述**：不得用数学符号或抽象公式替代技术语言，应以硬件运行过程描述技术方案
2. **不记载源程序**：专利保护技术方案而非代码，记载源程序会不当缩小保护范围
3. **提供主要流程图**：流程图是连接抽象程序功能与具体技术实施的桥梁，须覆盖核心步骤
4. **权利要求类型灵活**：可写成方法权利要求或产品权利要求（功能模块架构），保护范围实质相同
5. **充分公开标准**：公开重点在于程序指令硬件的运行过程，技术人员据此能编写出实现相同功能的程序即可

## 生物领域

### 创造性判断特点

- **基因发明**：已知结构基因可自然获得的突变基因，若来源相同、性质功能相同 → 不具备创造性
- **抗体发明**：针对已知抗原表位的单克隆抗体通常不具备创造性，但若产生不可预期的优异效果（如更广抗性）仍可具备
- **生物保藏**：涉及生物材料的发明，若公众无法获得该生物材料，需进行生物保藏
- **实验数据**：生物领域效果可预期性低，通常需要实验数据证实

### 撰写要点

1. **生物保藏要求**：涉及公众无法获得的生物材料时，须按规定进行保藏并在说明书中记载保藏信息
2. **效果数据要求严格**：生物发明技术效果可预期性低，需记载充分的实验过程和结果
3. **序列表要求**：涉及核苷酸或氨基酸序列的发明，须提交符合标准的序列表
4. **宽泛权利要求的限制**：生物领域宽泛权利要求（如属/种级别的范围）需充分实施例支撑

## 中药领域

### 创造性判断特点

- **组合物创造性**：药味及药量变化若仅遵循中药方剂一般组方规律（随证加减、数方加减、相互代用、药对配伍等），且未使功能、疗效、治法治则产生实质变化 → 不具备创造性
- **效果可预期性低**：中药组合物的效果通常需要实验证实，不能简单推论
- **参数/用量限定**：中药用量的具体选择若未产生预料不到效果，可能被认定为常规优化

### 撰写要点

1. **药材名称规范**：使用《中华人民共和国药典》记载的正名；使用俗名或异名时须在说明书中解释来源或功效以唯一确定
2. **功效数据充分**：中药发明技术效果可预期性低，说明书须记载定性或定量的药效实验过程和结果
3. **组方依据说明**：说明方剂组成的理论依据（君臣佐使、归经等），有助于论证非显而易见性
4. **制备工艺详述**：提取方法、工艺条件（温度、时间、溶剂等）需记载充分，避免公开不充分

## 机械领域

### 创造性判断特点

- **惯用手段替换常见**：螺栓改焊接、直齿轮改斜齿轮等常规连接/传动方式替换，通常被认定为惯用手段
- **结构变形**：将已知结构进行显而易见的变形（如圆形改方形、直线改曲线），若效果可预期则不具备创造性
- **功能扩展**：增加本领域常规功能模块（如增加传感器、显示屏），通常不具备创造性
- **位置调整**：部件位置变化若属常规设计选择且效果可预期，不具备创造性

### 撰写要点

1. **结构特征详述**：机械发明的核心在于结构特征，须详细描述各部件的形状、位置、连接关系和配合方式
2. **附图极为重要**：机械领域附图是理解技术方案的关键，须提供充分的结构示意图、剖视图、局部放大图
3. **权利要求层级设计**：独立权利要求概括核心结构，从属权利要求层层细化具体实施方式
4. **避免功能性限定**：尽量用结构特征而非功能性描述限定保护范围，减少清楚性争议

## 电学领域

### 创造性判断特点

- **电路改进**：用已知电路模块替代另一已知模块以实现相同功能，通常属惯用手段
- **参数优化**：在已知范围内调整电气参数（电压、频率、功率等），效果可预期的，不具备创造性
- **半导体器件**：结构接近的器件改进，若无预料不到效果，通常不具备创造性

### 撰写要点

1. **电路框图和时序图**：电学发明须提供充分的电路框图、信号流程图或时序图
2. **参数范围限定**：电气参数权利要求须有充分实施例支撑，特别是关键参数的端点值
3. **功能模块架构**：可采用功能模块+信号流向的方式撰写权利要求，兼顾保护范围和清楚性

## 通用三步法框架

> 适用于所有领域的创造性判断

```
第一步：确定最接近现有技术
  └─ 技术领域相同/相近优先，考量技术问题和发明构思

第二步：确定区别特征和实际解决的技术问题
  └─ 准确理解区别特征的技术功能，不能简单文字比对

第三步：判断是否显而易见（技术启示）
  └─ 站在本领域技术人员视角，避免"事后诸葛亮"
```

**辅助判断因素**：

- 人们一直渴望解决但始终未获成功的技术难题 → 支持创造性
- 克服了技术偏见 → 支持创造性
- 产生了预料不到的技术效果 → 支持创造性
- 商业成功（需证明与技术特征直接关联） → 辅助支持
