# 如何运用‘单对比文件+公知常识’模式来否定权利要求的创造性？

## 卡片内容

### 核心概念

该模式的核心逻辑在于，如果权利要求与最接近的现有技术之间存在区别特征，但该区别特征要么已被其他现有技术公开，要么属于本领域的公知常识，并且在采用该特征后并未产生预料不到的技术效果，那么本领域技术人员结合现有技术和公知常识可以显而易见地得到该权利要求的技术方案，从而认定该权利要求不具备创造性。

### 法律依据

该模式主要依据《专利法》第22条第3款的规定。在专利无效程序中，判断创造性的关键在于对区别特征是否属于公知常识或惯用手段的认定。

### 推理过程详解（三步法）

1. **确定最接近现有技术：** 合议组首先需要从证据中选择一篇与权利要求技术方案最为接近的对比文件作为判断创造性的起点。
2. **确定区别特征与技术问题：** 在明确最接近现有技术的基础上，需要找出权利要求中的区别特征，并基于该特征的作用确定其实际解决的技术问题。
3. **判断区别特征是否属于公知常识：** 这是核心环节。如果区别特征本身属于本领域的公知常识、惯用手段或常规选择，则无需引入其他对比文件，即可认定本领域技术人员有能力和动机将该区别特征应用于最接近现有技术。
4. **判断是否产生预料不到的技术效果：** 即使区别特征被认定为公知常识，如果将其与最接近现有技术结合后产生了预料不到的技术效果，权利要求仍可能具备创造性。

### 成功率分析

该模式是无效程序中最高效、最常用的否定路径。在大量样本中（超过一半），该模式的适用性极高，且全部无效的效率高达95.9%。这表明，在主张该模式时，需要有充分的证据来证明区别特征确实属于公知常识。

### 实务要点

**对于请求人：**
*   充分说明区别特征为何属于公知常识，最好提供教科书、工具书等证据支持。
*   预判专利权人可能的反驳方向，提前准备应对策略。
*   在口头审理中，要清晰阐述区别特征与最接近现有技术结合的显而易见性。

**对于专利权人：**
*   防御策略包括：强调区别特征的技术贡献，证明其解决了特定技术问题；提供实验数据或对比试验，证明区别特征产生了预料不到的技术效果；质疑请求人对公知常识的认定，要求其提供充分证据。
