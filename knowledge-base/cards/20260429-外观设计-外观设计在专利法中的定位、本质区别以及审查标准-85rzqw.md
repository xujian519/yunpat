## 卡片内容

### 核心定位

外观设计是专利保护的独立客体类型，但其本质上更接近美学设计而非技术方案。理解外观设计与发明、实用新型的本质区别，对于正确把握专利客体审查框架至关重要。

### 法律定义

根据《专利法》第2条第4款的规定，外观设计是指对产品的形状、图案或者其结合以及色彩与形状、图案的结合所作出的富有美感并适于工业应用的新设计。

### 与其他专利类型的并列关系

外观设计与发明和实用新型并列为三种专利保护客体。其核心差异在于保护对象和创造性标准：

*   **发明和实用新型：** 保护的是技术方案，核心在于技术功能的创新。
*   **外观设计：** 保护的是美学表达，核心在于视觉效果的创新。

### 本质属性的差异

外观设计本质上更接近**美学设计**，而非技术方案。这种判断基于以下几点：

1.  **保护对象不同：** 外观设计保护的是产品外观给人的视觉感受，而非产品的技术功能。
2.  **创造性标准不同：** 外观设计要求"富有美感"，而非传统技术方案的创造性。
3.  **侵权判断方法不同：** 外观设计侵权判断侧重于一般消费者的整体视觉效果，而非技术特征的比对。

### 与技术方案的区别

| 对比维度 | 发明/实用新型 | 外观设计 |
| :--- | :--- | :--- |
| 保护目标 | 技术功能 | 美学效果 |
| 核心要素 | 技术手段、技术问题、技术效果 | 形状、图案、色彩及其结合 |
| 创造性要求 | 创造性/技术进步 | 富有美感、与现有设计有明显区别 |
| 实用性要求 | 能够制造或使用并产生积极效果 | 适于工业应用 |
| 侵权判断 | 技术特征比对 | 整体视觉效果比对 |

### 审查标准的差异

*   **技术方案审查：** 关注是否利用自然规律、采用技术手段、解决技术问题、产生技术效果。
*   **外观设计审查：** 关注是否属于产品的形状、图案或其结合、是否富有美感、是否适于工业应用、以及是否与现有设计有明显区别。

### 与版权法的接近性

外观设计的制度原则更接近版权法，主要体现在：

1.  **保护表达而非思想：** 保护的是具体的设计表达，而非抽象的设计思想。
2.  **独创性要求：** 要求与现有设计有明显区别，类似于版权法中的独创性要求。
3.  **美学价值导向：** 强调美学价值而非技术价值。

### 制度演变

2020年《专利法》第四次修改对外观设计制度进行了调整，包括：

*   **保护期限延长：** 从10年延长至15年。
*   **局部外观设计：** 明确允许对产品局部申请外观设计专利。
*   **新颖性宽限期：** 完善了外观设计的新颖性宽限期制度。

### 审查要件

外观设计客体审查需依次通过以下步骤：

1.  **产品性：** 必须以产品为载体，不能脱离产品单独存在，且产品应能通过工业方法重复制造。
2.  **外观性：** 必须是产品的形状、图案或其结合，或色彩与形状、图案的结合，且能够为视觉所感知。
3.  **美感性：** 必须"富有美感"，美感判断以一般消费者的审美为标准，不要求达到艺术高度。
4.  **工业应用性：** 必须"适于工业应用"，能够通过工业方法批量生产，手工制作的艺术品通常不符合此要求。

### 排除情形

以下情形不属于外观设计保护客体：

*   纯艺术品：不能通过工业方法重复制造的手工艺品或纯美术作品。
*   功能性特征：仅由产品的功能决定的形状（如齿轮的齿形）。
*   标识设计：主要起标识作用的设计（归入第25条第1款第6项排除）。
*   违反公序良俗：带有暴力、淫秽内容的设计。

### 权利要求的协调

在同一产品上，发明/实用新型权利要求保护的是技术方案，外观设计保护的是美学表达。两者保护范围存在区别，侵权判断标准也不同。在实践中，如果形状同时具有功能性和美学性，可以同时申请发明/实用新型和外观设计专利。
