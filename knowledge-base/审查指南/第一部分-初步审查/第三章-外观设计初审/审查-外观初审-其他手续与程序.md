# 审查-外观初审-其他手续与程序

> **来源：** 《专利审查指南》第一部分第三章 §5、§12
> **对应法条：** 《专利法》第29条、第30条 / 《专利法实施细则》第34-35条、第40条
> **最新修订：** 2023年修订
> **关联页面：** [[审查-发明初审-其他手续与费用]]、[[审查-外观初审-图片照片与简要说明]]、[[外观-原理-初步审查]]

---

## 核心要点

外观设计专利申请的其他手续审查中，优先权规则有其特殊性：外国优先权和本国优先权期限均为6个月（发明/实用新型为12个月）；在先申请可以是发明、实用新型或外观设计专利申请。此外，外观设计分类采用洛迦诺分类法（非IPC），按产品用途确定分类号。局部外观设计和GUI设计有特殊的分类规则。

---

## 审查标准

### 一、委托专利代理机构（§5.1）

适用本部分第一章第6.1节的规定。与发明/实用新型相同。

### 二、要求优先权（§5.2）

#### 5.2.1 外国优先权

**期限**：自外观设计在外国第一次提出专利申请之日起**六个月**内。

**在先申请的范围**：可以是发明、实用新型或外观设计专利申请。

**在先申请文件副本**：应在提出在后申请之日起**三个月**内提交（发明/实用新型为16个月）。

**优先权转让证明**：申请人完全不一致的，应在提出在后申请之日起**三个月**内提交（发明/实用新型为16个月）。

**特殊规定**：
- 在先申请未包括简要说明的，申请人提交的简要说明未超出在先申请文件图片/照片表示范围的，不影响享有优先权。
- 在一件申请中可以要求一项或多项优先权。

#### 5.2.2 本国优先权

**期限**：自外观设计在中国第一次提出专利申请之日起**六个月**内。

**在先申请的范围**：可以是发明、实用新型或外观设计专利申请，不应当是分案申请。

**在先申请的条件**：
1. 在先申请的主题没有要求过优先权，或虽要求过但未享有优先权。
2. 该在先申请的主题尚未被授予专利权。
3. 在后申请是在其在先申请的申请日起六个月内提出的。

**视为撤回在先申请**：
- 申请人要求本国优先权的，在先申请自在后申请提出之日起即视为撤回。
- **例外**：外观设计专利申请的申请人要求以发明或实用新型专利申请作为本国优先权基础的，在先申请不视为撤回。
- 被视为撤回的在先申请不得请求恢复。

#### 5.2.3 优先权要求的撤回
适用第一章§6.2.4的规定。

#### 5.2.4 优先权要求费
适用第一章§6.2.5的规定。

#### 5.2.5 优先权要求的恢复
适用第一章§6.2.6.1节的规定。

### 三、不丧失新颖性的公开（§5.3）

适用第一章§6.3节的规定。四种情形（紧急状态公开、国际展览会、学术会议、他人泄露）的审查标准相同。

### 四、撤回专利申请声明（§5.4）

适用第一章§6.6节的规定。

### 五、著录项目变更（§5.5）

适用第一章§6.7节的规定。

### 六、外观设计分类（§12）

专利局采用**国际外观设计分类法**（洛迦诺分类法），以最新公布的《国际外观设计分类表》中文译本为工作文本。

#### 分类目的
1. 确定外观设计产品的类别属性。
2. 对外观设计专利进行归类管理。
3. 便于检索查询。
4. 按分类号顺序编排和公告外观设计专利文本。

#### 分类号格式
由"LOC""（版本号）""Cl.""大类号-小类号"组合而成。如：LOC（14）Cl.06-04。

#### 12.1 分类的依据
以产品名称、图片或照片以及简要说明中记载的产品用途为依据。

#### 12.2 分类的方法
- 遵循用途原则，不考虑制造材料。
- 按先大类再小类的顺序确定。
- **零部件**：有专属类别的分入专属类别（如汽车轮胎→12-15）；无专属类别的分入其上位产品所属类别（如打火机打火轮→27-05）。
- 因时代发展衍生出新用途的产品，仍保持传统用途的所属类别（如灯笼仍分入26大类照明设备）。

#### 12.3 分类号的确定

**单一用途产品**：
- 一件产品+用途单一 → 一个分类号。
- 同一产品多项设计+用途单一 → 一个分类号。
- 多件产品+用途相同单一 → 一个分类号（如枕套、床单、被罩→06-13）。

**多用途产品**：
- 一件产品为两个以上不同用途产品的组合体 → 多个分类号（家具组合体除外）。
- 如带有温度计的相框→06-07和10-04。
- 连体桌椅（家具组合体）→06-05。

**特殊产品**：
- 局部外观设计 → 产品整体和局部各对应的分类号。
- GUI设计 → 产品和GUI各对应的分类号。仅提交GUI不带有产品的，仅给GUI的分类号。

**分类的补正**：
- 产品名称、图片或照片不能确定用途且简要说明未写明 → 发出补正通知书。
- 产品名称、图片或照片确定的用途与简要说明明显不一致 → 发出补正通知书。

## 实务应对

### 代理人视角：外观设计手续办理要点

1. **优先权期限特殊**：外观设计优先权期限为6个月（而非12个月），在先申请文件副本和转让证明的提交期限为3个月（而非16个月）。代理人应特别注意外观设计优先权的更短期限。

2. **本国优先权的例外**：外观设计申请要求以发明或实用新型作为本国优先权基础时，在先的发明/实用新型申请不视为撤回（但反过来，以外观设计作为在先申请时则会视为撤回）。

3. **在先申请范围更广**：外观设计申请的在先申请可以是发明、实用新型或外观设计。发明申请附图显示的设计可以作为外观设计优先权的基础。

4. **分类以用途为准**：外观设计分类不考虑制造材料，以产品用途为依据。产品名称和简要说明中的用途描述直接影响分类。

5. **多用途产品**：一个产品具有多种用途的，应给出多个分类号（家具组合体除外）。

6. **局部外观设计和GUI设计分类**：需同时给出产品整体和局部/GUI的分类号。

## 审查要点速查

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 外观设计优先权超过6个月 | 视为未要求优先权 | 法第29条 |
| 在先申请文件副本超3个月未提交 | 发出视为未要求优先权通知书 | 细则第34条 |
| 本国优先权在先申请已被授权 | 视为未要求优先权 | 细则第35条 |
| 以发明申请为基础要求外观设计本国优先权 | 在先发明申请不视为撤回 | 细则第35条第3款 |
| 简要说明未写明产品用途 | 发出补正通知书 | 细则第31条 |
| 产品用途描述与名称/图片不一致 | 发出补正通知书 | 指南§12.3.4 |
| GUI设计未提交产品分类号 | 需补充产品和GUI的分类号 | 指南§12.3.3 |
