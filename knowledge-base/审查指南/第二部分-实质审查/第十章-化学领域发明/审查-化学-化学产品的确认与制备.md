# 审查-化学-化学产品的确认与制备

> **来源：** 《专利审查指南》第二部分第十章 §5-6
> **对应法条：** 《专利法》第二十二条第二款、第三款
> **最新修订：** 2023年修订（化合物新颖性判断标准完善）
> **关联页面：** [[审查-化学-不授予专利权的化学发明]]、[[审查-化学-化学方法发明与实用新型]]

---

## 核心要点

化学发明的新颖性和创造性审查有其特殊性。化合物的新颖性需要考虑结构信息的公开程度，组合物的新颖性与开放式/封闭式撰写方式密切相关。创造性审查中，化合物的结构差异和由此带来的预料不到的效果是关键判断因素。化学产品用途发明的新颖性不因产品已知而丧失，关键在于用途本身是否是新的。

---

## 审查标准

### 1. 化学发明的新颖性

#### 1.1 化合物的新颖性

**（1）基本规则**：如果对比文件中记载了化合物的化学名称、分子式（或者结构式）等结构信息，使所属技术领域的技术人员认为要求保护的化合物已经被公开，则该化合物不具备新颖性。但申请人能提供证据证明在申请日之前无法获得该化合物的除外。

如果依据对比文件中记载的结构信息不足以认定结构异同，但结合其他信息（物理化学参数、制备方法和效果实验数据等）综合考量后，所属技术领域的技术人员有理由推定二者实质相同，则要求保护的化合物不具备新颖性，除非申请人能证明结构确有差异。

**（2）通式与具体化合物**：
- 通式不能破坏该通式中一个具体化合物的新颖性；
- 一个具体化合物的公开使包括该具体化合物的通式权利要求丧失新颖性，但不影响其他化合物的新颖性；
- 一个范围的化合物能破坏该范围内两端具体化合物的新颖性；
- 如果两端化合物有几种异构体，不能破坏每个单独异构体的新颖性。

**（3）天然物质**：天然物质的存在本身并不能破坏发明物质的新颖性，只有对比文件中公开的与发明物质的结构和形态一致或者直接等同的天然物质，才能破坏新颖性。

#### 1.2 组合物的新颖性

**仅涉及组分时的新颖性判断**：

（i）发明专利申请为组合物乙（A+B），采用封闭式（"由A+B组成"），即使与组合物甲（A+B+C）解决的技术问题相同，仍有新颖性。

（ii）采用开放式（"含有A+B"），与组合物甲解决的技术问题相同，则无新颖性。

（iii）采用排除法（指明不含C），仍有新颖性。

**涉及组分含量时**：适用本部分第三章第3.2.4节的规定。

#### 1.3 用物理化学参数或制备方法表征的产品新颖性

**物理化学参数**：如果无法依据所记载的参数进行比较，不能确定区别，则推定不具备新颖性。

**制备方法**：新颖性审查应针对产品本身，而不是仅仅比较制备方法。方法不同不一定导致产品不同。如果仅仅是制备方法不同，没有表明带来功能、性质上的改变，则推定不具备新颖性。

#### 1.4 化学产品用途发明的新颖性

新产品的用途发明自然具有新颖性。已知产品不能因为提出新应用而被认为是新产品，但如果新用途本身是发明，则已知产品不能破坏该新用途的新颖性。

**医药用途发明的新颖性审查**：
（1）新用途与原已知用途是否实质上不同；
（2）新用途是否被原已知用途的作用机理、药理作用所直接揭示；
（3）新用途是否属于原已知用途的上位概念（已知下位用途可破坏上位用途的新颖性）；
（4）给药对象、给药方式等与使用有关的特征是否对制药过程具有限定作用（仅体现在用药过程中的区别不能使用途具有新颖性）。

### 2. 化学发明的创造性

#### 2.1 化合物的创造性

**（1）基本判断方法**：
- 确定要求保护的化合物与最接近现有技术化合物之间的结构差异；
- 基于结构改造所获得的用途和/或效果确定发明实际解决的技术问题；
- 判断现有技术整体上是否给出了通过这种结构改造以解决所述技术问题的技术启示。

如果所属技术领域的技术人员仅通过合乎逻辑的分析、推理或者有限的试验就可以进行结构改造，则认为存在技术启示。

**（2）预料不到的效果**：如果结构改造带来的用途改变和/或效果改进是预料不到的，则反映了化合物是非显而易见的，应认可其创造性。

**（3）已知趋势排除**：如果效果是已知的必然趋势所导致的，则没有创造性。

**（4）创造性判断示例**：

| 示例 | 结构差异 | 创造性判断 | 理由 |
|------|---------|-----------|------|
| 例1 | 母核结构不同，用途相同 | 具备创造性 | 现有技术不存在改造技术启示 |
| 例2 | 插入-CONH-，用途完全不同 | 具备创造性 | 无动机将抗菌素改造为抗糖尿病药 |
| 例3 | NH2与CH3置换（电子等排体），效果相当 | 无创造性 | 经典电子等排体置换有动机进行 |
| 例4 | -O-替换-NH-（电子等排体），活性提高40倍 | 具备创造性 | 取得预料不到的技术效果 |
| 例5（通式） | -S-替换-O-，性质接近 | 通式无创造性 | 有动机进行替换 |
| 例5（具体化合物） | 连接原子+取代基均不同，活性明显优 | 具备创造性 | 不存在提升活性的技术启示 |

#### 2.2 化学产品用途发明的创造性

**（1）新产品用途发明**：如果该用途不能从结构或组成相似的已知产品预见到，可认为有创造性。

**（2）已知产品用途发明**：如果新用途不能从产品本身的结构、组成、分子量、已知物理化学性质以及现有用途显而易见地得出或预见到，而是利用了新发现的性质并产生预料不到的技术效果，可认为有创造性。

### 3. 化学发明的实用性

#### 3.1 菜肴和烹调方法

不适于在产业上制造和不能重复实施的菜肴，不具备实用性。依赖于厨师的技术、创作等不确定因素导致不能重复实施的烹调方法不具备实用性。

#### 3.2 医生处方

医生处方和医生对处方的调剂以及仅仅根据医生处方配药的过程，均没有工业实用性，不能被授予专利权。

### 4. 化学发明的单一性

#### 4.1 马库什权利要求的单一性

如果一项申请在一个权利要求中限定多个并列的可选择要素，则构成马库什权利要求。

**当马库什要素是化合物时的单一性标准**：
- （1）所有可选择化合物具有共同的性能或者作用；和
- （2）所有可选择化合物具有共同的结构，该共同结构能构成与现有技术的区别特征并对共同性能是必不可少的；或在没有共同结构的情况下，所有要素应属于公认的同一化合物类别。

**示例分析**：
- 例1：3-吡啶基吲哚部分是区别于现有技术的共同结构，具有单一性。
- 例2：酯化成（I）具有抗热降解性能，酯化成（II）不具有，没有共同性能，不具有单一性。
- 例3：不同元环的杂环化合物，没有共同结构也不能预期相同表现，不具有单一性。
- 例4：马库什要素B分属不同类化合物（无机盐、有机盐、酰胺、腈、苯酚、胺、杂环），不具有单一性。
- 例5：X和X+A两种催化剂共同成分X，共同用于氧化，具有单一性。

#### 4.2 中间体与最终产物的单一性

**单一性条件**（需同时满足）：
- （i）中间体与最终产物有相同的基本结构单元，或化学结构在技术上密切相关；
- （ii）最终产物是直接由中间体制备的，或直接从中间体分离出来的。

## 实务应对

**代理人视角：**

1. **化合物新颖性的答辩策略**：如果审查员引用通式对比文件，应论证通式不能破坏具体化合物的新颖性。如果引用具体化合物，应从结构信息是否足以认定"实质相同"进行反驳，或提供证据证明结构确有差异。

2. **组合物权利要求的撰写与新颖性**：封闭式撰写有利于保持新颖性（即使与已知组合物相比少了某些组分），但保护范围较窄。开放式撰写保护范围宽，但可能丧失新颖性。应根据具体情况选择。

3. **化合物创造性答辩的核心**：强调结构改造带来了"预料不到的技术效果"。提供定量对比数据证明效果的显著差异（如例4中活性提高40倍）。避免在已知趋势方向上进行简单的延伸。

4. **马库什权利要求的单一性把控**：确保所有可选择要素具有共同性能和共同结构（或属于公认同一类别）。如果不同要素的性能差异大，应考虑分案申请。

5. **中间体与最终产物的策略**：如需在一件申请中同时保护中间体和最终产物，应确保它们之间满足"相同基本结构单元+直接制备"的双重要求。

## 审查要点速查

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 通式公开但不公开具体化合物 | 不破坏具体化合物新颖性 | 通式不能破坏其中具体化合物 |
| 具体化合物公开 | 破坏包含该化合物的通式新颖性 | 具体公开优先于概括 |
| 封闭式"A+B"vs已知"A+B+C" | 有新颖性 | 封闭式排除未指出组分 |
| 开放式"含有A+B"vs已知"A+B+C" | 无新颖性 | 开放式不排除其他组分 |
| 排除法"含A+B，不含C"vs已知"A+B+C" | 有新颖性 | 明确排除了C |
| 仅为制备方法不同，产品无功能/性质改变 | 推定无新颖性 | 方法不同不一定产品不同 |
| 结构改造+预料不到效果 | 具备创造性 | 非显而易见 |
| 电子等排体置换+效果相当 | 无创造性 | 有动机进行置换 |
| 电子等排体置换+效果显著提高 | 具备创造性 | 预料不到的技术效果 |
| 已知趋势方向的延伸 | 无创造性 | 必然趋势导致 |
| 菜肴和烹调方法 | 无实用性 | 不能重复实施 |
| 医生处方 | 无实用性 | 没有工业实用性 |
| 马库什要素无共同性能 | 无单一性 | 不满足单一性要求 |
| 中间体+最终产物（同结构+直接制备） | 有单一性 | 满足双重要求 |
