## 🚀 立即启动（4 条命令）

```bash
# 1. 启动数据库
cd packages/core/src/memory/long-term
docker-compose up -d

# 2. 初始化数据库
docker exec -it yunpat-postgres psql -U yunpat -d yunpat -f /docker_entrypoint_initdb.d/init.sql

# 3. 设置环境变量
export DATABASE_URL="postgres://yunpat:yunpat123@localhost:5432/yunpat"
```

## 💡 使用示例

```typescript
import { createMemoryLayer } from '@yunpat/core';

// 1. 初始化
const memory = await createMemoryLayer({
  databaseUrl: 'postgres://yunpat:yunpat123@localhost:5432/yunpat',
  vectorDimension: 1024, // BGE-M3
});

// 2. 添加记忆
await memory.addMemory({
  type: 'patent',
  content: '专利 CN123456 涉及人工智能技术',
  embedding: [0.1, 0.2, ...], // 1024 维向量
  metadata: { agent: 'writer', tags: ['AI', '专利'] },
});

// 3. 搜索记忆
const results = await memory.searchMemories(
  queryEmbedding,
  10,
  { types: ['patent'], tags: ['AI'] }
);

// 4. 图查询
const path = await memory.findShortestPath(
  personId,
  companyId,
  5
);
```

## ⚠️ 注意事项

1. **向量维度**: 必须与嵌入模型一致（BGE-M3: 1024）
2. **数据规模**: 建议 <50 万向量（超过需迁移）
3. **图深度**: 建议 <5 跳（避免性能问题）
4. **类型警告**: 有 Drizzle ORM 类型警告（不影响功能）
