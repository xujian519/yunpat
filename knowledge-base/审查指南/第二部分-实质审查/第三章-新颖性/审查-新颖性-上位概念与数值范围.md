# 审查-新颖性-上位概念与数值范围

> **来源：** 《专利审查指南》第二部分第三章 §3.2.2 具体（下位）概念与一般（上位）概念 + §3.2.4 数值和数值范围
> **对应法条：** 《专利法》第22条第2款
> **最新修订：** 无特殊修订
> **关联页面：** [[审查-新颖性-对比原则]]、[[审查-新颖性-相同与实质相同]]、[[新颖性-法条-新颖性判断]]

---

## 核心要点

**下位概念的公开破坏上位概念的新颖性**，但上位概念的公开不影响下位概念的新颖性。对于数值和数值范围，判断规则更为细致：对比文件的数值或范围落在要求保护的范围内、两者部分重叠或有共同端点时，破坏新颖性；但要求保护的数值或范围完全落在对比文件公开的范围内且无共同端点时，不破坏新颖性。

---

## 审查标准

### 一、具体（下位）概念与一般（上位）概念（§3.2.2）

#### 1.1 基本规则

如果要求保护的发明或者实用新型与对比文件相比，其区别仅在于前者采用**一般（上位）概念**，而后者采用**具体（下位）概念**限定同类性质的技术特征，则：

| 方向 | 新颖性影响 | 规则 |
|------|-----------|------|
| 下位→上位 | **破坏新颖性** | 具体（下位）概念的公开使采用一般（上位）概念限定的发明丧失新颖性 |
| 上位→下位 | **不影响新颖性** | 一般（上位）概念的公开不影响采用具体（下位）概念限定的发明的新颖性 |

#### 1.2 典型案例

**案例1：金属与铜**

| 对比项 | 内容 |
|--------|------|
| 对比文件公开 | 某产品是"用铜制成的" |
| 专利申请1 | "用金属制成的"同一产品 |
| 专利申请2 | "用铁制成的"同一产品 |

**分析：**
- "铜"是"金属"的下位概念，铜的公开破坏"金属"的新颖性——专利申请1**不具备新颖性**；
- 但铜的公开并不使铜以外的其他具体金属制成的产品丧失新颖性——专利申请2**具备新颖性**。

**案例2：卤素与氯/氟**

| 对比项 | 内容 |
|--------|------|
| 对比文件公开 | 某产品使用了"卤素"或"氟" |
| 专利申请 | 用"氯"代替"卤素"或"氟" |

**分析：**
- "卤素"是"氯"的上位概念，"氟"是与"氯"并列的具体概念；
- 上位概念"卤素"的公开不影响下位概念"氯"的新颖性；
- 并列概念"氟"的公开也不影响"氯"的新颖性；
- 专利申请**具备新颖性**。

#### 1.3 规则总结

```
下位概念的公开 ──→ 破坏上位概念的新颖性（✗）
上位概念的公开 ──→ 不影响下位概念的新颖性（✓）
并列概念的公开 ──→ 不影响其他并列概念的新颖性（✓）
```

### 二、数值和数值范围（§3.2.4）

如果要求保护的发明或者实用新型中存在以数值或者连续变化的数值范围限定的技术特征（例如部件的尺寸、温度、压力以及组合物的组分含量），而其余技术特征与对比文件相同，则其新颖性的判断依照以下规定。

#### 2.1 四种判断情形

**情形（1）：对比文件的数值/范围落在要求保护的范围内 → 破坏新颖性**

对比文件公开的数值或者数值范围落在限定的技术特征的数值范围内，将破坏要求保护的发明或者实用新型的新颖性。

> **例1：** 权利要求为铜基形状记忆合金，包含10%～35%（重量）的锌和2%～8%（重量）的铝。对比文件公开了包含20%锌和5%铝的铜基形状记忆合金。20%落在10%～35%内，5%落在2%～8%内，破坏新颖性。

> **例2：** 权利要求为热处理台车窑炉，拱衬厚度100mm～400mm。对比文件公开了180mm～250mm。180～250落在100～400内，破坏新颖性。

**情形（2）：部分重叠或有共同端点 → 破坏新颖性**

对比文件公开的数值范围与限定的技术特征的数值范围部分重叠或者有一个共同的端点，将破坏新颖性。

> **例1：** 权利要求烧成时间1小时～10小时。对比文件4小时～12小时。4～10小时重叠，破坏新颖性。

> **例2：** 权利要求喷枪功率20kW～50kW。对比文件50kW～80kW。50kW为共同端点，破坏新颖性。

**情形（3）：对比文件范围的端点破坏离散值中相同端点的新颖性**

对比文件公开的数值范围的两个端点将破坏限定的技术特征为离散数值并且具有该两端点中任一个的发明或者实用新型的新颖性，但**不破坏**该两端点之间任一数值的新颖性。

> **例如：** 权利要求干燥温度为40℃、58℃、75℃或100℃。对比文件公开40℃～100℃。对比文件破坏干燥温度为40℃和100℃时的新颖性，但不破坏58℃和75℃时的新颖性。

**情形（4）：要求保护的数值/范围落在对比文件范围内且无共同端点 → 不破坏新颖性**

限定的技术特征的数值或者数值范围落在对比文件公开的数值范围内，并且与对比文件公开的数值范围没有共同的端点，则对比文件**不破坏**新颖性。

> **例1：** 权利要求活塞环圆环直径95mm。对比文件70mm～105mm。95落在70～105内但非端点，不破坏新颖性。

> **例2：** 权利要求聚合度100～200。对比文件50～400。100～200落在50～400内且无共同端点，不破坏新颖性。

> **【审查员视角】**
> 数值范围的新颖性判断是审查员出错率较高的领域之一。常见偏差包括：（1）混淆"对比文件范围落在权利要求范围内"与"权利要求范围落在对比文件范围内"两种情形，将后者（情形4）错误地认定为破坏新颖性；（2）在判断"共同端点"时，忽视数值的精确对应关系，例如将"约50"与"50"视为共同端点；（3）对化学领域中"摩尔比""重量百分比"等不同计量单位的范围进行简单数字比较，而不考虑单位换算。代理人的有效抗辩策略是：首先绘制数值关系示意图，直观展示两个范围的相对位置；其次严格援引指南§3.2.4的四种情形，逐一对照本案属于哪种情形；对于化学组合物，若对比文件公开的是"A组分10-30wt%"而权利要求限定的是"A组分10-30mol%"，应指出计量单位不同，不能直接比较数字。此外，审查员有时会以"本领域技术人员容易想到选择该子范围"为由否定新颖性，但这是创造性判断的思路，代理人应明确指出新颖性判断不涉及"是否容易想到"，只涉及范围是否重叠或是否有共同端点。

#### 2.2 规则总结图

```
对比文件数值（CF）vs 要求保护数值（CL）：

CF的数值/范围落在CL范围内 ───────→ 破坏新颖性 ✓
CF范围与CL范围部分重叠 ──────────→ 破坏新颖性 ✓
CF范围与CL范围有共同端点 ────────→ 破坏新颖性 ✓
CF范围的端点 = CL离散值中的某点 ─→ 该离散值丧失新颖性 ✓
CF范围的端点 ≠ CL离散值 ────────→ 不破坏新颖性 ✗
CL数值落在CF范围内且无共同端点 ──→ 不破坏新颖性 ✗
```

#### 2.3 相关规定

- 有关数值范围的修改适用本部分第八章第5.2节的规定；
- 有关通式表示的化合物的新颖性判断适用本部分第十章第5.1节的规定。

## 实务应对

### 1. 上位与下位概念的撰写策略

- **独立权利要求**中可适当使用上位概念扩大保护范围，但需注意下位概念的现有技术可能破坏其新颖性；
- **从属权利要求**中使用下位概念或具体实施方案进行限定，即使独立权利要求因下位概念公开而丧失新颖性，从属权利要求仍可能维持有效；
- 撰写时应在说明书中提供足够的下位概念实施例，为后续修改提供支持。

### 2. 数值范围的撰写策略

代理人在撰写涉及数值范围的权利要求时，应注意：

- **端点选择：** 避免使用与现有技术文献中常见的数值范围端点相同的数值，以防止"共同端点"导致新颖性丧失；
- **多点离散值：** 如果使用离散数值限定（如40℃、58℃、75℃、100℃），应注意端点值可能被对比文件的连续范围覆盖；
- **说明书支持：** 在说明书中记载多个具体的数值点或子范围，为后续修改提供基础。

### 3. 面对数值范围新颖性质疑的应对

- 核实审查员引用的对比文件中的数值范围是否确实与权利要求存在重叠或共同端点；
- 若属于情形（4）（落入对比文件范围内但无共同端点），可直接引用规则主张不破坏新颖性；
- 考虑将权利要求的数值范围缩小到对比文件范围之外的子范围，但需确保说明书记载能提供支持。

### 4. 化学领域的特殊考量

对于化学领域发明（如组合物配比、化合物参数等），代理人还需注意：
- 通式化合物的新颖性判断有专门规定（第十章第5.1节）；
- 组合物中多个组分含量同时以数值范围限定时，需分别判断各组分的重叠情况。

### 审查意见通知书常见表述 → 代理人答复策略对照表

| 审查意见常见表述 | 代理人答复策略 | 援引依据 |
|-----------------|--------------|---------|
| "对比文件公开'铜'，权利要求'金属'，不具备新颖性" | 确认下位概念确实破坏上位概念的新颖性；退守至从属权利要求的下位概念 | 第二部分第三章§3.2.2 |
| "对比文件公开了20%，落在权利要求10%-35%范围内" | 若确实落入，不破坏新颖性的争辩空间较小；考虑将范围限缩至对比文件数值之外 | 第二部分第三章§3.2.4(1) |
| "对比文件范围1-10与权利要求范围4-12部分重叠" | 若审查员认定正确，可争辩将权利要求修改至不重叠的范围 | 第二部分第三章§3.2.4(2) |
| "权利要求95mm落在对比文件70-105mm范围内，无共同端点" | 明确指出指南规定此种情形不破坏新颖性；要求审查员纠正认定 | 第二部分第三章§3.2.4(4) |
| "离散值58℃落在对比文件40-100℃范围内" | 指出端点仅破坏相同端点的离散值，不破坏中间值的新颖性 | 第二部分第三章§3.2.4(3) |

### 具体争辩话术示例

**话术一：援引"落入但无共同端点"规则**
> "审查意见认为权利要求1中的数值[具体数值/范围]落在对比文件X公开的[对比文件范围]内，因此破坏新颖性。然而，根据《专利审查指南》第二部分第三章第3.2.4节(4)的规定，'限定的技术特征的数值或者数值范围落在对比文件公开的数值范围内，并且与对比文件公开的数值范围没有共同的端点，则对比文件不破坏新颖性'。本案中，权利要求限定的[数值/范围]虽然落在对比文件[范围]内，但两者没有共同的端点（权利要求端点为[A、B]，对比文件端点为[C、D]），因此对比文件不破坏本申请的新颖性。"

**话术二：端点覆盖规则的精确适用**
> "审查意见认为对比文件X公开的[范围端点]破坏了权利要求中离散值[端点值]的新颖性。申请人同意该端点值确实因共同端点而丧失新颖性。但权利要求中的其他离散值[中间值1]、[中间值2]并不属于对比文件X的端点，且对比文件X未明确公开这些具体数值。根据《专利审查指南》第二部分第三章第3.2.4节(3)的规定，'对比文件公开的数值范围的两个端点将破坏限定的技术特征为离散数值并且具有该两端点中任一个的发明或者实用新型的新颖性，但不破坏该两端点之间任一数值的新颖性'。因此，[中间值1]和[中间值2]仍具备新颖性。"

### 数值范围新颖性答复模板

```
一、关于上位/下位概念的判断（如适用）

审查意见认为对比文件X公开的[下位概念]破坏了权利要求[上位概念]的新颖性。申请人[同意/不同意]该认定。
[如不同意：对比文件X中的[术语]在本领域并不具有确定的、公知的下位概念含义，本领域技术人员不能从中直接确定出[下位概念]……]
[如同意但需退守：申请人同意该认定，但权利要求X（从属权利要求）进一步限定为[更下位概念]，对比文件X并未公开该限定，因此权利要求X具备新颖性。]

二、关于数值范围重叠的判断（如适用）

（一）落入范围内但有共同端点
审查意见认为对比文件X的[数值/范围]落在权利要求限定的[范围]内。申请人同意/不同意该认定。[如不同意：对比文件X实际公开的是[具体数值]，并不在权利要求范围内……]

（二）落入范围内但无共同端点
审查意见认为对比文件X破坏了权利要求的新颖性。然而，权利要求限定的[范围A-B]虽然落在对比文件X的[C-D]范围内，但两者没有共同端点（A≠C且B≠D）。根据指南第二部分第三章第3.2.4节(4)，对比文件不破坏新颖性。

（三）部分重叠
权利要求范围为[A-B]，对比文件范围为[C-D]，两者在[E-F]区间重叠。申请人同意该重叠部分丧失新颖性，但主张将权利要求修改至[A-E]或[F-B]（需说明书记载支持）。

（四）离散值与端点
对比文件端点[值1]破坏了权利要求离散值[值1]的新颖性，但未破坏离散值[值2]、[值3]的新颖性（根据指南§3.2.4(3)）。

三、修改建议（如适用）

若部分技术方案确实丧失新颖性，申请人愿意将权利要求修改为[修改后的方案]，该修改在说明书中已有充分支持（参见说明书第[X]页）。
```
