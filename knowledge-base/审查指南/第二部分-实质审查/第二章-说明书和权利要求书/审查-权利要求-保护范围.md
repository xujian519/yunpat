# 审查-权利要求-保护范围

> **来源：** 《专利审查指南》第二部分第二章 §3.1 权利要求（含 §3.1.1 权利要求的类型 + §3.1.2 独立权利要求和从属权利要求）

> **最新修订：** 本节自2010版以来未作修改

## 核心要点

权利要求书是确定专利保护范围的法律文件。权利要求按性质分为产品权利要求和方法权利要求两种基本类型。按结构分为独立权利要求和从属权利要求，独立权利要求从整体上反映技术方案并记载必要技术特征，从属权利要求通过附加技术特征对引用的权利要求作进一步限定。在确定保护范围时，权利要求中的所有特征均应予以考虑，每一个特征的实际限定作用应最终体现在所要求保护的主题上。

---

## 审查标准

### 一、权利要求书概述

权利要求书应当以说明书为依据，清楚、简要地限定要求专利保护的范围。

权利要求书应当记载发明或者实用新型的技术特征，技术特征可以是构成发明或者实用新型技术方案的组成要素，也可以是要素之间的相互关系。专利法第二十六条第四款和专利法实施细则第二十二条至第二十五条对权利要求的内容及其撰写作了规定。

一份权利要求书中应当至少包括一项独立权利要求，还可以包括从属权利要求。

### 二、权利要求的类型（§3.1.1）

按照性质划分，权利要求有两种基本类型，即物的权利要求和活动的权利要求，或者简单地称为产品权利要求和方法权利要求。第一种基本类型的权利要求包括人类技术生产的物（产品、设备）；第二种基本类型的权利要求包括有时间过程要素的活动（方法、用途）。属于物的权利要求有物品、物质、材料、工具、装置、设备等权利要求；属于活动的权利要求有制造方法、使用方法、通讯方法、处理方法以及将产品用于特定用途的方法等权利要求。

**类型区分的目的与实际限定作用**

在类型上区分权利要求的目的是为了确定权利要求的保护范围。通常情况下，在确定权利要求的保护范围时，权利要求中的所有特征均应当予以考虑，而每一个特征的实际限定作用应当最终体现在该权利要求所要求保护的主题上。

**方法特征表征的产品权利要求：**

例如，当产品权利要求中的一个或多个技术特征无法用结构特征并且也不能用参数特征予以清楚地表征时，允许借助于方法特征表征。但是，方法特征表征的产品权利要求的保护主题仍然是产品，其实际的限定作用取决于对所要求保护的产品本身带来何种影响。

**用途限定的产品权利要求：**

对于主题名称中含有用途限定的产品权利要求，其中的用途限定在确定该产品权利要求的保护范围时应当予以考虑，但其实际的限定作用取决于对所要求保护的产品本身带来何种影响。

例如，主题名称为"用于钢水浇铸的模具"的权利要求，其中"用于钢水浇铸"的用途对主题"模具"具有限定作用；对于"一种用于冰块成型的塑料模盒"，因其熔点远低于"用于钢水浇铸的模具"的熔点，不可能用于钢水浇铸，故不在上述权利要求的保护范围内。

然而，如果"用于……"的限定对所要求保护的产品或设备本身没有带来影响，只是对产品或设备的用途或使用方式的描述，则其对产品或设备例如是否具有新颖性、创造性的判断不起作用。例如，"用于……的化合物X"，如果其中"用于……"对化合物X本身没有带来任何影响，则在判断该化合物X是否具有新颖性、创造性时，其中的用途限定不起作用。

> **【审查员视角】**
> 审查员在判断用途限定和方法特征对产品权利要求的实际限定作用时，标准不一且主观性较强。部分审查员倾向于认为用途限定"通常不起作用"，在缺乏深入分析的情况下就直接否定其限定效果；而另一些审查员则可能过度解读用途限定，将其等同于结构限定。在方法特征表征的产品权利要求中，审查员有时会混淆"方法特征是否影响产品本身"与"方法步骤是否清楚"两个问题——前者关乎保护范围的实质限定作用，后者属于清楚性判断。代理人的有效抗辩策略是：当审查员否定用途限定的限定作用时，应详细论证该用途如何要求产品具有特定的结构、组成或性能参数（如"用于钢水浇铸的模具"必然要求模具具有耐高温特性和特定结构强度），从而证明用途限定对保护主题产生了实际影响；当审查员认定方法特征不具限定作用时，应说明该方法特征如何必然导致产品在微观结构、性能参数或组成上与对比文件产品不同。从审查员角度看，用途限定和方法特征的审查难点在于需要跨层推理（从用途/方法反推产品结构），审查员为避免复杂分析，往往倾向于简单否定，代理人应主动提供完整的技术逻辑链，降低审查员的分析成本，提高争辩成功率。

### 三、独立权利要求和从属权利要求（§3.1.2）

**独立权利要求**

独立权利要求应当从整体上反映发明或者实用新型的技术方案，记载解决技术问题的必要技术特征。

必要技术特征是指，发明或者实用新型为解决其技术问题所不可缺少的技术特征，其总和足以构成发明或者实用新型的技术方案，使之区别于背景技术中所述的其他技术方案。

判断某一技术特征是否为必要技术特征，应当从所要解决的技术问题出发并考虑说明书描述的整体内容，不应简单地将实施例中的技术特征直接认定为必要技术特征。

在一件专利申请的权利要求书中，独立权利要求所限定的一项发明或者实用新型的保护范围最宽。

**从属权利要求**

如果一项权利要求包含了另一项同类型权利要求中的所有技术特征，且对该另一项权利要求的技术方案作了进一步的限定，则该权利要求为从属权利要求。由于从属权利要求用附加的技术特征对所引用的权利要求作了进一步的限定，所以其保护范围落在其所引用的权利要求的保护范围之内。

从属权利要求中的附加技术特征，可以是对所引用的权利要求的技术特征作进一步限定的技术特征，也可以是增加的技术特征。

**并列独立权利要求**

一件专利申请的权利要求书中，应当至少有一项独立权利要求。当有两项或者两项以上独立权利要求时，写在最前面的独立权利要求被称为第一独立权利要求，其他独立权利要求称为并列独立权利要求。

审查员应当注意，有时并列独立权利要求也引用在前的独立权利要求，例如，"一种实施权利要求1的方法的装置，……"；"一种制造权利要求1的产品的 方法，……"；"一种包含权利要求1的部件的设备，……"；"与权利要求1的插座相配合的插头，……"等。这种引用其他独立权利要求的权利要求是并列的独立权利要求，而不能被看作是从属权利要求。对于这种引用另一权利要求的独立权利要求，在确定其保护范围时，被引用的权利要求的特征均应予以考虑，而其实际的限定作用应当最终体现在对该独立权利要求的保护主题产生了何种影响。

**形式上的从属权利要求**

在某些情况下，形式上的从属权利要求（即其包含有从属权利要求的引用部分），实质上不一定是从属权利要求。例如，独立权利要求1为："包括特征X的机床"。在后的另一项权利要求为："根据权利要求1所述的机床，其特征在于用特征Y代替特征X"。在这种情况下，后一权利要求也是独立权利要求。审查员不得仅从撰写的形式上判定在后的权利要求为从属权利要求。

## 实务应对

### 代理人撰写要点

**1. 权利要求类型的准确选择**

权利要求的类型直接影响保护范围。产品权利要求保护产品本身（无论谁制造、如何制造），方法权利要求保护特定方法步骤。实务中，如果发明既涉及产品又涉及其制造方法，建议同时撰写产品权利要求和方法权利要求，以获得更全面的保护。

**2. 用途限定的策略运用**

用途限定对保护范围的影响是"双刃剑"：
- 当用途限定对产品本身的结构、组成等带来了影响时（如"用于钢水浇铸的模具"限定了模具的耐高温特性），用途限定起到实际限定作用。
- 当用途限定仅仅描述使用方式而对产品本身无影响时（如"用于……的化合物X"），该用途限定在新颖性/创造性判断中不起作用。

代理人应根据具体技术内容决定是否在权利要求中加入用途限定。若要获得更宽的保护，应避免不必要的用途限定。

**3. 方法特征在产品权利要求中的使用**

方法特征表征的产品权利要求的保护主题仍然是产品本身。方法特征的实际限定作用取决于其对产品本身带来了何种影响。实务中，尽量用结构特征或参数特征来表征产品，仅在无法用上述特征清楚表征时才使用方法特征。

**4. 并列独立权利要求的撰写**

并列独立权利要求虽然引用了其他独立权利要求，但实质上仍是独立权利要求，不因引用关系而变为从属权利要求。常见的并列独立权利要求模式包括：
- "一种实施权利要求1的方法的装置"
- "一种制造权利要求1的产品的方法"
- "一种包含权利要求1的部件的设备"

撰写时应注意，这种引用方式虽然简化了表述，但在确定保护范围时仍需考虑被引用权利要求的全部特征。

**5. 必要技术特征的判断**

判断必要技术特征时，应从技术问题出发，考虑说明书的整体内容，而非简单照搬实施例。实务中常见错误是将实施例中的优选特征或附加特征写入独立权利要求，导致保护范围过窄。应区分"必要技术特征"与"优选技术特征"，前者写入独立权利要求，后者写入从属权利要求。

**6. 形式从属与实质独立的识别**

当权利要求的形式表现为从属（引用在前的权利要求），但实质上是用不同特征替换了被引用权利要求的核心特征时，该权利要求实质上是独立的。撰写时应注意这种情况，避免因形式上的引用关系导致审查中的认定错误。

### 具体争辩话术示例

**话术一：论证用途限定的实际限定作用**
> "审查意见认为权利要求中'用于……'的用途限定对所要求保护的产品没有带来影响，因此在新颖性/创造性判断中不起作用。然而，根据《专利审查指南》第二部分第二章第3.1.1节的规定，用途限定的'实际的限定作用取决于对所要求保护的产品本身带来何种影响'。本案中，用途'[用途]'要求产品必须具有[特定结构/组成/性能]，例如[举例说明]。若产品不具备该[结构/组成/性能]，则根本无法实现该用途。因此，该用途限定对产品权利要求的保护范围产生了实际的限定作用，在判断新颖性和创造性时应当予以考虑。"

**话术二：方法特征对产品本身的限定作用**
> "审查意见认为权利要求中的方法特征不具限定作用。然而，根据《专利审查指南》第二部分第二章第3.1.1节，方法特征表征的产品权利要求的'实际的限定作用取决于对所要求保护的产品本身带来何种影响'。本申请的方法[方法X]包含了[特定步骤/条件]，该方法必然导致产品在[微观结构/组成/性能]上产生[具体变化]。说明书中提供了对比实验数据（参见第[X]页），证明用方法X制得的产品与用其他方法制得的产品在[结构/性能]上存在显著差异。因此，该方法特征对产品权利要求具有实际的限定作用。"

### 权利要求保护范围争辩模板

```
一、关于用途限定限定作用的论证（如适用）

审查意见认为权利要求X中'[用途限定]'对[产品]本身没有带来影响。申请人认为该认定不当：

（一）用途要求特定结构
用途'[用途]'要求[产品]必须具有[特定结构/组成/性能参数]。具体而言：[详细说明该用途对产品结构的要求]。若产品不具备上述[结构/组成/性能]，则无法实现该用途。

（二）对比文件产品不在保护范围内
对比文件中的[产品]由于[原因]不具备[特定结构/组成/性能]，因此不能用于[该用途]，不在权利要求X的保护范围内。

二、关于方法特征限定作用的论证（如适用）

审查意见认为方法特征不具限定作用。申请人认为：

（一）方法必然导致产品结构变化
方法[方法X]中的[特定步骤/条件]使得产品在[微观结构/组成]上必然产生[具体变化]。说明书中已充分说明了该方法与产品结构之间的因果关系（参见第[X]页）。

（二）对比实验数据证明差异
申请人提供了对比实验数据，证明用方法X制得的产品与对比文件方法制得的产品在[性能/结构]上存在[量化差异]。

三、结论

综上所述，权利要求X中的[用途限定/方法特征]对产品保护范围具有实际的限定作用，审查意见的认定不能成立。
```

## 审查要点速查

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 权利要求书中无独立权利要求 | 不符合规定 | 细则23.1 |
| 产品权利要求中方法特征对产品无实际影响 | 方法特征不具限定作用 | §3.1.1 |
| "用于X的化合物"中用途对化合物无影响 | 用途限定在新颖性/创造性判断中不起作用 | §3.1.1 |
| 并列独立权利要求引用在前的独立权利要求 | 仍为独立权利要求，非从属 | §3.1.2 |
| 形式从属但实质替换了核心特征 | 实质为独立权利要求 | §3.1.2 |
| 实施例中的特征被写入独立权利要求 | 可能导致保护范围过窄 | §3.1.2 |
| 从属权利要求的保护范围宽于独立权利要求 | 不符合从属权利要求定义 | 细则23.3 |
| 产品权利要求中同时包含产品和方法 | 类型不清楚，要求修改 | §3.2.2 |
