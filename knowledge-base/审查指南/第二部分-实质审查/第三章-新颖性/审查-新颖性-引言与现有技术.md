# 审查-新颖性-引言与现有技术

> **来源：** 《专利审查指南》第二部分第三章 §1 引言 + §2.1 现有技术
> **对应法条：** 《专利法》第22条第1款、第2款、第5款
> **最新修订：** 2017年修订（互联网/在线数据库公开规定完善）
> **关联页面：** [[新颖性-法条-现有技术的定义]]、[[新颖性-法条-新颖性判断]]、[[审查-新颖性-抵触申请]]、[[审查-新颖性-对比原则]]

## 核心要点

新颖性是授予专利权的必要条件之一，其判断核心在于确定发明或实用新型是否属于**现有技术**，以及是否存在**抵触申请**。现有技术的时间界限为申请日（有优先权的指优先权日）以前，且以**公众能够得知**为标准，包括出版物公开、使用公开和以其他方式公开三种方式，均无地域限制。

---

## 审查标准

### 一、新颖性的概念（§2）

根据《专利法》第22条第2款，新颖性是指：

1. 该发明或者实用新型**不属于现有技术**；
2. 也没有任何单位或者个人就**同样的发明或者实用新型**在申请日以前向专利局提出过申请，并记载在申请日以后（含申请日）公布的专利申请文件或者公告的专利文件中。

上述第2项即"抵触申请"的概念，将在 [[审查-新颖性-抵触申请]] 中详细讨论。

### 二、现有技术的定义（§2.1）

根据《专利法》第22条第5款，现有技术是指**申请日以前在国内外为公众所知的技术**。

现有技术包括在申请日（有优先权的，指优先权日）以前在国内外：
- 出版物上公开发表的技术；
- 公开使用的技术；
- 以其他方式为公众所知的技术。

**核心标准：** 现有技术应当是在申请日以前**公众能够得知**的技术内容。换言之，现有技术应当在申请日以前处于能够为公众获得的状态，并包含有能够使公众从中得知实质性技术知识的内容。

**保密状态的排除：** 处于保密状态的技术内容不属于现有技术。保密状态包括：
- 受保密规定或者协议约束的情形；
- 社会观念或者商业习惯上被认为应当承担保密义务的情形（即**默契保密**）。

**泄密的处理：** 如果负有保密义务的人违反规定、协议或者默契泄露秘密，导致技术内容公开，使公众能够得知这些技术，这些技术即构成现有技术的一部分。

### 三、时间界限（§2.1.1）

现有技术的时间界限是**申请日**，享有优先权的，则指**优先权日**。

- 申请日以前公开的技术内容属于现有技术；
- **申请日当天公开的技术内容不包括在现有技术范围内**。

### 四、公开方式（§2.1.2）

现有技术公开方式包括**出版物公开**、**使用公开**和**以其他方式公开**三种，均无地域限制。

#### 4.1 出版物公开（§2.1.2.1）

专利法意义上的出版物是指记载有技术或者设计内容的独立存在的传播载体，并且应当表明或者有其他证据证明其公开发表或者出版的时间。

出版物可以是纸质出版物、视听资料，也可以是存在于互联网或者其他在线数据库中的资料等。

**（1）纸质出版物和视听资料**

纸质出版物包括：专利文献、科技杂志、科技书籍、学术论文、专业文献、教科书、技术手册、正式公布的会议记录或者技术报告、报纸、产品样本、产品目录、广告宣传册等。

视听资料包括：缩微胶片、影片、照相底片、录像带、磁带、唱片、光盘等。

纸质出版物和视听资料不受地理位置、语言或者获得方式的限制，也不受年代的限制。其是否能够获得与出版发行量多少、是否有人阅读过、申请人是否知道无关。

**例外：** 印有"内部资料""内部发行"等字样的纸质出版物和视听资料，确系在特定范围内发行并要求保密的，**不属于公开出版物**。

**公开日确定：** 纸质出版物的印刷日和视听资料的出版日视为公开日，有其他证据证明其公开日的除外。

**（2）存在于互联网或者其他在线数据库中的资料**

这是指以数据形式存储、以网络为传播途径的文字、图片、音视频等资料。

- 应当是通过合法途径能够获得的；
- 资料的获得与是否需要口令或者付费、是否有人阅读过无关。

**公开日确定规则：**
- 一般以发布日为准，有其他证据证明其公开日的除外；
- 以网络方式出版的书籍、期刊、学位论文等，公开日为网页上记载的网络发布日；
- 若同时具有内容相同的纸质出版物，也可以根据纸质出版物的印刷日确定，通常以能够确定的最早的公开日为准；
- 对于网页上未明确发布日或者发布日存疑的资料，可参考：日志文件中记载的发布日期和修改日期、搜索引擎给出的索引日期、互联网档案馆服务显示的日期、时间戳信息或者镜像网站上显示的复制信息的发布日期等信息。

**日期仅有年月的处理：** 印刷日、出版日或者发布日只写明年月或者年份的，以所写月份的最后一日或者所写年份的12月31日为公开日。

审查员认为出版物的公开日期存在疑义的，可以要求该出版物的提交人提出证明。

#### 4.2 使用公开（§2.1.2.2）

由于使用而导致技术方案的公开，或者导致技术方案处于公众可以得知的状态，即构成使用公开。

**使用公开的方式包括：** 制造、使用、销售、进口、交换、馈赠、演示、展出、招标投标等。只要通过上述方式使有关技术内容处于公众想得知就能够得知的状态，就构成使用公开，而不取决于是否有公众得知。

**例外：** 未给出任何有关技术内容的说明，以致所属技术领域的技术人员无法得知其结构和功能或者材料成分的产品展示，不属于使用公开。

**产品使用公开的特殊规则：** 如果使用公开的是一种产品，即使所使用的产品或者装置需要经过破坏才能够得知其结构和功能，也仍然属于使用公开。此外，放置在展台上、橱窗内公众可以阅读的信息资料及直观资料（如招贴画、图纸、照片、样本、样品等）也属于使用公开。

**公开日：** 使用公开是以公众能够得知该产品或者方法之日为公开日。

> **【审查员视角】**
> 在新颖性审查中，审查员对"使用公开"和"网络公开"的认定往往存在扩大化倾向。常见误区包括：（1）将展会上的外观展示或仅能看到外部结构的产品展示直接认定为使用公开，而忽略了"是否给出技术内容说明"的例外条件；（2）对网络资料的公开日认定过于宽松，仅凭网页上的发布日期就认定公开日，而不考虑该日期是否可被独立验证；（3）将"内部资料"形式发行的出版物一概认定为公开出版物，而不审查其是否确系在特定范围内发行并要求保密。代理人的有效抗辩策略是：针对使用公开，重点质疑公众是否能够"得知技术内容"而非仅仅"看到产品"；针对网络公开，要求审查员提供互联网档案馆（Wayback Machine）截图、搜索引擎索引日期等辅助证据，不能仅凭网页自载日期认定；针对出版物公开，若印有"内部资料"字样，应要求审查员证明其未受保密约束。此外，审查员有时会混淆"现有技术"与"抵触申请"的检索范围，代理人应注意审查员是否将申请日后公布的文献错误地作为现有技术引用。

#### 4.3 以其他方式公开（§2.1.2.3）

为公众所知的其他方式，主要是指口头公开等。包括：口头交谈、报告、讨论会发言、广播、电视、电影等能够使公众得知技术内容的方式。

**公开日确定：**
- 口头交谈、报告、讨论会发言以其发生之日为公开日；
- 公众可接收的广播、电视或者电影的报道，以其播放日为公开日。

## 实务应对

### 1. 申请前的保密管理

代理人应提醒申请人在提交专利申请前严格保密，避免因产品展示、学术论文发表、口头报告等行为构成使用公开或口头公开。特别是：
- **展会参展**需注意是否属于中国政府主办或承认的国际展览会（可主张宽限期）；
- **学术会议发表**需确认是否属于"规定的学术会议或技术会议"；
- **互联网披露**应特别注意，互联网公开一旦发生，几乎所有内容都可被检索到，且公开日确定规则已趋完善。

### 2. 现有技术抗辩的利用

在无效宣告程序中，应充分利用三种公开方式寻找现有技术证据：
- **出版物公开**：注意检索外文专利文献、学术论文、网络公开资料；
- **使用公开**：可通过销售发票、展会照片、公证书等证明公开使用；
- **其他方式公开**：如电视报道、广播节目等。

### 3. 公开日举证

涉及公开日争议时，代理人应注意：
- 网络资料的公开日可通过互联网档案馆（Wayback Machine）、日志文件等辅助证明；
- 印有"内部资料"的出版物，需审查其是否确系在特定范围内发行并要求保密；
- 仅标明年月的出版物，公开日默认为月末最后一天。

### 4. 优先权的有效利用

在外国首次申请后12个月内在中国提出申请，可享有优先权，将现有技术的时间界限提前至优先权日。代理人应及时提醒客户利用优先权制度。

### 审查意见通知书常见表述 → 代理人答复策略对照表

| 审查意见常见表述 | 代理人答复策略 | 援引依据 |
|-----------------|--------------|---------|
| "某网络资料公开日早于申请日，属于现有技术" | 要求审查员提供该网络资料的公开日证据（如互联网档案馆截图、搜索引擎索引日期）；质疑网页自载日期的可靠性 | 第二部分第三章§2.1.2.1 |
| "产品已在展会上展出，构成使用公开" | 核实展会展示是否给出了技术内容说明；若仅外观展示，不构成使用公开 | 第二部分第三章§2.1.2.2 |
| "内部资料属于公开出版物" | 要求审查员证明该资料未在特定范围内发行且未要求保密 | 第二部分第三章§2.1.2.1 |
| "申请日前6个月内的公开已丧失新颖性" | 核实是否属于四种法定宽限期情形之一；及时提出声明并提交证明文件 | 专利法第24条 |

### 具体争辩话术示例

**话术一：质疑网络公开日**
> "审查意见引用了[网站/网页]作为现有技术，并认定其公开日为[日期]。然而，该日期仅为网页上自行标注的发布日期，缺乏独立的第三方证据佐证。根据《专利审查指南》第二部分第三章第2.1.2.1节的规定，网络资料的公开日'一般以发布日为准，有其他证据证明其公开日的除外'，且'对于网页上未明确发布日或者发布日存疑的资料，可参考：日志文件中记载的发布日期和修改日期、搜索引擎给出的索引日期、互联网档案馆服务显示的日期、时间戳信息或者镜像网站上显示的复制信息的发布日期等信息'。审查意见未提供上述任何辅助证据，仅凭网页自载日期认定公开日，其证据不足以支持该认定。"

**话术二：反驳使用公开扩大化**
> "审查意见认为[产品]在[展会/销售/展示]中的公开构成了使用公开。然而，根据《专利审查指南》第二部分第三章第2.1.2.2节的规定，'未给出任何有关技术内容的说明，以致所属技术领域的技术人员无法得知其结构和功能或者材料成分的产品展示，不属于使用公开'。本案中，[展会/展示]仅涉及产品的[外观/外形/非技术信息]，未给出任何关于[具体技术内容]的说明，所属技术人员无法从该展示中得知产品的[结构/功能/材料成分]。因此，该展示不构成使用公开。"

### 现有技术与宽限期答复模板

```
一、关于[具体公开方式]的质疑

审查意见认为[某证据]构成现有技术，破坏了本申请的新颖性。申请人认为该认定不能成立：

（一）公开日质疑
[针对出版物公开：该证据的公开日存疑，审查意见未提供充分证据证明其公开日早于本申请申请日（优先权日）。]
[针对网络公开：网页自载日期不可靠，审查意见未提供互联网档案馆截图、搜索引擎索引日期等辅助证据。]
[针对使用公开：该使用行为未使技术内容处于公众想得知就能够得知的状态。]

（二）不属于现有技术
[针对保密状态：该技术内容在申请日前处于保密状态，不属于现有技术。]
[针对宽限期：该公开属于专利法第24条规定的[具体情形]，发生在申请日前6个月内，不丧失新颖性。申请人已在指定期限内提出声明并提交证明文件。]

二、结论

综上所述，[某证据]不构成破坏本申请新颖性的现有技术。
```

## 审查要点速查

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 申请日前在国外期刊发表的技术 | 属于现有技术，破坏新颖性 | §2.1 现有技术无地域限制 |
| 申请日当天公开的技术 | 不属于现有技术 | §2.1.1 申请日当天公开不纳入 |
| 印有"内部资料"且特定范围保密 | 不属于公开出版物 | §2.1.2.1 内部发行+保密要求 |
| 互联网公开资料（需付费获取） | 属于公开出版物 | §2.1.2.1 是否付费不影响公开性 |
| 展会上仅外观展示无技术说明 | 不属于使用公开 | §2.1.2.2 无法得知结构功能的不构成公开 |
| 产品需破坏才能获知内部结构 | 仍属于使用公开 | §2.1.2.2 使用公开的特殊规则 |
| 违反保密义务导致的技术泄露 | 构成现有技术 | §2.1 泄密使技术进入公有领域 |
| 保密协议约束下的技术交流 | 不属于现有技术 | §2.1 受保密协议约束处于保密状态 |
