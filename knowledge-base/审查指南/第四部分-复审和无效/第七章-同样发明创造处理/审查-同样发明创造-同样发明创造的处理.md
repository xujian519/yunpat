# 审查-同样发明创造-同样发明创造的处理

> **来源：** 《专利审查指南》第四部分第七章 §1-3
> **对应法条：** 《专利法》第9条 / 《专利法实施细则》第47条、第69条
> **最新修订：** 无重大修改
> **关联页面：** [[审查-外观-外观设计权利冲突与优先权]]、[[审查-无效-无效请求的形式审查]]、[[程序-无效宣告请求的审查]]

---

## 核心要点

被授予专利权的发明创造不符合专利法第九条（同样的发明创造只能授予一项专利权）的，属于无效宣告理由。对于发明和实用新型，"同样"是指要求保护的发明或者实用新型相同；对于外观设计，"同样"是指要求保护的产品外观设计相同或者实质相同。处理规则因专利权人是否相同、授权公告日是否相同而有所区别。

---

## 审查标准

### 一、"同样的发明创造"的界定

**发明和实用新型**：要求保护的发明或者实用新型相同，判断原则适用本指南第二部分第三章第6.1节的规定。

**外观设计**：要求保护的产品外观设计相同或者实质相同，判断适用本部分第五章的规定。

**转换适用**：任何单位或者个人以某项专利权与申请在先的另一项专利权构成同样的发明创造为由请求宣告无效的，如果申请在先的专利已构成现有技术（或现有设计）或者属于申请在先公开在后的专利，合议组可以依据专利法第二十二条（或第二十三条）的规定进行审查，而非专利法第九条。

#### 1.1 发明和实用新型"相同"的判断标准

判断两件发明或实用新型是否属于同样的发明创造，应当对比两者的权利要求书。如果两件专利申请或专利的权利要求书保护范围相同，则属于同样的发明创造。

**判断要点**：
- 比较对象是权利要求书，而非说明书；
- 比较的是保护范围，而非文字表述；
- 若一项专利的权利要求保护范围落入另一项专利的权利要求保护范围之内，且两者保护范围实质上相同，则可认定为同样的发明创造；
- 同样的发明创造判断与新颖性判断类似，但关注的是"同样的"而非"新颖的"。

#### 1.2 转换适用的情形

转换适用是同样发明创造处理中的重要规则：

| 情形 | 法律依据 | 说明 |
|------|---------|------|
| 申请在先的专利已构成现有技术 | 专利法第22条（新颖性/创造性） | 以新颖性等理由审查，通常更为简便 |
| 申请在先的专利属于申请在先公开在后 | 专利法第22条（抵触申请） | 同样以第22条审查 |
| 申请在先的专利尚未公开且仍有效 | 专利法第9条 | 依本章规定处理 |

转换适用的实务意义：如果申请在先的专利已经公开，其本身即可构成破坏在后专利新颖性的现有技术或抵触申请，此时以专利法第22条提出无效请求可能更为直接和简便。

### 二、专利权人相同的处理

#### 2.1 授权公告日不同（§2.1）

**请求宣告授权在前的专利权无效**：在不存在其他无效宣告理由或者其他理由不成立的情况下，合议组应当维持该项专利权有效。

**请求宣告授权在后的专利权无效**：合议组经审查后认为构成同样的发明创造的，应当宣告该项专利权无效。

**同日申请的实用新型和发明专利**：如果上述两项专利权为同一专利权人同日（仅指申请日）申请的一项实用新型专利权和一项发明专利权，专利权人在申请时根据专利法实施细则第四十七条第二款的规定作出过说明，且发明专利权授予时实用新型专利权尚未终止，在此情形下，专利权人可以通过放弃授权在前的实用新型专利权以保留被请求宣告无效的发明专利权。

#### 2.2 授权公告日相同（§2.2）

任何单位或者个人认为属于同一专利权人的具有相同申请日和相同授权公告日的两项专利权不符合专利法第九条第一款规定的，可以请求宣告其中一项专利权无效。

**仅针对一项专利权提出请求**：合议组经审查后认为构成同样的发明创造的，应当宣告被请求宣告无效的专利权无效。

**两项专利权均被提出请求**：一般应合并审理。经审查认为构成同样的发明创造的，合议组应当告知专利权人上述两项专利权构成同样的发明创造，并要求其选择仅保留其中一项专利权。

- 专利权人选择仅保留其中一项专利权的，在不存在其他无效宣告理由或者其他理由不成立的情况下，合议组应当维持该项专利权有效，宣告另一项专利权无效。
- **专利权人未进行选择的，合议组应当宣告两项专利权无效。**

### 三、专利权人不同的处理（§3）

任何单位或者个人认为属于不同专利权人的两项具有相同申请日的专利权不符合专利法第九条第一款规定的，可以分别请求宣告这两项专利权无效。

**两项专利权均被提出请求**：一般应合并审理。经审查认为构成同样的发明创造的，合议组应当告知两专利权人上述两项专利权构成同样的发明创造，并要求其协商选择仅保留其中一项专利权。

- 两专利权人经协商共同书面声明仅保留其中一项专利权的，在不存在其他无效宣告理由或者其他理由不成立的情况下，合议组应当维持该项专利权有效，宣告另一项专利权无效。
- **两专利权人协商不成未进行选择的，合议组应当宣告两项专利权无效。**

**仅针对一项专利权提出请求**：合议组经审查认为构成同样的发明创造的，应当告知双方当事人。专利权人可以请求宣告另外一项专利权无效，并与另一专利权人协商选择仅保留其中一项专利权。

- 专利权人请求宣告另外一项专利权无效的，按照前述规定处理。
- **专利权人未请求宣告另一项专利权无效的，合议组应当宣告被请求宣告无效的专利权无效。**

### 四、重复授权的判定标准详解

#### 4.1 重复授权的核心要件

构成专利法第九条所述的重复授权，需同时满足以下要件：

| 要件 | 内容 |
|------|------|
| 主体要件 | 涉及两项以上专利权 |
| 时间要件 | 具有相同的申请日（有优先权的，指优先权日） |
| 内容要件 | 属于同样的发明创造 |
| 状态要件 | 两项专利权均处于有效状态（或至少被请求宣告的专利处于有效状态） |

#### 4.2 "相同申请日"的理解

- "相同申请日"仅指日历日期相同，与申请时刻无关；
- 有优先权的，以优先权日为准判断是否相同；
- 同日申请但分案申请与原申请的申请日相同，不构成重复授权（因属同一申请）；
- 进入国家阶段的PCT申请，其申请日指国际申请日。

#### 4.3 重复授权与抵触申请的区别

| 比较维度 | 重复授权（法第9条） | 抵触申请（法第22条第2款） |
|---------|------------------|------------------------|
| 适用对象 | 两项专利权 | 专利申请与现有技术/抵触申请 |
| 时间关系 | 相同申请日 | 申请在先、公开在后 |
| 申请人关系 | 可以相同或不同 | 任何单位或个人 |
| 法律后果 | 只能保留一项 | 破坏新颖性 |
| 处理方式 | 选择保留或宣告无效 | 以新颖性为由驳回或无效 |

### 五、分案申请与重复授权的关系

#### 5.1 分案申请不构成重复授权

分案申请是从原申请中分出的申请，其申请日与原申请相同，但分案申请与原申请属于同一专利申请的分支，不构成专利法第九条意义上的"两项专利权"。

#### 5.2 分案申请之间的重复授权

若同一申请人针对同一原申请提出多项分案申请，且各分案申请授权后保护范围相同，则可能构成重复授权。此时应依据专利法第九条处理。

#### 5.3 分案申请与他人专利的重复授权

若分案申请的申请日与他人的专利申请日相同，且两者属于同样的发明创造，则可能构成重复授权。处理方式依据专利权人是否相同而定。

### 六、同日申请的协调处理

#### 6.1 同一申请人的同日申请

同一申请人在同日对同样的发明创造既申请实用新型专利又申请发明专利的，应当在申请时分别说明对同样的发明创造已申请了另一专利（专利法实施细则第47条第2款）。

**协调处理流程**：

```
同日申请发明+实用新型
    ↓
实用新型先授权（因初审周期短）
    ↓
发明专利经实质审查后符合授权条件
    ↓
申请人声明放弃实用新型专利权
    ↓
发明专利授权公告
    ↓
实用新型专利权自发明专利授权公告之日起终止
```

**注意事项**：
- 申请人必须在申请时作出说明，未说明的不能适用该协调机制；
- 发明专利授权时，实用新型专利权必须仍然有效（未终止、未放弃、未被无效）；
- 该协调机制仅适用于同一申请人的同日申请，不适用于不同申请人；
- 在无效程序中，若他人请求宣告发明专利无效，专利权人可主动提出放弃实用新型以保留发明。

#### 6.2 不同申请人的同日申请

不同申请人同日申请同样的发明创造，专利权授予最先申请的人。若两项均授权，则构成重复授权，应依本章§3的规定处理。

### 七、审查示例

**【示例1】同一专利权人、授权公告日不同**

专利权人甲于2020年1月1日申请专利A（发明），于2020年1月1日申请专利B（实用新型），专利A于2022年授权，专利B于2020年授权。两件专利的权利要求保护范围相同。

请求人请求宣告专利B（授权在后）无效。

处理结果：合议组审查后认为构成同样的发明创造，宣告专利B无效。

**【示例2】同一专利权人、授权公告日相同**

专利权人甲的两项专利权C和D具有相同的申请日和相同的授权公告日，权利要求保护范围相同。

请求人同时请求宣告C和D无效。

处理结果：合并审理，合议组告知甲选择保留一项。甲选择保留C，则维持C有效，宣告D无效。若甲未选择，则宣告C和D均无效。

**【示例3】不同专利权人**

专利权人甲的专利E和专利权人乙的专利F具有相同的申请日，权利要求保护范围相同。

请求人同时请求宣告E和F无效。

处理结果：合并审理，合议组告知甲和乙协商选择仅保留一项。甲和乙协商后书面声明保留E，则维持E有效，宣告F无效。若协商不成，则宣告E和F均无效。

**【示例4】仅请求宣告一项**

专利权人甲的专利G和专利权人乙的专利H具有相同申请日，权利要求保护范围相同。

请求人仅请求宣告G无效。

处理结果：合议组告知甲和乙。甲可以请求宣告H无效并与乙协商。若甲请求宣告H无效且双方协商保留G，则维持G有效，宣告H无效。若甲未请求宣告H无效，则合议组宣告G无效。

## 实务应对

**代理人视角：**

1. **同日申请的实用新型和发明专利**：对于同一专利权人同日申请的实用新型和发明专利，在无效宣告程序中，专利权人可以通过放弃实用新型来保留发明专利。代理人应在无效程序中主动提出这一选择，以最大程度保护客户利益。但需注意，该选择仅在申请时作出过说明且发明授权时实用新型仍有效的前提下才适用。

2. **未选择的严重后果**：无论专利权人是否相同，当两项专利构成同样发明创造时，如果权利人不进行选择（或协商不成），合议组将宣告两项专利权均无效。这是一个非常严重的后果，代理人务必提醒客户及时作出选择。特别是对于同一专利权人有两项授权公告日相同的专利时，若被同时请求无效，必须在指定期限内作出选择。

3. **请求对象的选择**：作为无效请求人，应仔细分析目标专利是否存在同日申请的相关专利。如果存在，应考虑同时请求宣告两项专利无效，否则仅宣告一项时另一项仍然有效。尤其针对同一专利权人授权公告日相同的两项专利，若仅请求宣告其中一项，该专利权人可能选择保留被请求宣告的一项（在授权公告日相同的情况下，选择保留被请求宣告的一项即可），导致无效请求的目的落空。

4. **转换适用专利法第二十二条/第二十三条**：如果申请在先的专利已构成现有技术，应以新颖性等理由而非专利法第九条提出无效请求，这可能更为简便。特别是当在先专利已经公开时，以新颖性为由只需证明公开时间和内容即可，无需考虑复杂的重复授权处理规则。

5. **合并审理的利用**：当两项专利均被提出无效宣告请求时，一般应合并审理。合并审理有助于提高效率，且可以集中解决两专利之间的关系问题。代理人应积极参与合并审理程序，充分陈述意见。

6. **分案申请的防范**：申请人在提交专利申请时，应避免对同样的发明创造提交多个独立的申请（包括分案申请），以免授权后构成重复授权。若确实需要分案，应确保分案申请的权利要求保护范围与原申请有实质性区别。

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 同一专利权人、授权在前、请求宣告在前的专利无效 | 维持在前的专利权有效 | §2.1 |
| 同一专利权人、授权在后、请求宣告在后的专利无效 | 宣告在后的专利权无效 | §2.1 |
| 同日申请实用新型和发明，放弃实用新型保留发明 | 允许（需在申请时作过说明） | §2.1 |
| 同一专利权人、授权公告日相同、未选择保留哪项 | 宣告两项均无效 | §2.2 |
| 不同专利权人、协商不成 | 宣告两项均无效 | §3 |
| 仅请求宣告一项、专利权人未请求宣告另一项 | 宣告被请求宣告的一项无效 | §3 |

| 同一申请人同日申请未作说明 | 不适用放弃实用新型保留发明的机制 | 细则第47条 |
| 发明专利授权时实用新型已终止 | 不适用放弃实用新型保留发明的机制 | §2.1 |
