# 专利权能概述

> **来源：** 崔国斌《专利法：原理与案例（第二版）》第9章
> **核心法条：** 《专利法》第11条
> **关联页面：** [[权能-原理-制造]]、[[权能-原理-使用]]、[[权能-原理-销售许诺销售]]、[[权能-原理-进口]]、[[权能-原理-方法专利的延伸保护]]

---

## 核心要点

《专利法》第11条规定了专利权的各项权能，包括制造、使用、许诺销售、销售、进口等。这些权能以"为生产经营目的"为前提，旨在为专利权人提供全面的法律保护。发明和实用新型专利与外观设计专利在权能内容上存在差异。

---

## 1. 专利权能的法律依据

### 1.1 《专利法》第11条

《专利法》（2008）第11条规定：

> 发明和实用新型专利权被授予后，除本法另有规定的以外，任何单位或者个人未经专利权人许可，都不得实施其专利，即不得为生产经营目的**制造、使用、许诺销售、销售、进口**其专利产品，或者**使用其专利方法**以及**使用、许诺销售、销售、进口依照该专利方法直接获得的产品**。

> 外观设计专利权被授予后，任何单位或者个人未经专利权人许可，都不得实施其专利，即不得为生产经营目的**制造、许诺销售、销售、进口**其外观设计专利产品。

### 1.2 权能的核心特征

从上述规定可以看出，专利权能具有以下核心特征：

**（1）以"为生产经营目的"为前提：** 《专利法》第11条将专利权人可以禁止的行为限制在以生产经营为目的的范围内，从而排除了部分对专利权人利益不构成实质性威胁的行为。这一限制旨在平衡专利权人的利益与社会公众的自由，避免专利权过度扩张。

**（2）权能的排他性：** 未经专利权人许可，任何单位或个人都不得实施上述行为。专利权是一种消极的权利，即禁止他人实施的权利，而非积极的使用权。专利权人自己实施专利还需要遵守其他法律法规的规定（如产品质量法、环境保护法等）。

**（3）法定的权利内容：** 专利权能由法律明确规定，专利权人不能在法定范围之外自行创设新的权能。例如，专利权人不能依据专利法禁止他人出租专利产品，因为出租权未被列入第11条规定的权能范围。

## 2. 发明和实用新型的权能

### 2.1 产品专利的权能

对于产品专利（包括发明和实用新型），专利权人享有以下五项权能：

| 权能 | 含义 |
|------|------|
| **制造** | 权利要求中所记载的产品技术方案被实现 |
| **使用** | 将专利产品用于其技术目的 |
| **许诺销售** | 做广告、在商店橱窗中陈列或者在展销会上展出等方式作出销售商品的意思表示 |
| **销售** | 交付产品并转让产品所有权 |
| **进口** | 将落入产品专利权利要求保护范围的产品从境外运进境内 |

### 2.2 方法专利的权能

对于方法专利，专利权人享有以下权能：

| 权能 | 含义 |
|------|------|
| **使用方法** | 权利要求记载的专利方法技术方案的每一个步骤均被实现 |
| **使用产品** | 使用依照该专利方法直接获得的产品 |
| **许诺销售产品** | 对依照该专利方法直接获得的产品作出销售的意思表示 |
| **销售产品** | 销售依照该专利方法直接获得的产品 |
| **进口产品** | 进口依照该专利方法直接获得的产品 |

方法专利原本只能直接限制他人未经许可使用该方法。这意味着权利人只能从源头上制止使用该方法的侵权行为，对于市场上的销售、使用、进口等环节缺乏干预手段。为此，专利法将方法专利保护延伸到依照该方法直接获得的产品。

## 3. 外观设计专利的权能

### 3.1 权能内容

外观设计专利权被授予后，任何单位或者个人未经专利权人许可，都不得为生产经营目的**制造、许诺销售、销售、进口**其外观设计专利产品。

### 3.2 与发明/实用新型的区别

外观设计专利的权能与发明/实用新型相比，**缺少"使用权能"**。即外观设计专利权人不享有禁止他人使用外观设计专利产品的权利。

这是因为，外观设计专利保护的是产品设计的美学功能，更接近版权意义上的禁止复制的保护，因而没有引入对使用行为的控制权。理论上，立法者如果要为外观设计专利权设置使用权权能，并没有不可逾越的障碍，但现行法并未赋予此项权能。

## 4. 各项权能之间的关系

### 4.1 制造权的核心地位

禁止他人制造专利产品是专利权最为核心的一项权能。如果没有制造行为，后续的使用、销售等行为就无从谈起。因此，控制制造行为是专利权人维护其市场地位的最根本手段。

### 4.2 许诺销售与销售的衔接

许诺销售权能旨在尽可能早地阻止那些为最终销售做准备的行为，带有明显的预防性质。许诺销售这一术语并非中国公众所熟悉的日常词汇，而是从国际公约中的"offering for sale"翻译而来。更准确的翻译可能是销售要约。当然，这里的要约与合同法上的要约没有必然联系。

一项行为只要让社会公众明确感知到行为人正在试图招揽侵权产品的潜在买主，就应该落入"许诺销售"的控制范围。

### 4.3 进口权的独立价值

在大多数情况下，进口的后续行为是国内销售或使用。如果没有单独的禁止进口权能，权利人有可能将进口行为解释为销售或使用的前奏，通过间接侵权制度阻止进口行为。设置单独的禁止进口权能，则避免了间接侵权制度本身的不确定性，强化了专利权人的控制能力。

---

## 5. "实施"专利的含义

《专利法》第11条使用了"实施其专利"的表述。所谓"实施"专利，对于不同类型的专利有不同的含义：

- **产品专利**：制造、使用、许诺销售、销售、进口专利产品；
- **方法专利**：使用方法，以及使用、许诺销售、销售、进口依照该方法直接获得的产品；
- **外观设计专利**：制造、许诺销售、销售、进口外观设计专利产品。

需要注意的是，"实施"专利的行为必须是"为生产经营目的"。如果行为人不以生产经营为目的，则不构成专利法意义上的"实施"。
