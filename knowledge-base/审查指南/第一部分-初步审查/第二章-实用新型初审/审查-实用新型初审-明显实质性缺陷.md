# 审查-实用新型初审-明显实质性缺陷

> **来源：** 《专利审查指南》第一部分第二章 §5-6、§8-14
> **对应法条：** 《专利法》第2条第3款、第5条、第9条、第19条、第22条、第25条、第31条第1款、第33条 / 《专利法实施细则》第11条、第23条、第49条
> **最新修订：** 2023年修订
> **关联页面：** [[审查-发明初审-明显实质性缺陷]]、[[审查-实用新型初审-申请文件审查]]

---

## 核心要点

实用新型专利申请的明显实质性缺陷审查与发明专利有较大重叠，但增加了实用新型特有的审查内容：保护客体的审查（仅保护产品形状/构造）、新颖性和创造性的审查（发明专利初审不做此项审查）。实用新型初审中，审查员可根据其获得的现有技术信息，审查申请是否明显不具备新颖性和创造性。修改超范围的判断基准为原说明书和权利要求书记载的范围。

---

## 审查标准

### 一、根据专利法第五条、第二十五条和细则第十一条的审查（§5）

对实用新型专利申请是否明显属于法第5条（违反法律/社会公德/妨害公共利益）、法第25条（不授予专利权的客体）规定的审查，参照指南第二部分第一章第3节和第4节的规定。

对实用新型专利申请是否符合细则第十一条（诚实信用原则）规定的审查，适用《规范申请专利行为的规定》。

### 二、根据专利法第二条第三款的审查（§6）

详见[[审查-实用新型初审-申请文件审查]]中"实用新型保护客体"部分。核心要点：
- 只保护产品，不保护方法。
- 仅保护形状、构造或其结合的改进。
- 必须是利用了自然规律的技术方案。

### 三、根据专利法第三十三条的审查（§8）

根据专利法第33条的规定，对申请文件的修改不得超出原说明书和权利要求书记载的范围。

**修改超范围的认定**：
- 加入了所属技术领域的技术人员不能从原说明书和权利要求书中直接地、毫无疑义地确定的内容。
- 从申请中删除某个或某些特征，也可能导致超范围。
- 说明书中补入原权利要求书中记载而原说明书中没有描述过的技术特征并作了扩大描述的。
- 补入原说明书和权利要求书中没有记载且借助附图不能毫无疑义确定的技术特征。

**注意**：
- 对明显错误的更正，不被认为超范围。
- 对于附图中明显可见并有唯一解释的结构，允许补入说明书并写入权利要求中。

#### 8.1 申请人主动修改
- 应当自申请日起两个月内提出。
- 超过两个月的修改，如果消除了原申请文件缺陷且具有授权前景，可以接受。
- 不予接受的，发出视为未提出通知书。

#### 8.2 针对通知书指出的缺陷进行修改
- 应审查修改是否超范围以及是否针对通知书指出的缺陷。
- 包含并非针对通知书所指缺陷的修改，如果符合法第33条且消除了缺陷、具有授权前景，可以接受。
- 不符合细则第57条第3款规定的修改文本，可以通知申请人提交新的修改文本。

#### 8.3 依职权修改
审查员在作出授予专利权通知前，可以对文字和符号的明显错误依职权修改：
1. **请求书**：修改地址中漏写、错写或重复填写的信息。
2. **说明书**：修改不适当的名称和/或技术领域；改正错别字、错误符号；修改不规范用语；增补遗漏的标题；删除不必要的文字说明。
3. **权利要求书**：改正错别字、错误标点符号、错误附图标记、附图标记增加括号。但可能引起保护范围变化的修改，不属于依职权修改范围。
4. **摘要**：修改不适当内容及明显错误，指定摘要附图。

### 四、根据专利法第三十一条第一款的审查（§9）

- 一件实用新型专利申请应当限于一项实用新型。
- 确定特定技术特征时一般依据申请文件中所描述的背景技术。
- 参照指南第二部分第六章第2节的规定。

### 五、根据专利法实施细则第四十九条的审查（§10）

分案申请的审查适用第一章§5.1的规定，同时参照指南第二部分第六章第3节的规定。

### 六、根据专利法第二十二条第二款和第三款的审查（§11）

**这是实用新型初审与发明专利初审的重要区别**：实用新型初审中，审查员对申请是否明显不具备新颖性和创造性进行审查。

- **新颖性审查**：审查员可根据其获得的有关现有技术或抵触申请的信息进行审查。参照指南第二部分第三章和第四部分第六章第3节。
- **创造性审查**：审查员可根据其获得的有关现有技术的信息进行审查。参照指南第四部分第六章第4节。

### 七、根据专利法第二十二条第四款的审查（§12）

实用性是指所申请的产品必须能够在产业中制造和应用，而且能够产生积极、有益的效果。参照指南第二部分第五章。

### 八、根据专利法第九条的审查（§13）

- 同样的发明创造只能授予一项专利权。
- 审查员可根据其获得的同样的发明创造的专利申请或专利进行审查。
- 参照指南第二部分第三章第6节。

### 九、根据专利法第十九条第一款的审查（§14）

与发明专利相同：在中国完成的实用新型向外国申请专利的，应当事先报经专利局进行保密审查。

## 实务应对

### 代理人视角：实用新型明显实质性缺陷的防范

1. **新颖性/创造性审查是实用新型初审的重点**：与发明专利不同，实用新型没有实质审查程序，因此审查员在初步审查中即会对明显不具备新颖性/创造性的申请提出审查意见。代理人在撰写时应进行充分的现有技术检索。

2. **保护客体**：确保权利要求仅涉及产品的形状、构造或其结合。方法步骤、材料配方、计算机程序等均不属于保护客体。若既包含形状/构造特征又包含方法/材料改进，将面临驳回。

3. **修改超范围**：实用新型的修改基准为"原说明书和权利要求书记载的范围"，采用"直接地、毫无疑义地确定"标准。注意：
   - 附图中明显可见并有唯一解释的结构可以补入。
   - 明显错误可以更正。
   - 但超出记载范围的修改将导致驳回。

4. **主动修改时机**：实用新型的主动修改期限为申请日起两个月内，与发明专利相同。超过两个月的修改只有在消除缺陷且有授权前景时才能被接受。

5. **单一性**：实用新型的特定技术特征确定一般依据背景技术。撰写时应确保独立权利要求之间的技术关联性。

## 审查要点速查

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 权利要求包含方法步骤（非已知方法名称） | 不属于保护客体，驳回 | 法第2条第3款 |
| 申请明显不具备新颖性 | 发出审查意见通知书 | 法第22条第2款 |
| 申请明显不具备创造性 | 发出审查意见通知书 | 法第22条第3款 |
| 修改超出原说明书和权利要求书记载范围 | 发出审查意见通知书 | 法第33条 |
| 同样的发明创造已有专利 | 驳回 | 法第9条 |
| 在中国完成的实用新型向外申请未保密审查 | 驳回 | 法第19条 |
| 主动修改超过两个月且无授权前景 | 发出视为未提出通知书 | 细则第57条 |
| 形式缺陷经两次补正仍未消除 | 驳回 | 细则第50条 |
