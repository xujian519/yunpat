# 不授予专利权的客体-技术方案的判断与实用新型客体

> **来源：** 《以案说法》第一章 §1，页1-6
> **核心法条：** 《专利法》第2条第2款、第3款
> **关联页面：** [[不授予专利权的客体-违反法律与社会公德]]、[[不授予专利权的客体-智力活动规则与科学发现]]、[[不授予专利权的客体-疾病的诊断和治疗方法]]、[[不授予专利权的客体-计算机领域的特殊问题]]

---

## 核心要点

专利法保护的基本属性是"技术方案"——必须采用技术手段、解决技术问题、获得技术效果。实用新型专利仅保护产品的形状和/或构造，对方法特征、材料特征有特殊限制。

---

## 1. 技术方案的判断

### 基本规则

根据《专利审查指南》第二部分第一章第2节，技术方案是对要解决的技术问题所采取的利用了自然规律的技术手段的集合。判断标准：

- **技术手段**：通常由技术特征来体现
- **技术问题**：从所采用的技术手段入手进行整体判断
- **技术效果**：符合自然规律的效果

**判断原则：** 将方案作为一个整体分析，不能仅凭技术参数/术语就认定构成技术手段，也不能仅凭非技术性内容就否定整个方案的技术性。

### 案例：第119361号复审决定（201310396828.9）

- **争议焦点：** 万能棋是否构成技术方案
- **决定要点：** 涉案申请对棋盘结构进行了改进（双面棋盘、正方形+等腰三角形布局），采用对棋盘结构进行改进的技术手段，解决了现有棋盘功能单一的技术问题，实现了棋盘功能多样化的技术效果，属于技术方案。
- **启示：** 对产品结构的物理改进，即使涉及棋类游戏，只要采用了技术手段解决技术问题，仍属于技术方案。

### 案例：第111139号复审决定（200510072481.8）

- **争议焦点：** 石墨铅笔分类的色彩图案标示法是否构成技术方案
- **决定要点：** 将铅笔硬度与色彩/图案的对应关系属于人为设定的规则，其"创意"仅来源于对个别自然现象的感官经验，而非具有普遍适用性的客观规律，不属于技术手段。
- **启示：** 仅仅根据个别自然现象的感官经验归纳得出的规则，未采用技术手段解决技术问题的方案，不属于技术方案。

### 案例：第87756号复审决定（200810185997.7）

- **争议焦点：** 自适应控制器设计方法是否构成技术方案
- **决定要点：** 虽然涉及计算方法，但这些方法利用了过程控制（前馈和反馈）数据处理基础上的自然规律，处理结果是对自适应控制器控制逻辑策略的优化。方案整体采用了技术手段，解决了"降低计算和存储要求"的技术问题，属于技术方案。
- **启示：** 计算方法如果利用了自然规律，其整体方案仍可构成技术方案。

### 案例：第29176号复审决定（200410049846.0）

- **争议焦点：** 直通式银行金融服务系统是否构成技术方案
- **决定要点：** 判断技术问题不应局限于申请人声称的问题（"提供资金汇划服务"），而应从技术手段入手整体判断。专线有线传输、身份识别符、密押设备等都是利用自然规律的技术手段，解决了数据传递安全性和效率的技术问题。
- **启示：** 判断技术问题应从技术手段入手作整体判断，不应仅局限于权利要求的主题或申请人声称的问题。

## 2. 实用新型专利保护客体的特殊问题

### 2.1 产品的形状和/或构造

"产品的形状"指产品所具有的、可以从外部观察到的确定的空间形状。粉末状或颗粒状物质因无确定形状，不属于实用新型保护客体。

### 案例：第19885号无效决定（200620127114.3）

- **争议焦点：** 颗粒硅肥是否属于实用新型保护客体
- **决定要点：** 硅肥为颗粒状的物质，没有确定的形状，也不包含构造特征。实用新型仅保护产品的形状和/或构造，因此不属于实用新型保护客体。
- **启示：** 粉末状、颗粒状物质因无确定形状，不属于实用新型保护客体。

### 2.2 包括方法特征的产品权利要求

如果实用新型权利要求既包括产品结构特征，又包括结构部件的加工装配方法，但实质上只是使用已知加工方法对其结构特征进行限定，则不属于对方法本身的改进，仍属于实用新型保护客体。

### 案例：第23819号无效决定（201120447218.3）

- **争议焦点：** 包含"卡压"等装配方法特征的权利要求是否属于实用新型保护客体
- **决定要点：** 使用"插入""卡压""之后"等词语，只是使用现有连接方法对结构特征进行限定，保护的是装配完成后的产品，属于实用新型保护客体。
- **启示：** 方法特征如果只是使用已知方法对结构进行限定，而非对方法本身的改进，不影响实用新型客体认定。

### 2.3 包括材料特征的产品权利要求

材料特征通常不属于产品形状、构造的范畴，但如果所包含的材料特征属于已知材料，则不影响技术方案成为实用新型保护客体。

### 案例：第28559号无效决定（201120154898.X）

- **争议焦点：** 限定"硅胶、皮革类耐高温材料"的桌垫是否属于实用新型保护客体
- **决定要点：** 限定的材料均属于现有技术中的已知材料，该方案是将已知材料应用于发热桌垫结构中，不是对材料本身的改进，属于实用新型保护客体。
- **启示：** 已知材料的应用不改变方案的实用新型客体性质，但材料本身的新发明不在实用新型保护范围内。

## 判断流程

```
权利要求是否构成技术方案？
├── 整体分析：是否采用技术手段？
│   ├── 是 → 是否解决技术问题？
│   │   ├── 是 → 是否获得技术效果？
│   │   │   ├── 是 → 属于技术方案 ✓
│   │   │   └── 否 → 不属于技术方案 ✗
│   │   └── 否 → 不属于技术方案 ✗
│   └── 否 → 不属于技术方案 ✗
└── 注意：不能仅凭技术术语认定，也不能因非技术内容否定
```
