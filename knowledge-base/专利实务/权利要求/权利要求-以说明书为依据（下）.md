# 权利要求 — 以说明书为依据（下）：功能性特征、马库什与数值范围

> **来源：** 《以案说法》第七章 §3.3-3.5，页283-290
> **核心法条：** 《专利法》第26条第4款 / 《专利审查指南》第二部分第二章第3.2.1节
> **关联页面：** [[权利要求-保护范围的确定]]、[[权利要求-以说明书为依据（上）]]、[[权利要求-必要技术特征]]、[[创造性-化学领域发明]]、[[审查-权利要求-说明书依据]]

## 核心要点

功能性特征原则上应理解为能够实现所述功能的所有实施方式。如果权利要求中限定的功能是以说明书实施例中记载的特定方式完成的，而所属领域技术人员并不能明了该功能还可以通过其他替代方式实现，则功能性特征的使用将导致权利要求得不到说明书的支持。马库什化合物权利要求和数值范围权利要求的支持判断，同样需要考察所属领域技术人员能否合理预期整个范围内的技术效果。

---

## 1. 功能性特征的支持问题

### 1.1 功能性特征的基本原则

功能性特征是指以功能或效果来限定技术特征的权利要求撰写方式。其支持判断的核心在于：

- 功能性特征原则上应当理解为能够实现所述功能的**所有实施方式**
- 如果说明书仅记载了一种实现方式，且所属领域技术人员不能明了其他替代方式，则权利要求得不到支持

### 1.2 功能性特征支持的判断标准

**判断步骤**：
1. 确定功能性特征所限定的功能
2. 分析说明书实施例中实现该功能的具体方式
3. 判断所属领域技术人员能否明了其他替代实现方式
4. 如能明了，则支持；如不能明了，则不支持

### 案例：第37170号复审决定（200780019411.3）— 透皮治疗系统的功能性特征

- **争议焦点：** 权利要求中功能性限定的"媒质"能否得到说明书的支持？
- **决定要点：** 权利要求1的技术方案实现发明目的的关键在于媒质具有"当从患者的皮肤上除去所述ITS时，该媒质能够以不取决于剥离方向的方式使所述活性物质与所述化学剂彼此接触并进行化学反应"的功能。"从说明书的记载可知，涉案申请的实施例是通过以下方式实现媒质的这一功能的，即：首先，媒质为硬质多棱角的锋利材料；其次，化学剂要采用例如溶液状态的流动形式；最后，化学剂与活性物质之间存在吸收性的纤维层。这三个特征对于媒质实现所述功能缺一不可。所属领域技术人员依据申请文件的记载不能明了还可以采用除实施例的特定方式之外的其他方式实现媒质的该功能。因此权利要求1得不到说明书的支持。"
- **启示：** 当功能性特征所限定的功能依赖于多个特定的实现条件，且所属领域技术人员无法预见其他实现方式时，功能性特征得不到支持。

### 案例：第33444号复审决定（200510092177.X）— 打印机时序安排的功能性限定

- **争议焦点：** 权利要求中"使打印机处于非操作模式的时间周期数最小化"和"使连续运行时间最大化"的功能性限定是否得到支持？
- **决定要点：** "权利要求1中的'使打印机处于非操作模式的时间周期数最小化'和'使连续运行时间最大化'是对所述时序安排所实现的效果进行的限定，属于功能性限定的技术特征，然而涉案申请说明书中仅仅给出了特定情况下高级工作的时序安排方法可以实现上述效果，例如，实施例中只有黑色打印和印刷色打印的最终编排结果，没有给出高级工作最佳化例程的工作原理，因此所属领域技术人员不能明了此功能是否可以采用说明书中未提到的其他替代方式来完成，权利要求1得不到说明书的支持。"
- **启示：** 功能性限定如果仅以效果描述为特征，而说明书未揭示实现该效果的基本原理或工作机制，则所属领域技术人员无法合理预期其他实现方式，权利要求得不到支持。

### 案例：第25815号无效决定（201120482733.5）— "上下机构"的功能性限定

- **争议焦点：** "上下机构"和"翻转机构"的功能性特征是否得到说明书支持？
- **决定要点：** 请求人认为"上下机构"和"翻转机构"均为功能性特征，应理解为覆盖了能够实现"上下"以及"翻转"功能的所有结构，然而说明书仅记载了一种实现"上下"功能的特定方式。复审决定认为："权利要求1的技术方案通过设置在机头座上的上下机构、翻转机构与具有上料框、下料框和雕铣台的移动工作台之间的配合，实现涉案专利说明书所记载的自动上下料功能。虽然权利要求1没有具体限定上下机构和翻转机构的具体结构，但权利要求1的整体技术方案已经通过各部件之间的配合关系限定了上下机构和翻转机构的工作方式及其与其他部件的相互关系，所属领域技术人员基于权利要求1的整体技术方案和说明书的教导，能够理解上下机构和翻转机构在所述技术方案中的具体作用和实现方式。"
- **启示：** 功能性特征是否得到支持，不仅取决于该特征本身，还取决于权利要求的整体技术方案是否已为所属领域技术人员提供了足够的技术教导。如果整体方案中各部件的配合关系已明确限定了功能性特征的具体实现方式，则可能得到支持。

## 2. 马库什化合物权利要求的支持问题

### 2.1 马库什权利要求的特点

马库什（Markush）权利要求是指在化学领域中，以可选列表形式限定化合物结构的权利要求，例如：

> "一种化合物，其结构式为……，其中R1选自氢、甲基、乙基、丙基……"

马库什权利要求的支持判断需考察：从具体实施例到马库什概括的范围，所属领域技术人员能否合理预期整个范围内的化合物均具有预期的技术效果。

### 2.2 马库什权利要求支持的判断因素

1. **结构相似性**：马库什列表中各选项之间的结构相似程度
2. **性能关联性**：结构变化与技术效果之间的关联是否可预期
3. **现有技术水平**：所属领域是否存在支持该概括的现有技术知识
4. **实施例的充分性**：说明书提供的实施例是否足以支持概括范围

### 案例：马库什权利要求概括过宽

在某案件中，权利要求保护一类喹诺酮类抗菌化合物，其中R1选自"氢、卤素、C1-C6烷基、C1-C6烷氧基、氨基、取代氨基……"等宽泛的取代基。说明书仅提供了R1为氢、甲基、甲氧基和氯的四个具体化合物的抗菌活性数据。决定认为："从说明书提供的四个具体化合物到权利要求概括的宽泛范围，所属领域技术人员无法合理预期所有马库什列表中的取代基变体均能保持相似的抗菌活性。特别是，对于'取代氨基'等结构变化较大的取代基，其对抗菌活性的影响无法从现有技术水平和说明书教导中合理预期。因此权利要求的概括超出了说明书公开的范围，得不到说明书的支持。"

- **启示：** 马库什权利要求的支持判断核心在于"结构-性能关系"的可预期性。当结构变化较大（如从简单烷基到复杂取代基）且缺乏现有技术或实验数据支持时，概括可能过宽。

## 3. 数值范围权利要求的支持问题

### 3.1 数值范围支持的基本原则

数值范围权利要求（如"温度为50-100℃""含量为1-10wt%"）的支持判断需考虑：

- 说明书是否提供了范围端点或范围内的实施例
- 所属领域技术人员能否合理预期范围内的所有数值均能实现预期技术效果
- 范围两端的数值是否存在技术上的特殊困难或异常

### 3.2 数值范围支持的常见情形

| 情形 | 是否通常支持 | 说明 |
|------|------------|------|
| 说明书提供范围内多个实施例 | ✅ 支持 | 实施例分布合理，可支持范围 |
| 说明书仅提供范围端点的实施例 | 通常✅支持 | 端点间的线性/单调关系可预期 |
| 说明书仅提供范围中某一点的实施例 | ⚠️ 可能不支持 | 需判断范围两端是否可预期 |
| 范围包含技术上的极端值 | ❌ 可能不支持 | 极端值可能产生异常效果 |

### 案例：数值范围端点的可预期性

在某案件中，权利要求限定"烧结温度为800-1200℃"，说明书仅提供了900℃和1100℃两个实施例。决定认为："所属领域技术人员基于说明书的教导和所属领域的现有技术水平，能够理解烧结温度与材料致密化程度之间存在正相关关系，在800-1200℃的范围内，随着温度升高，致密化程度逐渐增加。虽然在1200℃的极端高温下可能存在晶粒过度长大的风险，但所属领域技术人员能够根据现有技术水平合理控制保温时间以避免该问题。因此权利要求中800-1200℃的范围能够得到说明书的支持。"

- **启示：** 数值范围的支持判断需考虑范围内技术效果的连续性/单调性，以及所属领域技术人员对极端值的处理能力。

## 4. 实务要点速查

### 4.1 三类特殊权利要求的支持检查清单

```
功能性特征支持检查
├── 功能描述是否明确？
├── 说明书实现方式是否唯一？
├── 所属领域技术人员能否预见替代方式？
└── 整体技术方案是否提供足够教导？

马库什权利要求支持检查
├── 结构相似性是否足够？
├── 性能-结构关系是否可预期？
├── 实施例是否覆盖关键变体？
└── 现有技术是否支持概括？

数值范围支持检查
├── 端点值是否有实施例支持？
├── 范围内技术效果是否连续/单调？
├── 极端值是否存在技术障碍？
└── 所属领域技术人员能否合理预期？
```
