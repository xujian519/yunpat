- 修改后的接口期望不同的格式

**影响范围**:

- `packages/cli/src/commands/drafting-commands.ts`
- `packages/cli/src/commands.ts`

**解决方案**:

1. **方案 A**: 回退修改，保持原有接口
2. **方案 B**: 创建适配器，转换输入格式
3. **方案 C**: 修改 CLI 代码以适应新接口

**推荐**: 方案 A（回退修改），保持稳定性

### 2. 包版本问题

**问题描述**:

- 项目中存在两个相似的包：
  - `packages/agents/specification` (简化版)
  - `packages/agents/specification-drafter` (完整版，CLI 使用)

**影响**:

- 修改错误的包可能不会影响实际使用
- 需要明确哪个包是主要版本

**建议**:

- 保留两个版本，各有用途
- 在文档中说明区别
- 或者合并为一个包
