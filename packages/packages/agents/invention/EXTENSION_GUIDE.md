# 扩展领域支持说明

## 新增领域

### 1. 生物技术领域

**应用场景**：基因工程、蛋白质工程、细胞培养、生物制药等

**特殊要求**：

- 需要详细的实验数据支持
- 需要生物保藏信息
- 需要明确的生物学效果
- 需要考虑伦理审查

**撰写指南**：

- `撰写-化学-生物技术领域发明专利申请文件的撰写`
- `撰写-化学-审查-生物技术领域发明专利申请的审查`

**案例类型**：

- 基因编辑系统
- 抗体药物
- 疫苗开发
- 诊断试剂盒

### 2. 新材料领域

**应用场景**：高分子材料、纳米材料、复合材料、功能材料等

**特殊要求**：

- 需要详细的材料表征数据
- 需要明确的制备工艺
- 需要性能测试数据
- 需要应用场景说明

**撰写指南**：

- `撰写-化学-高分子化合物-撰写实例（上/中/下）`
- `撰写-化学-延伸-天然物质发明的专利保护`

**案例类型**：

- 可降解高分子材料
- 纳米复合材料
- 超导材料
- 智能材料

### 3. 医药领域

**应用场景**：化学药物、生物制品、中药、医疗器械等

**特殊要求**：

- 需要详细的药理毒理数据
- 需要临床试验数据
- 需要明确的适应症
- 需要考虑Bolar例外

**撰写指南**：

- `撰写-化学-组合物与药品发明专利申请文件的撰写`
- `撰写-化学-审查-药品`

**案例类型**：

- 新化合物
- 新型制剂
- 新适应症
- 联合用药

## 使用建议

1. **按领域测试**：每次测试时选择一个领域的所有案例
2. **按难度递进**：从简单案例开始，逐步到复杂案例
3. **对比验证**：与人工撰写的专利文件对比验证
4. **持续扩展**：根据实际需求添加更多案例
