# 侵权-原理-权利冲突抗辩

> **来源：** 崔国斌《专利法：原理与案例（第二版）》第10章 §4.4.6
> **核心法条：** 无直接规定；依据《专利法》先申请原则
> **关联页面：** [[侵权-原理-侵权基本原理]]、[[侵权-原理-先用权]]、[[侵权-法条-不视为侵权的行为]]

## 核心要点

在专利侵权诉讼中，被告以自己实施的方案已获得专利授权为由进行抗辩，通常被称为"权利冲突"抗辩。根据先申请原则，只要原告先于被告提出专利申请，被告的后专利权不能当然地使其行为免于侵权认定。

---

## 1. "权利冲突"问题的由来

在很多侵权案件中，被告常常以自己实施的方案已经获得专利授权为由，对原告的专利侵权指控进行抗辩，并引发所谓"权利冲突"的争论。

### 1.1 问题的本质

这种情形的本质是：原告拥有一项在先专利权，被告拥有一项在后专利权，被告实施其专利技术时落入了原告专利权的保护范围。被告主张自己也有专利权，因此不构成侵权。

### 1.2 专利权消极性的体现

**专利权的消极性**，决定着授予一项专利权，并不从法律上保证权利人可以实施该技术方案。也就是说，拥有专利权不等于可以自由实施该专利技术——如果该技术的实施落入他人更早的专利权范围，仍需获得该在先专利权人的许可。

---

## 2. 最高人民法院的立场

### 2.1 批复内容

在1993年，北京高院还专门就涉及此问题的天津市东郊农牧场诉中国人民解放军3608工厂案向最高人民法院提出请示。最高人民法院在《关于在专利侵权诉讼中原被告双方均拥有实用新型专利权应如何处理的批复》（1993年8月16日）中指出：

对于相同或者类似产品，不同的发明人都拥有专利权的有以下三种情形：

1. **技术方案有本质区别**：不同的发明人对该产品所作出的发明创造的发明点不同，他们的技术方案之间有本质区别
2. **从属专利**：在后的专利技术是对在先的专利技术的改进或改良，它比在先的专利技术更先进，但实施该技术，有赖于实施前一项专利技术
3. **重复授权**：因实用新型专利未经实质审查，前后两项实用新型专利的技术方案相同或者等同，后一项实用新型专利属于重复授权

### 2.2 处理规则

最高人民法院明确：

> 人民法院在审理专利侵权纠纷案件时，根据专利法规定的先申请原则，只要原告先于被告提出专利申请，则应当依据原告的专利权保护范围，审查被告制造的产品主要技术特征是否完全覆盖原告的专利保护范围。
>
> 在一般情况下，前述第一种情形由于被告发明的技术方案同原告发明的技术方案有本质的区别，故被告不构成侵权，后两种情形或者被告为了实施其从属专利而未经在先专利权人的许可，实施了在先的专利技术；或者由于前后两项实用新型专利的技术方案相同或者等同，被告对后一项重复授权专利技术的实施，均构成对原告专利权的侵犯。
>
> 因此，人民法院不应当仅以被告拥有专利权为由，不进行是否构成专利侵权的分析判断即驳回原告的诉讼请求，而应当分析被告拥有专利权的具体情况以及与原告专利权的关系，从而判定是否构成侵权。

## 3. 司法实践

### 3.1 保护在先权利原则

最高人民法院的解释毫不奇怪。现在，国内依据保护在先权利的原则，确认在后获得授权的实施者侵权的案例屡见不鲜。

### 3.2 典型案例

**案例：梧州制药（集团）股份有限公司保健食品罐头厂 v. 梧州市白鹤山保健品厂（广西南宁中院（1994）南市经初字第136号）**

法院认为：被告虽然对自己的龟苓膏产品也申请了外观设计专利，但比原告的外观设计专利要晚，根据先申请原则，确认被告的实施行为依然侵权。

---

## 4. 实务要点

### 4.1 对专利权人的建议

1. 在提起侵权诉讼前，检索被告是否拥有在后专利
2. 如果被告拥有在后专利，准备论证被告技术方案落入自己专利权保护范围
3. 强调先申请原则和专利权的消极性

### 4.2 对在后专利权人的建议

1. 拥有专利权不等于可以自由实施，实施前需进行FTO（自由实施）分析
2. 如果自己的专利技术依赖于在先专利，应寻求获得在先专利权人的许可
3. 在后专利虽然不能对抗在先专利，但可以在许可谈判中作为交叉许可的筹码

### 4.3 对重复授权的处理

如果后专利与先专利技术方案相同或等同，后专利属于重复授权。在先专利权人可以：
1. 在侵权诉讼中主张侵权
2. 向专利复审委员会请求宣告后专利无效

## 判断流程

```
权利冲突抗辩判断
├── 第一步：确定申请时间
│   ├── 原告专利申请日 > 被告专利申请日 → 不适用先申请原则
│   └── 原告专利申请日 < 被告专利申请日 → 继续判断
├── 第二步：技术方案比对
│   ├── 被告技术方案与原告专利有本质区别？
│   │   ├── 是 → 不构成侵权
│   │   └── 否 → 继续判断
│   ├── 被告技术方案落入原告专利保护范围？
│   │   ├── 是 → 继续判断
│   │   └── 否 → 不构成侵权
│   └── 被告技术方案属于原告专利的从属专利？
│       ├── 是 → 实施需获许可，构成侵权
│       └── 否 → 继续判断
└── 第三步：重复授权审查
    └── 前后两项专利技术方案相同或等同？
        ├── 是 → 后专利属于重复授权，构成侵权
        └── 否 → 根据具体情况判断
```
