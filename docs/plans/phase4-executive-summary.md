## 🎯 核心目标

实现**OrchestratorAgent（中枢层）**，作为整个系统的智能调度中心：

1. **意图识别** - 理解用户需求，分类为9种意图类型
2. **任务规划** - 自动生成多步骤执行计划
3. **HITL交互** - 关键节点人机交互确认
4. **结果聚合** - 整合多Agent输出结果
5. **异常降级** - 优雅处理错误和边缘情况

---

## 📐 架构设计

```
┌─────────────────────────────────────────────────────────────┐
│                   OrchestratorAgent                          │
│                  (中枢层 - Layer 0)                          │
├─────────────────────────────────────────────────────────────┤
│  • Context Manager（上下文管理）                             │
│  • 5-Call LLM Pipeline（意图识别→任务规划→HITL→聚合→降级）   │
│  • EventBus Communication（事件总线）                        │
│  • Routing Logic（路由逻辑：简单直达 vs 复杂编排）            │
└─────────────────────────────────────────────────────────────┘
                          ↓
              专业层 → 执行层 → 工具层
```
