# 中国对生物技术的专利保护

生物技术是21世纪最具发展潜力的技术领域之一。随着人类基因组计划的实施和生物技术的飞速发展，其技术成果的专利保护出现了许多新问题，引起了国内外专利界的激烈争论。

## 一、生物技术的特点

1. **发展快**：转基因水稻、棉花已大面积种植；转基因动物如克隆羊的出现使克隆技术上新台阶；人类基因组图谱的公布使生物技术进入全新阶段
2. **意义重大**：转基因植物为解决温饱问题提供根本出路；生物酶应用有望取代污染严重的化工行业；人类基因组计划有望攻克疑难病症
3. **特殊问题多**：引发法律、政治、经济、宗教、社会公德及伦理道德等棘手问题

## 二、中国对生物技术专利保护的历史

生物技术领域的发明大致可分为六类：
1. 转基因动物和植物品种发明
2. 微生物及遗传物质发明
3. 生物制品发明
4. 获得生物体的生物学方法或遗传工程学方法发明
5. 微生物学方法发明
6. 基因治疗方法发明

**历史演变**：
- **1985-1992年**：由于《专利法》第25条规定对"药品和用化学方法获得的物质"以及"动物和植物品种"均不授予专利权，仅可对第(4)、(5)类发明授予专利
- **1993年1月1日**：修改后的《专利法》删除上述限制，第(2)、(3)类发明也可获得专利保护
- **1994年1月1日**：加入《专利合作条约》（PCT）
- **1995年7月1日**：成为《布达佩斯条约》成员国
- **1997年10月1日**：《植物新品种保护条例》施行
- **2001年7月1日**：《专利法》第二次修改施行，进一步加强保护力度
- **2009年10月1日**：《专利法》第三次修改施行，增加遗传资源保护和披露条款

## 三、中国对生物技术专利保护的现状

### 1. 对动物和植物品种不给予专利保护

不论是用传统生物学方法还是通过基因工程DNA重组技术得到的转基因动植物，目前在中国都不给予专利保护。但符合条件的植物新品种可通过《植物新品种保护条例》保护。

### 2. 微生物菌种和遗传物质属于可授予专利的主题

微生物既不属于动物也不属于植物品种。依据《专利法实施细则》和《专利审查指南》第二部分第十章，微生物菌种（包括细菌、真菌、放线菌、病毒、动植物细胞系、质粒、原生动物和藻类）属于可授予专利的主题。遗传物质如基因、DNA、RNA和染色体等属于生物化学物质，也可以被授予专利。

**例外**：未经人类技术处理而存在于自然界的微生物菌种和遗传物质属于科学发现；不能重复得到的微生物菌种和遗传物质不具备实用性。

### 3. 生物制品可以得到产品专利

生物制品（疫苗、抗毒血清、类毒素、抗菌素等）可以获得产品专利。

- 新的化学物质：产品权利要求应当用分子式或结构式、DNA核苷酸顺序、物理化学及性能参数和/或制备方法来定义
- 药物制剂：产品权利要求中除写明组成外，还应当写明其应用范围（可治疗的病症）

### 4. 获得生物体的方法必须是可以重复再现的非生物学方法

生物学方法在很大程度上依赖随机因素，所产生的个体差异很大，不能满足实用性要求的再现性。例如辐照饲养法生产高产牛奶乳牛的方法、改进饲养方法生产瘦肉型猪的方法、采用基因重组技术生产转基因动植物或微生物的方法，都可以依法被授予方法专利。

### 5. 微生物学方法必要时要保藏菌种

如果所使用的微生物是公众不能得到的，应当按照《专利法实施细则》第24条的规定到保藏单位保藏。公众不能得到的包括：个人或单位持有且不对公众公开发放的菌种；或通过筛选、突变等手段新创制的微生物。

### 6. 疾病的诊断和治疗方法不授予专利

包括基因治疗在内的所有疾病的诊断和治疗方法在中国都得不到专利保护。

**例外**：为诊断和治疗疾病而使用的药物和医疗器具；美容方法；物品消毒方法；在人或动物体之外进行的化验方法；尸体的测试及处理方法；非诊断和治疗目的的测定生理参数的方法等。

### 7. 依赖遗传资源完成的发明需要披露直接来源和原始来源

《专利法》第26条第5款规定，依赖遗传资源完成的发明创造，申请人应当在专利申请文件中说明该遗传资源的直接来源和原始来源；无法说明原始来源的，应当陈述理由。对违反法律、行政法规的规定获取或者利用遗传资源完成的发明创造，不授予专利权。

## 四、生物技术对专利保护提出的挑战

### 1. 关于人体基因是否属于科学发现

**观点**：对人类基因组图谱的测定和绘制属于科学发现，不能授予专利权。但从客观存在的全长DNA序列中选择特定片段，第一次用技术手段将其分离或克隆出来，使其显示出特有的应用价值（例如制备治疗疑难病症的生物药品），则属于技术发明，可以授予专利权。

这与天然物质的规定一致：仅仅发现天然物质的存在属于科学发现；通过技术手段第一次将其分离出来，改变了其存在状态或内部结构，从而使其显示出过去未曾显示的使用价值，就可以视为发明。

### 2. 关于动物和植物品种的概念

欧洲专利局将"品种"狭义解释为动物分类学"种"的概念，从而可以对转基因动植物授予专利。但笔者认为这种做法在法律概念上存在严重逻辑缺陷，容易造成混乱。建议通过修改《专利法》或《专利审查指南》，将"动物和植物品种"限定为"用传统生物学方法得到的动物和植物品种"。

### 3. 关于基因治疗方法的专利保护

目前在中国，讨论对疾病的诊断和治疗方法进行专利保护似乎还为时过早。但如果某项基因治疗方法能够转换成基因治疗药物或器械等产品形式，则可以按照现行专利法规得到变相的专利保护。

### 4. 关于伦理道德标准

《专利法》第5条规定，对违反国家法律、社会公德或者妨害公共利益的发明创造不授予专利权。《欧洲专利公约实施细则》已明确规定以下不授予专利权：克隆人类的方法；改变人类遗传特性的方法；人类胚胎在工业或商业上的应用；使动物遭受痛苦而没有任何明显医学益处的改变动物遗传特性的方法及所得动物。

### 5. 关于实用性和专利的授权范围

- **实用性**：我国应当从严掌握，只有当所申请的基因具有通过生物学实验证明的确定功能时才能认为具备实用性
- **授权范围**：由于我国生物技术领域总体技术水平不够高，判断可实施尺度可以紧一些，批准专利的授权范围相对小一些，这符合我国的国情

**遗传资源保护**：我国是生物遗传资源大国，应当采取多管齐下的方针，既要保护资源，也要积极开发利用资源，使其尽快转化为自主知识产权。
