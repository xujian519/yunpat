## 📚 使用指南

### 快速开始

#### 方式1: 非交互式（适合自动化）

```bash
yunpat draft \
  --title "一种基于深度学习的图像识别方法" \
  --field "人工智能" \
  --disclosure examples/disclosure-example.md \
  --output report.md
```

#### 方式2: 交互式（推荐，适合首次使用）

```bash
yunpat draft-interactive \
  --title "一种基于深度学习的图像识别方法" \
  --field "人工智能" \
  --disclosure examples/disclosure-example.md
```

然后按照提示选择操作：

- `y` - 通过并保存
- `c` - 修正内容
- `s` - 补充信息
- `r` - 重新分析
- `q` - 取消

### 工作流程示例

```bash
# 第一次分析
yunpat draft-interactive --title "..." --field "..." --disclosure ...

# 查看结果，选择 "s" 补充信息
> 输入补充信息: 该方法还支持迁移学习

# 查看改进后的结果，选择 "c" 修正
> 输入修正: keyFeatures: 迁移学习支持

# 确认无误，选择 "y" 保存
✓ 结果已保存到: data/drafts/invention-understanding-2026-05-03.md
✓ JSON数据已保存到: data/drafts/invention-understanding-2026-05-03.json
```
