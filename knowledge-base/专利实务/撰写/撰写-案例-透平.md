# 撰写-案例-透平

> **来源：** 《发明和实用新型专利申请文件撰写案例剖析（第2版）》第二部分案例四，吴观乐主编，知识产权出版社，2004年
> **类型：** 发明专利申请（产品：透平/涡轮机械）
> **关联页面：** [[撰写-权利要求书撰写步骤]]、[[撰写-说明书撰写步骤]]

---

## 核心要点

本案例训练复杂机械系统的权利要求层次设计。透平作为由多个部件组成的复杂系统，其权利要求书需要构建从系统整体到关键部件、再到具体结构特征的多层次保护体系。撰写时需注意各部件之间的配合关系、功能性限定与结构性限定的平衡，以及说明书对复杂系统工作原理的充分公开。

---

## 一、案情简介

**发明主题：** 透平（涡轮机械）

**技术背景：** 透平是将流体能量转换为机械功的旋转式动力机械，广泛应用于发电、航空、化工等领域。现有透平在效率、可靠性、维护性等方面存在改进空间。

**技术方案：** 对透平的叶片结构、流道设计、密封系统或冷却系统进行改进，以提高效率、降低噪音或延长寿命。

**主要训练点：**
1. 复杂系统的权利要求层次设计（系统→子系统→部件→结构）
2. 功能性限定与结构性限定的选择
3. 多个独立权利要求的布局（系统权利要求+部件权利要求）
4. 说明书中复杂系统工作原理的清晰表达

---

## 二、撰写思路

### （一）权利要求书的层次设计

**层次一：系统整体**
```
1. 一种透平，包括壳体、转子、叶片和密封装置，
   其特征在于：……（系统级别的改进）
```

**层次二：关键部件**
```
2. 一种用于权利要求1所述透平的叶片，其特征在于：……
3. 一种用于权利要求1所述透平的密封装置，其特征在于：……
```

**层次三：具体结构**
```
4-10. 从属权利要求：叶片的具体形状、密封装置的具体结构等
```

### （二）说明书的撰写要点

1. **技术领域：** 涡轮机械，特别是……透平
2. **背景技术：** 现有透平的结构和工作原理，存在的效率或可靠性问题
3. **发明内容：** 
   - 系统级别的改进及其效果
   - 关键部件的改进及其效果
4. **具体实施方式：** 透平的整体结构图、关键部件的详细图、工作过程描述

---

## 三、典型案例特征

| 特征 | 撰写要点 |
|------|---------|
| 多层次保护 | 系统→部件→结构，层层递进 |
| 功能性限定 | 对于复杂配合关系，可适当使用功能性限定 |
| 附图重要性 | 复杂机械系统必须借助附图清晰表达各部件关系 |
| 工作原理 | 说明书中需清楚说明流体流动路径和能量转换过程 |

---

## 四、启示

1. **从整体到局部**：复杂系统的权利要求书应从系统整体开始，逐步深入到关键部件和具体结构。
2. **单一性与关联性**：系统权利要求与部件权利要求之间应满足单一性要求，体现技术上的关联。
3. **充分公开**：复杂系统的工作原理和结构关系必须在说明书中充分公开，以支持权利要求中的概括性描述。
