# 不授予专利权的客体-疾病的诊断和治疗方法

> **来源：** 《以案说法》第一章 §3.3，页15-28
> **核心法条：** 《专利法》第25条第1款第（3）项
> **关联页面：** [[不授予专利权的客体-智力活动规则与科学发现]]、[[不授予专利权的客体-动物和植物品种]]、[[不授予专利权的客体-计算机领域的特殊问题]]

---

## 核心要点

疾病的诊断和治疗方法不授予专利权。诊断方法需同时满足"以有生命的人体或动物体为对象"和"以获得疾病诊断结果或健康状况为直接目的"两个条件。判断关键在于检测结果是"诊断结果"还是"中间结果"，以及方法是否实质上限制了医生的治疗自由。

---

## 1. 疾病的诊断方法

### 判断标准

一项与疾病诊断有关的方法如果同时满足以下两个条件，则属于疾病的诊断方法：
1. **以有生命的人体或动物体为对象**
2. **以获得疾病诊断结果或健康状况为直接目的**

### 1.1 诊断结果

如果专利申请文件中明确记载了检测结果直接用于诊断特定疾病，通常无法将检测结果与诊断用途割裂。即使申请人在审查程序中删除了所有诊断相关记载，所属领域技术人员基于原始申请文件仍有充分理由认定检测方法的直接目的在于诊断疾病。

### 案例：第15588号复审决定（02825879.7）

- **争议焦点：** 检测碱性鞘磷脂酶的体外方法是否属于诊断方法
- **决定要点：** 说明书记载了该酶在肠道癌变中的重要作用，其过量排泄可作为结肠直肠癌变的诊断标记。即使申请人删除了所有诊断信息，基于原始申请文件的记载，该检测信息的直接目的仍是获得诊断结果。
- **启示：** 原始申请文件的记载是判断的直接依据，后续删除诊断相关内容不影响认定。

### 生物学物质与诊断结果

肿瘤标志物、染色体拷贝数、外源病菌/病毒等与疾病关联程度较高的生物学物质，其检测信息往往足以判断是否患病，通常应认定为"诊断结果"。

### 案例：第34984号复审决定（200410074306.8）

- **争议焦点：** 检测血清中胰腺癌差异蛋白的方法是否属于诊断方法
- **决定要点：** 蛋白含量明显降低与胰腺癌直接相关，所属领域技术人员可根据该标志物含量诊断或辅助诊断胰腺癌，属于疾病诊断方法。
- **启示：** 特定疾病标志物的检测方法通常属于诊断方法。

### 1.2 中间结果

如果检测方法的直接目的只是获取不具有疾病诊断意义的信息，或者对脱离人体的组织、体液或排泄物进行处理以获取中间结果信息，则不属于疾病的诊断方法。

### 案例：第77730号复审决定（200880023632.2）

- **争议焦点：** 检测体液中维生素B6数量的方法是否属于诊断方法
- **决定要点：** 维生素B6参与体内多种生理和代谢过程，饮食、生活习惯等因素都可能影响其浓度，根据体液浓度既不能直接确定患病情况，也不足以判定健康状况。该检测结果属于"中间结果"。
- **启示：** 与多种非疾病因素相关的检测指标，通常只能获得"中间结果"，不属于诊断方法。

### 案例：第67292号复审决定（200910258228.X）

- **争议焦点：** 测量血样谐振频率的方法是否属于诊断方法
- **决定要点：** 谐振频率仅是反映止血特性的参数之一，不能反映全部止血特性；判断凝血功能障碍需要多项筛查试验，仅凭该数据不能直接诊断疾病。
- **启示：** 如果需要结合其他多项检测才能做出诊断，单项检测结果属于"中间结果"。

### 1.3 同一主体

如果检测结果并非针对离体样品来源的同一主体，而是通过统计分析获得规律用于不特定个体，则不属于诊断方法。

### 案例：第16198号复审决定（03136093.9）

- **争议焦点：** 体外确定精神分裂症易感基因的方法是否属于诊断方法
- **决定要点：** 该方法以已确定的患者离体样品为对象，分析结果用于判断一般人群的易感性，诊断结果并非针对离体样品检测方法的同一主体。
- **启示：** 以已知患者样品为对象，结论用于不特定群体的统计分析方法，不属于诊断方法。

### 1.4 健康状况

"健康状况"主要体现在影响人体或动物体生存与发育的机体状态。对健康状况的评估如果包含患病风险度、是否处于健康或亚健康状态以及治疗效果等内容，则属于疾病诊断的范畴。

### 案例：第34233号复审决定（200610023414.1）

- **争议焦点：** 判别上皮细胞癌症性质的方法是否属于诊断方法
- **决定要点：** 根据该方法得到的癌细胞比例分布，至少能直接获得本体是否存在癌变风险、是否已发生癌变等健康状况结论，属于疾病诊断方法。
- **启示：** 即使不能确定具体分型分期，只要能获得健康状况结论就属于诊断方法。

### 1.5 排除式限定

如果检测方法既可以用于诊断目的也可以用于非诊断目的，可以采用"用于非诊断目的"等限定来避免属于诊断方法的缺陷。但如果实施该方法客观上就实现了诊断目的，或所属领域技术人员无法确定其还可用于非诊断领域，则即便采用排除式限定，仍属于诊断方法。

### 案例：第93858号复审决定（200910221879.1）

- **争议焦点：** "非诊断性显像方法"是否仍属于诊断方法
- **决定要点：** 尽管权利要求主题限定为"非诊断性"，但该显像方法利用放射性同位素标示物选择性地被存在机能障碍的线粒体摄取的原理来侦测疾病，实质上仍以获得同一主体疾病诊断结果为直接目的。
- **启示：** 排除式限定不能改变方法的实质目的，形式上的限定不足以规避诊断方法的认定。

### 1.6 制药用途权利要求

"瑞士型权利要求"（物质X在制备用于治疗疾病Z的药物中的应用）保护的是药物制造方法，而非疾病的诊断与治疗方法。

### 案例：第103924号复审决定（201180024021.1）

- **争议焦点：** DNA聚合酶在制备试剂盒中的应用是否涉及诊断方法
- **决定要点：** 该权利要求属于制药用途权利要求，试剂盒属于药用试剂，不涉及疾病诊断方法。
- **启示：** 制药用途权利要求保护的是制造方法，即使检测用试剂盒也不属于诊断方法。

## 2. 疾病的治疗方法

### 2.1 以治疗为目的的方法

判断时需从行为主体、所针对的对象、目的和结果几个方面考量，特别需要注意该方法是否实际影响了医生在治疗疾病中选择各种方法和条件的自由。

### 案例：第95595号复审决定（201010218853.4）

- **争议焦点：** 连续性血液净化局部枸橼酸抗凝补钙计算方法是否属于治疗方法
- **决定要点：** 该方法只能在治疗过程中进行计算，属于治疗方法不可缺少的一部分，授权必然会限制医生选择抗凝方式的自由。
- **启示：** 伴随治疗过程进行的计算方法如果构成治疗的必要组成部分，属于治疗方法。

### 案例：第74262号复审决定（200480009006.X）

- **争议焦点：** 测定面罩中呼吸气流的方法是否属于治疗方法
- **决定要点：** 对气流的测量不属于正压通气治疗方法本身，授权不会限制医生使用面罩进行治疗的自由；数据处理步骤也与治疗无关。
- **启示：** 如果方法不限制医生治疗选择的自由，不属于治疗方法。

### 2.2 具有治疗性质的方法

包括为实施外科手术治疗采用的辅助方法、手术规划和手术导航方法等。

### 案例：第101855号复审决定（201310064967.1）

- **争议焦点：** 医学过程模拟方法是否属于治疗方法
- **决定要点：** 该方法是以真实病人的心脏图像为基础，为实际手术确定治疗方案，是外科手术前的步骤和组成部分，属于实施外科手术治疗方法所采用的辅助方法。
- **启示：** 手术规划和模拟方法如果针对真实患者并为实际手术做准备，属于具有治疗性质的方法。

### 2.3 美容方法

分为"单纯的美容方法"（不介入人体或不产生创伤）和"包含治疗手段的美容方法"。前者不属于治疗方法，后者属于治疗方法。

### 案例：第37259号复审决定（200580021133.6）

- **争议焦点：** 减少皱纹的面部皮肤美容处理方法是否属于治疗方法
- **决定要点：** 皱纹不是疾病，该方法以降低皱纹为目的，是对面部的修饰，通过电磁辐射照射皮肤不涉及创伤性或介入性处置，属于单纯美容方法。
- **启示：** 非治疗目的且无创伤性的美容方法不属于治疗方法。

## 判断流程

```
是否属于疾病的诊断方法？
├── 以有生命的人体/动物体为对象？
│   └── 否 → 不属于（可能是中间结果/制药用途）✓
├── 直接目的是诊断结果还是中间结果？
│   ├── 诊断结果 → 属于 ✗
│   └── 中间结果 → 不属于 ✓
├── 排除式限定能否改变实质？
│   ├── 客观上实现诊断目的 → 仍属于 ✗
│   └── 可用于非诊断目的 → 不属于 ✓
└── 制药用途权利要求 → 不属于诊断方法 ✓

是否属于疾病的治疗方法？
├── 是否限制医生治疗选择的自由？
│   ├── 是 → 属于 ✗
│   └── 否 → 不属于 ✓
├── 是否为手术的辅助方法？
│   └── 是 → 具有治疗性质 ✗
└── 美容方法？
    ├── 单纯美容 → 不属于 ✓
    └── 包含治疗手段 → 属于 ✗
```
