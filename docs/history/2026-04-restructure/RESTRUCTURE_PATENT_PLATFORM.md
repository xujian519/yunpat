## 🎯 核心模块设计

### 1. 专利撰写智能体 (PatentWriter)

**功能**：

- ✅ 技术方案理解（权利要求设计）
- ✅ 说明书生成（背景技术、发明内容、附图说明）
- ✅ 权利要求书撰写（独立权利要求、从属权利要求）
- ✅ 申请文件生成（请求书、摘要、附图）

**核心能力**：

```typescript
class PatentWriter extends Agent {
  // 技术方案分析
  protected async plan(invention, context) {
    // 1. 理解技术方案
    // 2. 检索现有专利
    // 3. 设计保护范围
  }

  // 撰写申请文件
  protected async act(plan, context) {
    // 1. 撰写权利要求
    // 2. 撰写说明书
    // 3. 生成附图说明
  }

  // 质量检查
  protected async reflect(draft, context) {
    // 1. 格式检查
    // 2. 逻辑一致性检查
    // 3. 可专利性评估
  }
}
```

**输入**：

```json
{
  "invention": {
    "title": "一种基于深度学习的图像识别方法",
    "field": "计算机视觉",
    "applicant": "某某科技公司",
    "inventors": ["张三", "李四"],
    "abstract": "本发明公开了一种...",
    "detailedDescription": "具体实现方式...",
    "drawings": ["图1", "图2"]
  }
}
```

**输出**：

```json
{
  "patentApplication": {
    "title": "一种基于深度学习的图像识别方法",
    "abstract": "...",
    "claims": [
      {
        "type": "independent",
        "number": "1",
        "content": "一种基于深度学习的图像识别方法，其特征在于..."
      },
      {
        "type": "dependent",
        "number": "2",
        "content": "根据权利要求1所述的方法，其特征在于..."
      }
    ],
    "description": "...",
    "drawings": "..."
  }
}
```

- ✅ 答复书生成（法律论证）
