# 证据的认定-举证责任与证明标准

> **来源：** 《以案说法》第十章 §2-3，页330-342
> **核心法条：** 《专利法》《专利审查指南》举证责任相关规定
> **关联页面：** [[证据-书证与证人证言]]、[[证据-鉴定意见与域外证据]]、[[证据-互联网证据的特殊规则]]、[[证据-举证责任分配与证明标准]]、[[证据的认定-证据资格与证明力]]、[[证据的认定-举证期限与证据调查]]、[[现有技术和现有设计-出版物公开与使用公开]]、[[专利判决/程序与证据/程序与证据-证据规则]]

## 核心要点

举证责任由主张方承担，没有证据或证据不足的承担不利后果。使用公开由请求人举证；专利权人主张售后修改或保密义务的，举证责任转移至专利权人。证明标准要求证据能清楚反映待证事实，证据之间需相互印证形成完整链条。

---

## 1. 举证责任

### 基本规则

当事人对自己提出的主张有提供证据进行证明的责任。没有证据或证据不足以证明事实主张的，负有举证责任的当事人承担不利后果。

### 使用公开的举证

请求人主张使用公开应举证证明。发票产品型号与实物型号一致的，可相互印证构成使用公开。

### 案例：第27858号无效决定（201420054693.8）

- **争议焦点：** 专利权人主张售后维修改变了产品结构
- **决定要点：** 证据6（发票）与证据7（公证保全实物）相互印证，可认定产品在申请日前构成使用公开。专利权人虽主张结构变化但提交的图纸无签名、公章、日期等信息。同一产品型号一般对应相同产品结构，专利权人未证明结构变化起止时间和替换事实。
- **启示：** 产品型号一般对应相同结构，专利权人主张售后修改需承担举证责任，仅有来源不明的图纸不能完成举证。

### 保密义务抗辩

专利权人主张合同双方存在保密义务的，应提供相关证据。如果能证明供需双方因保密义务不构成使用公开，则举证责任转回请求人。

### 案例：第21507号无效决定（02159947.5）

- **争议焦点：** 合同约定的产品是否构成使用公开
- **决定要点：** 专利权人提交的反证（保密协议书、技术协议书等）与合同可相互呼应，证明协议需方负有保密义务，购买和使用行为不构成专利法意义上的使用公开。在请求人未提供其他证据的情况下，使用公开主张不予支持。
- **启示：** 保密义务一旦被证明，使用公开的主张即不能成立，举证责任转回请求人。

## 2. 证明标准

### 2.1 达到证明标准

证据必须能**清楚地反映待证事实**。多份证据相互印证、符合认知习惯的，一般认为达到证明标准。

**域外出版物的公开时间：** 即使公证认证完备，还需核实公开时间。出版单位证言、交易记录、供货时间等可用于佐证。

**多份证人证言的使用公开：** 如果证人身份、活动性质与出证内容符合认知习惯，且证言内容相互印证，达到证明标准。

**有瑕疵的交易证据：** 瑕疵符合生活、交易习惯，不违反法律，多份证据相互印证的，达到证明标准。

**生效裁判文书：** 人民法院生效判决所确认的事实可直接作为认定依据，达到证明标准。

### 2.2 未达到证明标准

**随印产品说明书：** 缺乏核实手段、证明力弱的，不能单独证明公开时间和真实性。

**庭审/调查笔录：** 仅证明庭上作过陈述，不证明陈述内容为真；未被生效判决采纳的，不足以证明待证事实。

**保密主张不合理：** 召开媒体新闻发布会推广新产品后又主张保密的，违反常理，保密主张不予支持。

### 案例：第14484号无效决定（200430088722.4）- 证明标准部分

- **争议焦点：** 专利权人声称新闻发布会属于保密状态
- **决定要点：** 通过召开新闻发布会、邀请媒体记者推广新车型的方式宣传新产品，其后又主张该发布会处于保密状态，不符合常理。发布会显然是为了推广宣传而非保密。
- **启示：** 公开宣传活动后主张保密的，需举证证明保密措施的存在，仅凭单方主张不足以证明。

**用户可修改网站证据：** 用户可自行上传、修改、删除产品信息的网站（如Made-in-China.com），无其他佐证的不能确认公开时间。

## 判断流程

```
举证责任分配
├── 使用公开 → 请求人举证
│   ├── 发票+实物型号一致 → 印证使用公开
│   └── 专利权人主张修改 → 专利权人举证
├── 保密义务 → 专利权人举证
│   ├── 证明保密协议存在 → 举证转回请求人
│   └── 未证明 → 使用公开成立
└── 证明标准
    ├── 多份证据相互印证 → 达到标准 ✓
    ├── 符合认知习惯+无相反证据 → 达到标准 ✓
    └── 孤证/矛盾/违反常理 → 未达标准 ✗
```
