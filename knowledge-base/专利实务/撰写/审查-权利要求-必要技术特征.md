> **来源：** 《机械领域专利申请文件的撰写与审查（第2版）》第三章"五、与专利审查及专利保护有关问题的讨论"之(十四)谈权利要求中的"必要技术特征"，张荣彦著，知识产权出版社，2006年，页200-215

> **关联页面：** [[权利要求-必要技术特征]]、[[撰写-机械-权利要求书撰写-基本问题]]、[[审查-权利要求-技术特征的认定]]

## 核心要点

必要技术特征是发明或者实用新型为解决其技术问题所不可缺少的技术特征，其总和足以构成发明或者实用新型的保护客体，使之区别于其他技术方案。准确判断必要技术特征是撰写独立权利要求和进行专利审查的基础。

---

## 一、必要技术特征的定义

### （一）法律依据

《专利法实施细则》第21条第2款规定：

> 独立权利要求应当从整体上反映发明或者实用新型的技术方案，记载解决技术问题的**必要技术特征**。

### （二）定义

**必要技术特征**是指发明或者实用新型为解决其技术问题所**不可缺少**的技术特征，其总和足以构成发明或者实用新型的保护客体，使之区别于其他技术方案。

### （三）关键要素

| 要素 | 内容 |
|------|------|
| 不可或缺性 | 缺少该技术特征，就无法解决发明提出的技术问题 |
| 整体性 | 全部必要技术特征的总和构成完整的技术方案 |
| 区别性 | 必要技术特征的总和使发明区别于其他技术方案 |

---

## 二、必要技术特征的确定方法

### （一）从技术问题出发

**基本方法：** 从说明书提出的技术问题出发，反推解决该技术问题所**必须**包括的全部技术特征。

**步骤：**
1. 明确发明要解决的技术问题
2. 分析说明书中描述的技术方案
3. 逐一判断每个技术特征是否为解决该技术问题所必需
4. 将必要技术特征写入独立权利要求

### （二）案例分析

#### 案例一：便携式牙刷

**技术问题：** 不取出牙膏软袋即可将牙膏挤在刷毛上，使用方便。

**技术特征分析：**

| 技术特征 | 是否必要 | 理由 |
|---------|---------|------|
| 牙刷本体 | ✅ | 基本组成部分 |
| 盒体 | ✅ | 基本组成部分 |
| 牙膏软袋 | ✅ | 基本组成部分 |
| 盒体顶壁上有空腔 | ✅ | 携带时容纳刷毛 |
| **牙膏软袋出膏口位置与刷毛空腔位置相对应** | ✅ **关键** | **不对应则无法挤牙膏到刷毛上** |
| **盒体有挤压牙膏软袋的装置** | ✅ **关键** | **无挤压装置则无法挤出牙膏** |
| 活动连接 | ✅ | 实现牙刷本体与盒体的连接 |

**结论：** "牙膏软袋出膏口位置与刷毛空腔位置相对应"和"盒体有挤压牙膏软袋的装置"是解决技术问题的关键必要技术特征，必须写入独立权利要求。

#### 案例二：磁化防垢除垢器

**技术问题：** 能在管道中产生足够的磁通密度，使水很好地磁化，无漏磁。

**技术特征分析：**

| 技术特征 | 是否必要 | 理由 |
|---------|---------|------|
| 外壳 | ✅ | 基本结构 |
| 管道 | ✅ | 基本结构 |
| 永磁磁块 | ✅ | 产生磁场的核心部件 |
| **相邻两对磁块之间磁场基本相互垂直** | ✅ **关键** | **确保无漏磁** |
| **包覆磁块的铁皮外表面与外壳内壁之间有间隙** | ✅ **关键** | **确保磁通密度** |
| 外壳外表面涂防护漆 | ❌ | 与磁化效果无关，仅防锈 |
| 不超过5对磁块 | ❌ | 两对以上即可解决问题 |

**结论：** "相邻两对磁块之间磁场基本相互垂直"和"包覆磁块的铁皮外表面与外壳内壁之间有间隙"是必要技术特征；"涂防护漆"和"不超过5对"是非必要技术特征。

## 三、必要技术特征与创造性判断的关系

### （一）两者的联系

必要技术特征的确定与创造性判断密切相关：

- **必要技术特征越少** → 独立权利要求的保护范围越宽 → 越容易被现有技术覆盖 → 创造性越低
- **必要技术特征越多** → 独立权利要求的保护范围越窄 → 越不容易被现有技术覆盖 → 创造性越高

### （二）平衡保护范围与创造性

撰写独立权利要求时，需要在保护范围和创造性之间取得平衡：

1. **尽可能减少必要技术特征** → 获得较宽的保护范围
2. **但减少后的技术方案必须具备新颖性和创造性** → 确保能够授权

## 四、常见错误

### （一）缺少必要技术特征

**表现：** 独立权利要求未包含解决技术问题的全部必要技术特征。

**后果：**
- 不符合《专利法实施细则》第21条第2款的规定
- 审查员将发出审查意见，要求补充必要技术特征
- 授权后，可能成为无效宣告的理由

### （二）写入非必要技术特征

**表现：** 独立权利要求中写入了与发明技术问题无关的技术特征。

**后果：**
- 缩小保护范围
- 损害专利权人利益
- 竞争对手容易规避

### （三）将附加技术特征误认为必要技术特征

**表现：** 将优选方案中的技术特征误认为必要技术特征，写入独立权利要求。

**后果：** 保护范围不必要地缩小。

## 五、实务建议

### （一）撰写前的分析

在撰写独立权利要求前，应当：
1. 仔细阅读技术交底书，理解发明的技术问题
2. 列出所有技术特征
3. 逐一判断每个技术特征是否为解决技术问题所必需
4. 将必要技术特征写入独立权利要求，将非必要特征写入从属权利要求

### （二）审查中的应对

收到"缺少必要技术特征"的审查意见时：
1. 审查员认定的必要技术特征是否确实必要
2. 如果确实必要，从说明书中提取并补充到独立权利要求
3. 如果不必要，向审查员陈述理由
