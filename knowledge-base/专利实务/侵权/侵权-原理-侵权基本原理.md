# 侵权-原理-侵权基本原理

> **来源：** 崔国斌《专利法：原理与案例（第二版）》第10章 §1
> **核心法条：** 《专利法》（2008）第11条

---

## 核心要点

《专利法》第11条确立了专利权人对其专利的独占实施权，任何单位或个人未经许可以生产经营目的实施专利，即构成侵权。专利侵权行为分为直接侵权与间接侵权，直接侵权又可分为字面侵权与等同侵权。

---

## 1. 专利侵权的法律依据

### 1.1 《专利法》第11条

《专利法》（2008）第11条规定：

> 发明和实用新型专利权被授予后，除本法另有规定的以外，任何单位或者个人未经专利权人许可，都不得实施其专利，即不得为生产经营目的制造、使用、许诺销售、销售、进口其专利产品，或者使用其专利方法以及使用、许诺销售、销售、进口依照该专利方法直接获得的产品。
>
> 外观设计专利权被授予后，任何单位或者个人未经专利权人许可，都不得实施其专利，即不得为生产经营目的制造、许诺销售、销售、进口其外观设计专利产品。

### 1.2 核心要素解析

依据上述规定，专利侵权的构成需要满足以下要件：

1. **有效的专利权**：侵权指控的前提是存在一项有效的专利
2. **未经专利权人许可**：行为人未获得专利权人的授权
3. **以生产经营为目的**：行为具有商业性质
4. **实施了专利**：从事了第11条所列明的一项或多项行为

### 1.3 "生产经营目的"的理解

"生产经营目的"是专利侵权判断的第一步。如果行为不以生产经营为目的，则不构成专利侵权，后续审查即无必要。

关于"生产经营目的"的详细讨论，参见[[权能-原理-生产经营目的]]。简言之：

- **典型情形**：为营利而制造、销售专利产品；在企业生产中使用专利方法等
- **非典型情形**：个人消费性使用、纯粹科学研究（特定条件下）等可能不构成"生产经营目的"

需要注意的是，"生产经营目的"的认定不应过于狭窄。即使行为主体是科研机构或非营利组织，如果其行为具有商业化意图或最终服务于商业目的，仍可能被认定为"生产经营目的"。

## 2. 侵权行为的分类

### 2.1 直接侵权与间接侵权

专利侵权行为从理论上分为两大类别：

**直接侵权行为**：行为人从事《专利法》第11条所禁止的直接侵害专利权的违法行为，比如制造、使用或销售专利产品。

**间接侵权行为**：行为人没有直接侵害专利权，但是诱使直接侵权行为发生，或者在明知或者应知的情况下为直接侵权行为提供实质性的帮助。

在司法实践中，法院偶尔也会直接确认间接侵权行为，但更多的时候是依据所谓的共同侵权的规则来处理间接侵权案件。

### 2.2 直接侵权的两种类型

专利直接侵权又可以分成两种类型：**字面侵权**和**等同侵权**。

**字面侵权（相同侵权）**：被控侵权的产品或方法直接落入该专利权利要求字面描述的范围。中国专利法接受字面侵权，没有争议。

**等同侵权**：被控侵权的产品或方法并没有落入该专利权利要求字面描述的范围，但是该产品或方法与权利要求所描述的技术方案实质等同。

中国专利法是否应该承认等同侵权，理论上存在一定的争论。因为等同侵权实际上是在一定程度上扩大了专利权的保护范围，同时也给专利权保护范围带来较大的不确定性。不过，在中国司法实践中，法院已经判决了大量的等同侵权的案例。对于等同侵权，后文将专章讨论。

## 3. 侵权判断的基本步骤

判断一项行为是否构成专利侵权，大致要经过下面的步骤：

1. **判断是否以生产经营为目的**：如果答案是否定的，则不构成侵权
2. **对权利要求进行解释**：明确专利权的保护范围
3. **对比被控侵权方案和权利要求**：看前者是否落入后者所主张的字面或等同范围
4. **审查法定抗辩或免责事由**：被控侵权者是否具有法定的抗辩或免责事由

上述步骤只是理论上的概括，并非法定要求。在实践中，如果被控侵权者有法定的抗辩事由，侵权审查可能直接终止，而无须考虑其是否以生产经营为目的，更无须进行复杂的权利要求解释和比对。

详细讨论参见[[侵权-原理-侵权判断步骤]]。

## 判断流程

```
专利侵权判断流程
├── 第一步：有效专利权存在？
│   └── 否 → 不构成侵权
├── 第二步：未经专利权人许可？
│   └── 否 → 不构成侵权
├── 第三步：以生产经营为目的？
│   └── 否 → 不构成侵权
├── 第四步：是否落入权利要求范围？
│   ├── 字面侵权？
│   └── 等同侵权？
│       └── 否 → 不构成侵权
└── 第五步：是否存在法定抗辩？
    ├── 是 → 不视为侵权/免责
    └── 否 → 构成侵权
```
