# 实用性-原理-再现性要求

> **来源：** 《专利法：原理与案例（第二版）》第3章 §2.3，《专利审查指南》第二部分第五章第3.2.1节
> **核心法条：** 《专利法》第22条第4款
> **关联页面：** [[实用性-原理-实用性概述]]、[[实用性-原理-审查程序与证明责任]]、[[实用性-原理-缺乏实用性的典型情形]]、[[实用性-原理-实用性审查的实务要点]]

---

## 核心要点

再现性是实用性的核心要件之一，要求所属技术领域的技术人员根据公开的技术内容能够重复实施技术方案，且实施结果相同。再现性与成品率低有本质区别：前者是在满足全部技术条件下仍无法重复实现结果，后者是能够重复实施但因技术条件控制不当导致成品率低。

---

## 1. 再现性的定义

### 1.1 法律要求

《专利审查指南》（2010）第二部分第五章第3.2.1节规定：

> 具有实用性的发明或者实用新型专利申请主题，应当具有再现性。反之，无再现性的发明或者实用新型专利申请主题不具备实用性。

### 1.2 再现性的含义

**再现性**，是指所属技术领域的技术人员，根据公开的技术内容，能够重复实施专利申请中为解决技术问题所采用的技术方案。

这种重复实施不得依赖任何随机的因素，并且实施结果应该是相同的。

### 1.3 与"能够制造或者使用"的关系

《专利审查指南》明确：

> 专利法第二十二条第四款所说的"能够制造或者使用"是指发明或者实用新型的技术方案具有在产业中被制造或使用的可能性。满足实用性要求的技术方案不能违背自然规律并且应当具有再现性。

再现性是"能够制造或者使用"的内在要求之一。如果一个技术方案在产业中无法被重复制造或使用，那么它就不具备在产业中制造或使用的可能性。

---

## 2. 再现性与成品率低的本质区别

### 2.1 核心区别

审查员应当注意，申请发明或者实用新型专利的产品的**成品率低**与**不具有再现性**是有本质区别的：

| 对比项 | 成品率低 | 无再现性 |
|-------|---------|---------|
| 能否重复实施 | 能够重复实施 | 无法重复实施 |
| 原因 | 实施过程中未能确保某些技术条件 | 技术方案本身固有的缺陷 |
| 举例 | 环境洁净度不够、温度控制不当 | 依赖随机因素、原理不可行 |
| 解决方式 | 改善工艺条件 | 无法通过改善条件解决 |
| 实用性判断 | 仍具备实用性 | 不具备实用性 |

### 2.2 成品率低的情形

成品率低是指：能够重复实施，只是由于实施过程中未能确保某些技术条件（例如环境洁净度、温度等）而导致成品率低。

**关键特征：**
- 技术方案本身是正确的
- 实施原理是可行的
- 问题在于工艺控制
- 通过改善条件可以提高成品率

### 2.3 无再现性的情形

无再现性是指：在确保发明或者实用新型专利申请所需全部技术条件下，所属技术领域的技术人员仍不可能重复实现该技术方案所要求达到的结果。

**关键特征：**
- 技术方案本身存在缺陷
- 实施原理可能不可行
- 依赖随机因素
- 无法通过改善条件解决

---

## 3. 再现性的审查标准

### 3.1 判断主体

再现性的判断主体是**所属技术领域的技术人员**。这一标准人员应当具备：

- 该领域的普通技术知识
- 常规实验手段
- 根据公开内容实施技术方案的能力

### 3.2 判断依据

再现性的判断依据是**公开的技术内容**，包括：

- 权利要求书
- 说明书
- 附图
- 实施例

### 3.3 判断标准

**正面标准：**
- 所属技术领域的技术人员能够重复实施
- 不依赖任何随机因素
- 实施结果相同

**反面标准：**
- 依赖随机因素
- 需要特殊技能或经验
- 结果不可预测
- 在全部技术条件下仍无法重复

### 3.4 与说明书公开程度的关系

《专利审查指南》特别指出：

> 因不能制造或者使用而不具备实用性是由技术方案本身固有的缺陷引起的，与说明书公开的程度无关。

这意味着：
- 如果技术方案本身具有再现性，即使说明书公开不够详细，也不应以此为由否定实用性（而应适用充分公开条款）
- 如果技术方案本身不具有再现性，即使说明书公开再详细，也不能挽救其实用性

## 4. 典型无再现性情形的分析

### 4.1 依赖随机因素的技术方案

如果技术方案的实施结果依赖于不可控的随机因素，则不具备再现性。

**典型情形：**
- 利用特定自然条件且不可复制的方案
- 依赖个体特殊反应的方法
- 未明确关键参数导致结果不确定的方案

### 4.2 违背自然规律的技术方案

违背自然规律的技术方案当然也不具备再现性，因为自然规律决定了其不可能实现。

**典型情形：**
- 永动机类发明
- 效率超过100%的能量转换装置
- 违背热力学定律的方案

### 4.3 利用独一无二的自然条件的产品

具备实用性的发明或者实用新型专利申请不得是由自然条件限定的独一无二的产品。利用特定的自然条件建造的自始至终都是不可移动的唯一产品不具备实用性。

**典型情形：**
- 利用各大洋之间海峡的大洋暖流循环工程
- 利用特定地质条件的建筑方案
- 依托特定气候条件的农业方法

**注意：** 不能因为上述利用独一无二的自然条件的产品不具备实用性，而认为其构件本身也不具备实用性。

### 4.4 人体或动物体的非治疗目的外科手术方法

非治疗目的的外科手术方法，由于是以有生命的人或者动物为实施对象，无法在产业上使用，因此不具备实用性。

**典型情形：**
- 为美容而实施的外科手术方法
- 采用外科手术从活牛身体上摘取牛黄的方法
- 为辅助诊断而采用的外科手术方法

### 4.5 测量人体或动物体在极限情况下的生理参数的方法

测量人体或动物体在极限情况下的生理参数需要将被测对象置于极限环境中，这会对人或动物的生命构成威胁，不同的人或动物个体可以耐受的极限条件是不同的，需要有经验的测试人员根据被测对象的情况来确定其耐受的极限条件，因此这类方法无法在产业上使用，不具备实用性。

**典型情形：**

（1）通过逐渐降低人或动物的体温，以测量人或动物对寒冷耐受程度的测量方法；

（2）利用降低吸入气体中氧气分压的方法逐级增加冠状动脉的负荷，并通过动脉血压的动态变化观察冠状动脉的代偿反应，以测量冠状动脉代谢机能的非侵入性的检查方法。

## 5. 实务要点

### 5.1 申请阶段的注意事项

**对于申请人：**

1. **充分公开实施条件：** 在说明书中详细记载实施技术方案所需的全部技术条件，包括温度、压力、时间、配比等关键参数。

2. **提供多个实施例：** 通过多个实施例证明技术方案的可重复性，避免仅依赖单一实施例。

3. **避免依赖随机因素：** 技术方案的设计应避免依赖不可控的随机因素。如果必须依赖某些变量，应明确其控制范围。

4. **区分成品率问题：** 如果存在成品率低的问题，应在申请文件中说明原因，并证明通过改善条件可以提高成品率。

### 5.2 审查阶段的判断要点

**对于审查员：**

1. **区分成品率低与无再现性：** 这是审查中最容易混淆的问题。应仔细分析申请人无法重复实施的原因。

2. **考虑全部技术条件：** 判断无再现性时，应假设在满足专利申请所需全部技术条件下进行判断。

3. **不混淆充分公开与实用性：** 再现性是由技术方案本身固有的缺陷引起的，与说明书公开的程度无关。

4. **参考公知常识：** 在判断技术方案是否具有再现性时，可以参考该领域的公知常识和常规技术手段。

### 5.3 无效宣告程序中的攻防策略

**对于请求人（主张不具备实用性）：**

1. 提供证据证明按照说明书公开的技术方案无法重复实施
2. 证明技术方案依赖随机因素
3. 证明在全部技术条件下仍无法达到预期结果

**对于专利权人（主张具备实用性）：**

1. 提供实验数据证明技术方案的可重复性
2. 解释成品率低的原因及改善措施
3. 区分技术方案本身的缺陷与实施条件的不足

## 判断流程

```
再现性审查流程
├── 第一步：技术方案是否依赖随机因素？
│   ├── 是 → 不具备再现性 ✗
│   └── 否 → 继续
├── 第二步：所属领域技术人员能否重复实施？
│   ├── 否 → 不具备再现性 ✗
│   └── 是 → 继续
├── 第三步：实施结果是否相同？
│   ├── 否（结果不可预测） → 不具备再现性 ✗
│   └── 是 → 继续
├── 第四步：是否为成品率低？
│   ├── 是（能改善） → 具备再现性 ✓
│   └── 否 → 继续
└── 结论
    ├── 具备再现性 → 继续审查其他实用性要件
    └── 不具备再现性 → 不具备实用性，驳回/无效
```
