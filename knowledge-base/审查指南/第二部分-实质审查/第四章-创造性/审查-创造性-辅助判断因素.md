# 审查-创造性-辅助判断因素

> **来源：** 《专利审查指南》第二部分第四章第3.2.2节、第5节、第6.1-6.3节
> **对应法条：** 《专利法》第22条第3款
> **最新修订：** 本节内容自2010版发布以来未作修改。
> **关联页面：** [[创造性-辅助判断因素]]

## 核心要点

本页汇总创造性判断中的辅助因素和审查注意事项。显著的进步以有益的技术效果为核心判断标准；当发明属于解决长期技术难题、克服技术偏见、取得预料不到的效果或在商业上获得成功等情形时，审查员不应轻易否定创造性。审查时应避免"事后诸葛亮"，创立发明的途径（艰辛或偶然）不影响创造性评价。

---

## 审查标准

### 一、显著的进步的判断（§3.2.2）

在评价发明是否具有显著的进步时，主要应当考虑发明是否具有**有益的技术效果**。以下情况，通常应当认为发明具有有益的技术效果，具有显著的进步：

1. **效果改善型：** 发明与现有技术相比具有更好的技术效果，例如质量改善、产量提高、节约能源、防治环境污染等。
2. **异途同归型：** 发明提供了一种技术构思不同的技术方案，其技术效果能够基本上达到现有技术的水平。
3. **趋势引领型：** 发明代表某种新技术发展趋势。
4. **利弊权衡型：** 尽管发明在某些方面有负面效果，但在其他方面具有明显积极的技术效果。

### 二、判断创造性时需考虑的其他因素（§5）

发明是否具备创造性，通常应当根据"三步法"等审查基准进行审查。但当申请属于以下情形时，审查员应当予以考虑，**不应轻易作出发明不具备创造性的结论**。

#### 5.1 发明解决了人们一直渴望解决但始终未能获得成功的技术难题

如果发明解决了人们一直渴望解决但始终未能获得成功的技术难题，这种发明具有突出的实质性特点和显著的进步，具备创造性。

**【案例】** 自有农场以来，人们一直期望解决在农场牲畜（如奶牛）身上无痛而且不损坏牲畜表皮地打上永久性标记的技术问题。某发明人基于冷冻能使牲畜表皮着色这一发现而发明的一项冷冻"烙印"的方法成功解决了该问题，该发明具备创造性。

#### 5.2 发明克服了技术偏见

**技术偏见**是指在某段时间内、某个技术领域中，技术人员对某个技术问题普遍存在的、偏离客观事实的认识，它引导人们不去考虑其他方面的可能性，阻碍人们对该技术领域的研究和开发。如果发明克服了这种技术偏见，采用了人们由于技术偏见而舍弃的技术手段，从而解决了技术问题，则该发明具备创造性。

**【案例】** 对于电动机的换向器与电刷间界面，通常认为越光滑接触越好，电流损耗也越小。一项发明将换向器表面制出一定粗糙度的细纹，结果电流损耗更小，优于光滑表面。该发明克服了技术偏见，具备创造性。

#### 5.3 发明取得了预料不到的技术效果

发明取得了预料不到的技术效果，是指发明同现有技术相比，其技术效果产生**"质"的变化**（具有新的性能），或者产生**"量"的变化**（超出人们预期的想象）。这种变化对所属技术领域的技术人员来说，事先无法预测或者推理出来。

当发明产生了预料不到的技术效果时，一方面说明发明具有显著的进步，同时也反映出发明的技术方案是非显而易见的，具有突出的实质性特点，该发明具备创造性。

#### 5.4 发明在商业上获得成功

当发明的产品在商业上获得成功时，如果这种成功是由于**发明的技术特征直接导致**的，则反映发明具有有益效果且非显而易见，具备创造性。但是，如果商业上的成功是由于其他原因所致（如销售技术的改进或广告宣传），则不能作为判断创造性的依据。

### 三、审查创造性时应当注意的问题（§6）

#### 6.1 创立发明的途径

不管发明者在创立发明的过程中是历尽艰辛，还是唾手而得，都**不应当影响对该发明创造性的评价**。绝大多数发明是发明者创造性劳动的结晶，是长期科学研究或者生产实践的总结，但也有一部分发明是偶然作出的。

**【案例】** 公知的汽车轮胎具有很好的强度和耐磨性能，它曾经是由于一名工匠在准备黑色橡胶配料时，把决定加入3%的碳黑错用为30%而造成的。事实证明，加入30%碳黑生产出来的橡胶具有原先不曾预料到的高强度和耐磨性能，尽管它是由于操作者偶然的疏忽而造成的，但不影响该发明具备创造性。

#### 6.2 避免"事后诸葛亮"

审查发明的创造性时，由于审查员是在了解了发明内容之后才作出判断，因而容易对发明的创造性估计偏低，从而犯"事后诸葛亮"的错误。审查员应当牢牢记住，对发明的创造性评价是由发明所属技术领域的**技术人员依据申请日以前的现有技术与发明进行比较**而作出的，以减少和避免主观因素的影响。

> **【审查员视角】**
> "事后诸葛亮"是审查员最难自我觉察的系统性偏差。其典型表现包括：（1）在已知发明方案后，觉得"这很简单，我也会做"，从而低估发明的创造性高度；（2）在检索到多篇对比文件后，将各文件中的碎片信息拼接起来，重构出一条看似合理的"技术演进路径"，然后断言发明只是该路径上的自然延伸；（3）面对长期未解决的技术难题，在发明提出后觉得"道理 obvious"，却忽视了此前整个行业都未能突破的事实。代理人的有效抗辩策略是：在意见陈述中引入"时间回溯法"——将审查员的推理链条置于申请日（优先权日）之前的时空背景下，追问"当时的技术人员面对的是什么问题""现有技术给出了什么教导""当时的行业共识是什么"。如果发明解决了长期渴望解决但始终未能成功的技术难题，或克服了根深蒂固的技术偏见，这本身就是对"事后诸葛亮"最强有力的反驳。此外，代理人在答复时应主动援引§5.1-§5.4的辅助判断因素，不要等待审查员自行考虑——实践中审查员往往机械地走完三步法就得出结论，对辅助因素的审查流于形式。

#### 6.3 对预料不到的技术效果的考虑

在创造性的判断过程中，考虑发明的技术效果有利于正确评价发明的创造性：

- 如果发明与现有技术相比具有**预料不到的技术效果**，则不必再怀疑其技术方案是否具有突出的实质性特点，可以确定发明具备创造性。
- 但是，如果通过"三步法"等方法可以判断出发明的技术方案对本领域的技术人员来说是**非显而易见的**，且能够产生有益的技术效果，则发明具备创造性，**此种情况不应强调发明是否具有预料不到的技术效果**。

## 实务应对

**代理人实务要点：**

1. **主动主张辅助因素：** 在答复审查意见时，除了常规的"三步法"反驳外，应主动审视发明是否落入上述四种辅助判断因素（长期技术难题、技术偏见、预料不到的效果、商业成功）中的任何一种，作为补充论据。
2. **预料不到的效果需要证据支撑：** 仅在说明书中泛泛而谈"取得了预料不到的效果"往往不够。应提供对比实验数据、定量分析或本领域技术人员的常识性预期差距作为佐证。如果说明书中未记载，可在答复审查意见时补充实验数据（需注意补充数据的可接受性）。
3. **商业成功论证需谨慎：** 以商业成功作为创造性论据时，必须建立技术特征与商业成功之间的因果链。准备市场调研报告、用户反馈、销售数据对比等材料，排除营销因素。
4. **克服技术偏见的举证策略：** 需要证明（1）偏见确实存在（引用教科书、技术手册、行业共识等文献）；（2）本发明采用了被偏见排除的技术路线；（3）取得了积极效果。在撰写申请文件时即应考虑收录反映技术偏见的背景技术描述。
5. **偶然发明的创造性保护：** 发明的偶然性不影响创造性，但说明书中应完整记录发现过程和技术效果的验证，避免被认定为"显而易见的试错"。

| 审查意见常见表述 | 代理人答复策略 | 援引依据 |
|-----------------|--------------|---------|
| "发明未取得预料不到的技术效果，因而不具备创造性" | 指出预料不到的效果是辅助因素而非必要条件；若三步法已认定非显而易见且有有益效果，则具备创造性 | 第二部分第四章§6.3 |
| "发明仅取得常规技术效果，未克服技术偏见" | 提供本领域技术文献证明偏见存在；说明发明如何突破偏见并取得效果 | 第二部分第四章§5.2 |
| "商业成功与发明技术特征无关" | 建立技术特征→产品性能→市场认可→商业成功的完整因果链；提供第三方数据 | 第二部分第四章§5.4 |
| "发明是常规技术发展的必然结果" | 若发明代表了新技术发展趋势，可直接主张其具备显著的进步 | 第二部分第四章§3.2.2(3) |
| "审查员已通过三步法否定创造性，辅助因素无需考虑" | 明确指出指南§5规定'当申请属于以下情形时，审查员应当予以考虑，不应轻易作出发明不具备创造性的结论' | 第二部分第四章§5 |

### 具体争辩话术示例

**话术一：援引辅助判断因素作为补充论据**
> "即使审查员通过三步法认为发明不具备突出的实质性特点，本发明仍属于《专利审查指南》第二部分第四章第5节规定的应当予以考虑的情形。[选择适用：§5.1解决了长期渴望解决但始终未能成功的技术难题/§5.2克服了技术偏见/§5.3取得了预料不到的技术效果/§5.4在商业上获得成功]。根据指南的明确规定，'当申请属于以下情形时，审查员应当予以考虑，不应轻易作出发明不具备创造性的结论'。审查意见未对上述辅助因素进行充分考量，请求审查员予以补充审查。"

**话术二：预料不到的技术效果的举证框架**
> "本发明取得了预料不到的技术效果，具体表现为：（1）[质变：具有新的性能]或[量变的量化数据，超出本领域技术人员合理预期的范围]；（2）该效果在申请日（优先权日）之前无法从现有技术中预测或推理出来；（3）提供了对比实验数据作为证据[附上数据表]。根据《专利审查指南》第二部分第四章第5.3节的规定，'当发明产生了预料不到的技术效果时，一方面说明发明具有显著的进步，同时也反映出发明的技术方案是非显而易见的，具有突出的实质性特点'。"

**话术三：区分"辅助因素"与"必要条件"**
> "审查意见认为发明未产生预料不到的技术效果，因而不具备创造性。该观点混淆了'预料不到的技术效果'与'创造性'的关系。根据《专利审查指南》第二部分第四章第6.3节的规定，'如果通过'三步法'等方法可以判断出发明的技术方案对本领域的技术人员来说是非显而易见的，且能够产生有益的技术效果，则发明具备创造性，此种情况不应强调发明是否具有预料不到的技术效果'。预料不到的技术效果是辅助判断因素，而非创造性的必要条件。"

```

[针对审查意见的三步法分析进行常规反驳，此处略]

二、关于辅助判断因素的补充论证

退一步讲，即使审查员对三步法的分析成立，本发明仍属于《专利审查指南》第二部分第四章第5节规定的不应轻易否定创造性的情形，具体论证如下：

（一）[选择一项或多项适用]

□ §5.3 预料不到的技术效果：本发明相对于现有技术产生了[质变/量变]，具体数据为[实验数据]，该效果超出了本领域技术人员的合理预期。
□ §5.4 商业成功：本发明的产品自[时间]上市以来，[销售数据/市场份额/用户反馈]，该商业成功是由发明的[具体技术特征]直接导致的，而非营销因素。

（二）法律依据
根据《专利审查指南》第二部分第四章第5节的规定，'当申请属于以下情形时，审查员应当予以考虑，不应轻易作出发明不具备创造性的结论'。审查意见未对上述辅助因素进行审查，请求予以补充考虑。

三、结论

综上所述，无论从三步法分析还是辅助判断因素的角度，本发明均具备突出的实质性特点和显著的进步，符合专利法第22条第3款的规定。
```

## 审查要点速查

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 发明解决了长期渴望解决但未获成功的技术难题 | 具备创造性 | §5.1，无需进一步论证显而易见性 |
| 发明克服了技术偏见，采用被舍弃的技术手段 | 具备创造性 | §5.2，需证明偏见确实存在且被克服 |
| 发明取得预料不到的技术效果（质变或量变） | 具备创造性 | §5.3，效果需超出技术人员合理预期 |
| 商业成功且由技术特征直接导致 | 可辅助论证创造性 | §5.4，需排除非技术因素 |
| 商业成功仅因营销手段所致 | 不能作为创造性依据 | §5.4 |
| 发明系偶然发现（如操作失误） | 不影响创造性评价 | §6.1，创立途径不影响判断 |
| 审查员"事后诸葛亮"式否定 | 应予纠正 | §6.2，应基于申请日前的现有技术判断 |
| 通过三步法已认定非显而易见且有有益效果 | 具备创造性，不要求预料不到的效果 | §6.3 |
