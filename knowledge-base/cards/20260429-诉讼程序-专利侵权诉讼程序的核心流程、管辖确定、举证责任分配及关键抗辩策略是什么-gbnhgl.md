# 专利侵权诉讼程序的核心流程、管辖确定、举证责任分配及关键抗辩策略是什么？

## 卡片内容

### 核心回答
专利侵权诉讼程序是一个多阶段、复杂的法律过程，涉及从管辖确定到证据收集、抗辩策略（如无效宣告）以及诉讼时效等多个环节。专利权人需要证明有效的专利权和侵权事实，而被控侵权人则可以通过各种方式进行防御。

### 诉讼管辖
1. **地域管辖**：专利侵权案件的地域管辖遵循一般民事侵权案件的原则，主要由**被告所在地**或**侵权行为地**的法院管辖。侵权行为地特指制造、使用、销售等侵权行为的实施地。如果权利人同时起诉制造者和销售者，可通过选择销售地来确定管辖法院。
2. **确认不侵权之诉**：确认不侵犯专利权诉讼也应依照《民事诉讼法》第29条确定地域管辖，即由侵权行为地或被告所在地法院受理。
3. **管辖冲突处理**：为避免同一事实在不同法院重复审理，涉及同一事实的确认不侵权诉讼和专利侵权诉讼，若由不同法院提起，应依法移送管辖合并审理，后受理的法院向先受理的法院移送。
4. **级别管辖**：专利纠纷第一审案件，由各省、自治区、直辖市人民政府所在地的中级人民法院和最高人民法院指定的中级人民法院管辖。

### 举证责任
1. **一般规则**：专利侵权诉讼的举证责任遵循“谁主张，谁举证”的一般原则（依据《民事诉讼法》第64条），专利权人需证明两项主要内容：**有效的专利权**和**侵权事实**。
2. **专利权效力的证明**：对于发明专利，专利登记簿副本或专利权利证书即是有效权利的证明。在被控侵权人启动无效宣告程序前，法院应承认专利权效力；在被控侵权人启动无效宣告程序后，法院仍可不中止侵权诉讼。
3. **新产品制造方法专利的举证责任倒置**：对于新产品制造方法专利，被告（制造同样产品的单位或个人）有义务提供其产品制造方法不同于专利方法的证明（依据《专利法》第61条第1款）。若权利人证明了有效专利权、产品是新产品，且被告有制造该产品，法院可推定被告使用了专利方法，要求被告举证推翻此推定。
4. **商业秘密保护**：在涉及新产品制造方法专利案件中，被告提交的证据可能涉及商业秘密。法院应采取适当措施保护被告的商业秘密（如不公开审理、限制参与人范围），但证据的质证仍需依法进行，不能以涉及商业秘密为由拒绝质证。

### 陷阱取证
专利权人通过“陷阱取证”引诱他人陷入侵权状态，法院认为这不被禁止，但需区分：
* **有效陷阱取证**：仅为核实被控方是否具备侵权生产设备或能力，不构成许可。
* **无效陷阱取证**：主动提供技术方案（如图纸）诱导被控方生产特定产品，且该产品与被控方原有生产无关，则可能被认定为不构成侵权。

### 诉讼中止
被告常以请求专利复审委宣告专利无效为由，请求法院中止侵权诉讼。法院对中止的裁定有条件：
* **可中止情形**：如果所涉专利是发明专利，或复审委审查维持有效的实用新型/外观设计专利，法院可以不中止诉讼。
* **通常应中止情形**：如果所涉专利是未经复审的实用新型或外观设计专利，法院通常应当中止诉讼，除非存在以下例外情形（如权利人未发现无效事由、被告证据不足、被告请求的无效理由明显不充分等）。

### 诉讼时效
1. **一般侵权时效**：专利侵权的诉讼时效为两年，自专利权人或利害关系人得知或应当得知侵权行为之日起计算。
2. **持续侵权处理**：即使最初的侵权行为已超逾诉讼时效，如果侵权行为在起诉之日仍在继续，权利人仍可从起诉之日往前推算两年追究责任。
3. **发明专利临时保护期**：对于发明专利，自专利申请公布至专利权授予前使用该发明而未支付适当使用费，权利人要求支付使用费的诉讼时效为两年，但若权利人于专利权授予日前已得知或应当得知，则自专利权授予之日起计算。
