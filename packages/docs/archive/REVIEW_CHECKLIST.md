## 🔍 审查方法

### 自动化检查

```bash
# 1. 运行文档同步检查
node scripts/check-doc-sync.js

# 2. 检查文档链接
npm run check-links

# 3. 检查文档语法
npm run lint-docs
```

### 手动检查

1. **阅读文档**
   - 从用户角度阅读文档
   - 尝试按照文档操作
   - 记录不清晰的地方

2. **对比代码**
   - 打开相关源代码
   - 对比文档中的描述
   - 验证接口定义

3. **运行示例**
   - 复制示例代码
   - 在实际环境中运行
   - 验证输出结果
