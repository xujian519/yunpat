# 审查-发明初审-明显实质性缺陷

> **来源：** 《专利审查指南》第一部分第一章 §7-8
> **对应法条：** 《专利法》第2条第2款、第5条、第19条、第25条、第31条第1款、第33条 / 《专利法实施细则》第11条、第20条、第22条
> **最新修订：** 2023年修订（新增细则第11条诚实信用原则审查）
> **关联页面：** [[审查-发明初审-申请文件审查]]、[[程序-原理-初步审查]]

---

## 核心要点

发明专利申请的明显实质性缺陷审查覆盖七类情形：不符合发明定义（法第2条第2款）、违反法律和社会公德（法第5条）、向外申请未保密审查（法第19条）、不属于授权客体（法第25条）、缺乏单一性（法第31条）、修改超范围（法第33条）、说明书和权利要求书的撰写缺陷（细则第20条、第22条）。审查员仅在缺陷"明显"且影响公布时才予以处理，其余留待实质审查。

---

## 审查标准

### 一、审查定位

明显实质性缺陷审查与实质审查有本质区别：
- **初步审查**仅关注"明显"存在的缺陷，即不经过深入检索和分析即可发现的缺陷。
- 缺陷只有当其"明显存在并影响公布时"，才需指出和处理。
- 部分问题可留待实质审查时处理。

### 二、根据专利法第二条第二款的审查（§7.1）

**法条依据**：专利法第二条第二款——发明是指对产品、方法或者其改进所提出的新的技术方案。

**审查标准**：
- 申请文件描述了"发明"的部分技术特征的，审查员可以不判断技术方案是否完整，也可以不判断能否实施。
- 但申请文件**仅描述了某些技术指标、优点和效果**，而对解决技术问题的技术方案未作任何描述，甚至未描述任何技术内容的，应当发出审查意见通知书。
- 申请人未在指定期限内答复的，发出视为撤回通知书；陈述意见或补正后仍不符合规定的，可以作出驳回决定。

### 三、根据专利法第五条的审查（§7.2）

**法条依据**：专利法第五条——对违反法律、社会公德或者妨害公共利益的发明创造，以及违反法律、行政法规的规定获取或者利用遗传资源并依赖该遗传资源完成的发明创造，不授予专利权。

**审查内容**：
1. **是否明显违反法律**：参照指南第二部分第一章第3节。
2. **是否明显违反社会公德**。
3. **是否明显妨害公共利益**。
4. **遗传资源的获取或利用是否明显违反法律、行政法规**。

**处理方式**：审查员认为属于上述情形之一的（如"一种吸毒工具""一种赌博工具及其使用方法"），应当发出审查意见通知书，说明理由，通知申请人陈述意见或删除相应部分。申请人按审查员意见删除相应部分的，为使上下文达到文字连贯性而增加必要的文字应当允许。

**注意**：违反专利法第五条的发明创造，**不包括仅其实施为法律所禁止的发明创造**（细则第10条）。

### 四、根据专利法第十九条第一款的审查（§7.3）

**法条依据**：专利法第十九条第一款——申请人将在中国完成的发明向外国申请专利的，应当事先报经专利局进行保密审查。

**"在中国完成的发明"**：技术方案的实质性内容在中国境内完成的发明。

**审查标准**：审查员有理由认为申请人违反上述规定向外国申请专利的，对于其在国内就相同的发明提出的专利申请，应当发出审查意见通知书。申请人陈述理由不足以说明不属于上述情形的，可以以不符合专利法第十九条第一款为理由驳回。

### 五、根据专利法第二十五条的审查（§7.4）

**法条依据**：专利法第二十五条——下列各项不授予专利权：
1. 科学发现
2. 智力活动的规则和方法
3. 疾病的诊断和治疗方法
4. 动物和植物品种
5. 原子核变换方法以及用原子核变换方法获得的物质

**审查标准**：
- 参照指南第二部分第一章第4节的规定。
- 审查员认为专利申请的**全部内容**属于上述情形之一的（如"一颗新发现的小行星""一种人体疾病的诊断方法"），应当发出审查意见通知书。
- 审查员认为专利申请的**部分内容**属于上述情形之一，而又难以从申请中分割出来时，在初步审查中**可不作处理**，留待实质审查时处理。

### 六、根据专利法第三十一条第一款的审查（§7.5）

**法条依据**：专利法第三十一条第一款——一件发明专利申请应当限于一项发明，属于一个总的发明构思的两项以上的发明，可以作为一件申请提出。

**审查标准**：
- 初步审查中，**只有当**一件专利申请包含了两项以上完全不相关联的发明时，审查员才需发出审查意见通知书。
- 申请人无正当理由而又拒绝对其申请进行修改的，审查员可以作出驳回决定。

### 七、根据专利法第三十三条的审查（§7.6）

**法条依据**：专利法第三十三条——对专利申请文件的修改不得超出原说明书和权利要求书记载的范围。

**初步审查中的审查范围**：
- 只有当审查员发出了审查意见通知书要求申请人修改时，才需对修改是否明显超范围进行审查。
- **修改明显超范围的情形**：
  - 修改了数据或扩大了数值范围；
  - 增加了原说明书中没有相应文字记载的技术方案的权利要求；
  - 增加原说明书或权利要求中没有记载的发明的实质内容。

**主动修改的处理**：
- 申请人根据细则第57条提出主动修改文本的，审查员仅需核实提出时机是否符合规定。
- 符合规定的，作出合格的处理意见后存档；不符合规定的，作出供实质审查参考的处理意见后存档。
- 对主动修改文本的**内容不进行审查**，留待实质审查时处理。

### 八、根据专利法实施细则第二十条的审查（§7.7）

**审查内容**：
- 说明书中不得使用与技术无关的词句、商业性宣传用语以及贬低或诽谤他人或他人产品的词句。
- 客观地指出背景技术所存在的技术问题不应当认为是贬低行为。
- 说明书中应当记载发明的技术内容。

**审查界限**：只要说明书中描述了发明的部分技术特征，并且形式上符合§4.2的规定，对其他实质性问题不必审查，留待实质审查时处理。

### 九、根据专利法实施细则第二十二条的审查（§7.8）

**审查内容**：
- 权利要求书应当记载发明的技术特征。
- 不得使用与技术方案内容无关的词句（如"请求保护该专利的生产、销售权"）。
- 不得使用商业性宣传用语，不得使用贬低他人或他人产品的词句。

### 十、根据专利法实施细则第十一条的审查（§7.9）

对发明专利申请是否符合专利法实施细则第十一条（诚实信用原则）规定的审查，适用《规范申请专利行为的规定》。

### 十一、依职权修改（§8）

审查员可以在初步审查合格之前依职权修改申请文件中文字和符号的明显错误，并通知申请人。

**常见情形**：
1. **请求书**：修改地址中漏写、错写或重复填写的省/市/邮政编码等信息。
2. **权利要求书和说明书**：改正明显的文字错误和标点符号错误，修改明显的文本编辑错误，删除明显多余的信息。但可能导致原始申请文件记载范围发生变化的修改，不属于依职权修改的范围。
3. **摘要**：添加明显遗漏的内容，改正明显的文字错误，删除明显多余的信息，指定摘要附图。

## 实务应对

### 代理人视角：避免明显实质性缺陷

1. **确保说明书有技术方案**：不要仅写技术效果和指标，必须至少描述一个技术方案的基本框架，否则可能被以法第2条第2款驳回。

2. **避免法第5条风险**：申请名称和内容不得涉及赌博工具、吸毒器具、违反社会公德的内容。如审查员要求删除，配合删除即可，为上下文连贯增加必要文字是允许的。

3. **保密审查（法第19条）**：如涉及在中国完成的发明向外国申请，必须先经保密审查。初步审查中审查员如有理由认为违反此规定，可在国内申请中驳回。

4. **法第25条排除客体**：科学发现、智力活动规则和方法、疾病诊断和治疗方法等不属于授权客体。但部分内容难以分割的，初步审查中可不作处理。

5. **修改超范围**：答复审查意见通知书的修改必须严格在原说明书和权利要求书记载范围内。主动修改（申请日起两个月内）的内容在初步审查中不审查，留待实质审查。

6. **善用依职权修改**：明显的文字错误、标点错误等，审查员可以依职权修改。代理人可在提交时主动修正，减少补正通知。

## 审查要点速查

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 说明书仅描述技术效果而未描述技术方案 | 发出审查意见通知书 | 法第2条第2款 |
| 申请涉及赌博工具、吸毒器具 | 发出审查意见通知书，通知删除或陈述意见 | 法第5条 |
| 在中国完成的发明向外国申请未做保密审查 | 可以驳回 | 法第19条 |
| 申请明显属于科学发现或疾病诊断方法 | 发出审查意见通知书 | 法第25条 |
| 包含两项以上完全不相关的发明 | 发出审查意见通知书，通知修改 | 法第31条第1款 |
| 修改明显超出原说明书和权利要求书记载范围 | 发出审查意见通知书 | 法第33条 |
| 说明书含商业性宣传用语或贬低他人词句 | 发出审查意见通知书 | 细则第20条 |
| 权利要求含与技术无关的词句 | 发出审查意见通知书 | 细则第22条 |
| 涉及虚假发明人等违反诚实信用原则的行为 | 适用《规范申请专利行为的规定》 | 细则第11条 |
