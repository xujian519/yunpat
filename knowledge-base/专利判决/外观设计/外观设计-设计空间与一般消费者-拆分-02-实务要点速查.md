# 实务要点速查

> **拆分自：** [[外观设计-设计空间与一般消费者]]

---

## 实务要点速查

| 情形 | 裁判倾向 | 关键依据 | 代表案例 |
|------|---------|---------|---------|
| 主张设计空间小但未提交充分现有设计证据 | 不予支持 | 司法解释（一）第14条 | （2024）最高法知行终1151号 |
| 产品受功能限制基本构成近似，但各部分仍有设计变化余地 | 认定存在一定设计空间 | 整体观察、综合判断 | （2025）最高法知行终64号 |
| 设计空间基于模仿的其他品类产品（如传统打字机）主张 | 不予支持，应基于产品本身 | 司法解释（一）第14条 | （2024）最高法知行终1275号 |
| 国家标准对产品部分结构有要求，但未限定整体外形 | 认定具有较大设计空间 | 产品功能、技术标准因素 | （2024）最高法知行终737号 |
| 一般消费者为专业从业人员（如建筑从业者） | 即使限定为专业人员，细微差别仍不足以产生明显区别的判断 | 一般消费者知识水平和认知能力 | （2024）最高法知行终737号 |
| 产品各部分设计存在多种替代方案（如按摩器按摩头形状、手柄造型等） | 认定设计空间较大 | 现有设计的整体状况 | （2019）最高法民再142号 |
| 设计空间较大，被诉侵权设计与专利设计仅有局部细微差别 | 认定近似 | 司法解释（一）第14条 | （2019）最高法民再142号 |
| 区别点位于使用时不容易观察到的部位（如产品背面、底面） | 认定对整体视觉效果影响较小 | 整体观察、综合判断 | （2024）最高法知行终1275号 |
| 兼具功能性和装饰性的设计特征 | 不能因功能性否定对视觉效果的影响 | 外观设计定义 | （2020）最高法知行终628号 |
| 结构复杂的日用产品（如办公座椅） | 通常认定设计空间较大 | 现有设计的整体状况 | （2023）最高法知行终1289号 |
| 专利权评价报告与无效决定结论冲突 | 评价报告无拘束力 | 专利法第66条第2款 | （2024）最高法知行终1275号 |
