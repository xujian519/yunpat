# 审查-单一性-审查示例

> **来源：** 《专利审查指南》第二部分第六章 §2.2.2 单一性审查的方法和举例
> **对应法条：** 《专利法》第31条第1款；《专利法实施细则》第39条
> **最新修订：** 无修改
> **关联页面：** [[审查-单一性-基本概念与审查原则]]、[[审查-单一性-特定技术特征]]、[[单一性-特定技术特征与特定领域判断]]

---

## 核心要点

审查指南通过13个典型案例详细说明了同类独立权利要求（例1-7）、不同类独立权利要求（例8-13）以及从属权利要求的单一性判断方法。核心规则是：**相同的特定技术特征使权利要求之间具有单一性；不相关联的特定技术特征导致缺乏单一性**。当第一独立权利要求被否定后，需重新判断其余权利要求之间的单一性。

---

## 审查标准

### 一、同类独立权利要求的单一性（§2.2.2.1）

#### 例1：传送带（相同特征分析）

| 权利要求 | 技术特征 |
|---------|---------|
| 权利要求1 | 传送带X，特征为A |
| 权利要求2 | 传送带Y，特征为B |
| 权利要求3 | 传送带Z，特征为A和B |

**现有技术：** 没有公开具有特征A或B的传送带，A与B不相关。

**分析：**
- 权利要求1和2：没有相同或相应的技术特征 → **无单一性**
- 权利要求1和3：共享特定技术特征A → **有单一性**
- 权利要求2和3：共享特定技术特征B → **有单一性**

#### 例2：发射器与接收器（相应特征分析）

| 权利要求 | 技术特征 |
|---------|---------|
| 权利要求1 | 发射器，视频信号的时轴扩展器 |
| 权利要求2 | 接收器，视频信号的时轴压缩器 |
| 权利要求3 | 传送视频信号的设备（包括发射器和接收器） |

**分析：**
- 权利要求1的时轴扩展器与权利要求2的时轴压缩器相互关联、不能分开使用，构成**相应的特定技术特征** → 权利要求1与2**有单一性**
- 权利要求3包含了1和2的特定技术特征 → 与1和2均有单一性

#### 例3：插头与插座（配套产品）

| 权利要求 | 技术特征 |
|---------|---------|
| 权利要求1 | 插头，特征为A |
| 权利要求2 | 插座，特征与A相应 |

**分析：** 插头和插座是相互关联且必须同时使用的两种产品，特征A与其相应特征构成**相应的特定技术特征** → **有单一性**。

#### 例4：控制电路（不相关特征）

| 权利要求 | 技术特征 |
|---------|---------|
| 权利要求1 | 直流电动机控制电路，特征A |
| 权利要求2 | 直流电动机控制电路，特征B |
| 权利要求3 | 设备，包括具有特征A的控制电路的直流电机 |
| 权利要求4 | 设备，包括具有特征B的控制电路的直流电机 |

**现有技术：** 特征A和B分别是体现发明贡献的技术特征，且A与B完全不相关。

**分析：**
- 权利要求1与3之间、权利要求2与4之间：有相同的特定技术特征 → **有单一性**
- 权利要求1与2或4之间、权利要求3与2或4之间：无相同或相应的特定技术特征 → **无单一性**

#### 例5：灯丝-灯泡-探照灯（产品层级）

| 权利要求 | 技术特征 |
|---------|---------|
| 权利要求1 | 灯丝A |
| 权利要求2 | 用灯丝A制成的灯泡B |
| 权利要求3 | 装有灯丝A灯泡B和旋转装置C的探照灯 |

**现有技术：** 灯丝A是新的并具有创造性。

**分析：** 三项权利要求具有**相同的特定技术特征**灯丝A → **有单一性**。

#### 例6：制造方法（产品作为特定技术特征）

| 权利要求 | 技术特征 |
|---------|---------|
| 权利要求1 | 制造产品A的方法B |
| 权利要求2 | 制造产品A的方法C |
| 权利要求3 | 制造产品A的方法D |

**现有技术：** 产品A是新的并具有创造性。

**分析：** 产品A是三项方法权利要求的**相同的特定技术特征** → **有单一性**。

**注意：** 如果产品A是已知的，则不能作为特定技术特征，需重新判断三项方法权利要求的单一性。

#### 例7：树脂组合物（表面相同但特定技术特征不同）

| 权利要求 | 技术特征 |
|---------|---------|
| 权利要求1 | 树脂组合物：树脂A + 填料B + 阻燃剂C |
| 权利要求2 | 树脂组合物：树脂A + 填料B + 抗静电剂D |

**现有技术：** A、B、C、D分别已知，AB组合不体现发明贡献；ABC组合形成性能良好的不易燃树脂组合物，ABD组合形成性能良好的防静电树脂组合物。

**分析：** 尽管两项权利要求都包括相同的特征A和B，但A、B及AB组合都不体现发明对现有技术的贡献。权利要求1的特定技术特征是**ABC组合**，权利要求2的特定技术特征是**ABD组合**，两者不相同也不相应 → **无单一性**。

### 二、不同类独立权利要求的单一性（§2.2.2.2）

#### 例8：化合物-方法-用途（两种情况对比）

| 权利要求 | 内容 |
|---------|------|
| 权利要求1 | 化合物X |
| 权利要求2 | 制备化合物X的方法 |
| 权利要求3 | 化合物X作为杀虫剂的应用 |

**第一种情况：化合物X具有新颖性和创造性**

化合物X是三项权利要求相同的特定技术特征 → 权利要求1至3**有单一性**。

**第二种情况：化合物X不具有新颖性或创造性**

权利要求1不能被授权。权利要求2和3之间的相同技术特征仍为化合物X，但化合物X对现有技术没有作出贡献，不是特定技术特征，且权利要求2和3之间也没有相应的特定技术特征 → 权利要求2和3**缺乏单一性**。

#### 例9：不锈钢带与制造方法

权利要求1为高强度耐腐蚀不锈钢带产品，权利要求2为其制造方法。

**分析：** 产品的特定技术特征是伸长率为0.2%时屈服强度超过50kg/mm2，方法中的工艺步骤正是为生产出具有该屈服强度的不锈钢带而采用，虽然权利要求2的文字未体现，但从说明书中可以清楚看出 → 工艺步骤与产品的强度特征构成**相应的特定技术特征** → **有单一性**。

#### 例10：涂料-喷涂方法-喷涂设备（三角关系）

| 权利要求 | 特定技术特征 |
|---------|-------------|
| 权利要求1（含物质X的涂料） | 含X的涂料 |
| 权利要求2（喷涂方法） | 含X的涂料 + 电极装置A |
| 权利要求3（喷涂设备） | 电极装置A |

**分析：**
- 权利要求1与2：含X的涂料是相同的特定技术特征 → **有单一性**
- 权利要求2与3：电极装置A是相同的特定技术特征 → **有单一性**
- 权利要求1与3：缺乏相同或相应的特定技术特征 → **缺乏单一性**

#### 例11：纺织材料处理方法-产品-设备

**分析：** 方法的特定技术特征是选用特殊涂料A的特定工艺条件，产品的特定技术特征是该方法处理得到的纺织材料。方法与产品具有相应的特定技术特征 → **有单一性**。喷涂机与方法和产品无相应的特定技术特征 → **无单一性**。

#### 例12：制造方法-设备（多个设备）

| 权利要求 | 内容 |
|---------|------|
| 权利要求1 | 制造方法，包括步骤A和B |
| 权利要求2 | 为实施步骤A专门设计的设备 |
| 权利要求3 | 为实施步骤B专门设计的设备 |

**分析：** 步骤A和B分别体现发明贡献。权利要求1与2之间、权利要求1与3之间有单一性。权利要求2与3之间不存在相同或相应的特定技术特征 → **无单一性**。

#### 例13：燃烧器综合示例

| 权利要求 | 内容 | 特定技术特征 |
|---------|------|-------------|
| 权利要求1 | 燃烧器（正切方向进料口） | 正切方向进料口 |
| 权利要求2 | 制造方法（正切方向进料口步骤） | 正切方向进料口 |
| 权利要求3 | 制造方法（浇铸工序） | 浇铸工序 |
| 权利要求4 | 制造设备（装置X） | 装置X（与正切进料口相应） |
| 权利要求5 | 制造设备（自动控制装置D） | 装置D |
| 权利要求6 | 用燃烧器制造碳黑的方法 | 正切方向进料口 |

**分析：** 权利要求1、2、4、6有单一性（都涉及正切方向进料口）。权利要求3、5与1/2/4/6之间以及权利要求3与5之间均**无单一性**。

### 三、从属权利要求的单一性（§2.2.2.3）

#### 3.1 一般原则

凡符合规定的从属权利要求，与其所引用的独立权利要求之间**不存在缺乏单一性的问题**，即使该从属权利要求还包含着另外的发明。

#### 3.2 形式从属实质独立

**例如：** 权利要求1为接触器，具有特征A、B和C；权利要求2为"权利要求1的接触器"，而其中特征C由特征D代替。由于权利要求2并没有包括权利要求1的全部特征（缺少C），因此不是从属权利要求，而是独立权利要求，应按同类独立权利要求的单一性审查原则来判断。

#### 3.3 独立权利要求被否定后从属权利要求的单一性

**例如：**
- 权利要求1：显示器，具有特征A和B；
- 权利要求2：权利要求1所述显示器，具有另一特征C；
- 权利要求3：权利要求1所述显示器，具有另一特征D。

**第一种情况：** 权利要求1具有新颖性和创造性 → 权利要求1、2、3具有单一性。

**第二种情况：** 权利要求1不具有创造性，特征C和D分别是对现有技术作出贡献的技术特征，且C和D完全不相关。权利要求2和3实际上应视为独立权利要求，C与D不相同也不相应 → 权利要求2和3**无单一性**。

## 实务应对

### 1. 从示例中学习撰写策略

代理人可参考审查示例来优化权利要求结构：
- **例5策略：** 如果核心发明是某个组件（如灯丝A），应将使用该组件的产品作为从属权利要求或并列独立权利要求，确保单一性；
- **例6策略：** 制造方法类申请中，应以新产品作为核心，多个方法共享产品这一特定技术特征；
- **例10策略：** 多项不同类权利要求之间的单一性呈"链式"关系（A-B有单一性，B-C有单一性，但A-C不一定有单一性），撰写时应确保所有需要保护的权利要求之间有直接的特定技术特征关联。

### 2. 避免例7中的"陷阱"

例7表明，即使权利要求之间存在大量共同技术特征，如果这些共同特征不构成"特定技术特征"（即不对现有技术作出贡献），仍然缺乏单一性。代理人应：
- 识别真正的核心创新点（特定技术特征）；
- 确保多项权利要求之间在核心创新点上有关联。

### 3. 从属权利要求的防护策略

为防止独立权利要求被否定后从属权利要求之间缺乏单一性，代理人应：
- 确保从属权利要求之间的区别特征与独立权利要求的核心特征相关联；
- 在从属权利要求中引入独立权利要求的特征，避免成为"实质上的独立权利要求"；
- 设置多层次的保护体系，减少单一性风险。

## 审查要点速查

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 共享核心创新特征的多项权利要求 | 有单一性 | 例1、5、6 |
| 相互配套的产品特征（插头-插座） | 有单一性 | 例3 相应特征 |
| 共同特征仅为已知技术（树脂组合物） | 无单一性 | 例7 特定技术特征不同 |
| 产品新颖时，产品+方法+用途 | 有单一性 | 例8 第一种情况 |
| 产品已知时，方法+用途之间 | 可能无单一性 | 例8 第二种情况 |
| A-B有单一性，B-C有单一性，A-C | 不一定有单一性 | 例10 链式关系 |
| 独立权利要求被否定后，从属权利要求 | 需重新判断 | 从属权利要求示例第二种情况 |
| 形式从属但实质独立 | 按独立权利要求判断 | 缺少全部特征的情形 |
