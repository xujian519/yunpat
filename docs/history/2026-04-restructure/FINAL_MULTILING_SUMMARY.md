## 🎯 实际应用场景

### 场景 1: Agent 执行流程

```
用户请求
  ↓
TypeScript AgentService (编排层)
  ↓ gRPC
Rust VectorService (核心引擎) - 向量检索
  ↓ gRPC
Python Tools (工具层) - ML 推理
  ↓
结果返回
```

### 场景 2: 高并发任务调度

```
Agent 请求
  ↓
TypeScript AgentService
  ↓ gRPC
Rust SchedulerService (核心引擎)
  - 优先级队列
  - 并发调度
  - 超时管理
  ↓
任务完成通知
```

### 场景 3: ML 工具调用

```
Agent 需要文本嵌入
  ↓ gRPC
Python Tools Service (隔离)
  - BERT 模型推理
  - 资源限制（2核、4GB）
  - 超时保护
  ↓
嵌入向量返回
```
