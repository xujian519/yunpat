# 专利无效宣告程序的核心流程、理由分类、请求人资格限制及与侵权诉讼的衔接机制是什么？

## 卡片内容

### 核心回答

专利无效宣告程序旨在为纠正错误专利授权设置的行政性纠正机制。任何认为专利权授予不符合专利法规定的单位或个人，均有权请求专利复审委员会宣告该专利权无效。该程序不仅涉及实体性要件的审查，也涵盖了程序性义务的核查。

### 1. 无效宣告制度概述

**制度目的：** 专利审查过程复杂，为避免审查错误，该程序提供了一种纠正机制。**启动方式：** 根据《专利法》第45条规定，自国务院专利行政部门公告授予专利权之日起，任何单位或个人均可请求专利复审委员会宣告该专利权无效。

### 2. 无效宣告的理由

无效宣告的理由主要分为实体性缺陷和程序性缺陷：

**实体性理由：**
*   **保护客体缺陷：** 专利不属于专利法意义上的发明（如违反第2条）、违法或违反公德（第5条第1款）、或属于法定排除客体（第25条）。
*   **三性缺陷：** 专利不具备实用性、新颖性或创造性（涉及第22条、第23条）。
*   **公开不充分：** 存在未充分公开的问题（如违反第26条第3款和第4款）。
*   **修改超范围：** 专利权利要求可能不符合专利法第33条的规定。

**程序性理由：**
*   违反保密审查规定（如第20条第1款）。
*   违反遗传资源披露义务（如第5条第2款和第26条第5款）。
*   重复授权问题（如违反第9条）。

### 3. 请求人资格限制

**一般规则：** 原则上，任何人均可提出无效宣告请求，旨在鼓励对不应获得授权的专利提出挑战。

**例外情形（不予受理）：** 《专利审查指南》对请求人资格设置了限制，包括但不限于：
1.  请求人不具备民事诉讼主体资格。
2.  以外观设计与他人先取得的合法权利冲突为由请求无效，但请求人无法证明其为先取得人或利害关系人。
3.  专利权人针对其专利权提出无效宣告请求且请求宣告全部无效，或提交的证据不是公开出版物，或请求人不是共有专利权所有人的全部权利人。
4.  多个请求人共同提出无效宣告请求（除非是所有专利权人针对其共有专利权提出的除外）。

**专利权人自己提出请求：** 许可权利人可以主动提出无效请求，目的是为专利权人提供机会，使其在无效程序中有机会修改其权利要求。

### 4. 无效程序的中止

**中止情形：** 在专利复审委员会的无效程序中，若涉及专利权属争议，根据原中国专利局第31号公告，应当先中止无效程序。

**滥用问题：** 实践中存在权利人利用“权属争议”为由，通过第三方提出争议，从而拖延专利被宣告无效的时间，导致专利权人救济受阻。如何有效防止权利人滥用中止程序仍是实践中的难题。

### 5. 专利无效诉讼

**诉讼当事人：**
*   专利复审委员会对无效宣告请求进行审查后，将作出宣告无效或维持专利效力的决定。
*   若请求人或专利权人对该决定不服，可在收到通知后3个月内向法院提起诉讼。
*   法院受理案件后，应当通知无效宣告请求程序的对方当事人作为第三人参加诉讼。

**诉讼结果处理：**
*   法院撤销专利复审委员会的决定时，复审委员会需重新作出决定。
*   法院不能直接替代复审委员会的审查决定。

### 6. 无效宣告与侵权程序的衔接

**侵权诉讼的中止：**
*   **发明专利或经复审委维持的实用新型/外观设计专利：** 法院可以不中止诉讼。
*   **普通的实用新型与外观设计专利：** 法院应当中止诉讼程序。

**程序分离的制度背景：** 依据中国专利法，专利权只能通过无效宣告程序来消灭专利权，法院不能在侵权诉讼中直接宣告专利权无效。这种安排的理论基础是行政行为公定力，但存在效率低下问题，可能导致判决侵权后专利被宣告无效，或侵权人拖延无效宣告程序，使专利权人救济受阻。
