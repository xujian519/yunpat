## 📊 关键指标

### 工具复用率

**计算公式**:

```
工具复用率 = (复用工具数 / 总工具数) × 100%
```

**目标**: ≥80%

**工具分类**:

- 直接复用：无需修改，直接使用
- 修改后复用：需要少量修改
- 不复用：功能不合适或质量问题

### 测试覆盖率

**计算公式**:

```
测试覆盖率 = (测试代码行数 / 总代码行数) × 100%
```

**目标**:

- 工具层单元测试覆盖率：≥90%
- 工具层集成测试覆盖率：≥90%

### 性能指标

**目标**:

- 工具响应时间：<2s
- 专利下载速度：≥1MB/s
- 并发支持：≥10个并发请求
