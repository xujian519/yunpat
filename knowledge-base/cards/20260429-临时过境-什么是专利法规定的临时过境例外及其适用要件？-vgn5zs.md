# 什么是专利法规定的临时过境例外及其适用要件？

- 来源问题: 什么是专利法规定的临时过境例外及其适用要件？
- 质量分: 0.9000
- 生成时间: 2026-04-29T10:29:21.823Z
- 概念: 临时过境
- 领域: 侵权抗辩
- 版本: 1

## 卡片内容

### 核心回答

临时过境例外规定，为外国运输工具临时通过中国领土时，若为运输工具自身需要而在其装置和设备中使用有关专利，则不视为侵犯专利权。该例外旨在便利国际交通，消除专利权对运输工具正常运营的障碍。

### 法律依据

根据《专利法》（2008）第69条第（三）项，明确了这一例外情形。该条款规定，临时通过中国领土的外国运输工具，若依据所属国与中国签订的协议、共同参加的国际公约，或依照互惠原则，为运输工具自身需要而在其装置和设备中使用有关专利，则不视为侵犯专利权。

### 立法目的

该临时过境例外主要源于《巴黎公约》第5条之3的规定。其立法目的在于为国际商业交往中频繁穿越过境的运输工具消除专利障碍，确保专利权不会阻碍运输工具的正常运行，从而保障国际运输的顺畅进行。

### 适用要件

为认定临时过境例外适用，需要满足以下核心要素：

**1. 临时性：** 运输工具的通过必须是临时的或偶然的。判断标准包括：
*   按照规定的航线进入境内作短暂停留后离开，通常应视为临时进入。
*   特定交通工具的频率或持续性不影响进入本身的临时性。
*   应考量运输工具控制者在入境时的主观意愿，即是否准备在合理的时间内离开。

**2. 外国运输工具：** 临时例外主要针对外国运输工具。在国内永久注册的运输工具，即使在境内外来往，也可能不享受此项例外。

**3. 依据协议、条约或互惠原则：** 适用该例外必须满足以下条件之一：
*   所属国与中国签订了相关协议。
*   双方共同参加了国际公约（如《巴黎公约》）。
*   依据互惠原则进行认定。

**4. 为运输工具自身需要：** 该要求对专利技术的使用范围进行了实质性限制。判断标准在于：
*   使用专利技术应当是维持运输工具正常运行所必需的。
*   不包括利用运输工具搭载的设备从事与运输无关的专利实施行为。
*   对于运输工具携带的作业工具，若临时进入的义务涉及承揽装卸货物的行为，可能存在争议，需判断是否属于“为运输工具自身需要”。

### 实务要点

**对中国专利权人：** 在司法实践中，临时过境例外非常罕见，专利权人通常无需过度担忧外国运输工具对其专利权的侵害。

**对国际运输企业：**
1. 确保运输工具在境外注册。
2. 在中国境内的停留应保持临时性。
3. 专利技术的使用应严格限于运输工具自身运行的需要。
4. 避免利用运输工具搭载的设备在中国境内从事与运输无关的专利实施活动。

### 判断流程

判断流程涉及主体审查、临时性审查、依据审查和目的审查四个主要环节。例如，在主体审查中，中国注册的运输工具不适用该例外；在临时性审查中，若不符合临时性要求，则不适用例外；在依据审查中，若不满足协议或公约等条件，则不适用例外；在目的审查中，若不符合“为运输工具自身需要”的要求，则不适用例外。
