# 创造性-预料不到的技术效果与实验数据

> **来源：** 《以案说法——专利复审、无效典型案例指引》第四章 §4.3（深入专题）
> **核心法条：** 《专利法》第22条第3款
> **关联页面：** [[创造性-辅助判断因素]]、[[创造性-创造性判断实务要点]]、[[创造性-分析推理与有限试验]]、[[创造性-技术启示的判断]]、[[创造性-概述与三步法框架]]、[[审查-创造性-辅助判断因素]]

---

## 核心要点

预料不到的技术效果的判断核心在于：以所属领域技术人员的视角，将发明实际取得的技术效果与其基于现有技术和普通技术知识所能预期的效果进行比较，超越预期方可认定。该判断与实验数据密切相关，但补充实验数据受先申请制原则限制，对比试验数据则需满足代表性要求，方能作为认定依据。

---

## 1. 预料不到的技术效果的判断标准

### 1.1 判断视角：所属领域技术人员

判断预料不到的技术效果，核心在于**以所属领域技术人员的视角**判断发明取得的技术效果是否预料不到。具体而言，是将发明实际取得的技术效果与所属领域技术人员在其掌握的知识和能力的基础上预期的技术效果进行比较，确认发明的技术效果是否超越了技术人员的预期。

对发明实际取得的技术效果的认定，需要在专利文件公开的事实基础上认真研究案件中出现的各种证据；而对于技术人员能够预期的技术效果，则需在全面考虑申请日前该领域现有技术状况的基础上，结合其所掌握的普通技术知识和能力来判断。

有些情况下，如果现有技术未对其所公开的技术方案以及技术手段的技术效果进行描述，则专利文件中记载的针对现有技术进行的分析和研究结果可以帮助判断者更加准确地了解现有技术发展水平，有助于对可预期的技术效果做出准确认定。

### 1.2 数量差异或新性质≠必然预料不到

确定是否属于预料不到的技术效果，应当着重考量所属领域技术人员对技术效果的**预判能力**，据此明确其所能够预期的技术效果，继而确定发明的技术效果是否达到预料不到的程度。因此，技术效果是否达到所属领域技术人员预料不到的程度，**既不必然取决于效果数量上的差异，也不必然取决于性质或用途上的差异**。将发明的技术效果与现有技术的技术效果进行对比，即便是出现数量级上的差异或者出现新的性质或用途，也不必然意味着产生的是预料不到的技术效果。

### 1.3 结构改造中的预料不到效果

即使是基于已知理论的结构改造，若效果远超预期，仍可能构成预料不到的技术效果。

#### 案例：第95778号复审决定（20098013907.6）

- **争议焦点：** 嘌呤6位以-O-替代-NH-是否产生预料不到的技术效果
- **决定要点：** 涉案申请与对比文件1的区别特征仅在于化合物结构的嘌呤6位上的基团被限定为氧代基团（-O-），而对比文件1中的相应位置为氨基取代基（-NH-）。根据所属领域的公知常识，一般认为-O-与-NH-为生物电子等排体，二者的结构和性质类似，因此技术人员可以预期的技术效果是-O-与-NH-相互替换后在药理活性方面不会产生特别明显的变化。

  然而，实验证据中选取的对比对象（涉案申请实施例43化合物与对比文件1化合物26）结构差异仅在于嘌呤6位上以-O-替代了-NH-，在此情形下，涉案申请实施例43的化合物在HCT-116细胞系中抑制癌细胞生长活性（GI₅₀）方面比化合物26强大约40倍，且在其他多种癌细胞上也都体现出了这种活性差异。涉案申请说明书中的其他实施例也同样证实将嘌呤6位的-NH-用-O-替换后确实能够明显提高抑制癌细胞生长的活性。因此，将嘌呤6位的-NH-替换成-O-能够大幅提高先导化合物的生物活性，而这种活性的提高并非在申请日时考虑将电子等排理论直接运用于对比文件1时就能够预料得到的。
- **启示：** 生物电子等排原理虽是结构改造的常用理论，但具体替换后的活性变化仍需实验验证。当实际效果远超该理论应用时的可预期范围时，构成预料不到的技术效果。这一案例也提示，在化学药物领域，**不能仅凭理论上的等效性就否定结构微小改变可能带来的巨大活性差异**。

### 1.4 组合的预料不到效果

#### 案例：第21779号无效决定（201010190797.8）

- **争议焦点：** 淀粉+微晶纤维素混合赋形剂的抗吸湿效果是否预料不到
- **决定要点：** 权利要求1与证据3的区别特征是药物组合物中所使用的赋形剂不同，涉案专利中使用淀粉+微晶纤维素（1:1~1:2），证据3中使用的是单一赋形剂。根据专利文件的记载，使用乳糖、淀粉、糖粉、糊精、微晶纤维素、碳酸氢钙与浸膏粉逐一配伍测定吸湿性，结果显示吸湿率分别为10.4%、10.1%、11.2%、9.85%、10.5%、10.6%，而使用淀粉：微晶纤维素的比例为1:2和1:1的混合赋形剂时，吸湿百分率为4.21%和4.05%，抗吸湿性效果显著提高。

  根据涉案专利说明书中给出的实验结果，可以确认的是现有技术中赋形剂的吸湿效果较差。在此基础上，所属领域技术人员可以预期即便将赋形剂组合，其吸湿率也大致保持在10%左右的水平，不会有明显改变。然而，专利权人通过其自身研究所发现的特定配比的混合赋形剂导致了吸湿性大幅下降，则该技术效果超出了预期的技术效果，属于预料不到的技术效果。
- **启示：** 当组合后产生了远超各组分单独使用时可预期水平的协同效果时，即使组合本身看似简单，也可能构成预料不到的技术效果。关键在于**技术人员基于现有技术对组合效果的预判能力**——如果现有技术已给出各组分效果的大致范围，而实际组合效果显著跳出该范围，则具有预料不到性。

### 1.5 不构成预料不到的情形

并非所有改进都能构成预料不到的技术效果。以下两种常见情形需特别注意：

- **常规变化的可预期效果：** 所属领域对于常见剂型、常见辅料的改变所带来的技术效果通常是技术人员可以预期的。例如第15409号无效决定中，片剂改为颗粒剂总有效率提高6%，因属常规剂型变化且省略减压干燥步骤的效果也可预期，故不构成预料不到的技术效果。（详见[[创造性-辅助判断因素|辅助判断因素]]）

- **新效果与已知效果存在关联：** 当发明产生的新效果与已知效果之间存在关联，技术人员能够从已知效果预期新效果时，则不属于预料不到。例如第52675号复审决定中，添加抗冲击改性剂后改善雾度水平（即降低透明度），因技术人员公知添加组分会影响透明性，该效果可通过逻辑分析和推理预知，故不构成预料不到的技术效果。（详见[[创造性-辅助判断因素|辅助判断因素]]）

## 2. 补充实验数据的限制

预料不到的技术效果的判断往往与实验数据关系十分密切。除记载在原始申请文件中的实验数据外，申请人/专利权人有时会采取补充实验数据的方式证明发明技术方案取得了预料不到的技术效果。针对补充实验数据的审查，除关注该数据能否证明申请人/专利权人所主张的技术效果成立之外，**还应根据先申请制的要求**，判断这种基于补充数据提出的技术效果是否属于所属领域技术人员能够从原始申请文件公开的内容中得到的技术效果。

### 2.1 先申请制原则与信息边界

我国专利制度采用的是先申请原则，判断专利申请能否授权应以其申请日提交的文件所记载的内容作为基础。因此，补充实验数据不能为专利申请文件引入新的信息。如果补充实验数据所证明的事实超出了所属领域技术人员基于原始申请文件公开的内容能够获知的范畴，则该补充实验数据不能作为主张专利权的基础。

这一原则的法理基础在于：专利权的保护范围以申请日公开的内容为边界，如果允许通过补充数据引入申请日未公开的新信息，则相当于变相扩大了专利权的保护范围，损害了公众利益。

### 2.2 原始文件未记载具体程度的效果

#### 案例：第21646号无效决定（97180299.8）

- **争议焦点：** 通过补充实验数据证明化合物抑制TNFα活性高三个数量级是否可接受
- **决定要点：** 涉案专利保护的具体化合物与证据6公开的化合物EM-12的区别仅在于化合物的异二氢吲哚环4位是否具有氨基取代基。专利权人提交的反证5记载了涉案专利化合物抑制TNFα活性的IC₅₀值为100nM，沙利度胺的IC₅₀值为200μM，而化合物EM-12具有与沙利度胺相似的活性，专利权人主张据此可推知涉案专利化合物抑制TNFα的活性高出化合物EM-12三个数量级，具有预料不到的技术效果。

  决定认为，本案中是否能够接受专利权人通过补充实验数据而提出的发明具有的技术效果，主要看补充后的内容是否为专利申请文件引入了新的信息。涉案专利说明书中仅记载式I化合物被用来抑制TNFα的不期望有的作用，以及这些化合物还可以用于需要抑制TNFα产生的人以外的哺乳动物的兽医治疗，既未进一步具体描述，也未给出任何实验证据证实式I化合物产生的是何种程度的抑制效果。在此情况下，所属领域技术人员仅能够意识到涉案专利化合物具有一般的TNFα抑制效果，而不会确信其具有超出常规水平的活性或者其他预料不到的技术效果。反证5用来证明涉案专利化合物的技术效果实质上明显优于所属领域技术人员所认识到的现有技术的一般水平，由于该效果是所属领域技术人员在阅读原申请文件后无法获知的，因此不能作为确定涉案专利获得了预料不到的技术效果的依据。
- **启示：** 原始说明书仅泛泛记载用途而未给出任何实验证据、未记载效果程度时，技术人员只能认识到"一般水平"的效果。此时，即便补充数据证明效果远超现有技术，也因引入了原始文件无法获知的新信息而不能被接受。**在专利撰写阶段，如预期依赖技术效果主张创造性，应在原始申请文件中至少记载部分实验数据或效果描述，为后续可能的补充数据留出空间。**

### 2.3 原始文件未记载比较关系和验证条件

#### 案例：第27508号无效决定（200480034152.8）

- **争议焦点：** 通过补充实验数据证明药物制剂具有预料不到的稳定性是否可接受
- **决定要点：** 涉案专利保护的药物制剂与证据1的区别在于等渗剂种类有所差异。专利权人在提交的补充实验数据中详细对比了两种等渗剂对于药物制剂稳定性的影响，拟结合证人证言证明涉案专利的制剂具有预料不到的稳定性。

  决定认为，如果补充实验数据所证明的事实超出了所属领域技术人员基于原始申请文件公开的内容能够获知的范畴，则该补充实验数据不能作为主张专利权的基础。涉案专利说明书中仅提到，所述等渗剂对药物制剂的稳定性没有影响，既未涉及任何将该等渗剂与其他候选等渗剂的效果进行比较的信息，也未验证制剂的稳定性，甚至也没有记载检测制剂稳定性的实验对象、方法和条件，导致所属领域的技术人员无法通过原始申请文件的记载确认该制剂是否具有预料不到的稳定性效果，同时亦无法利用原始申请文件记载的信息进行有效验证。因此专利权人不能将该补充实验数据提出的技术效果作为判断预料不到的技术效果的基础。
- **启示：** 不仅效果的"有无"需要在原始文件中有所体现，用于验证该效果的**比较关系、实验对象、方法和条件**也应在原始文件中有所记载或至少可被技术人员推导。如果原始文件完全没有提供可供验证的框架，补充数据即便再详实也无法被接受。

## 3. 对比试验数据的选择要求

### 3.1 围绕主张的技术效果选择对比对象

对比试验数据中对比对象的选择应当围绕申请人或专利权人所主张的技术效果。对比试验的核心目的是客观地反映权利要求技术方案相对于现有技术的真实技术效果，因此对比对象的选择必须具有**代表性**和**合理性**。

### 3.2 个别实施例不能代表整体保护范围

#### 案例：第88160号复审决定（200780046315.8）

- **争议焦点：** 以特定实施例的反应时间短于对比文件主张预料不到的技术效果是否成立
- **决定要点：** 权利要求1的保护范围涵盖了包括实施例1~5在内的多种技术方案，请求人主张涉案申请的反应时间为4小时，明显短于对比文件1的13小时，因此取得了预料不到的技术效果。然而，根据涉案申请说明书的记载，实施例1~5完成反应所需的时间分布在大约3~13小时的范围内，但各不相同，例如实施例2为13小时，并不比对比文件1（13小时）和对比文件2（3~18小时）更短。并且也没有证据表明，在相同的条件下，涉案申请的反应时间会明显短于对比文件1。

  决定认为，特定的实施例并不能代表要求保护的技术方案的整体状况，故不能认为涉案申请权利要求取得了预料不到的技术效果。
- **启示：** 在主张预料不到的技术效果时，**不能仅以个别效果最好的实施例代表整个权利要求的保护范围**。如果权利要求涵盖的多种技术方案效果差异很大，而仅选取其中效果最优的个别方案与现有技术对比，这种对比试验无法客观反映技术方案的整体状况，也就无法证明整个权利要求范围取得了预料不到的技术效果。同理，仅选择现有技术中效果最差的方案进行对比，也同样不具有说服力。

## 判断要点

```
预料不到的技术效果与实验数据认定流程
│
├── 第一步：判断是否属于预料不到的技术效果
│   ├── 以技术人员视角比较"实际效果"与"预期效果"
│   ├── 预期效果基于：申请日前现有技术 + 普通技术知识 + 能力
│   ├── 数量差异/新性质 ≠ 必然预料不到
│   ├── 常规变化的可预期效果 → 不构成
│   └── 新效果与已知效果可关联推导 → 不构成
│
├── 第二步：审查实验数据的来源与合法性
│   ├── 原始申请文件中的实验数据
│   │   └── 直接作为认定依据
│   │
│   ├── 补充实验数据
│   │   ├── 能否证明技术效果成立？
│   │   └── 技术效果能否从原始申请文件公开内容中得到？
│   │       ├── 是 → 可作为认定依据
│   │       └── 否 → 引入新信息，不可接受
│   │
│   └── 对比试验数据
│       ├── 对比对象是否围绕主张的技术效果？
│       ├── 是否以个别实施例代表整体保护范围？
│       │   └── 是 → 不具代表性，不可接受
│       └── 是否仅选择现有技术中最差方案对比？
│           └── 是 → 不具代表性，不可接受
│
└── 第三步：综合认定
    ├── 效果超出预期 + 数据来源合法 → 支持创造性
    └── 任一步骤不满足 → 不构成预料不到的技术效果
```
