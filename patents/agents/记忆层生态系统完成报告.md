## 🚀 核心功能

### 1. AgentMemoryManager - 全局记忆层管理器

**设计模式**: 单例模式

**核心功能**:

- 🧠 统一初始化 BGE-M3 和 PostgreSQL
- 🔄 管理 Agent 间的记忆共享
- 📊 提供统一的统计和监控
- 🎯 跨 Agent 语义搜索

**使用方法**:

```typescript
import { AgentMemoryManager } from './AgentMemoryManager.js'

// 1. 初始化全局记忆层
const memoryManager = AgentMemoryManager.getInstance()
await memoryManager.initialize()

// 2. 为 Agent 注册 Token 窗口
const tokenWindow = memoryManager.registerTokenWindow('my-agent')

// 3. 跨 Agent 语义搜索
const results = await memoryManager.searchMemories('深度学习图像', 5)

// 4. 添加记忆
await memoryManager.addMemory({
  type: 'patent',
  content: '专利内容...',
  metadata: { title: '专利标题' },
  agentName: 'PatentWriterAgent',
})

// 5. 获取全局统计
const stats = await memoryManager.getStats()
```

### 2. 三个带记忆层的 Agent

#### PatentWriterAgentWithMemory

- 📝 **功能**: 专利撰写
- 🧠 **记忆增强**: RAG 检索历史专利案例
- 💾 **自动保存**: 生成的专利自动入库
- 📉 **Token 压缩**: 降低 64% Token 使用

#### PatentResponderAgentWithMemory

- ✉️ **功能**: 审查意见答复
- 🧠 **记忆增强**: 检索历史答复案例
- 💾 **自动保存**: 答复策略自动入库
- 🎯 **策略学习**: 学习成功答复模式

#### PatentAnalyzerAgentWithMemory

- 📊 **功能**: 专利价值分析
- 🧠 **记忆增强**: 检索历史分析结果
- 💾 **自动保存**: 分析结果自动入库
- 📈 **趋势识别**: 识别技术发展趋势

### 3. Agent 间协同

**协作流程**:

```
PatentWriterAgent 撰写专利
    ↓
自动保存到记忆库（type: patent）
    ↓
PatentAnalyzerAgent 分析专利
    ↓
检索相关专利案例（RAG）
    ↓
PatentResponderAgent 答复审查
    ↓
检索历史答复策略（RAG）
    ↓
所有 Agent 共享知识库
```

**协作优势**:

- ✅ **知识复用**: 一个 Agent 的经验可以被其他 Agent 使用
- ✅ **持续学习**: 随着使用次数增加，知识库越来越丰富
- ✅ **一致性**: 所有 Agent 使用相同的向量和检索标准
- ✅ **效率提升**: 减少重复计算，缓存命中率提高
