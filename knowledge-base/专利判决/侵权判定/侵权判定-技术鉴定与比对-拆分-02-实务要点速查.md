# 实务要点速查

> **拆分自：** [[侵权判定-技术鉴定与比对]]

---

## 核心要点

专利侵权案件中的技术鉴定与比对是确定被诉侵权产品是否落入专利权保护范围的关键环节。法院在技术事实查明中采用"技术特征对比法"，将被诉侵权技术方案与专利权利要求记载的技术特征逐一比对。技术鉴定并非侵权判定的必经程序，法院可以结合技术调查官参与、当事人举证和庭审查明自行完成技术比对。当事人单方委托鉴定机构出具的书面意见经审查可予采纳，法院委托鉴定机构的鉴定意见需经质证后方可采信。鉴定程序的合法性、鉴定机构的资质、鉴定方法的科学性是审查鉴定意见的三大核心要素。

## 实务要点速查

| 情形 | 裁判倾向 | 关键依据 | 代表案例 |
|------|---------|---------|---------|
| 法院摇号确定鉴定机构 | 程序合法，鉴定意见可采信 | 民事诉讼法第79条 | （2024）最高法知民终900号 |
| 当事人协商选定鉴定机构 | 一般认可，证明力较高 | 民事诉讼法第79条 | （2021）最高法知民终1400号 |
| 单方委托鉴定意见经审查合格 | 可予采纳 | 最高人民法院关于民事诉讼证据的若干规定第41条 | （2022）最高法知民终2100号 |
| 诉前形成的鉴定意见 | 需审查与本案的关联性 | 民事诉讼法证据规则 | （2021）最高法知民终2037号 |
| 鉴定机构未实际测量仅凭推理 | 有合理性时可采信 | 专利侵权判定司法解释 | （2023）最高法知民终833号 |
| 另案鉴定意见直接引用认定本案事实 | 可能构成程序瑕疵 | 民事诉讼法质证程序 | （2019）最高法知民终831号 |
| 鉴定意见缺乏测试方法和数据 | 不予采信或要求补充鉴定 | 司法鉴定程序通则第22条 | （2016）最高法民再85号 |
| 鉴定超期 | 可能构成程序违法 | 人民法院司法鉴定工作暂行规定第21条 | （2016）最高法民再85号 |
| 不同检测机构结果存在偏差 | 合理偏差不否定证明力 | 证据审查规则 | （2019）最高法知民终702号 |
| 技术调查官参与技术事实查明 | 合法有效的辅助方式 | 最高人民法院关于知识产权法庭若干问题的规定 | （2020）最高法知民终659号 |
| 鉴定人对同类产品做过鉴定 | 不当然构成回避事由 | 民事诉讼法回避制度 | （2021）最高法知民终1400号 |
| 技术比对范围超出本案争议范围 | 超出部分的鉴定费用可酌减 | 诉讼费用负担规则 | （2022）最高法知民终2100号 |
