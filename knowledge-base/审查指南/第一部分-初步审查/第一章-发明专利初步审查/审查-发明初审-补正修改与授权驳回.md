# 审查-发明初审-补正修改与授权驳回

> **来源：** 《专利审查指南》第一部分第一章 §3-4（审查程序部分）、§8（依职权修改）
> **对应法条：** 《专利法》第33条、第34条 / 《专利法实施细则》第50条、第57条
> **最新修订：** 2023年修订（完善依职权修改范围）
> **关联页面：** [[审查-发明初审-申请文件审查]]、[[审查-发明初审-明显实质性缺陷]]、[[程序-原理-初步审查]]

## 核心要点

发明专利初步审查的补正修改与授权驳回程序是整个初步审查流程的核心操作环节。审查员对可补正缺陷发出补正通知书，对不可补正的实质性缺陷发出审查意见通知书，经补正合格的发出初步审查合格通知书进入公布程序，经补正/陈述仍不合格的作出驳回决定。驳回决定须经听证程序，且申请人可提出复审请求。

---

## 审查标准

### 一、初步审查合格（§3.1）

经初步审查，申请文件符合专利法及其实施细则有关规定并且不存在明显实质性缺陷的专利申请，包括经过补正符合初步审查要求的专利申请，应当认为初步审查合格。

审查员应当发出**初步审查合格通知书**，指明公布所依据的申请文本，之后进入公布程序。

### 二、申请文件的补正（§3.2）

对于存在可以通过补正克服的缺陷的专利申请，审查员应当**进行全面审查**，并发出补正通知书。

补正通知书应当：
1. 指明专利申请存在的缺陷。
2. 说明理由。
3. 指定答复期限。

### 三、明显实质性缺陷的处理（§3.3）

对于存在不可能通过补正方式克服的明显实质性缺陷的专利申请，审查员应当发出**审查意见通知书**。

审查意见通知书应当：
1. 指明专利申请存在的实质性缺陷。
2. 说明理由。
3. 指定答复期限。

**审查界限**：对于申请文件中存在的实质性缺陷，只有其**明显存在并影响公布时**，才需指出和处理。

### 四、通知书的答复（§3.4）

申请人在收到补正通知书或审查意见通知书后，应当在指定期限内补正或陈述意见。

**补正要求**：
- 应当提交补正书和相应修改文件替换页。
- 对申请文件的修改，应当针对通知书指出的缺陷进行。
- 修改的内容不得超出申请日提交的说明书和权利要求书记载的范围。

**逾期处理**：
- 申请人期满未答复的，发出视为撤回通知书或其他通知书。
- 因正当理由难以在指定期限内答复的，可以提出延长期限请求（适用指南第五部分第七章第4节）。
- 因不可抗拒事由或其他正当理由耽误期限导致申请被视为撤回的，可以在规定期限内向专利局提出恢复权利的请求（适用指南第五部分第七章第6节）。

### 五、申请的驳回（§3.5）

#### 驳回条件

审查员可以在以下情形下作出驳回决定：

1. **实质性缺陷驳回**：申请文件存在明显实质性缺陷，在审查员发出审查意见通知书后，经申请人陈述意见或修改后仍然没有消除的。

2. **形式缺陷驳回**：申请文件存在形式缺陷，审查员针对该缺陷已发出过**两次补正通知书**，经申请人陈述意见或补正后仍然没有消除的。

#### 驳回决定正文

驳回决定正文应当包括三部分内容：

**（1）案由部分**
- 简述被驳回申请的审查过程。

**（2）驳回的理由部分**
- 说明驳回的事实、理由和证据。

**（3）决定部分**
- 明确指出该专利申请不符合专利法及其实施细则的相应条款。
- 说明根据专利法实施细则第五十条第二款的规定驳回该专利申请。

#### 驳回的法律依据

驳回的法律依据应当包含在专利法实施细则第五十条所列的法律条款中，主要包括：
- 专利法第二条第二款、第五条、第十七条、第十八条第一款、第十九条第一款、第二十五条、第三十一条第一款、第三十三条。
- 专利法实施细则第十一条、第二十条、第二十二条。

### 六、前置审查与复审后的处理（§3.6）

- 申请人对驳回决定不服的，可以在规定期限内提出复审请求。
- 对复审请求的前置审查及复审后的处理，参照指南第二部分第八章第8节的规定。

### 七、依职权修改（§8）

根据专利法实施细则第五十七条第四款的规定，审查员可以在初步审查合格之前依职权修改申请文件中文字和符号的明显错误，并通知申请人。

#### 依职权修改的常见情形

**（1）请求书**
- 修改申请人地址或联系人地址中漏写、错写或重复填写的省（自治区、直辖市）、市、邮政编码等信息。

**（2）权利要求书和说明书**
- 改正明显的文字错误和标点符号错误。
- 修改明显的文本编辑错误。
- 删除明显多余的信息。
- **注意**：可能导致原始申请文件记载范围发生变化的修改，不属于依职权修改的范围。

**（3）摘要**
- 添加明显遗漏的内容。
- 改正明显的文字错误和标点符号错误。
- 删除明显多余的信息。
- 指定摘要附图。

### 八、审查流程总结

```
受理 → 初步审查
  ├─ 合格 → 初步审查合格通知书 → 进入公布程序
  ├─ 可补正缺陷 → 补正通知书（最多两次）
  │   ├─ 补正合格 → 初步审查合格通知书
  │   ├─ 补正不合格（两次后仍存在）→ 驳回决定
  │   └─ 期满未答复 → 视为撤回通知书
  └─ 明显实质性缺陷 → 审查意见通知书
      ├─ 陈述/修改后消除 → 初步审查合格通知书
      ├─ 陈述/修改后未消除 → 驳回决定
      └─ 期满未答复 → 视为撤回通知书

驳回决定 → 复审请求 → 前置审查 → 复审决定
```

## 实务应对

### 代理人视角：补正修改的策略要点

1. **全面审查原则**：审查员在发出补正通知书时应当进行全面审查，因此代理人提交申请时应尽量一次性消除所有形式缺陷，避免多次补正。第一次补正通知书指出全部缺陷，但形式缺陷经两次补正仍未消除即可驳回。

2. **针对性修改**：答复补正通知书或审查意见通知书时，修改必须针对通知书指出的缺陷。超出范围的修改不予接受。

3. **修改不超范围**：所有修改不得超出原说明书和权利要求书记载的范围。这是专利法第33条的核心要求。

4. **利用依职权修改**：明显的文字错误和标点符号错误，审查员可以依职权修改。代理人可在申请中附带简要说明，提示审查员可以依职权修改的内容。

5. **驳回的应对**：收到驳回决定后，应在规定期限内提出复审请求。驳回决定必须已经过听证（即驳回所依据的事实、理由和证据已通知过申请人），否则程序违法。

6. **期限管理**：
   - 补正通知书的答复期限应严格遵守，逾期导致视为撤回。
   - 如有正当理由，应及时提出延长期限请求或恢复权利请求。

## 审查要点速查

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 申请文件无缺陷或缺陷已消除 | 发出初步审查合格通知书 | 法第34条 |
| 存在可补正缺陷 | 发出补正通知书 | 细则第50条 |
| 形式缺陷经两次补正仍未消除 | 可以作出驳回决定 | 细则第50条第2款 |
| 实质性缺陷经陈述/修改仍未消除 | 可以作出驳回决定 | 细则第50条第2款 |
| 期满未答复通知书 | 发出视为撤回通知书 | 细则第50条 |
| 修改超出原说明书和权利要求书记载范围 | 发出审查意见通知书 | 法第33条 |
| 审查员发现明显文字/标点错误 | 可以依职权修改并通知申请人 | 细则第57条第4款 |
| 驳回决定包含未通知过的新事实/理由 | 违反听证原则，复审可撤销 | 指南§2 |
