# 外观设计侵权：双方专利与无效处理

> **来源：** 《侵权判定指南(2017)理解与适用》第四章，第95-96条
> **核心法条：** 《专利法》第二十三条；《侵犯专利权司法解释（二）》第二条
> **关联页面：** [[外观设计侵权-种类相同与虚实对比]]、[[侵权判定-全面覆盖原则]]

---

## 核心要点

当侵权诉讼中原被告双方均拥有外观设计专利权时，法院不应当仅以被告拥有专利权为由驳回原告诉讼请求，而应分析双方专利的具体关系。若原告专利申请日在先且双方设计构成相同或相近似，可认定被告实施其专利的行为侵犯了在先的外观设计专利权。在外观设计专利侵权诉讼中，若专利被宣告无效，参照发明和实用新型专利的相关规定处理，可裁定驳回起诉。

---

## 第95条 双方外观设计专利的比对

### 条文原文

> 专利权人、被诉侵权人的外观设计专利申请均已被授权，且专利权人的外观设计专利的申请日早于被诉侵权人的外观设计专利的申请日，如果被诉侵权人的外观设计与专利权人的外观设计构成相同或相近似的，则可以认定被诉侵权人实施其外观设计专利的行为，侵犯了在先的外观设计专利权。

### 理解与适用

由于我国对实用新型和外观设计专利申请采用初步审查制，难免出现不同申请人就同一设计方案先后提出申请的情况。本条源于最高人民法院1993年8月16日《关于在专利侵权诉讼中当事人均拥有专利权应如何处理问题的批复》的精神。

**"双方都有专利"的三种情形：**

| 情形 | 关系 | 是否构成侵权 |
|------|------|------------|
| 双方发明创造点不同，有本质区别 | 独立设计 | 不构成侵权 |
| 被告专利是对原告专利的改进/改良（从属专利） | 从属关系 | 实施从属专利需经在先权利人许可，否则构成侵权 |
| 前后两项专利技术方案相同或等同 | 重复授权 | 后一项的实施构成侵权 |

**适用要点：**

1. **先申请原则**：只要原告先于被告提出专利申请，应依据原告的专利权保护范围进行审查
2. **不因被告有专利而免责**：法院不应当仅以被告拥有专利权为由不进行侵权分析即驳回原告诉讼请求
3. **"虚"与"实"的比对**：
   - "虚"：专利公告中的图片或照片
   - "实"：被诉侵权物
   - 当被诉侵权人的外观设计与在先外观设计构成相同或相近似，且被告确系实施其自己专利的，可推定侵权

**"在先权利"的限定：** 本条中的"在先"不宜称为《专利法》第二十三条所称的"在先权利"。专利法意义的"在先权利"特指商标权、著作权、企业名称权等专利法以外的其他权利，"在先的专利权"在专利法体系内已属于"现有技术"或"现有设计"的范畴。

**两种时间关系：**
- 专利权人的外观设计专利授权日早于被诉侵权人的申请日 → 构成现有设计
- 专利权人的外观设计专利授权日晚于被诉侵权人的申请日 → 构成抵触设计

## 第96条 终审判决作出前外观设计专利被宣告无效

### 条文原文

> 在终审判决作出前，权利人所主张的外观设计被专利复审委员会（现专利复审和无效审理部）宣告无效的，可以参照本指南第9、10条处理。

### 理解与适用

本条明确了外观设计专利侵权及无效案件并发时的处理方式，与发明及实用新型案件采用统一做法。

**先行裁驳、另行起诉制度：**

《侵犯专利权司法解释（二）》第二条设计了这一制度：
- 专利复审委员会（现专利复审和无效审理部）宣告专利权无效后，审理侵权纠纷案件的法院**可以**裁定驳回起诉
- 无需等待专利行政诉讼的最终结果
- 若无效决定后续被生效行政判决撤销，权利人可以**另行起诉**

**为什么采用"裁定驳回起诉"而非"判决驳回诉讼请求"？**

主要是考虑若无效决定后续被行政裁判推翻，权利人仍可通过另行起诉获得救济。如果采用判决驳回，则可能需要通过审判监督程序，程序更为复杂。

**二审阶段的处理：**

如果在二审阶段专利被宣告无效：
- 二审法院可以径行裁定驳回起诉
- 在综合考虑在案证据、涉案专利技术难度、被告抗辩理由等因素，且当事人请求的情况下，可以裁定中止二审案件的审理

**程序统一性：** 虽然外观设计在侵权判定方面有着独特的保护体系，但在司法程序问题上，外观设计专利案件属于专利案件，应采用专利类案件的统一做法。在没有法律法规及司法解释对其作出特殊规定的前提下，不应因案件涉及外观设计而作出特殊处理。

## 本页实务要点总结

### 双方均有专利时的诉讼策略

| 角色 | 建议策略 |
|------|---------|
| 原告（在先专利权人） | 重点证明被告设计与原告设计构成相同或相近似 |
| 被告（在后专利权人） | 可提出现有设计抗辩或申请宣告原告专利无效 |

### 专利被宣告无效后的应对

| 阶段 | 处理方式 |
|------|---------|
| 一审期间被宣告无效 | 法院可裁定驳回起诉 |
| 二审期间被宣告无效 | 二审法院可径行裁定驳回起诉 |
| 无效决定被行政判决撤销 | 权利人可另行起诉 |
| 当事人请求中止 | 法院可综合考虑后裁定中止 |

## 本页关键概念索引

| 概念 | 含义 |
|------|------|
| 先申请原则 | 两个以上的申请人分别就同样的发明创造申请专利的，专利权授予最先申请的人 |
| 从属专利 | 在后专利是对在先专利技术的改进或改良，实施在后专利有赖于实施在先专利 |
| 重复授权 | 前后两项专利的技术方案相同或等同，后一项属于重复授权 |
| 先行裁驳 | 专利被宣告无效后法院可先行裁定驳回起诉，无需等待行政诉讼结果 |
| 另行起诉 | 无效决定被撤销后，权利人可重新提起侵权诉讼 |
| 抵触设计 | 在先申请但在后公开的外观设计，可用于否定在后申请的新颖性 |
