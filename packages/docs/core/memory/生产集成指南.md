## 核心组件

### 1. BGE-M3 嵌入服务 ⭐

**配置**：

```bash
# 服务地址
Base URL: http://localhost:8009/v1
API 密钥: xj781102@
模型: bge-m3-mlx-8bit
向量维度: 1024
```

**使用方式**：

```typescript
import { createBGEM3Client } from '@yunpat/core'

const bgeClient = createBGEM3Client({
  apiKey: 'xj781102@',
})

// 单个文本向量化
const embedding = await bgeClient.embed('专利撰写的关键在于权利要求书的撰写')
console.log(embedding.length) // 1024

// 批量向量化
const embeddings = await bgeClient.embedBatch(['文本1', '文本2', '文本3'])
```

**性能指标**：

- 单个向量化：50-100ms
- 批量处理：100-200 QPS
- 缓存命中：<1ms（快 100 倍）

### 2. PostgreSQL 向量存储 ⭐

**配置**：

```bash
# Docker 启动
cd packages/core/src/memory/long-term
docker-compose up -d

# 连接信息
Host: localhost:5432
Database: yunpat
User: yunpat
Password: yunpat123
```

**核心功能**：

- HNSW 索引（向量相似度搜索）
- JSONB 索引（元数据过滤）
- ACID 事务（数据一致性）

**使用方式**：

```typescript
import { PostgresVectorStore } from '@yunpat/core'

const vectorStore = new PostgresVectorStore({
  databaseUrl: 'postgres://yunpat:yunpat123@localhost:5432/yunpat',
  vectorDimension: 1024,
})

await vectorStore.initialize()

// 添加记忆
const embedding = await bgeClient.embed('专利内容...')
await vectorStore.upsert({
  type: 'patent',
  content: '专利内容...',
  embedding: embedding,
  metadata: { patentId: 'CN123456' },
})

// 语义检索
const queryEmbedding = await bgeClient.embed('如何撰写专利？')
const results = await vectorStore.search(queryEmbedding, 5, {
  types: ['patent'],
  threshold: 0.7,
})
```

### 3. Token 窗口管理 ⭐

**核心功能**：

- 滑动窗口算法（保留最近 N 条对话）
- Token 估算（中英文混合）
- 语义摘要（压缩历史对话）
- 重要性评分（智能过滤）

**使用方式**：

```typescript
import { createTokenWindowManager } from '@yunpat/core'

const tokenWindow = createTokenWindowManager({
  maxTokens: 4000,
  reservedTokens: 500,
  enableSummary: true,
})

// 应用滑动窗口
const { messages, stats } = await tokenWindow.slideWindow(dialogueHistory)

console.log(`压缩比例: ${(stats.compressionRatio * 100).toFixed(2)}%`)
// 输出: 35.77%（降低 64.23%）

// Token 预测
const prediction = await tokenWindow.predictNextTokens(
  messages,
  800 // 预估响应 Token 数
)

if (prediction.willExceedLimit) {
  console.log('警告：即将超限！')
  for (const action of prediction.recommendedActions) {
    console.log(`建议：${action}`)
  }
}
```

### 4. 上下文管理 ⭐

**核心功能**：

- 多轮对话拼接
- 系统提示注入
- 格式化输出（Markdown/纯文本）
- Token 统计

**使用方式**：

```typescript
import { createContextManager } from '@yunpat/core'

const contextManager = createContextManager({
  maxTokens: 4000,
  systemPrompt: '你是一个专业的专利撰写助手。',
})

// 构建上下文
const { context, stats } = await contextManager.buildContext(messages, {
  asMarkdown: true,
  includeRole: true,
})

console.log(context)
console.log(`Token 使用: ${stats.totalTokens}`)
```

### Step 3: 集成到 Agent

```typescript
// 1. 导入组件
import { createBGEM3Client } from '@yunpat/core'
import { PostgresVectorStore } from '@yunpat/core'
import { createTokenWindowManager } from '@yunpat/core'

// 2. 初始化（在 Agent 构造函数中）
class MyAgent extends Agent<Input, Output> {
  private bgeClient: any
  private vectorStore: any
  private tokenWindow: any

  async init?(): Promise<void> {
    this.bgeClient = createBGEM3Client({
      apiKey: 'xj781102@',
    })

    this.vectorStore = new PostgresVectorStore({
      databaseUrl: 'postgres://yunpat:yunpat123@localhost:5432/yunpat',
      vectorDimension: 1024,
    })

    await this.vectorStore.initialize()

    this.tokenWindow = createTokenWindowManager({
      maxTokens: 4000,
      enableSummary: true,
    })
  }

  // 3. 使用记忆层
  protected async act(plan: Plan, context: ExecutionContext): Promise<Result> {
    // 3.1 语义检索
    const queryEmbedding = await this.bgeClient.embed(userQuery)
    const relevantDocs = await this.vectorStore.search(queryEmbedding, 5, { types: ['patent'] })

    // 3.2 构建 RAG 上下文
    const ragContext = relevantDocs
      .map((doc) => `[参考] ${doc.content.slice(0, 100)}...`)
      .join('\n')

    // 3.3 管理 Token 窗口
    const { messages: compressed } = await this.tokenWindow.slideWindow(conversationHistory)

    // 3.4 调用 LLM
    const prompt = `
参考信息：
${ragContext}

对话历史：
${compressed.map((m) => `${m.role}: ${m.content}`).join('\n')}

当前任务：${task}
`

    const response = await this.callLLM(prompt)

    // 3.5 保存到记忆库
    const responseEmbedding = await this.bgeClient.embed(response)
    await this.vectorStore.upsert({
      type: 'conversation',
      content: response,
      embedding: responseEmbedding,
      metadata: { agent: 'MyAgent', timestamp: new Date() },
    })

    return response
  }
}
```

## 使用示例

### 示例 1：智能专利撰写（RAG 增强）

**场景**：用户要求撰写专利，系统自动检索相关案例

```typescript
// 1. 用户输入
const userInput = {
  inventionTitle: '基于注意力机制的医学图像分析',
  technicalField: '医学影像分析',
  inventionContent: '使用多头注意力机制处理 CT 图像...',
}

// 2. 检索相关专利
const queryEmbedding = await bgeClient.embed(
  `${userInput.inventionTitle} ${userInput.technicalField}`
)

const relevantPatents = await vectorStore.search(queryEmbedding, 5, {
  types: ['patent'],
})

// 3. 构建 RAG 提示词
const ragPrompt = `
参考以下专利案例：

${relevantPatents
  .map(
    (p, i) => `
案例 ${i + 1}（相似度：${(p.similarity * 100).toFixed(1)}%））：
${p.content.slice(0, 200)}...
`
  )
  .join('\n')}

请根据上述案例风格，撰写以下专利：
${userInput.inventionTitle}
${userInput.technicalField}
${userInput.inventionContent}
`

// 4. 调用 LLM
const patentContent = await llm.generate(ragPrompt)

// 5. 保存到记忆库
const patentEmbedding = await bgeClient.embed(patentContent)
await vectorStore.upsert({
  type: 'patent',
  content: patentContent,
  embedding: patentEmbedding,
  metadata: {
    title: userInput.inventionTitle,
    field: userInput.technicalField,
    createdAt: new Date(),
  },
})
```

## 最佳实践

### 1. 向量维度管理

**原则**：所有向量必须使用相同维度

```typescript
// ✅ 正确：统一使用 1024 维
const bgeClient = createBGEM3Client({
  apiKey: 'xj781102@',
})

const vectorStore = new PostgresVectorStore({
  vectorDimension: 1024, // 必须匹配 BGE-M3
})

// ❌ 错误：维度不匹配
const vectorStore = new PostgresVectorStore({
  vectorDimension: 1536, // 与 BGE-M3 不匹配！
})
```

### 2. Token 窗口配置

**推荐配置**（DeepSeek）：

```typescript
const tokenWindow = createTokenWindowManager({
  maxTokens: 4000, // DeepSeek 上下文窗口
  reservedTokens: 500, // 预留响应空间
  enableSummary: true, // 启用语义摘要
  compressionRatio: 0.6, // 目标压缩到 60%
})
```

**推荐配置**（通义千问）：

```typescript
const tokenWindow = createTokenWindowManager({
  maxTokens: 8000, // Qwen 长上下文
  reservedTokens: 1000, // 预留更多空间
  enableSummary: true,
})
```

### 3. 缓存策略

**向量化缓存**：

```typescript
// 自动缓存重复文本
const bgeClient = createBGEM3Client()

// 第一次调用（生成向量，100ms）
await bgeClient.embed('专利撰写')

// 第二次调用（使用缓存，<1ms）
await bgeClient.embed('专利撰写') // 快 100 倍
```

**数据库连接缓存**：

```typescript
// 使用连接池
const vectorStore = new PostgresVectorStore({
  databaseUrl: 'postgres://yunpat:yunpat123@localhost:5432/yunpat?pool_size=10',
})
```

### 4. 错误处理

**优雅降级**：

```typescript
async function safeSearch(query: string) {
  try {
    // 尝试语义搜索
    const results = await vectorStore.search(queryEmbedding, 5)
    return { results, method: 'semantic' }
  } catch (error) {
    console.warn('语义搜索失败，降级到关键词搜索')

    // 降级到关键词搜索
    const results = await keywordSearch(query)
    return { results, method: 'keyword' }
  }
}
```

## 性能优化

### 1. 批量操作

```typescript
// ❌ 慢：逐个处理
for (const doc of documents) {
  await vectorStore.upsert(doc) // 100ms × 100 = 10 秒
}

// ✅ 快：批量处理
await vectorStore.upsertBatch(documents) // 2 秒（快 5 倍）
```

### 2. 元数据预过滤

```typescript
// ❌ 慢：先计算相似度再过滤
const results = await vectorStore.search(queryEmbedding, 100)
const filtered = results.filter((r) => r.metadata.type === 'patent')

// ✅ 快：先过滤再计算相似度
const results = await vectorStore.search(queryEmbedding, 100, {
  types: ['patent'], // 先缩小范围
})
```

### 3. 相似度阈值

```typescript
// 高精度场景（专利检索）
const threshold = 0.8

// 宽松场景（推荐系统）
const threshold = 0.6

// 应用
const results = await vectorStore.search(queryEmbedding, 10, {
  threshold,
})
```

### 4. 定期归档

```typescript
// 归档 30 天前的数据
const archivedCount = await vectorStore.archiveOldMemories(30)
console.log(`已归档 ${archivedCount} 条记忆`)
```
