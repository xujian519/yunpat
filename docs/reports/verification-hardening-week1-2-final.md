## 🔧 优化详情

### 1. 测试超时优化（packages/core/vitest.config.ts）

**修改内容**:
```diff
-   testTimeout: 120000,  // 120 秒
+   testTimeout: 180000,  // 180 秒（3 分钟）
-   maxConcurrency: 2,  // 并发数为 2
+   maxConcurrency: 1,  // 降为 1，避免 API 速率限制
```

**优化原因**:
1. **真实 LLM API 响应时间**: 某些 LLM 提供商响应较慢，120 秒可能不足
2. **API 速率限制**: 降低并发数从 2 到 1，避免触发 API 速率限制
3. **稳定性提升**: 更保守的超时设置提高测试稳定性
