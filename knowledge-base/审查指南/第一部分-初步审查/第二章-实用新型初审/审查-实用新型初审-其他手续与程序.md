# 审查-实用新型初审-其他手续与程序

> **来源：** 《专利审查指南》第一部分第二章 §4 + 第一章相应条款
> **对应法条：** 同第一章相应条款
> **最新修订：** 2023年修订
> **关联页面：** [[审查-发明初审-其他手续与费用]]、[[审查-实用新型初审-申请文件审查]]

---

## 核心要点

实用新型专利申请的其他手续和程序审查与发明专利基本一致，均适用第一章的相应规定，包括：委托专利代理机构、要求优先权、不丧失新颖性的公开、撤回专利申请声明、著录项目变更等。由于实用新型没有实质审查程序，因此不涉及实质审查请求和提前公布声明。实用新型经初步审查合格即授权，程序流程相对简洁，但代理人和申请人仍需注意各环节的期限、费用和衔接要求。

---

## 审查标准

### 一、委托专利代理机构（§4.1）

适用本部分第一章第6.1节的规定。

**要点**：
- 外国人、港澳台申请人（在内地无居所/营业所的）必须委托代理机构。
- 中国内地单位和个人可以委托代理机构。
- 委托的代理机构仅限一家，代理师不得超过两名。

**实用新型初审中的特殊注意**：

在实用新型初审中，审查员发现上述申请人申请专利和办理其他专利事务时未委托专利代理机构的，应当发出审查意见通知书，通知申请人在指定期限内答复。申请人在指定期限内未答复的，其申请被视为撤回；申请人陈述意见或者补正后，仍然不符合规定的，该专利申请应当被驳回。

### 二、要求优先权（§4.2）

适用本部分第一章第6.2节的规定。

**要点**：
- 外国优先权：发明/实用新型在先申请之日起12个月内。
- 本国优先权：发明/实用新型在先申请之日起12个月内。
- 在先申请文件副本在优先权日起16个月内提交（外国优先权）。
- 本国优先权：在先申请自在后申请提出之日起即视为撤回。

**实用新型优先权的特殊规则**：

| 优先权类型 | 在先申请类型 | 在后申请类型 | 说明 |
|-----------|-------------|-------------|------|
| 外国优先权 | 发明或实用新型 | 实用新型 | 允许 |
| 本国优先权 | 发明 | 实用新型 | 允许 |
| 本国优先权 | 实用新型 | 实用新型 | 允许 |
| 本国优先权 | 外观设计 | 实用新型 | 不允许 |

### 三、不丧失新颖性的公开（§4.3）

适用本部分第一章第6.3节的规定。

**四种情形**（申请日前6个月内）：
1. 国家紧急状态下为公共利益首次公开。
2. 中国政府主办或承认的国际展览会上首次展出。
3. 规定的学术会议或技术会议上首次发表。
4. 他人未经申请人同意泄露其内容。

**证明材料要求**：
- 国际展览会证明材料应当由展览会主办单位或组委会出具，注明展期、地点、名称及展出日期、形式和内容；
- 学术会议证明材料应当由国务院主管部门或全国性学术团体出具；
- 他人泄露的证明材料应注明泄露日期、方式、内容，并由证明人签字或盖章。

### 四、撤回专利申请声明（§4.4）

适用本部分第一章第6.6节的规定。

**要点**：
- 授予专利权之前可随时撤回。
- 应提交全体申请人签字或盖章的声明。
- 撤回不得附有任何条件。

**撤回后的处理**：
- 撤回专利申请的生效日为手续合格通知书的发文日；
- 对于已公布的实用新型专利申请（实用新型无公布程序，故不涉及），但如涉及同日申请的发明专利已公布，撤回实用新型后发明继续审查；
- 申请人无正当理由不得要求撤销撤回声明；但在申请权非真正拥有人恶意撤回后，真正拥有人可要求撤销（需提交生效法律文书）。

### 五、著录项目变更（§4.5）

适用本部分第一章第6.7节的规定。

**要点**：
- 申请人/专利权人变更需提交证明文件。
- 发明人变更需在规定期限内提出。
- 涉及外国人的权利转让需技术出口许可证/登记证。

**著录项目变更的主要类型及要求**：

| 变更类型 | 所需证明文件 |
|---------|-------------|
| 个人更名 | 户籍管理部门证明 |
| 企业更名 | 工商行政管理部门证明 |
| 权利转移（转让/赠与） | 双方签字的转让合同或赠与合同 |
| 权利转移（权属纠纷） | 法院判决书/调解书、仲裁裁决书或调解书 |
| 权利转移（涉外转让） | 技术出口许可证或登记证 + 转让合同 |
| 发明人变更 | 全体申请人和变更前后全体发明人签字的证明 |
| 代理机构变更 | 解除委托声明 + 新委托书 |

### 六、实用新型与发明专利程序的差异

| 程序事项 | 发明专利 | 实用新型专利 |
|---------|---------|-------------|
| 实质审查请求 | 需要在3年内提出 | 无实质审查程序 |
| 提前公布声明 | 适用 | 不适用（无公布程序） |
| 新颖性/创造性审查 | 在实质审查中进行 | 在初步审查中进行 |
| 保护期限 | 20年（申请日起） | 10年（申请日起） |
| 授权后程序 | 公布→实质审查→授权 | 初步审查→直接授权 |
| 审查周期 | 较长（通常2-4年） | 较短（通常6-12个月） |
| 授权后稳定性 | 经过实质审查，相对稳定 | 未经实质审查，稳定性较弱 |

### 七、实用新型初审的程序流程

实用新型专利申请的初步审查遵循以下程序：

```
受理 → 初步审查 → [补正/审查意见] → 初步审查合格 → 授予专利权通知
   ↓                      ↓
不受理/视为未提出    驳回决定（经两次补正仍未消除缺陷，或陈述意见后仍未消除实质缺陷）
```

#### 7.1 授予专利权通知

实用新型专利申请经初步审查没有发现驳回理由的，审查员应当作出授予实用新型专利权通知。能够授予专利权的包括：
- 不需要补正就符合初步审查要求的专利申请；
- 经过补正符合初步审查要求的专利申请。

授予专利权通知书除收件人信息、著录项目外，还应指明授权所依据的文本和实用新型名称。审查员依职权修改的，还应当写明依职权修改的内容。

#### 7.2 申请文件的补正

初步审查中，对于申请文件存在可以通过补正克服的缺陷的专利申请，审查员应当进行全面审查，并发出补正通知书。

补正通知书应包括：
- 指出补正通知书所针对的是申请人何时提交的何种文件；
- 明确具体地指出申请文件中存在的缺陷，并指出其不符合的条款；
- 明确具体地说明审查员的倾向性意见和可能的建议；
- 指定答复期限；
- 提示补正时的文件种类。

#### 7.3 明显实质性缺陷的处理

初步审查中，对于申请文件存在不可能通过补正方式克服的明显实质性缺陷的专利申请，审查员应当发出审查意见通知书。

审查意见通知书应包括：
- 指出审查意见通知书所针对的是申请人何时提交的何种文件；
- 明确具体地指出申请文件中存在的缺陷，并指出其不符合的条款，必要时结合证据分析；
- 说明审查员准备驳回的倾向性意见；
- 指定答复期限。

#### 7.4 通知书的答复

申请人在收到补正通知书或者审查意见通知书后，应当在指定的期限内补正或者陈述意见。申请人对专利申请进行补正的，应当提交补正书和相应修改文件替换页。对申请文件的修改，应当针对通知书指出的缺陷进行。修改的内容不得超出申请日提交的说明书和权利要求书记载的范围。

申请人期满未答复的，审查员应当根据情况发出视为撤回通知书或者其他通知书。

#### 7.5 申请的驳回

申请文件存在明显实质性缺陷，在审查员发出审查意见通知书后，经申请人陈述意见或者修改后仍然没有消除的，或者申请文件存在形式缺陷，审查员针对该缺陷已发出过两次补正通知书，经申请人陈述意见或者补正后仍然没有消除的，审查员可以作出驳回决定。

驳回决定正文应当包括案由、驳回的理由和决定三部分内容。

### 八、费用审查

实用新型专利申请的费用审查适用本部分第一章及专利法实施细则的相关规定。

#### 8.1 实用新型涉及的主要费用

| 费用种类 | 缴纳期限 | 未缴纳后果 |
|---------|---------|-----------|
| 申请费 | 申请日起两个月内或收到受理通知书之日起15日内 | 视为撤回 |
| 优先权要求费 | 缴纳申请费同时 | 视为未要求优先权 |
| 申请附加费 | 申请日起两个月内或收到受理通知书之日起15日内 | 视为撤回 |
| 著录事项变更费 | 提出请求之日起一个月内 | 视为未提出变更 |
| 专利权评价报告请求费 | 提出请求时 | 视为未提出 |
| 年费 | 授权当年起每年申请日前 | 专利权终止 |

#### 8.2 费用减缴

符合专利收费减缴办法规定的申请人，可以请求减缴部分专利收费。实用新型专利申请的申请费和年费均可减缴。

**费用减缴的条件**（满足其一即可）：
- 上年度月均收入低于5000元（年6万元）的个人；
- 上年度企业应纳税所得额低于100万元的企业；
- 事业单位、社会团体、非营利性科研机构。

两个或两个以上个人或单位为共同专利申请人或共有专利权人的，应当分别符合上述条件。

### 九、期限审查

实用新型专利申请涉及的期限主要包括：

#### 9.1 法定期限

| 期限事项 | 期限 | 起算日 |
|---------|------|--------|
| 缴纳申请费 | 2个月 | 申请日或收到受理通知书之日 |
| 主动修改 | 2个月 | 申请日 |
| 答复补正通知书 | 2个月 | 收到通知书之日 |
| 答复审查意见通知书 | 2个月 | 收到通知书之日 |
| 提出复审请求 | 3个月 | 收到驳回决定之日 |
| 分案申请（收到授权通知书后） | 2个月 | 收到授予专利权通知书之日 |
| 分案申请（收到驳回决定后） | 3个月 | 收到驳回决定之日 |

#### 9.2 期限的延长与恢复

- **延长期限请求**：申请人因正当理由难以在指定期限内作出答复的，可以提出延长期限请求；
- **权利恢复**：因不可抗拒事由或其他正当理由耽误期限导致专利申请被视为撤回的，可以在规定期限内提出恢复权利请求；
- **视为未要求优先权的恢复**：符合特定情形的，可依据专利法实施细则第6条请求恢复。

### 十、与其他程序的衔接

#### 10.1 复审程序

因不符合专利法及其实施细则的规定，专利申请被驳回，申请人对驳回决定不服的，可以在规定的期限内提出复审请求。对复审请求的前置审查及复审后的处理，参照本指南第二部分第八章第8节的规定。

#### 10.2 分案申请

适用第一章§5.1的规定。分案申请的类别应与原申请一致（均为实用新型）。分案申请的各种法定期限从原申请日起算。已经届满或不足两个月的期限，可自分案申请递交日起两个月内补办。

**分案申请的时机**：
- 最迟在收到授予专利权通知书之日起两个月期限届满前提出；
- 收到驳回决定之日起三个月内（不论是否提出复审请求）可提出；
- 复审期间、收到复审决定之日起三个月内、行政诉讼期间均可提出。

#### 10.3 专利权评价报告

实用新型专利授权后，专利权人或利害关系人可以请求国家知识产权局作出专利权评价报告。专利权评价报告是人民法院或管理专利工作的部门审理、处理专利侵权纠纷的证据，主要用于评估实用新型专利权的稳定性。

**请求专利权评价报告的主体及时机**：
- 专利权人（包括共有专利权的部分专利权人）；
- 利害关系人（包括专利实施独占许可合同的被许可人、由专利权人授予起诉权的专利实施普通许可合同的被许可人）；
- 在授权公告后即可提出请求。

#### 10.4 同日申请的协调

同一申请人在同日（仅指申请日）对同样的发明创造既申请实用新型专利又申请发明专利的，应当在申请时分别说明对同样的发明创造已申请了另一专利。

在实用新型授权后、发明专利审查过程中，若发明专利也符合授权条件，申请人可以通过放弃实用新型专利权来保留发明专利权。

## 实务应对

### 代理人视角：实用新型手续办理注意点

1. **无实质审查程序**：实用新型经初步审查合格即授权，没有公布和实质审查阶段。因此不需要提交实质审查请求，也没有提前公布声明。代理人应告知申请人，实用新型授权较快但未经实质审查，授权后可能被无效。

2. **优先权**：实用新型的优先权规则与发明专利相同，但实用新型只能要求发明或实用新型在先申请的优先权，不能要求外观设计在先申请的优先权（作为本国优先权基础时）。

3. **分案申请**：适用第一章§5.1的规定。分案申请的类别应与原申请一致。分案申请的各种法定期限从原申请日起算。已经届满或不足两个月的期限，可自分案申请递交日起两个月内补办。

4. **期限计算**：分案申请的各种法定期限从原申请日起算。已经届满或不足两个月的期限，可自分案申请递交日起两个月内补办。代理人应注意期限计算，避免因期限届满而导致权利丧失。

5. **费用**：分案申请视为一件新申请收取各种费用。实用新型专利授权后需按时缴纳年费，逾期未缴纳的，专利权将终止。代理人应建立年费监控机制，提醒专利权人及时缴费。

6. **主动修改时机**：申请人可以自申请日起两个月内对实用新型专利申请文件主动提出修改。超过两个月的主动修改，如果消除了原申请文件存在的缺陷且具有授权前景，可以接受；否则视为未提出。代理人应把握两个月的主动修改窗口期。

7. **答复策略**：收到补正通知书或审查意见通知书后，应在指定期限内答复。对于形式缺陷，应尽快补正；对于实质缺陷，应充分陈述意见并提供支持证据。若第一次答复未消除缺陷，可能收到第二次补正通知书或审查意见通知书，若仍未消除，则可能面临驳回。

8. **专利权评价报告的运用**：在涉及实用新型专利的侵权诉讼中，专利权评价报告是重要的证据材料。代理人应在提起侵权诉讼前或同时，协助专利权人申请专利权评价报告，以评估专利权的稳定性。

## 审查要点速查

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 外国申请人未委托代理机构 | 发出审查意见通知书，逾期不补正则驳回 | 法第18条 |
| 实用新型提交实质审查请求 | 不适用（无实质审查程序） | 法第35条 |
| 优先权声明未在请求书中提出 | 视为未要求优先权 | 法第30条 |
| 撤回申请声明未附全体申请人签字 | 发出视为未提出通知书 | 细则第41条 |
| 著录项目变更涉及外国人受让方 | 需提交技术出口许可/登记证 | 法第10条 |
| 分案申请类别与原申请不一致 | 不符合规定 | 细则第48条 |
| 主动修改超过两个月期限 | 如消除缺陷且有授权前景可接受，否则视为未提出 | 细则第57条 |
| 两次补正仍未消除形式缺陷 | 可驳回 | 细则第50条 |
| 审查意见后未消除实质缺陷 | 可驳回 | 细则第50条 |
| 未按期缴纳申请费 | 视为撤回 | 细则第112条 |
| 未按期缴纳年费 | 专利权终止 | 法第44条 |
| 期满未答复通知书 | 视为撤回 | 细则第6条 |
