## 💡 核心技术栈

### 架构设计

```
┌─────────────────────────────────────────────────────────┐
│  应用层 (PatentWriterAgent, etc.)                       │
└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────┐
│  记忆层 (Memory Layer)                                   │
│  ┌──────────────────────────────────────────────────┐   │
│  │ BGE-M3 集成                                      │   │
│  │ - 文本向量化 (1024维)                             │   │
│  │ - 智能缓存                                        │   │
│  └──────────────────────────────────────────────────┘   │
│  ┌──────────────────────────────────────────────────┐   │
│  │ PostgreSQL + pgvector                           │   │
│  │ - 向量存储 (HNSW索引)                             │   │
│  │ - 图存储 (递归CTE)                                │   │
│  │ - 元数据过滤 (JSONB)                              │   │
│  └──────────────────────────────────────────────────┘   │
│  ┌──────────────────────────────────────────────────┐   │
│  │ Token 窗口管理                                   │   │
│  │ - 滑动窗口算法                                    │   │
│  │ - 上下文压缩                                      │   │
│  │ - Token 预测                                      │   │
│  └──────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────┘
```

### 技术选型

| 组件 | 技术 | 理由 |
|------|------|------|
| **向量数据库** | PostgreSQL + pgvector | 成熟稳定、ACID、无需额外服务 |
| **图数据库** | SQL/PGQ (PostgreSQL) | 统一技术栈、降低复杂度 |
| **嵌入模型** | BGE-M3 (1024维) | 中文优化、性能好、本地部署 |
| **ORM** | Drizzle ORM | 类型安全、性能好 |
| **索引** | HNSW (余弦距离) | 高性能近似最近邻搜索 |
| **缓存** | LRU (内存) | 简单高效、命中率25%+ |

### 使用示例

#### 1. BGE-M3 文本向量化

```typescript
import { createBGEM3Client } from '@yunpat/core';

const bgeClient = createBGEM3Client({
  apiKey: 'xj781102@',
});

const embedding = await bgeClient.embed('专利撰写的关键在于权利要求书的撰写');
console.log(embedding.length); // 1024
```

#### 2. Token 窗口压缩

```typescript
import { createTokenWindowManager } from '@yunpat/core';

const tokenWindow = createTokenWindowManager({
  maxTokens: 4000,
  enableSummary: true,
});

const { messages, stats } = await tokenWindow.slideWindow(dialogueHistory);
console.log(`Token 降低: ${(stats.compressionRatio * 100).toFixed(2)}%`);
```

#### 3. 语义检索

```typescript
import { PostgresVectorStore } from '@yunpat/core';

const vectorStore = new PostgresVectorStore({
  databaseUrl: 'postgres://yunpat:yunpat123@localhost:5432/yunpat',
  vectorDimension: 1024,
});

await vectorStore.initialize();

// 添加文档
const embedding = await bgeClient.embed('专利内容...');
await vectorStore.upsert({
  type: 'patent',
  content: '专利内容...',
  embedding: embedding,
  metadata: { patentId: 'CN123456' },
});

// 检索
const queryEmbedding = await bgeClient.embed('如何撰写专利？');
const results = await vectorStore.search(queryEmbedding, 5);
```
