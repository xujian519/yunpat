## 使用示例

```typescript
import { InventionUnderstandingAgent } from '@yunpat/agents/invention'

// 创建 Agent
const agent = new InventionUnderstandingAgent({
  name: 'invention-understanding',
  description: '发明理解智能体',
  enableKnowledgeGraph: true,
})

// 使用案例
const result = await agent.execute(ceramicValveExample)

console.log('发明构思数量:', result.inventionConcepts.length)
result.inventionConcepts.forEach((concept, index) => {
  console.log(`\n三元组 ${index + 1}:`)
  console.log(`  问题: ${concept.technicalProblem}`)
  console.log(`  特征: ${concept.keyFeatures.join(', ')}`)
  console.log(`  效果: ${concept.technicalEffects.join(', ')}`)
  console.log(`  置信度: ${(concept.confidence * 100).toFixed(0)}%`)
})
```

## 预期输出验证

每个案例都应满足：

✅ **多组三元组**：至少2组问题-特征-效果三元组
✅ **逻辑一致性**：问题-特征-效果一一对应
✅ **具体性**：技术特征具体，技术效果可量化
✅ **对比性**：技术效果与现有技术有明确对比
✅ **完整性**：包含所有必需字段
