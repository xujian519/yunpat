# 什么是合法来源抗辩？其适用条件、举证责任及司法认定标准是什么？

## 卡片内容

### 核心回答

合法来源抗辩是一种抗辩权利，允许销售者、使用者或许诺销售者在特定条件下，免除其对专利侵权产品的赔偿责任。该抗辩的核心在于证明所销售或使用的侵权产品具有“合法来源”，并且该行为人“不知道”该产品是未经专利权人许可而制造的。

**抗辩的法律效果**：如果抗辩成立，行为人可以免除赔偿责任，但请注意，该抗辩**不免除**其停止侵权的责任。

### 法律依据

**《专利法》（2008）第70条**明确规定了该抗辩的法律基础：
> 为生产经营目的使用、许诺销售或者销售不知道是未经专利权人许可而制造并售出的专利侵权产品，能证明该产品合法来源的，不承担赔偿责任。

**适用主体限制**：该抗辩仅适用于**使用者、许诺销售者或销售者**。制造专利产品的行为人即使不知情，仍需承担损害赔偿责任。

### 司法实践中的认定标准

**1. 主观要件（“不知道”）**
行为人必须处于一种“不知道”的状态，即确实不知道该产品是侵权产品，且没有合理的理由应当知道。这通常指向行为人具有善意。

**2. 客观要件（“合法来源”）**
行为人必须能够提供证据，证明该产品是通过正当的商业渠道获得的，例如从正规供应商处采购或拥有合法的进货凭证等。

**3. 举证责任**
主张合法来源抗辩的销售者或使用者，需要承担举证责任，包括证明产品的来源渠道合法、提供相应的交易凭证（如合同、发票等），以及证明交易行为的真实发生。

**4. 认定“不知道”的因素**
法院在判断行为人是否“不知道”时，会综合考虑以下因素：
*   产品的来源渠道是否正规。
*   产品价格是否明显低于市场价格。
*   行为人是否在收到侵权通知后仍继续销售。
*   行为人作为专业经营者是否尽到了合理的审查义务。

**5. 加盟店与普通零售商的区别**
在实践中，法院可能根据销售者的类型进行区分：
*   **特许加盟店**：由于与品牌方有紧密合作，法院可能对其注意义务提出更高的要求，其抗辩的成立可能受到更严格的审查。
*   **普通零售商**：通常只需要证明其从上游供应商处合法采购即可。

### 典型案例分析

**案例：广东联邦家私集团有限公司 v. 陈军等（最高人民法院（2009）民申字第1495号）**

该案例强调了合法来源抗辩的举证责任和证明标准的严格性。法院认定，仅是“主张”合法来源抗辩是不够的，必须提供足以“证明”的证据。在涉及特许加盟店的情况下，法院对合法来源证据的审查会更加严格，以确保抗辩的有效性。

### 判断流程

**合法来源抗辩判断流程**
1. **主体审查**：
    *   制造者：不适用合法来源抗辩。
    *   销售者/使用者/许诺销售者：继续判断。
2. **主观要件审查**：
    *   是否知道是侵权产品？
        *   是 $\rightarrow$ 抗辩不成立。
        *   否 $\rightarrow$ 继续判断。
    *   是否应当知道？（合理注意义务）
        *   是 $\rightarrow$ 抗辩可能不成立。
        *   否 $\rightarrow$ 继续判断。
3. **客观要件审查**：
    *   能否证明合法来源？
        *   否 $\rightarrow$ 抗辩不成立。
        *   是 $\rightarrow$ 抗辩成立。

4. **法律效果**：
    *   不承担赔偿责任。
    *   但仍需停止侵权。
