## 🚀 部署指南

### 快速开始

```bash
# 1. 配置环境变量
cp .env.example .env
# 编辑.env，设置API密钥

# 2. 一键部署
make deploy

# 3. 验证部署
make health
make status
```

### 服务访问

| 服务        | 地址                  | 说明                   |
| ----------- | --------------------- | ---------------------- |
| Grafana监控 | http://localhost:3001 | 监控面板 (admin/admin) |
| Prometheus  | http://localhost:9090 | 指标收集               |

### 常用命令

```bash
make deploy     # 完整部署
make start      # 启动服务
make stop       # 停止服务
make restart    # 重启服务
make logs       # 查看日志
make status     # 查看状态
make health     # 健康检查
```
