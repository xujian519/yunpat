# 审查-单一性-基本概念与审查原则

> **来源：** 《专利审查指南》第二部分第六章 §1 引言 + §2.1-2.2 单一性的基本概念与审查
> **对应法条：** 《专利法》第31条第1款；《专利法实施细则》第39条
> **最新修订：** 无修改
> **关联页面：** [[单一性-法条-合案申请与分案]]、[[审查-单一性-特定技术特征]]、[[审查-单一性-审查示例]]

---

## 核心要点

单一性要求一件专利申请应当限于一项发明或者实用新型，属于一个**总的发明构思**的两项以上发明或实用新型才能合案申请。"总的发明构思"体现为权利要求中包含使各项发明在技术上相互关联的**相同或者相应的特定技术特征**，其中特定技术特征是指每一项发明作为整体对**现有技术作出贡献**的技术特征。缺乏单一性不影响专利的有效性，但会导致审查中的修改和分案要求。

---

## 审查标准

### 一、引言（§1）

专利申请应当符合专利法及其实施细则有关单一性的规定：
- 《专利法》第31条第1款及其实施细则第39条对发明或者实用新型专利申请的单一性作了规定；
- 《专利法实施细则》第48条、第49条对不符合单一性的专利申请的分案及其修改作了规定。

**适用范围：**
- 本章的单一性规定主要涉及发明专利申请，其中基本概念和原则也适用于实用新型专利申请；
- 外观设计专利申请单一性的审查，适用第一部分第三章第9节；
- 化学领域发明专利申请单一性审查的特殊问题，适用本部分第十章第8节。

### 二、单一性的基本概念（§2.1）

#### 2.1 单一性要求（§2.1.1）

根据《专利法》第31条第1款，单一性是指：

> 一件发明或者实用新型专利申请应当**限于一项发明或者实用新型**，属于一个**总的发明构思**的两项以上发明或者实用新型，可以作为一件申请提出。

也就是说，如果一件申请包括几项发明或者实用新型，则只有在所有这几项发明或者实用新型之间有一个**总的发明构思**使之相互关联的情况下才被允许。

**单一性要求的主要原因：**

| 原因 | 说明 |
|------|------|
| 经济原因 | 防止申请人只支付一件专利的费用而获得几项不同发明的保护 |
| 技术原因 | 便于专利申请的分类、检索和审查 |

**重要规则：缺乏单一性不影响专利的有效性，因此缺乏单一性不应当作为专利无效的理由。**

#### 2.2 总的发明构思（§2.1.2）

根据《专利法实施细则》第39条，属于一个总的发明构思的两项以上发明或者实用新型，应当在**技术上相互关联**，包含一个或者多个**相同或者相应的特定技术特征**。

**特定技术特征的定义：**

特定技术特征是指每一项发明或者实用新型作为整体，对**现有技术作出贡献**的技术特征。

#### 2.3 关键概念解析

**"技术上相互关联"：** 属于一个总的发明构思的两项以上发明在技术上必须相互关联，这种相互关联是以**相同或者相应的特定技术特征**表示在权利要求中的。

**"特定技术特征"的理解：**

| 要点 | 说明 |
|------|------|
| 专门概念 | 特定技术特征是专门为评定专利申请单一性而提出的一个概念 |
| 贡献标准 | 体现发明对现有技术作出贡献的技术特征 |
| 新创性关联 | 使发明相对于现有技术具有新颖性和创造性的技术特征 |
| 整体考量 | 应当从每一项要求保护的发明的整体上考虑后加以确定 |

**因此，专利法第31条第1款所称的"属于一个总的发明构思"就是指具有相同或者相应的特定技术特征。**

### 三、判断框架

```
单一性判断流程：

1. 确定各项权利要求中记载的技术方案
          ↓
2. 识别各项技术方案之间的相同或相应技术特征
          ↓
3. 判断这些技术特征是否为"特定技术特征"
   （即是否体现对现有技术的贡献）
          ↓
4. 如果存在相同或相应的特定技术特征 → 具有单一性
   如果不存在 → 不具有单一性
```

### 四、单一性审查原则详解（§2.2.1）

审查员在审查发明专利申请的单一性时，应当遵循以下基本原则：

**原则一：以权利要求为基础**

根据专利法第31条第1款及其实施细则第39条所规定的内容，判断一件专利申请中要求保护的两项以上发明是否满足发明单一性的要求，就是要看权利要求中记载的技术方案的实质性内容是否属于一个总的发明构思，即判断这些权利要求中是否包含使它们在技术上相互关联的一个或者多个相同或者相应的特定技术特征。这一判断是根据权利要求的内容来进行的，必要时可以参照说明书和附图的内容。

**原则二：六种可允许的权利要求组合方式**

属于一个总的发明构思的两项以上发明的权利要求可以按照以下六种方式之一撰写；但是，不属于一个总的发明构思的两项以上独立权利要求，即使按照所列举的六种方式中的某一种方式撰写，也不能允许在一件申请中请求保护：

| 序号 | 组合方式 | 说明 |
|------|---------|------|
| (i) | 不能包括在一项权利要求内的两项以上产品或者方法的同类独立权利要求 | "同类"指类型相同（均为产品或均为方法） |
| (ii) | 产品和专用于制造该产品的方法的独立权利要求 | "专用"方法使用的结果就是获得该产品 |
| (iii) | 产品和该产品的用途的独立权利要求 | 用途必须由该产品的特定性能决定 |
| (iv) | 产品、专用于制造该产品的方法和该产品的用途的独立权利要求 | 三者的组合 |
| (v) | 产品、专用于制造该产品的方法和为实施该方法而专门设计的设备的独立权利要求 | 设备对现有技术的贡献须与方法对应 |
| (vi) | 方法和为实施该方法而专门设计的设备的独立权利要求 | 设备须能实施该方法且贡献对应 |

**原则三：六种方式并非穷举**

以上列举了六种可允许包括在一件申请中的两项以上同类或不同类独立权利要求的组合方式及适当的排列次序，但是，所列六种方式并非穷举，也就是说，在属于一个总的发明构思的前提下，除上述排列组合方式外，还允许有其他的方式。

**原则四：权利要求排列形式不影响单一性判断**

不同类独立权利要求之间是否按照引用关系撰写，只是形式上的不同，不影响它们的单一性。例如，与一项产品A独立权利要求相并列的一项专用于制造该产品A的方法独立权利要求，可以写成"权利要求1的产品A的制造方法……"也可以写成"产品A的制造方法……"

**原则五：并列技术方案的单一性标准一致**

评定两项以上发明是否属于一个总的发明构思，无须考虑这些发明是分别在各自的独立权利要求中要求保护，还是在同一项权利要求中作为并列选择的技术方案要求保护。对于上述两种情况，均应当按照相同的标准判断其单一性。后一种情况经常出现在马库什权利要求中，关于马库什权利要求单一性的审查，适用本部分第十章第8.1节的规定。此外，权利要求的排列次序也不应当影响发明单一性的判断。

**原则六：独立权利要求之间的单一性优先审查**

一般情况下，审查员只需要考虑独立权利要求之间的单一性，从属权利要求与其所从属的独立权利要求之间不存在缺乏单一性的问题。但是，在遇有形式上为从属权利要求而实质上是独立权利要求的情况时，应当审查其是否符合单一性规定。

如果一项独立权利要求由于缺乏新颖性、创造性等理由而不能被授予专利权，则需要考虑其从属权利要求之间是否符合单一性的规定。

### 五、特定技术特征的认定标准

#### 5.1 认定方法

特定技术特征的认定需要经过以下步骤：

1. **确定现有技术**：通过检索确定最接近的现有技术；
2. **找出区别特征**：将发明的技术方案与现有技术进行比较，找出区别特征；
3. **判断贡献性**：判断这些区别特征是否使发明相对于现有技术具有新颖性和创造性；
4. **确认特定技术特征**：那些体现对现有技术作出贡献的区别特征即为特定技术特征。

#### 5.2 "相同"与"相应"的特定技术特征

| 类型 | 含义 | 示例 |
|------|------|------|
| 相同的特定技术特征 | 两项发明中完全相同的体现对现有技术作出贡献的技术特征 | 两项权利要求均包含"特征A"，且特征A均体现对现有技术的贡献 |
| 相应的特定技术特征 | 两项发明中的技术特征虽然不同，但存在密切的技术关联，缺一不可 | 插头和插座中相互对应的特征 |

#### 5.3 检索前后的单一性判断

某些申请的单一性可以在检索现有技术之前确定，而某些申请的单一性则只有在考虑了现有技术之后才能确定：

- **检索前可判断**：当一件申请中不同的发明明显不具有一个总的发明构思时，则在检索之前即可判断其缺乏单一性。例如一件申请中包括了除草剂和割草机两项独立权利要求，由于两者之间没有相同或者相应的技术特征，更不可能有相同或者相应的特定技术特征，因而明显不具有单一性。

- **检索后才能判断**：由于特定技术特征是体现发明对现有技术作出贡献的技术特征，是相对于现有技术而言的，只有在考虑了现有技术之后才能确定，因此，不少申请的单一性问题常常要在检索之后才能作出判断。

当申请与现有技术比较后，在否定了第一独立权利要求的新颖性或创造性的情形下，与其并列的其余独立权利要求之间是否还属于一个总的发明构思，应当重新确定。

### 六、分案申请的时机与策略

#### 6.1 分案的几种情况

一件申请有下列不符合单一性情况的，审查员应当要求申请人对申请文件进行修改（包括分案处理），使其符合单一性要求：

**情况一：原权利要求书中包含不符合单一性规定的两项以上发明**

原始提交的权利要求书中包含不属于一个总的发明构思的两项以上发明的，应当要求申请人将该权利要求书限制至其中一项发明（一般情况是权利要求1所对应的发明）或者属于一个总的发明构思的两项以上的发明，对于其余的发明，申请人可以提交分案申请。

**情况二：修改后增加的独立权利要求与原权利要求书中的发明之间不具有单一性**

在审查过程中，申请人在修改权利要求时，将原来仅在说明书中描述的发明作为独立权利要求增加到原权利要求书中，或者在答复审查意见通知书时修改权利要求，将原来仅在说明书中描述的发明作为独立权利要求替换原独立权利要求，而该发明与原权利要求书中的发明之间缺乏单一性。在此情况下，审查员一般应当要求申请人将后增加或替换的发明从权利要求书中删除。申请人可以对该删除的发明提交分案申请。

**情况三：独立权利要求之一缺乏新颖性或创造性，其余的权利要求之间缺乏单一性**

某一独立权利要求（通常是权利要求1）缺乏新颖性或创造性，导致与其并列的其余独立权利要求之间，甚至其从属权利要求之间失去相同或者相应的特定技术特征，即缺乏单一性，因此需要修改，对于因修改而删除的主题，申请人可以提交分案申请。

#### 6.2 分案申请应当满足的要求

| 要求 | 内容 |
|------|------|
| 文本要求 | 分案申请应当在其说明书的起始部分说明本申请是哪一件申请的分案申请，并写明原申请的申请日、申请号和发明创造名称 |
| 内容限制 | 分案申请的内容不得超出原申请记载的范围 |
| 权利要求书 | 分案以后的原申请与分案申请的权利要求书应当分别要求保护不同的发明 |
| 说明书 | 分案申请的说明书可以仍然是原申请的全部内容，也可以只保留与分案发明相关的内容 |

#### 6.3 分案申请的时机

- 申请人最迟应当在收到专利局对原申请作出授予专利权通知书之日起两个月期限（即办理登记手续的期限）届满之前提出分案申请；
- 对于审查员已发出驳回决定的原申请，自申请人收到驳回决定之日起三个月内，不论申请人是否提出复审请求，均可以提出分案申请；
- 在提出复审请求以后的复审期间、收到复审决定之日起三个月内以及对复审决定不服提起的行政诉讼期间，申请人也可以提出分案申请；
- 针对一件申请，可以提出一件或者一件以上的分案申请；针对一件分案申请还可以以原申请为依据再提出一件或者一件以上的分案申请。

## 实务应对

### 1. 申请撰写时的单一性规划

代理人在撰写专利申请文件时，应从一开始就考虑单一性问题：

- **独立权利要求的设置：** 确保多项独立权利要求之间存在相同或相应的特定技术特征；
- **从属权利要求的布局：** 合理设置从属权利要求，确保独立权利要求被否定后，从属权利要求之间仍能保持单一性；
- **说明书的撰写：** 在说明书中充分描述各项发明之间的技术关联性。

### 2. 特定技术特征的识别

在撰写过程中，代理人应主动识别发明对现有技术作出贡献的技术特征：
- 核心发明点是什么；
- 各项独立权利要求是否共享该核心发明点；
- 如果有不同的发明点，它们之间是否存在技术上的关联。

### 3. 分案策略的提前规划

考虑到单一性可能在审查过程中被质疑，代理人应：
- 在说明书中记载足够多的技术方案，为可能的分案提供支持；
- 预估可能被要求分案的技术方案，提前准备分案申请的文本；
- 了解分案申请的时间要求和内容限制；
- 注意分案申请视为一件新申请收取各种费用。

### 4. 面对单一性质疑的应对

- 审查特定技术特征的认定是否正确；
- 检查审查员是否充分考虑了说明书和附图中的技术关联性；
- 如果确实缺乏单一性，优先保留核心发明，将其余发明作为分案申请；
- 注意：提出分案申请是申请人自愿的行为，审查员只要求申请人将不符合单一性要求的两项以上发明改为一项发明或属于一个总的发明构思的几项发明，至于是否分案由申请人自行决定。

### 5. 单一性审查中的举证

- 若审查员认定缺乏单一性，通常需要说明现有技术的状况以及为什么共同的技术特征不构成特定技术特征；
- 申请人可以通过补充检索报告或技术说明，证明共同特征相对于现有技术具有新颖性和创造性，从而主张该特征为特定技术特征。

## 审查要点速查

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 多项独立权利要求共享核心发明点 | 具有单一性 | §2.1.2 相同的特定技术特征 |
| 多项权利要求之间无任何共同技术特征 | 不具有单一性 | §2.1.2 无技术关联 |
| 共同特征属于惯用技术手段 | 不具有单一性 | §2.1.2 惯用手段不构成特定技术特征 |
| 共同特征是对现有技术的贡献 | 具有单一性 | §2.1.2 特定技术特征 |
| 产品权利要求与专用方法权利要求 | 可能具有单一性 | §2.2.1(ii) 相应的特定技术特征 |
| 缺乏单一性的已授权专利 | 不构成无效理由 | §2.1.1 单一性不影响有效性 |
| 独立权利要求1被否定后，从属权利要求之间可能缺乏单一性 | 需重新判断 | §2.2.1(5) 从属权利要求的单一性 |
| 检索前明显不相关的发明（如除草剂+割草机） | 明显不具有单一性 | §2.2.1(6) 检索前的判断 |
| 形式上从属但实质上独立的权利要求 | 需按独立权利要求审查单一性 | §2.2.1(5) 形式从属实质独立 |
| 分案申请超出原申请记载范围 | 不符合规定 | §3.2 分案申请的内容限制 |
