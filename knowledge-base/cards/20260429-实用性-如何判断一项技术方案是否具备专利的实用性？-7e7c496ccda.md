# 如何判断一项技术方案是否具备专利的实用性？

## 卡片内容

### 核心概念

实用性是专利授权审查中的一项关键性判断标准，旨在确保申请的技术方案是具有实际应用价值和可实施性的。审查员需要根据《专利审查指南》列举的六种典型情形来评估技术方案的实用性。如果技术方案属于以下任何一种情形，则不具备实用性。

### 实用性判断的六大维度

1. **无再现性**
   - **定义**：指所属技术领域的技术人员根据公开的技术内容，无法重复实施专利申请中为解决技术问题所采用的技术方案，或实施结果依赖于随机因素。
   - **区分**：成品率低（实施过程中可能因条件未达标导致成品率低）与无再现性（在确保全部技术条件下，技术人员仍无法重复实现所需结果）存在本质区别。
   - **判断要点**：需考察说明书是否充分公开了实施条件，以及实施过程中是否存在必须依赖的随机因素。

2. **违背自然规律**
   - **定义**：发明或实用新型专利申请应当符合自然规律。违背自然规律的发明不具备实用性。
   - **常见情形**：包括但不限于：
     - **永动机类**：直接违背能量守恒定律（如第一类永动机）。
     - **效率超标**：声称的机械能输出大于输入电能，或效率超过100%的能量转换装置，违反热力学第二定律。
     - **其他违反物理学/化学定律**：如声称接收天空电荷或分离磁场产生能量等。

3. **利用独一无二的自然条件的产品**
   - **定义**：不应由自然条件限定的独一无二的产品构成专利。如果产品自始至终都依赖于特定的自然条件，则不具备实用性。
   - **区别**：即使方法涉及特定地理环境，但如果方法本身不依赖特定自然条件，仍可能具备实用性。

4. **人体或动物体的非治疗目的的外科手术方法**
   - **定义**：以有生命的人或动物为实施对象，但目的并非治疗疾病，则不具备实用性。
   - **典型情形**：
     - **美容手术**：为美容目的实施的外科手术。
     - **摘取物质**：采用外科手术从活体动物身体上摘取物质的方法。
     - **辅助诊断手术**：为辅助诊断而采用的外科手术方法。
   - **例外**：如果方案包含的简单介入性处置步骤个体差异小，且不依赖专业技能，则可能不被认定为不具备实用性。

5. **测量人体或动物体在极限情况下的生理参数的方法**
   - **定义**：测量人体或动物体在极限环境下的生理参数的方法，通常会对人或动物的生命构成威胁，且个体差异大，难以标准化实施。
   - **典型情形**：
     - **体温耐受测量**：通过降低体温来测量对寒冷的耐受程度。
     - **冠状动脉负荷测量**：通过动态观察动脉血压变化来测量冠状动脉代谢机能。
   - **区别**：正常的、可标准化的医学检查方法，即使涉及侵入性操作，如果不会威胁生命，则可能具备实用性。

6. **无积极效果**
   - **定义**：技术方案应当能够产生预期的积极效果。明显无益、脱离社会需要，或明显浪费能源/资源，不具备实用性。
   - **修订说明**：2006年修订前曾列举“严重污染环境、严重浪费能源或者资源、损害人身体健康”等情形，现已调整为更侧重于“明显无益、脱离社会需要”的判断，以避免与《专利法》第5条的客体审查产生重叠。

### 实用性判断流程

判断流程通常按以下顺序进行：
1. **再现性**：检查是否能重复实施。
2. **自然规律**：检查是否违反了物理或化学基本定律。
3. **自然条件**：检查是否依赖于独一无二的自然条件。
4. **外科手术**：检查是否涉及非治疗目的的外科手术。
5. **生理参数测量**：检查是否涉及对生命极限的测量。
6. **积极效果**：检查是否能产生预期的积极效果。

### 关键法律依据

该判断主要依据《专利法》第22条第4款。
