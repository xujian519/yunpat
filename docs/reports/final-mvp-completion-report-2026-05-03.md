## 💡 核心创新

### 1. 垂直切片策略

不是一次性构建所有功能，而是：
- Phase 1: 发明理解 → 可验收
- Phase 2: 检索策略 → 可验收
- Phase 3: 权利要求 → 可验收
- Phase 4: 整合 → 端到端

**优势**: 快速验证、风险可控、反馈及时

### 2. 沉睡能力唤醒

框架已有ApprovalFlow/CheckpointManager，但Agent未调用：
- Phase 0: 集成到Agent基类
- Phase 1-4: 充分利用这些能力
- **结果**: 从"能力丰富但沉睡"到"能力完全激活"

### 3. 人机协作设计

每个关键步骤都有审批点：
- 发明理解确认
- 检索策略确认
- 权利要求确认
- 支持5种操作：通过/修正/补充/重来/取消

### 4. 专业模板系统

使用宝宸知识库提炼的专业模板：
- 01-claims-generation.md（权利要求）
- 02-specification-drafting.md（说明书）
- 确保输出质量符合专利法要求

## 🚀 使用指南

### 快速开始

#### 方式1: 端到端（推荐）

```bash
# 设置API密钥
export DEEPSEEK_API_KEY=your_key

# 运行完整流程
yunpat draft-full \
  --disclosure examples/disclosure-example.md \
  --title "一种基于深度学习的图像识别方法" \
  --field "人工智能" \
  --output patent-application.md
```

#### 方式2: 分步执行

```bash
# Step 1: 发明理解
yunpat draft \
  --title "..." \
  --field "..." \
  --disclosure ... \
  --output invention.json

# Step 2: 检索策略
yunpat search \
  --invention-json invention.json \
  --output search.json

# Step 3: 权利要求
yunpat claims \
  --invention-json invention.json \
  --search-json search.json \
  --output claims.json
```

#### 方式3: 交互式（最佳体验）

```bash
yunpat draft-interactive \
  --title "..." \
  --field "..." \
  --disclosure ...
```

然后按照提示选择操作（y/c/s/r/q）
