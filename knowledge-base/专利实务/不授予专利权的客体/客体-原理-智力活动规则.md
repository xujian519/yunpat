# 客体-原理-智力活动规则

> **来源：** 《专利法：原理与案例（第二版）》第2章 §4.1-4.2，崔国斌著
> **核心法条：** 《专利法》第25条第1款第（2）项
> **关联页面：** [[客体-原理-基本原理]]、[[客体-原理-计算程序]]、[[客体-原理-商业方法]]

---

## 核心要点

智力活动的规则和方法不授予专利权。但如果权利要求中包含了技术特征，就不应当仅依据第25条排除其获得专利权的可能性。

---

## 1. 排除智力活动规则的原因

### 1.1 抽象思想不保护

智力活动的规则和方法本质上属于抽象思想、思维理论，不满足专利法对技术方案的要求。专利法保护的是利用自然规律解决技术问题的技术方案，而不是纯粹的思维活动。

### 1.2 防止垄断思维

如果将智力活动的规则和方法纳入专利保护范围，将导致对人类思维活动的垄断，阻碍知识的传播和交流。这与专利法鼓励创新、促进技术进步的立法目的相悖。

### 1.3 区分技术方案

智力活动的规则和方法通常不涉及技术手段，不解决技术问题，不产生技术效果。它们属于人类思维活动的范畴，不属于专利法意义上的技术方案。

---

## 2. 智力活动规则的范围

### 2.1 基本判断

判断一项权利要求是否属于智力活动规则和方法，应从权利要求的全部内容判断。即使主题名称看似智力活动规则，只要权利要求中包含了技术特征，就不应当依据第25条排除其授权可能性。

### 2.2 典型情形

以下通常被认为是智力活动的规则和方法：
1. **智力训练方法**：如教育方法、记忆方法
2. **游戏方法**：如棋牌游戏规则、体育竞技方法
3. **商业经营方法**：如商业计划、营销策略
4. **管理方法**：如企业管理、项目管理方法
5. **纯粹的数学运算方法**：如数学公式、算法
6. **纯粹的逻辑推理方法**：如推理规则、论证方法

### 2.3 包含技术特征的情形

如果权利要求中包含了技术特征，则可能不再属于智力活动规则的排除范围：

**关键考虑因素：**
1. 是否利用了自然规律？
2. 是否采用了技术手段？
3. 是否解决了技术问题？
4. 是否产生了技术效果？

**示例：**
- 带有计算机系统的教学方法 → 可能包含技术特征
- 利用光学辨识装置的阅读方法 → 可能包含技术特征
- 结合传感器和数据分析的管理方法 → 可能包含技术特征

## 3. 单纯智力活动规则的判断

### 3.1 基本标准

如果要求保护的方法在使用过程中必须直接借助于人的感知力、理解力、决策力或完全依赖于相关经验、态度等人的主观意识，不受自然规律约束，则属于单纯的智力活动规则和方法。

### 3.2 关键特征

**依赖人的因素：**
- 感知力：依赖人的感觉器官
- 理解力：依赖人的思维活动
- 决策力：依赖人的判断和选择
- 经验：依赖人的实践经验
- 态度：依赖人的主观意识

**不受自然规律约束：**
- 不涉及物理、化学等自然规律
- 不涉及技术手段的应用
- 不产生技术效果

---

## 4. 参数指标的质量控制方法

### 4.1 一般原则

对于由参数指标及其阈值选择限定的产品质量控制方法，不能因为包含某些人为设定的指标就一概认定其属于智力活动规则。

### 4.2 判断标准

如果指标不是主观随意选择，而是由产品自身性能决定，则可认定利用了自然规律：

**考虑因素：**
1. **指标来源**：是主观设定还是产品性能决定？
2. **科学依据**：是否有科学原理支撑？
3. **技术效果**：对产品质量、性能是否有技术性作用？
4. **可重复性**：是否可以客观地重复实施？

### 4.3 典型案例

**案例：卡介菌培养基的马铃薯质控方法**

- **争议焦点**：该质量控制方法是否属于智力活动规则？
- **分析**：直链淀粉占总淀粉比例这一参数指标及阈值的选择均以卡介菌生长状态为标准
- **结论**：遵循了自然规律，对技术问题和技术效果发挥着技术性作用，不是人为的任意选择
- **启示**：参数指标选择如果基于产品客观性能而非主观随意设定，则不属于智力活动规则

---

## 5. 利用自然规律的方法

### 5.1 基本原则

利用自然规律对客观世界进行改造的活动，不属于智力活动的规则和方法。

### 5.2 判断要素

**利用自然规律：**
- 涉及物理、化学、生物等自然规律
- 通过技术手段应用自然规律
- 客观规律约束活动过程

**改造客观世界：**
- 改变物质的状态或性质
- 创造新的产品或方法
- 产生实际的技术效果

### 5.3 典型案例

**案例：全球尺度强地震趋势预测方法**

- **争议焦点**：该方法是否属于智力活动规则？
- **分析**：地震发生有其自然规律，该方法利用统计方法从历史地震中统计出相关性的概率分布，再根据相关性这一自然规律推算未来地震概率
- **结论**：利用了自然规律，不属于智力活动规则
- **启示**：基于自然规律的统计分析预测方法，只要利用了自然规律而非纯粹的人为规则，就具有可授权的可能性

---

## 6. 除主题名称外实质为智力活动规则

### 6.1 基本原则

如果权利要求的主题名称为产品，但对其进行限定的全部内容均属于信息的表达、标记或编排方式等规则，且未对该产品的物理结构产生任何影响，则实质上属于智力活动的规则和方法。

### 6.2 判断标准

**主题名称与实质内容的关系：**
- 主题名称：产品
- 限定内容：信息的表达、标记或编排方式
- 物理结构：未发生变化

**关键问题：**
1. 权利要求的实质内容是什么？
2. 是否对产品的物理结构产生影响？
3. 是否包含技术特征？

### 6.3 典型案例

**案例：姓氏定位通讯录**

- **争议焦点**：该产品是否属于智力活动规则？
- **分析**：姓氏拼音检索表及姓氏收录空间设计仅属于特定信息的检索编排方法，通过人的思维、推理实现特定信息在载体上的标识和设计，并未涉及通讯录载体的技术特征
- **结论**：本质仍属于智力活动规则和方法
- **启示**：即使权利要求主题为产品，如果限定内容仅为信息编排方式且未影响产品物理结构，仍属于智力活动规则

---

## 判断流程

```
智力活动规则审查
├── 第一步：是否包含技术特征？
│   ├── 包含技术特征 → 不属于智力活动规则 ✓
│   └── 不包含技术特征 → 继续
├── 第二步：是否利用自然规律？
│   ├── 利用自然规律 → 不属于智力活动规则 ✓
│   └── 未利用自然规律 → 继续
├── 第三步：是否依赖人的主观意识？
│   ├── 依赖感知力、理解力、决策力 → 属于智力活动规则 ✗
│   ├── 完全依赖经验、态度 → 属于智力活动规则 ✗
│   └── 不依赖人的主观意识 → 继续
├── 第四步：主题名称为产品？
│   ├── 否 → 属于智力活动规则 ✗
│   └── 是 → 继续
├── 第五步：限定内容是否仅信息编排？
│   ├── 是，未影响物理结构 → 实质属于智力活动规则 ✗
│   └── 否，影响物理结构 → 不属于智力活动规则 ✓
└── 结论
    ├── 属于智力活动规则 → 不授予专利权 ✗
    └── 不属于智力活动规则 → 进入技术方案审查
```
