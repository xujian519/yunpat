# 高分子化合物-说明书撰写（中）- 参数与效果确认

> 本章内容来源于张清奎《化学领域发明专利申请的文件撰写与审查》（第3版）第二部分 第三章。

## 2. 可依据制备方法、物理化学参数，和/或用途和/或使用效果的实验数据确认高分子化合物的情形

有些情况下，高分子化合物的制备方法、物理化学参数，和/或用途和/或使用效果的实验数据隐含了该高分子化合物的某些结构和/或组成特征，使得所属技术领域的技术人员能够清楚地确认该高分子化合物，并将其与现有技术的已知产品相区别，则它们也可以作为确认该高分子化合物的依据。

### （1）依据制备方法确认高分子化合物

对于要求保护的高分子化合物，如果所属技术领域的技术人员根据其反应原理、制备工艺（反应原料、加料顺序、反应步骤和条件等）能够并且只能得到所述结构和/或组成的高分子化合物，则所述制备方法可以作为确认相应高分子化合物的依据，说明书中可以不记载该高分子化合物的结构确认数据。

**例如**，权利要求请求保护一种丁二烯-异戊二烯-苯乙烯嵌段共聚物，其特征在于该共聚物具有如下结构：S-I-B-I-S，其中S为聚苯乙烯片段，B为聚丁二烯骨架，I为聚异戊二烯嵌段，其中苯乙烯含量为10wt%~50wt%，异戊二烯含量为10wt%~75wt%，丁二烯含量为10wt%~75wt%。

说明书记载了这是一种典型的活性阴离子聚合。首先丁二烯在引发剂作用下反应，形成聚丁二烯链段B，然后加入异戊二烯，在聚丁二烯内段的两端形成聚异戊二烯链段I，最后加入苯乙烯，在聚异戊二烯链段两端形成聚苯乙烯链段，由制备方法中加料的先后顺序可以导出嵌段共聚物中各重复单元的连接关系为S-I-B-I-S，而且根据单体的加入量可以控制嵌段共聚物中各单体单元的含量，根据引发剂的用量可以计算共聚物的理论平均分子量。这种情况下，说明书中可以不提供确认该嵌段共聚物结构的数据，因为制备方法中单体的用量、单体的加入顺序已经隐含了产物的结构和/或组成，因此可以认为请求保护的嵌段共聚物的结构能够被清楚地确认。

### （2）结合制备方法和物理化学参数确认高分子化合物

对于请求保护的高分子化合物，说明书记载了该高分子化合物的制备方法和物理化学参数，但没有记载该高分子化合物的结构确认数据，如果所属技术领域的技术人员结合制备方法和物理化学参数必然能够得到该高分子化合物，且该物理化学参数是该高分子化合物的特定结构、组成、性能和/或效果的直接反映，则即使说明书中不记载该高分子化合物的结构和/或组成确认数据，该高分子化合物也被认为能够被清楚地确认。

**例如**，权利要求请求保护一种含二氮杂萘结构的聚醚酮，其特征在于构成该聚醚酮的重复结构单元为：

```
    O     O
    ||    ||
-[-O-Ar-O-Ar-C-O-]-
```

其中R₁、R₂为甲基。

说明书中记载了聚合物的结构式，但是未记载该聚合物的结构确认数据。说明书中还记载以碱金属或碱土金属的盐类为催化剂，通过如下的含二氮杂萘结构的双酚单体：

```
      R₁
      |
   HO-Ar-OH
      |
      R₂
```

其中R₁、R₂为甲基；

与二卤二苯酮为原料在高温下进行聚合反应制备所述聚醚酮，并且在实施例中给出了聚醚酮的玻璃化转变温度。

双酚单体中的酚羟基与二卤二苯酮中的卤素之间的反应是现有技术已知的反应，并且反应位点唯一。另外，说明书还记载了该聚合物产品的玻璃化转变温度及其测量方法，由于玻璃化转变温度与聚合物的结构及分子量密切相关，是评价聚合物耐热性的重要指标，因此，结合该聚合物的制备方法和物理化学参数（玻璃化转变温度）能够清楚地确认得到了所述结构的聚合物。

### （3）结合制备方法与用途和/或使用效果的实验数据确认高分子化合物

对于请求保护的高分子化合物，说明书记载了该高分子化合物的制备方法及其具体用途和/或使用效果的实验数据，但没有记载该高分子化合物的结构确认数据，如果所属技术领域的技术人员结合其制备方法和效果必然能够得到该高分子化合物，且能够确认该效果是由该高分子化合物直接带来的，则即使说明书中不记载该高分子化合物的结构和/或组成确认数据，该高分子化合物也被认为能够清楚地确认。

**例如**，权利要求请求保护一种具有通式所示结构的弱碱性高分子树脂：

```
       (CH₃)₃N⁺HCO₃⁻
            |
   -[-Ar-CH₂-]-ₙ
```

说明书中没有记载该高分子树脂的结构确认数据，只记载了所述弱碱性高分子树脂的结构式、制备方法以及反应原理，并记载了树脂成膜后的电导率以及透气率。

根据该反应原理可以看出，树脂结构中的聚合度n是由作为原料的聚合物决定的，其数值大小取决于所选择的原料聚合物的分子量，而该原料可以通过市售途径获得。制备弱碱性树脂通常采用氯甲基化和季铵化，而氯甲基化是亲电取代反应，CH₂Cl将进攻苯环的富电位。由于甲基是吸电子基，酚氧基是斥电子基，与-O-相连的苯环邻位（a,b位）的电子相对于与-C(O)-相连的苯环邻位比较富集，因此可以确定将在苯环的a,b位发生氯甲基化。另外，说明书记载了效果数据电导率，说明树脂中含有离子化结构，离子化后产生了离子交换能力，由使用效果的实验数据说明确实得到了具有所述离子结构的树脂，因此，由所述制备方法和使用效果的实验数据可以确认要求保护的树脂的结构。

应当注意，高分子学科与其他化学学科一样，是一门实验科学，在多数情况下，仅仅依据聚合物的制备方法、物理化学参数，和/或用途和/或使用效果的实验数据，无法直接或难以确认高分子化合物的结构，因此，申请人在说明书中应当尽可能记载高分子化合物的结构确认数据。

**反例**：申请要求保护"由下述通式1表示的重复单元组成的聚苯并咪唑-苯甲酰胺共聚物"：

```
      N
      ||
   -[-Ar-]-ₓ-[-Ar-CO-NH-Ar-]-₁₀₀₋ₓ
```

其中，x为0.1~99.9。

根据说明书的记载，现有技术中适用于燃料电池的电解质膜所采用的聚苯并咪唑电解质膜的质子传导率不高，本发明将聚苯并咪唑改造成由苯并咪唑重复单元和苯甲酰胺重复单元组成的共聚物。说明书在制备例中记载了聚苯并咪唑-苯甲酰胺共聚物的制备方法，具体地"在220℃下加热烧瓶中的95g多磷酸，向其中加入58g的5wt%的二氨基苯甲酸，然后搅拌。10分钟后，加入0.25g的5wt%的氨基苯甲酸。随着聚合进行，溶液变成暗红色高粘性的溶液"。在试验例中分别测定了由所述共聚物制备的电解质膜的质子传导率、粘度和机械强度，另外，说明书还记载了聚合物电解质膜的质子传导率-温度相关性和热稳定性。但申请说明书中没有公开该聚合物的结构确认数据。

根据说明书实施例中有关共聚物聚合步骤的描述，合成该共聚物的反应原料是二氨基苯甲酸和氨基苯甲酸，根据本领域技术人员的普通技术知识，二氨基苯甲酸是获得聚苯并咪唑嵌段的原料，之后添加的氨基苯甲酸中的氨基和羧基分别与聚苯并咪唑的活性终端羧基或氨基缩合反应形成另一聚合嵌段，而这样形成的聚合物显然与权利要求1所述由通式1表示的重复单元组成的聚苯并咪唑-苯甲酰胺共聚物的结构不符，因为通式1中显示的结构为氨基苯甲酸的苯基与苯并咪唑的苯环直接相连，本领域技术人员根据说明书记载的反应原理和本领域的普通技术知识可知，该苯并咪唑嵌段末端应当残留有羧基或氨基才能与氨基苯甲酸的氨基或羧基进一步缩合，两嵌段之间必须通过酰胺键相连。因此，本领域技术人员根据说明书制备例的反应步骤难以获得请求保护的权利要求中通式1所示的共聚物结构。另外，说明书中关于包含所制备聚合物的电解质膜的质子传导率、粘度、机械强度和热稳定性等效果实验数据也不能证明请求保护的权利要求中通式1所表示的共聚物结构。因此，说明书中对聚合物结构、制备方法和效果实验数据的记载没有达到使所属技术领域的技术人员能够确认该聚合物的程度。
