# 许诺销售侵权行为的认定

> **来源：** 《侵权判定指南(2017)理解与适用》第五章，第107条
> **核心法条：** 《专利法》第11条第1款、第2款
> **关联页面：** [[其他侵权-销售与许诺销售侵权]]、[[其他侵权-制造侵权行为]]、[[其他侵权-使用侵权行为]]、[[侵权判定-全面覆盖原则]]

---

## 核心要点

许诺销售是指在实际销售行为发生前，被诉侵权人作出销售侵犯他人专利权产品意思表示的行为。许诺销售的认定需满足两项要件：行为人具有实际销售目的，且产品处于能够销售的状态。我国专利法历经三次修改，逐步将许诺销售行为纳入发明、实用新型和外观设计专利权的禁止范围，以实现与TRIPS协议的接轨。

---

## 第107条 许诺销售

### 条文原文

> 在销售侵犯他人专利权的产品行为实际发生前，被诉侵权人作出销售侵犯他人专利权产品意思表示的，构成许诺销售。
>
> 以做广告、在商店橱窗中陈列、在网络或者在展销会上展出等方式作出销售侵犯他人专利权产品的意思表示的，可以认定为许诺销售。

### 理解与适用

#### 一、许诺销售的立法沿革

TRIPS协议第28条第1款规定，专利权人享有独占权，包括制止第三方未经专利权人许可而许诺销售专利产品或者依照专利方法直接获得的产品。我国为了与TRIPS协议的规定相一致，在2000年修改《专利法》时增加了许诺销售的规定，但彼时仅限于发明和实用新型专利。2008年修改《专利法》时，进一步将许诺销售纳入外观设计专利权的侵权类型。

**2008年修改的立法背景：** 随着我国外观设计专利申请量的日益增长，展会经济蓬勃发展，展会上侵犯知识产权的现象较为突出。由于2000年《专利法》未对外观设计许诺销售行为予以规范，导致展会期间对外观设计专利权的保护弱于发明和实用新型专利权。因此，2008年修法增加了外观设计的许诺销售禁止权。

#### 二、许诺销售的认定要件

**（一）许诺销售者具备实际销售目的**

实际销售目的是许诺销售的主观要件。许诺销售是为销售目的而向特定或非特定主体作出的愿意销售或将要销售专利产品的表示行为。如果行为人进行演示、展览的目的不是销售，而是诋毁有关专利技术等，则不构成许诺销售，可能构成不正当竞争。

**（二）产品处于能够销售的状态**

被诉侵权人不但应当具有即将销售侵犯专利权产品的明确意思表示，而且在作出该意思表示之时其产品应当处于能够销售的状态。专利法禁止许诺销售的目的在于尽可能早地制止侵权产品的交易，使专利权人在被诉侵权产品扩散之前就有可能制止侵权。

**（三）许诺销售行为的方式**

本条第二款以列举方式说明常见的许诺销售行为：做广告、在商店橱窗中陈列、在网络展出、在展销会上展出等。随着网络购物的普及，网络许诺销售行为日益盛行，此次修订特意增加"网络"这一新型许诺销售方式。

#### 三、许诺销售与销售的关系

许诺销售发生在实际销售之前，是销售行为的前期行为。一旦实际销售发生，许诺销售就转化为销售行为。在司法实践中，关于如何确认许诺销售的产品是侵权产品，原告往往遇到取证难的问题。法官应当坚持"谁主张谁举证"的证据审查规则，如果原告证据不充分，不能依照其提供的证据确认侵权产品，则其侵权指控不能被支持。

#### 四、仅存在许诺销售行为的法律责任

关于仅存在许诺销售行为时是否应当承担损害赔偿责任，司法实践中存在争议：

- **观点一：** 许诺销售尚未造成实际损害，仅需承担停止侵权及赔偿合理开支的责任
- **观点二：** 许诺销售同样可能给专利权人造成实质性损害（如交易机会丧失、专利产品价格下滑），应承担实际损失的赔偿责任

在沈其衡诉北京天汇万博科技发展有限公司侵犯实用新型专利权纠纷案中，法院认为：鉴于被告实施的是许诺销售行为，未通过该行为获得直接经济利益，亦未给原告造成直接经济损失，故不支持经济损害赔偿请求，但支持了合理费用赔偿请求。

### 典型案例：伊莱利利公司诉甘李药业有限公司发明专利侵权纠纷案

- **审理法院：** 北京市高级人民法院
- **案号：** （2007）高民终字第1598号
- **争议焦点：** 药品注册申请及网络宣传行为是否构成许诺销售
- **裁判要点：** 许诺销售以销售产品为直接目的。甘李公司的被控侵权产品尚未取得药品注册，现有证据不能证明其将在专利权有效期内从事生产、销售行为，故不构成许诺销售。
- **启示：** 许诺销售的认定要求产品处于能够销售的状态，且行为人具有在专利有效期内销售的目的。仅凭网站宣传不足以认定许诺销售成立。

### 典型案例：沈其衡诉北京天汇万博科技发展有限公司侵犯实用新型专利权纠纷案

- **审理法院：** 北京市高级人民法院
- **案号：** （2009）高民终字第4011号
- **争议焦点：** 仅存在许诺销售行为时的民事责任范围
- **裁判要点：** 许诺销售行为未给专利权人造成直接经济损失的，不支持经济损害赔偿请求，但应赔偿权利人为制止侵权支付的合理费用。
- **启示：** 在仅存在许诺销售的案件中，法院一般判令停止许诺销售行为并赔偿合理开支，但通常不判决赔偿实际损失，除非有证据证明实际损失存在。

## 判断流程

```
是否作出销售侵权产品的意思表示
        ↓
   是 → 是否具有实际销售目的
              ↓
         是 → 产品是否处于能够销售状态
                    ↓
               是 → 构成许诺销售侵权
```
