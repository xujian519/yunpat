# 撰写-权利要求书撰写步骤

> **来源：** 《发明和实用新型专利申请文件撰写案例剖析（第2版）》第一部分第三章，吴观乐主编，知识产权出版社，2004年，页41-54
> **核心法条：** 《专利法》第26条第4款、《专利法实施细则》第21-23条
> **关联页面：** [[撰写-权利要求书撰写要求]]、[[撰写-说明书撰写步骤]]、[[书籍/撰写案例剖析-吴观乐-全书骨架]]、[[书籍/撰写案例剖析-吴观乐-全书骨架]]

## 核心要点

权利要求书的撰写可归纳为五个步骤：理解技术内容→确定最接近现有技术→确定必要技术特征→撰写独立权利要求→撰写从属权利要求。撰写独立权利要求时应尽可能用上位概念或并列概括的方式，以获得较宽的保护范围。

---

## 一、撰写权利要求书的五个步骤

在理解发明或实用新型技术内容的实质和对相关现有技术进行调研后，就可以开始着手撰写权利要求书。一般说来，权利要求书的撰写可按下述五个步骤进行。

### 步骤一：深入理解发明，列出主要技术特征

在理解发明和实用新型的基础上，找出其**主要技术特征**，弄清各技术特征之间的关系。

**操作要点：**
- 仔细阅读技术交底书或发明人提供的技术资料
- 列出发明或实用新型的所有技术特征
- 分析各技术特征之间的结构关系、连接关系或作用关系
- 区分哪些是解决技术问题的必要技术特征，哪些是附加技术特征

### 步骤二：确定最接近的现有技术

根据检索和调研得到的现有技术，确定与本发明或实用新型**最接近的现有技术**。

**判断标准：**
- 技术领域相同或相近
- 技术问题相同或相近
- 技术特征相同或相近最多

### 步骤三：确定必要技术特征并概括

根据最接近的现有技术，进一步确定本发明或实用新型要**解决的技术问题**，从而列出本发明或实用新型为解决此技术问题所必须包括的**全部必要技术特征**，并应当尽可能用上位概念或并列概括的方式加以概括，以使其具有较宽的保护范围。

**上位概括的方法：**
- 将具体结构特征概括为功能性特征
- 将特定材料概括为材料类别
- 将具体尺寸概括为一个范围
- 将特定结构概括为更通用的结构形式

### 步骤四：撰写独立权利要求

与最接近的现有技术作比较，将它们**共同的必要技术特征**写入独立权利要求的**前序部分**，本发明或实用新型区别于最接近的现有技术的**必要技术特征**写入**特征部分**，从而完成独立权利要求的撰写。

**格式：**
```
一种[产品/方法名称]，包括[前序部分：与最接近现有技术共有的必要技术特征]，
其特征在于：[特征部分：区别于最接近现有技术的必要技术特征]。
```

### 步骤五：撰写从属权利要求

对其他附加技术特征进行分析，将那些有可能对申请的**创造性起作用**的技术特征作为对本发明或实用新型进一步限定的附加技术特征，写成相应的从属权利要求。

**从属权利要求的布局策略：**
- 将具体实施方式中的优选方案写成从属权利要求
- 将可能对创造性有贡献的附加技术特征写成从属权利要求
- 按照保护范围从大到小、重要性从高到低的顺序排列

## 二、案例：便携式牙刷的撰写示范

### （一）案情介绍

**发明名称：** 便携式旅行牙刷

**技术方案：** 该便携式旅行牙刷由牙刷本体、盒体和牙膏软袋组成。牙刷本体与盒体用铰链连接。盒体形状是细长方体，顶壁上有一个形状、大小与牙刷刷毛相应的空腔，当牙刷折叠起来放置时，牙刷刷毛正好落入此刷毛空腔内。盒体底部开有孔，置于盒底的牙膏软袋压板上有一凸块从孔中伸出。牙膏采用软袋包装，牙膏软袋放置在压板上，牙膏出膏口开在牙膏软袋上侧与刷毛空腔位置相对应处。出膏口上有螺纹，与牙膏旋盖的螺纹匹配。盒体一端有端盖，端盖内壁上有2至4个凸起，它与盒体侧端外壁上的凹坑相卡紧。

**使用方法：** 使用时，将牙刷本体转动一个角度，打开牙膏旋盖，再将牙刷本体转回来，使刷毛靠在出膏口上，按一下压板凸块，把牙膏挤在刷毛上，再将牙刷本体转动180度伸直，盖上牙膏旋盖即可刷牙。

### （二）现有技术检索

经检索和在市场上调查，找到三种便携式牙刷：

1. **第一种：** 由牙刷本体和带有空腔的牙刷柄组成。携带时牙刷本体放入牙刷柄的空腔中，使用时取出牙刷本体，将牙膏挤在刷毛上即可刷牙。缺点：牙膏和牙刷分别携带，使用不便。

2. **第二种：** 由盒体、普通牙刷、小袋牙膏组成，用完即弃。缺点：不能重复使用，浪费资源。

3. **第三种（最接近现有技术）：** 由盒体、普通牙刷、置于盒体中的牙膏软袋组成，盒体顶壁上有刷毛空腔。缺点：使用时必须取出牙膏软袋，将牙膏挤在刷毛上后再放回去，使用不方便。

### （三）确定必要技术特征

将本申请的技术方案与最接近的现有技术（第三种）进行比较：

**共有的必要技术特征（前序部分）：**
- 牙刷本体
- 兼作刷柄的盒体
- 置于盒体中的牙膏软袋
- 盒体顶壁上有供刷毛放入的空腔

**区别的必要技术特征（特征部分）：**
- 牙膏软袋出膏口位置与刷毛空腔位置相对应
- 盒体有一个挤压牙膏软袋的装置

**效果：** 不取出牙膏软袋即可将牙膏挤在刷毛上，使用方便。

### （四）独立权利要求的撰写

```
1. 一种便携式牙刷，由牙刷本体、兼作刷柄的盒体和置于
该盒体内的牙膏软袋组成，该牙刷本体与该盒体之间为活动连
接，其特征在于：所述盒体顶壁上有一个形状、大小与牙刷本体
上的刷毛相应的空腔，携带时该牙刷本体上的刷毛正好位于此空
腔内；所述牙膏软袋的出膏口位置与该刷毛空腔位置相对应；
所述盒体有一个挤压牙膏软袋的装置。
```

### （五）从属权利要求的撰写

```
2. 根据权利要求1所述的便携式牙刷，其特征在于：所述
挤压牙膏软袋的装置是一块位于所述牙膏软袋下方的、带凸块的
压板，该压板上的凸块从盒体底壁上的孔中伸出。

3. 根据权利要求1所述的便携式牙刷，其特征在于：所述
挤压牙膏软袋的装置是一块位于远离出膏口那一端、可向出膏口
方向移动的移动板。

4. 根据权利要求1所述的便携式牙刷，其特征在于：所述
盒体底壁上开有孔，该挤压牙膏软袋的装置是位于所述牙膏软袋
下方的、带凸块的压板，该压板上的凸块从盒体底壁上的孔中
伸出。
```

### （六）保护范围的优化

在独立权利要求中，"挤压牙膏软袋的装置"采用了**功能性限定**，而不是具体的挤压结构（如带凸块的压板）。这样撰写的独立权利要求保护范围较宽，涵盖了所有能够实现挤压牙膏软袋功能的结构。

但为了保护具体实施方式中的优选方案，可以将带凸块的压板等具体结构写入从属权利要求，形成**多层次的保护体系**。

## 三、并列独立权利要求的撰写示例

### 案例：沸腾液体传热壁

该申请案涉及一种用于沸腾液体的传热壁，包括产品、制造方法和专用设备三个发明点。可以撰写三个并列独立权利要求：

**产品权利要求：**
```
1. 一种用于沸腾液体的传热壁，传热壁外表面下方有许多
平行、细长的通道，外表面上沿着通道间隔地开有小孔，使通道
与传热壁外部相通，其特征在于：所述小孔中有一个从孔壁向孔
中心伸出的非对称突起，它在小孔横截面上的投影面积与小孔横
截面的面积比为0.4-0.8。
```

**方法权利要求：**
```
2. 一种制造权利要求1所述沸腾液体传热壁的方法，先在
管壁上形成彼此间隔很近的、顶部带有多个均匀分布切口的肋片，
再将肋片端部折弯，并与相邻肋片搭接，从而在外表面下方形成
平行细长的通道，肋片切口处成为沿通道间隔设置的小孔，其特
征在于：所述带切口的肋片是按照下述工艺步骤制得的，先在金
属管外表面上形成多条浅沟槽，此后用铲刮刀具沿其表面铲刮起
带切口的肋片，在此同时该铲刮刀具的后缘挤压该金属管上尚未
被铲刮起的外表面，使与其相邻的浅沟槽变形，在其底部部分形
成隆起，从而在铲刮下一个肋片时，该肋片的切口内有一个隆起
部分，这样在形成肋片后和当肋片折弯与相邻肋片搭接时，该隆
起部分就成为小孔内从孔壁向孔中心伸出的非对称突起。
```

**设备权利要求：**
```
3. 实现权利要求2所述制造沸腾液体传热壁方法中的专用
铲刮刀具，该刀具端部表面是倾斜的，形成锐角的铲刮前刃，其
特征在于：在所述刀具端部的后缘有一个高出端部斜面、可对尚
未铲刮起的金属管外表面起挤压作用的突出部分。
```

**单一性分析：** 这三个独立权利要求属于一个总的发明构思。方法权利要求的特征部分正是为了使产品上的小孔中形成非对称突起；设备权利要求的特征部分正是反映了为使该方法能在小孔中形成非对称突起而对刀具结构作出的改进。因此，三个独立权利要求从实质上看满足了单一性要求。

## 四、撰写权利要求书的注意事项

### 1. 保护范围的平衡

- 独立权利要求应当保护范围尽可能宽，但不得超出说明书公开的范围
- 从属权利要求应当对独立权利要求的保护范围作进一步限定，形成多层次保护
- 具体实施方式中的每个重要变型都应当考虑写入从属权利要求

### 2. 功能性限定的使用

- 功能性限定可以扩大保护范围，但必须以说明书中有具体实施方式支持为前提
- 对于机械领域的发明，如果说明书只给出一种具体结构，而独立权利要求使用了功能性限定，可能导致权利要求得不到说明书的支持

### 3. 避免保护范围过窄

在撰写独立权利要求时，若局限于最佳实施方式，往往会使保护范围过窄。应当仔细分析具体实施方式，确定哪些是解决技术问题必不可少的技术特征，在这些特征之中是否有一部分可用类似的结构达到相同或相近的效果，可否用概括性的描述来表达。
