# 如何判断一项技术方案是否具备专利的实用性？

## 卡片内容

### 核心概念

专利的实用性是专利授权审查中的一项关键性判断，旨在确保所申请的技术方案具有实际的技术价值和可操作性。审查指南明确列举了6种不具备实用性的典型情形，这些情形本质上是技术方案本身的固有缺陷，与说明书的公开程度无关。

### 6种不具备实用性的典型情形

1. **无再现性**：
   - **定义**：指所属技术领域的技术人员根据公开的技术内容，无法重复实施专利申请中为解决技术问题所采用的技术方案，或实施结果依赖于随机因素。
   - **区分**：与“成品率低”不同，无再现性意味着在确保全部技术条件的情况下，技术人员仍无法重复实现所要求达到的结果。
   - **判断要点**：需考察说明书是否充分公开了实施条件，实施过程中是否存在必须依赖的随机因素，以及技术人员是否能在相同条件下重复实施并获得相同结果。

2. **违背自然规律**：
   - **定义**：发明或实用新型专利申请应当符合自然规律。违背自然规律的发明不具备实用性，因为它在本质上是不可实施的。
   - **常见情形**：包括但不限于：
     - **永动机类**：如直接违背能量守恒定律（第一类永动机），或从单一热源完全转换为有用功而不产生其他影响（第二类永动机），这违反了能量守恒定律和热力学第二定律。
     - **效率超标**：声称机械能输出大于输入电能的电动机，或声称效率超过100%的能量转换装置。
     - **其他违背定律**：如声称能够接收天空电荷（违背静电平衡理论）或分离磁场产生能量等缺乏科学依据的方案。

3. **利用独一无二的自然条件的产品**：
   - **定义**：如果产品是由特定自然条件限定的，且该产品自始至终都是不可移动的唯一产品，则不具备实用性。
   - **区别**：与构件实用性不同，即使产品依赖特定自然条件，其构件本身如果可以独立制造和使用，则其构件可能具备实用性。
   - **典型情形**：如利用各大洋之间海峡的暖流循环工程，或依托特定地质条件的建筑方案。

4. **人体或动物体的非治疗目的的外科手术方法**：
   - **定义**：以有生命的人或动物为实施对象，但目的并非治疗疾病，则不具备实用性。
   - **典型情形**：
     - **美容手术**：为美容目的实施的外科手术。
     - **摘取物质**：采用外科手术从活体动物身体上摘取物质的方法。
     - **辅助诊断**：为辅助诊断而采用的外科手术方法（如在进行主要检查前采用的步骤）。
   - **例外**：如果方案包含的简单介入性处置步骤个体差异小，且不必须依靠医护人员的专业技能就能实施，则不宜认定为此类。

5. **测量人体或动物体在极限情况下的生理参数的方法**
   - **定义**：测量人体或动物体在极限情况下的生理参数需要将被测对象置于极限环境中，这会对生命构成威胁，且不同个体对极限条件的耐受度存在差异，导致无法标准化实施。
   - **典型情形**：
     - **体温耐受测量**：通过逐渐降低体温来测量人或动物对寒冷的耐受程度。
     - **冠状动脉负荷测量**：通过动态观察动脉血压变化来测量冠状动脉的代谢机能，该方法需要将被测对象置于有生命危险的极限环境中。
   - **区别**：与正常的、可标准化的医学检查方法不同，后者如果不会对生命构成威胁且可以标准化实施，则可能具备实用性。

6. **无积极效果**：
   - **定义**：技术方案应当能够产生预期的积极效果。明显无益、脱离社会需要的发明不具备实用性。
   - **历史演变**：审查指南曾列举“严重污染环境、严重浪费能源或损害人身体健康”等情形，但后续修订倾向于将此类判断留给市场机制，更侧重于“明显无益”或“脱离社会需要”。
   - **案例**：如通过路灯杆喷气消除城市空气污染的方法，如果仅能控制有限气流，无法实现清除所有污染物的预期效果，则不具备实用性。
