# 证据-鉴定意见与域外证据

> **来源：** 《以案说法》第十章 §1.3、§1.5，页323-324、328-330
> **核心法条：** 《专利审查指南》第四部分第八章证据规定
> **关联页面：** [[证据-书证与证人证言]]、[[证据-互联网证据的特殊规则]]、[[证据-举证责任分配与证明标准]]、[[证据的认定-证据资格与证明力]]、[[证据的认定-举证期限与证据调查]]、[[程序-无效宣告请求的审查]]、[[专利判决/程序与证据/程序与证据-证据规则]]

---

## 核心要点

鉴定意见的采信取决于鉴定机构资质、程序合法性和检材可靠性；域外证据需经所在国公证及中国使领馆认证，但公证认证仅证明取得途径合法，不当然证明内容真实。二者均需在形式合法性之外，进一步审查实质真实性。证据形式瑕疵在特定条件下可通过补正方式完善。

---

## 1. 鉴定意见

### 基本规则

鉴定意见是具有专门知识和技能的人员，对与案件有关的专门性问题，运用其专门知识或借助仪器设备，进行分析、鉴别作出的结论。对鉴定意见的审查可以考虑鉴定机构、鉴定人员的资质，鉴定程序、鉴定的依据以及所使用的技术手段等方面。

**鉴定机构资质：** 鉴定机构应具备法定资质，鉴定人员应具有相应执业资格。鉴定程序应由当事人协商确定或经审理机关指定。协商不成的，由专利复审委员会（现专利复审和无效审理部）指定有资质的鉴定机构。

**证据保全类鉴定意见：** 用于证据保全的鉴定意见不同于公证机关的公证。对其真实性的判断，应考虑检材的来源是否合法、真实，检材是否可以再现，鉴定人是否接受质询等。如果作为鉴定对象的检材已灭失或无法验证，且鉴定人未能到庭接受质询，鉴定意见不足以证明被鉴定对象的真实性。

### 案例：第4694号无效决定（96233118.X）

- **争议焦点：** 委托中国计量科学研究院对芬兰马克钞票防伪安全线进行鉴定的程序及结论是否合法有效
- **决定要点：** 由于双方当事人均未确定鉴定机构，专利复审委员会遂委托中国计量科学研究院进行鉴定。中国计量科学研究院是中华人民共和国国家法定计量检定机构，其鉴定人员、鉴定程序、鉴定依据、所使用的科学技术手段以及鉴定结论，无论从形式还是内容上均符合法律规定。双方当事人质证后均未提出有效的反驳意见，因此对该鉴定意见予以采信。
- **启示：** 在专利无效程序中，当事人协商不成时由专利复审委员会指定鉴定机构是标准程序。国家法定计量检定机构出具的鉴定意见，在形式和程序均合法的情况下，若无有效反驳意见，一般予以采信。

### 案例：第26939号无效决定（201430428191.2）

- **争议焦点：** 单方启动的网页证据保全鉴定意见能否证明网页内容的真实性
- **决定要点：** 作为鉴定对象的检材是司法鉴定意见正确与否的关键因素之一。与证据1相关的检材通过互联网已不能查询到，即检材灭失，当事人无法对其质证，检材是否在当时是互联网客观存在的网页信息无从查证。鉴定报告中的检材由请求人利用自己的电脑提交，其电脑是否经过清洁无法确认。同时，鉴定程序由请求人单方启动，鉴定人未能到庭接受质询。综合上述因素，仅凭证据2所示鉴定报告不足以证明证据1所示网页内容的真实性。
- **启示：** 证据保全类鉴定意见的核心在于检材的可靠性和程序的公正性。单方启动、检材来源不明、鉴定人未出庭、检材已灭失的鉴定意见，证明力极弱。实务中应尽量通过公证或双方认可的程序固定证据。

## 2. 域外证据

### 基本规则

**公证认证要求：** 除有特别规定，中华人民共和国领域外形成的证据应当经所在国公证机关进行公证，并经中华人民共和国驻该国使领馆予以认证，或者履行中华人民共和国与该所在国订立的有关条约中规定的证明手续。未履行公证认证手续的域外证据，对其真实性通常不予认可；但是，如果其待证事实得到其他证据的印证，则可以认可该证据的真实性。

**公证认证的效力边界：** 履行了公证认证手续并不意味着该域外证据必然具有真实性。对域外证据的真实性应从证据的表现形式、来源、内容等方面进行审核。公证认证仅证明证据的取得途径可以得到确认，不当然证明内容真实。

**国内公共渠道例外：** 如果域外证据能够从除香港、澳门、台湾地区外的国内公共渠道获得，则不需办理公证认证手续。例如通过国家图书馆馆际互借获取的外国文献。

**其他证据印证例外：** 根据《专利审查指南》的相关规定，对于域外证据，当"有其他证据足以证明该证据真实性的"，当事人可以在无效宣告程序中不办理相关的证明手续。

**港澳台证据的特殊处理：** 在香港、澳门、台湾地区形成的证据，属于域外证据的范畴，应当按照相关规定办理证明手续。在香港形成的证据，应当经中国委托公证人公证并由中国法律服务（香港）有限公司加章转递；在澳门形成的证据，应当经中国委托公证人公证并由中国法律服务（澳门）有限公司加章转递；在台湾地区形成的证据，应当经台湾地区的公证机构公证，并由海峡两岸交流基金会进行认证。在实务中，港澳台证据的证明手续与外国证据存在差异，代理人在提交此类证据时应特别注意对应程序要求。

### 案例：第13188号无效决定（99812498.2）

- **争议焦点：** 未履行公证认证手续的域外书籍复印件是否一律不予认可
- **决定要点：** 证据6属于未履行公证认证手续的域外证据，在证明形式上存在瑕疵。但是，鉴于证据6的作者、出版商、版次、相关页码与本专利说明书"发明背景"部分所引用的《抗生素生物技术》一书及其相关页码完全一致，并且证据6相关部分内容也与说明书所涉及和概括的内容对应一致，因此本专利说明书"发明背景"部分描述的上述内容能够佐证证据6的真实性。
- **启示：** 未履行公证认证手续的域外证据并非绝对排除。如果该证据的内容能被本专利说明书或其他在案证据所印证，真实性仍可得到认可。代理实务中若因客观原因无法及时办理公证认证，应积极寻找其他印证途径。

### 案例：第20736号无效决定（200730170517.6）

- **争议焦点：** 履行了公证认证手续的域外画册是否当然具有真实性
- **决定要点：** 附件5是由美国加利福尼亚州政府出具的公证书以及中华人民共和国驻旧金山总领事馆出具的认证手续，形式上无明显瑕疵。但附件5的公证书仅对"Tog Wu对附件3之画册的封面页和版权信息页进行了翻译"这一事项进行公证，该公证手续可以证明Tog Wu进行了所述翻译行为，但对于附件3之画册的来源无证明作用。在无其他证据加以证明的情况下，仅凭附件5的公证认证手续不足以证明附件3的真实性。
- **启示：** 公证认证的内容必须与待证事实匹配。仅证明翻译行为的公证，不能证明被翻译文件本身的来源和真实性。代理实务中应确保公证内容覆盖证据来源、内容真实性等关键要素。

### 案例：第14712号无效决定（01815839.0）

- **争议焦点：** 通过国内公共图书馆获取的外文文献是否仍需公证认证
- **决定要点：** 请求人提交了盖有"上海图书馆上海科学技术情报研究所文献服务部"印章的《馆外索取证明》，证明该文献服务部以馆际互借的方式从国家图书馆索取。该文件的ISBN号为1-886362-40-8，版权所有人为Iron & Steel Society。上述《馆外索取证明》已表明该文献属于可以通过国内公共渠道获得的公开出版物，因此即使未履行公证认证手续，也认可其真实性。
- **启示：** 通过国内公共图书馆馆际互借等合法公共渠道可获取的域外文献，属于《专利审查指南》规定的例外情形，无需办理公证认证。代理实务中应优先尝试通过国内图书馆获取域外文献。

### 案例：第34304号无效决定（201310630670.7）

- **争议焦点：** 通过互联网搜索引擎获取的域外书籍片段及后续补交的国内复制证明能否确认真实性
- **决定要点：** 对比文件5是从互联网搜索获得的外文书籍片段，专利权人质疑其真实性且认为属于域外证据未办理公证认证。请求人随后补交了国家图书馆科技查新中心出具的文献复制证明。经对比，证明文件与证据内容实质相同，ISBN号一致。国家图书馆科技查新中心是国家科技部认可的查新机构，其出具的文献复制证明真实可信。根据《专利审查指南》规定，有其他证据足以证明该证据真实性的，当事人可以不办理相关证明手续。
- **启示：** 对于通过互联网获取的域外出版物片段，可通过国家图书馆等权威机构出具文献复制证明来补强其真实性，从而免除公证认证手续。补交证明时应确保与原始证据的内容一致性。

## 3. 证据形式瑕疵的补正规则

### 基本规则

在无效宣告程序中，当事人可以在口头审理辩论终结前补交完善证据法定形式的相关证明文件，如公证书、原件等。但补交的范围仅限于完善原有证据的法定形式，不能实质上构成新的证据。

**可补正的情形：** 保藏证明的后续公证认证、外文文献的翻译件补充、原件与复印件一致性证明等。

**不可补正的情形：** 如果补充的证据证明了原证据未涉及的新来源、公开方式、公开时间或技术内容，则实质上已超出完善法定形式的范畴，相当于另行提出新的证据。

### 补正与重新举证的界限

| 情形 | 是否属于补正 | 处理方式 |
|------|------------|---------|
| 为已有复印件补充公证 | 是 | 口审辩论终结前可接受 |
| 为保藏证明补充签名人身份公证 | 是 | 综合对照后认可 |
| 补充外文证据的中文译文 | 是 | 口审辩论终结前可接受 |
| 为已有公证书补充翻译件 | 是 | 视为完善形式 |
| 证明新的公开方式或来源 | 否 | 视为新证据，受举证期限限制 |
| 证明新的公开时间 | 否 | 视为新证据，受举证期限限制 |
| 补充原证据未涉及的技术内容 | 否 | 视为新证据，受举证期限限制 |

**实务提示：** 补正的本质是"完善形式"而非"补充实质"。代理人在举证期限届满后补交文件时，应严格评估该补交是否超出了完善法定形式的范畴。若补交内容涉及新的待证事实、新的公开时间或新的技术内容，则面临不予接受的风险。建议在举证期限内尽可能提交完整的证据材料，避免依赖期限后的补正。
