## 核心审查标准

外观设计相同或近似的判断建立在两个前提之上：（1）对比产品属于相同或相近种类；（2）在整体观察、综合判断的基础上，判断是否具有明显区别。产品种类的认定以用途是否相同或相近为准，而非仅看名称或分类号。

---

## 决定要点

### 要点一：产品种类相同或相近的认定

认定产品种类是否相同或相近，应当以产品的用途是否相同或相近为准。判断依据包括：

1. **产品名称和分类**：参考国际外观设计分类表（洛迦诺分类）
2. **产品功能**：产品的实际功能和用途
3. **销售和使用情况**：产品在市场上的实际销售渠道和使用场景
4. **一般消费者的认知**：一般消费者是否认为属于同类产品

### 要点二：相同外观设计的认定

被诉设计与授权外观设计在整体视觉效果上**无差异**的，认定为相同。主要包括：

1. 形状、图案、色彩等全部设计要素完全一致
2. 仅在产品大小、材料等不影响整体视觉效果的因素上有差异

### 要点三：近似外观设计的认定

被诉设计与授权外观设计在整体视觉效果上**无实质性差异**的，认定为近似。判断时需考虑：

1. **更容易观察到的部位**：产品正常使用时容易被直接观察到的部位相对于其他部位更具影响
2. **区别于现有设计的特征**：授权外观设计区别于现有设计的设计特征相对于其他设计特征更具影响
3. **设计空间的考量**：设计空间较大的领域，一般消费者不容易注意到较小差异；设计空间较小的领域，较小差异也可能对整体视觉效果产生显著影响

### 要点四：不同类型外观设计的对比

| 对比情形 | 判断标准 |
|---------|---------|
| 形状对比 | 以形状为主要判断依据 |
| 形状+图案 | 综合考量形状和图案 |
| 形状+图案+色彩 | 三要素综合判断（色彩保护需在简要说明中声明） |
| 成套产品 | 各单件产品分别对比 |
| 组件产品 | 组合状态和使用状态分别对比 |
| 变化状态产品 | 所有状态综合判断 |

## 审查要点速查

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 产品名称不同但用途相同 | 认定为相近种类 | 第23条+审查指南 |
| 产品用途完全不同 | 不构成相近种类 | 第23条 |
| 整体视觉效果无差异 | 认定相同 | 第23条 |
| 整体视觉效果无实质性差异 | 认定近似 | 第23条 |
| 设计空间大，细微差异 | 一般不构成近似 | 审查指南 |
| 设计空间小，细微差异 | 可能构成近似 | 审查指南 |
