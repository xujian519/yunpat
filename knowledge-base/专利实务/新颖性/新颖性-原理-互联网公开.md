# 新颖性-原理-互联网公开

> **来源：** 崔国斌《专利法：原理与案例（第二版）》第4章 §3.1
> **核心法条：** 《专利法》第22条第5款；《专利审查指南》第二部分第三章第2.1.2.3节
> **关联页面：** [[新颖性-原理-书面公开]]、[[新颖性-原理-公开时间的判断]]、[[新颖性-原理-新颖性审查的实务要点]]

## 核心要点

互联网公开属于专利法意义上的出版物公开，其公开日的认定以信息处于公众想得知就能够得知的状态为准。互联网公开不受地理位置、语言或获得方式的限制，但网络内容的易变性和可删除性给举证带来了特殊挑战。

---

## 1. 互联网公开的法律地位

### 1.1 属于出版物公开

依据《专利审查指南》，专利法意义上的出版物包括"以互联网或其他在线数据库形式存在的文件等"。因此，互联网公开在性质上属于**书面公开（出版物公开）**的一种特殊形式，而非"其他方式公开"。

互联网作为公开媒介的特征：
- 信息传播不受地理限制，全球可访问
- 不受传统出版周期约束，可即时发布
- 信息形式多样：网页、电子书、数据库、论坛帖子、博客、视频等
- 可复制性强，传播速度快

### 1.2 与传统出版物的区别

| 特征 | 传统出版物 | 互联网公开 |
|------|----------|-----------|
| 载体 | 纸质、胶片、磁带等实体 | 数字化信息 |
| 获取方式 | 图书馆、书店等物理渠道 | 网络访问 |
| 公开时间认定 | 印刷日期、发行日期 | 上传时间、发布时间 |
| 稳定性 | 相对稳定，不易篡改 | 易修改、易删除 |
| 举证难度 | 相对容易（实物保存） | 相对困难（需公证保全） |

## 2. 互联网公开的认定标准

### 2.1 "为公众所知"的标准

互联网公开同样适用"公众想得知就能够得知"的标准。关键在于：
- 信息是否发布在**对公众开放**的平台上
- 是否需要特殊权限（如密码保护、会员限制）才能访问
- 是否处于公众可以通过常规搜索手段发现的状态

### 2.2 需要密码或权限访问的情形

如果网络内容需要密码或特定权限才能访问，是否构成公开需要具体分析：
- **完全开放访问**：构成公开
- **注册即可访问（无需审核）**：一般构成公开
- **需要审批或特定身份**：可能不构成公开
- **仅限特定群体内部访问**：不构成公开

### 2.3 搜索引擎收录的影响

网页被搜索引擎收录通常可以作为该网页已经公开的一个佐证，但并非决定性因素。关键在于该网页在申请日之前是否确实处于公众可访问的状态。

## 3. 互联网公开时间的判断

### 3.1 公开日的认定难点

互联网内容的公开日认定面临以下挑战：
- 网页显示的时间可能被修改
- 论坛帖子的发布时间可能被后台更改
- 网络数据库的更新频率和具体时间点难以精确确定
- 网站迁移或改版可能导致原始发布时间信息丢失

### 3.2 举证要求

在专利无效宣告程序或侵权诉讼中，主张互联网公开的一方通常需要提供：

1. **网页截图或打印件**：显示网页内容和访问时间
2. **网络公证**：由公证机关对网页内容进行保全证据公证
3. **第三方存档证明**：如互联网档案馆（Internet Archive）的存档记录
4. **后台数据**：由网站运营方提供的信息发布记录

### 3.3 境外网站内容的举证

境外网站上的公开内容属于域外证据。依据相关规定，域外证据应当经所在国公证机关证明并经中国驻该国使领馆认证。但在以下情况下可以简化：
- 能够从国内公共渠道获得的国外专利文件
- 有其他证据足以证明真实性
- 对方当事人认可真实性

## 4. 实务要点

### 4.1 网络公开的稳定性

考虑到网络内容容易被修改或删除，在专利申请日前对可能构成现有技术的网络信息进行及时保全非常重要。对于专利权人而言，如果依赖网络公开作为现有技术抗辩或无效宣告的证据，应当尽早进行公证保全。

### 4.2 社交媒体与学术平台

近年来，社交媒体（如微博、微信公众号）、学术预印本平台（如中国知网的网络首发）等新型网络发布渠道日益增多。这些内容如果处于公众可自由访问的状态，同样可以构成专利法意义上的公开。但需要注意：
- 微信公众号文章需要判断是否对所有人开放
- 学术平台的"在线首发"论文通常视为已公开
- 社交媒体的私密或限定分享内容不构成公开

### 4.3 动态网页与数据库

对于动态生成的网页内容或在线数据库，公开时间的认定更为复杂。需要考察：
- 该内容是否在申请日之前就已经存在于数据库中
- 公众是否可以通过常规查询方式检索到该内容
- 数据库的更新机制和日志记录

### 4.4 与"其他方式公开"的区分

需要特别注意的是，互联网公开属于"出版物公开"的范畴，而非"其他方式公开"。这一区分在2008年《专利法》修订前具有实际意义，因为修订前"其他方式公开"仅限于国内地域范围，而出版物公开没有地域限制。将互联网公开认定为出版物公开，意味着即使该信息存储在境外服务器上，也能够构成中国专利法意义上的现有技术。

杭州顶津食品有限公司诉专利复审委员会案（（2007）一中行初字第73号）虽然涉及的是在台湾地区播出的电视广告，但法院在判决中区分了"出版物公开"和"其他方式公开"：电视广告不属于出版物公开，而属于"其他方式"公开。由此可以类推，互联网上的信息资料因其具有"记载有技术内容的独立存在的传播载体"的属性，应当归入出版物公开。

### 4.5 网络首发论文与学术预印本

随着学术出版模式的变革，"网络首发"（Online First）已成为期刊论文常见的发布方式。中国知网、万方等数据库的网络首发论文通常在正式排版前即上传至网络，供公众检索和下载。此类网络首发论文如果处于公众可自由访问的状态，一般应当认定为已经公开，其网络首发日期可以作为公开日。

学术预印本平台（如arXiv、bioRxiv、SSRN等）上的论文同样构成专利法意义上的公开，前提是平台对公众开放访问。需要注意的是，部分预印本平台允许作者在论文正式发表前进行修改或撤回，这种情况下公开日的认定可能需要结合平台的版本控制记录进行判断。

### 4.6 电子证据的保全方式

在涉及互联网公开的专利案件中，电子证据的保全至关重要。常见的保全方式包括：

- **公证保全**：由公证机关对网页内容进行实时保全，出具公证书。这是最常用、证明力最强的保全方式。
- **时间戳认证**：由可信时间戳服务机构对电子数据的内容和时间进行认证，成本较低，适用于大量网页的批量保全。
- **区块链存证**：利用区块链技术的不可篡改性对电子证据进行存证，逐渐被司法实践所接受。
- **互联网档案馆存档**：Wayback Machine等互联网存档服务的快照记录可以作为公开时间的辅助证据，但证明力相对较弱，通常需要与其他证据结合使用。

无论采用何种保全方式，都应当尽可能完整地记录网页的URL、访问时间、页面内容以及周围环境（如浏览器地址栏、系统时间等），以确保证据链的完整性。
