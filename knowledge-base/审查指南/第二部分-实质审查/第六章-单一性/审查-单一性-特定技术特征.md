# 审查-单一性-特定技术特征

> **来源：** 《专利审查指南》第二部分第六章 §2.2 单一性的审查
> **对应法条：** 《专利法》第31条第1款；《专利法实施细则》第39条
> **最新修订：** 无修改
> **关联页面：** [[审查-单一性-基本概念与审查原则]]、[[审查-单一性-审查示例]]、[[单一性-特定技术特征与特定领域判断]]

---

## 核心要点

单一性审查的核心是判断权利要求中是否包含使各项发明在技术上相互关联的**相同或者相应的特定技术特征**。审查员可按照六种允许的独立权利要求组合方式检查形式合规性，但更关键的是判断实质上的技术关联。特定技术特征是**相对于现有技术**而言的，因此部分申请的单一性判断需要在检索现有技术之后才能确定。

---

## 审查标准

### 一、审查原则（§2.2.1）

审查员在审查发明专利申请的单一性时，应当遵循以下基本原则：

#### 1.1 判断标准

判断一件专利申请中要求保护的两项以上发明是否满足单一性要求，就是要看权利要求中记载的技术方案的实质性内容是否属于一个总的发明构思，即判断这些权利要求中是否包含使它们在技术上相互关联的一个或者多个**相同或者相应的特定技术特征**。

- 这一判断是根据**权利要求**的内容来进行的；
- 必要时可以参照**说明书和附图**的内容。

#### 1.2 允许的独立权利要求组合方式

属于一个总的发明构思的两项以上发明的权利要求可以按照以下**六种方式之一**撰写。但须注意，即使按照这六种方式撰写，如果不具有相同或相应的特定技术特征，仍然不能允许在一件申请中请求保护：

| 编号 | 组合方式 | 说明 |
|------|---------|------|
| (i) | 同类独立权利要求 | 不能包括在一项权利要求内的两项以上产品或者方法的同类独立权利要求 |
| (ii) | 产品 + 专用制造方法 | 产品和专用于制造该产品的方法的独立权利要求 |
| (iii) | 产品 + 用途 | 产品和该产品的用途的独立权利要求 |
| (iv) | 产品 + 方法 + 用途 | 产品、专用于制造该产品的方法和该产品的用途的独立权利要求 |
| (v) | 产品 + 方法 + 设备 | 产品、专用于制造该产品的方法和为实施该方法而专门设计的设备的独立权利要求 |
| (vi) | 方法 + 设备 | 方法和为实施该方法而专门设计的设备的独立权利要求 |

**各组合方式的具体说明：**

**第(i)种——同类独立权利要求：** "同类"是指独立权利要求的类型相同，即一件专利申请中所要求保护的两项以上发明仅涉及产品发明，或者仅涉及方法发明。只要有一个或者多个相同或者相应的特定技术特征使多项产品类或方法类独立权利要求之间在技术上相关联，则允许。

**第(ii)至第(vi)种——不同类独立权利要求的组合：**

- **"专用"方法：** 该方法使用的结果就是获得该产品，两者在技术上相关联。但"专用"并不意味着该产品不能用其他方法制造。
- **产品用途：** 该用途必须是由该产品的特定性能决定的，在技术上相关联。
- **"专门设计"的设备：** 除了该设备能够实施该方法外，该设备对现有技术作出的贡献还必须与该方法对现有技术作出的贡献相对应。"专门设计"并不意味着该设备不能用来实施其他方法，或者该方法不能用其他设备来实施。

**不同类独立权利要求之间是否按照引用关系撰写，只是形式上的不同，不影响单一性。**

**注意：** 以上六种方式并非穷举，在属于一个总的发明构思的前提下，还允许有其他的组合方式。

#### 1.3 其他审查原则

**（4）排列方式和并列选择：**
- 评定单一性无须考虑发明是在各自的独立权利要求中还是在同一项权利要求中作为并列选择的技术方案要求保护；
- 权利要求的排列次序也不应当影响单一性的判断。

**（5）独立权利要求与从属权利要求：**
- 一般情况下，审查员只需要考虑**独立权利要求**之间的单一性；
- 从属权利要求与其所从属的独立权利要求之间不存在缺乏单一性的问题；
- 但形式上为从属权利要求而实质上是独立权利要求的，应当审查其单一性；
- 如果一项独立权利要求因缺乏新颖性、创造性而不能被授予专利权，则需要考虑其从属权利要求之间是否符合单一性规定。

**（6）检索前后的判断时机：**
- 某些申请的单一性可以在**检索现有技术之前**确定；
- 某些申请的单一性则只有在**考虑了现有技术之后**才能确定。

### 二、单一性审查的方法（§2.2.2）

#### 2.1 检索前的初步判断

在对包含在一件申请中的两项以上发明进行检索之前，应当首先判断它们之间是否**明显不具有单一性**。

**明显不具有单一性的情形：**
- 几项发明没有包含相同或相应的技术特征；或
- 所包含的相同或相应的技术特征均属于本领域**惯用的技术手段**。

上述情形下不可能包含相同或相应的体现发明对现有技术作出贡献的特定技术特征，因而明显不具有单一性。

> **例如：** 一件申请中包括除草剂和割草机两项独立权利要求，两者之间没有相同或者相应的技术特征，更不可能有相同或者相应的特定技术特征，检索前即可判断缺乏单一性。

#### 2.2 检索后的分析方法

对于不明显缺乏单一性的两项以上发明，通常采用以下分析方法：

**步骤一：** 将第一项发明的主题与相关的现有技术进行比较，确定体现发明对现有技术作出贡献的**特定技术特征**。

**步骤二：** 判断第二项发明中是否存在一个或者多个与第一项发明**相同或者相应的特定技术特征**，从而确定这两项发明是否在技术上相关联。

**步骤三：**
- 如果存在相同或相应的特定技术特征 → 属于一个总的发明构思 → **具有单一性**；
- 如果不存在 → 不属于一个总的发明构思 → **不具有单一性**。

#### 2.3 现有技术变化对单一性的影响

由于特定技术特征是相对于现有技术而言的，当申请与现有技术比较后，在**否定了第一独立权利要求的新颖性或创造性**的情形下，与其并列的其余独立权利要求之间是否还属于一个总的发明构思，应当**重新确定**。

## 实务应对

### 1. 独立权利要求的组合撰写

代理人在撰写多项独立权利要求时，应：
- 优先按照六种允许的组合方式安排权利要求结构；
- 确保不同独立权利要求之间具有相同或相应的特定技术特征；
- 在说明书中明确描述各项发明之间的技术关联性。

### 2. "专用"和"专门设计"的合理运用

- 产品与制造方法组合时，方法应为"专用于"制造该产品，但并非排他性的；
- 方法与设备组合时，设备应为实施方法而"专门设计"，且设备对现有技术的贡献应与方法对现有技术的贡献相对应；
- 在说明书中充分说明产品-方法、方法-设备之间的技术关联。

### 3. 面对单一性审查意见的应对

**步骤一：** 核实审查员认定的"特定技术特征"是否正确：
- 审查员是否正确理解了发明对现有技术的贡献；
- 共同技术特征是否确实不构成"特定技术特征"（即是否属于惯用手段）。

**步骤二：** 如果审查员在否定第一独立权利要求后重新判断单一性：
- 检查从属权利要求或并列独立权利要求中是否存在新的特定技术特征；
- 考虑修改权利要求以引入或突出特定技术特征。

**步骤三：** 如果确实缺乏单一性：
- 选择保留最重要的发明，删除其余发明；
- 对删除的发明及时提出分案申请。

### 4. 马库什权利要求的注意

对于化学领域的马库什权利要求，单一性判断有特殊规定（第十章第8.1节）。代理人应注意马库什权利要求中各可选择要素之间是否具有相同或相应的特定技术特征。

## 审查要点速查

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 产品+专用制造方法（共享特定技术特征） | 具有单一性 | §2.2.1 组合方式(ii) |
| 产品+非专用方法 | 需进一步判断 | "专用"要求技术关联 |
| 两项产品权利要求共享核心创新特征 | 具有单一性 | §2.2.1 组合方式(i) |
| 共同特征仅为惯用技术手段 | 不具有单一性 | §2.2.2 惯用手段不构成特定技术特征 |
| 第一独立权利要求被否定后 | 需重新判断单一性 | §2.2.1(6) |
| 不同类权利要求引用关系撰写 | 不影响单一性 | §2.2.1 引用关系仅为形式 |
| 从属权利要求vs所从属的独立权利要求 | 不存在单一性问题 | §2.2.1(5) |
