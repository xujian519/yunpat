## 审查要点速查

| 情形 | 审查结论 | 关键依据 |
|------|---------|---------|
| 与对比文件技术内容完全相同 | 不具备新颖性 | §3.2.1 相同内容 |
| 与对比文件仅文字表述不同 | 不具备新颖性 | §3.2.1 简单文字变换 |
| 对比文件使用公知术语涵盖权利要求特征 | 不具备新颖性 | §3.2.1 直接确定的内容 |
| 区别仅在于螺钉→螺栓固定 | 不具备新颖性 | §3.2.3 惯用手段置换 |
| 区别在于替换后产生不同技术效果 | 需进一步分析 | 超出惯用手段置换范畴 |
| 对比文件可推导但需推测 | 不属于相同 | §3.2.1 需直接确定 |
| 对比文件公开下位概念，权利要求用上位概念 | 丧失新颖性 | §3.2.2 下位破坏上位 |
| 对比文件公开上位概念，权利要求用下位概念 | 不丧失新颖性 | §3.2.2 上位不破坏下位 |
| 对比文件数值落在权利要求数值范围内 | 丧失新颖性 | §3.2.4(1) |
| 数值范围部分重叠或有共同端点 | 丧失新颖性 | §3.2.4(2) |
| 权利要求数值落在对比文件范围内且无共同端点 | 不丧失新颖性 | §3.2.4(4) |
