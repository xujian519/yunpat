# 高分子化合物-权利要求书（上）- 结构表征

> 本章内容来源于张清奎《化学领域发明专利申请的文件撰写与审查》（第3版）第二部分 第三章。

## 二、权利要求书的撰写

高分子化合物发明专利申请的权利要求书，应当在申请说明书满足《[[专利法]]》第26条第3款规定的前提下，根据现有技术状况，选择适当要素的组合，确切地限定请求保护的高分子化合物，以确定权利要求的保护范围。下面举例说明高分子化合物权利要求的一般撰写方式。

### （一）采用结构和/或组成特征表征

权利要求限定的特征应当与权利要求的类型相适应，高分子化合物通常应当用其结构和/或组成特征来描述。采用结构特征进行表征是最直接、最明确的产品限定方式，它使得所属技术领域的技术人员能够根据权利要求的文字描述，在化学结构上确定产品的保护边界。一般来说，对于主链由一种链节重复结构单元构成的基本上线性的聚合物，至少应当用重复单元数和/或分子量进行表征。这是因为，链节结构单元是高分子化合物的基本结构单元，链节结构不同的聚合物具有不同的性质。即使链节结构单元相同的聚合物，由于所含链节结构单元的数目（聚合度）或分子量不同，聚合物的性质也不同。另外，平均分子量相同的聚合物，分子量分布不同，聚合物的性质也不一样。因此，有时根据发明所要解决的技术问题不同，需加上[[个人笔记/个人笔记-说明书和权利要求书-说明书应当充分公开]]这一技术特征来限定请求保护的高分子化合物。

链节结构单元是高分子化合物的基本结构单元，链节结构不同的聚合物具有不同的性质。即使链节结构单元相同的聚合物，由于所含链节结构单元的数目（聚合度）或分子量不同，聚合物的性质也不同。另外，平均分子量相同的聚合物，分子量分布不同，聚合物的性质也不一样。因此，有时根据发明所要解决的技术问题不同，需加上[[个人笔记/个人笔记-说明书和权利要求书-说明书应当充分公开]]这一技术特征来限定请求保护的高分子化合物。

**【例1】** 一种主链由下述重复结构单元构成的分子量为10 000~100 000的聚酰胺：

```
   O         O
   ||        ||
-[ -C-(CH₂)₄-C-NH-(CH₂)₆-NH- ]-
```

**【例2】** 一种用下列结构式表示的聚醚：

```
CH₃
|
HO-(-CH₂CH₂O-)ₐ-(-CHCH₂O-)ᵦ-(-CH₂CH-)꜀-CH₂CH₂OH
```

式中a为50~70，b为20~30，c为30~50的整数。

**【例3】** 在汽车制造工业中，粘性结合的金属零件需要经过清洗、磷化处理和浸渍打底操作，现有技术的端羟基液体聚丁二烯橡胶粘合剂可能因这些操作中所用的处理剂而从粘合接缝中漂洗出来，因此，需要改进其粘结强度。另外，为了提高汽车在事故中的抗冲击能力，要求粘合剂在低温下表现出弹性特性和冲击韧性特性，而现有技术的端羟基液体聚丁二烯橡胶粘合剂均没有表现出这一性能。为此，发明人发现，通过对现有技术的具有特定端羟基的液体聚丁二烯进行改进，设定分子量分布在特定范围内，可改进其强度，同时为改进该液体聚丁二烯在分子量较高情况下，粘度过大带来的加工问题，作为分子链主体的聚丁二烯要求具有一定摩尔比的1,2-和1,4-结构，这也是使其区别于现有技术已知产品的特征所在。

因此，申请人可以将请求保护的权利要求撰写为："一种端羟基液体聚丁二烯，其数均分子量为500~5 000，分子量分布为1.5~3.0，聚丁二烯链中1,2-结构与1,4-结构的摩尔比为1:3~1:5"。

申请人通过用分子量、分子量分布、端基结构以及聚丁二烯链的1,2-和1,4-结构的摩尔比，并结合主题名称聚丁二烯描述请求保护的产品，所属技术领域的技术人员据此能够清楚地确定其请求保护的范围。该案例说明，当发明点在于对已知聚合物进行改进以获得新的性能时，权利要求应当包含那些能够体现该改进点的关键结构参数，从而确保权利要求的保护范围与发明贡献相匹配。

---

对于主链由两种或两种以上不同链节结构单元构成的共聚物，申请人至少应当用不同重复结构单元的名称（分子式或结构式）、及其比例与排列状态，和共聚物的分子量进行表征。

两种单体A和B共聚合得到的共聚物，按两种链节连接的序列不同，可分为[[个人笔记/个人笔记-说明书和权利要求书-说明书应当充分公开|交替]]、[[个人笔记/个人笔记-说明书和权利要求书-说明书应当充分公开|无规]]、[[个人笔记/个人笔记-说明书和权利要求书-说明书应当充分公开|嵌段]]、[[个人笔记/个人笔记-说明书和权利要求书-说明书应当充分公开|接枝]]共聚物等。不同序列构成的共聚物性质不同。例如，由苯乙烯（S链段）和丁二烯（B链段）两种单体链节组成的嵌段共聚物，在总链节比（SB）相同的情况下，以SBS和BSB序列排列的不同的嵌段共聚物，其物理性能，如断裂伸张、冲击强度、透光率等有较大差别。因此，这类聚合物除了表述清楚构成共聚物的重复结构单元及其比例关系外，还应当说明各重复单元的排列状态。

**【例4】** 一种用式A-O-CH₂CH₂-OCO-Ar-CO-表示的结构单元和用式B-O-CH₂CH₂-OCO-Ar-CO-表示的结构单元，以4~6:6~4的摩尔比线形无规排列的分子量为50 000~100 000的聚酯。

该案例中，申请人给出了重复单元的结构式、排列状态（无规）及其比例关系，以及所得聚酯的分子量，因此所属技术领域的技术人员能够清楚地确定该共聚酯的保护范围。

**【例5】** 一种由下述结构式I和II表示的结构单元构成的平均分子量为7 000~100 000的交替共聚物：

结构式I：
```
   -[-CH₂-CH-]-
         |
         O-CO-(CH₂)ₙ-CH₃
```

式中n为2~28的整数；

结构式II：
```
   -[-CH₂-CH-]-
         |
         O-CO-Ar-CO-O-Ar-O-...
```

该案例中，申请人给出了重复单元的结构式、重复单元的排列状态（交替共聚物）、聚合物的分子量，并且根据重复单元交替共聚的排列状态可以确定二者的比例关系，因此所属技术领域的技术人员能够清楚地确定该交替共聚物的保护范围。

---

## 小结

采用结构和/或组成特征表征是高分子化合物权利要求撰写的基本原则。申请人在撰写时应当根据高分子化合物的具体结构类型（线性均聚物、共聚物、支化或交联聚合物等），选择恰当的结构参数进行限定。对于线性聚合物，重复单元和分子量是基础；对于共聚物，还需明确各单元的比例和排列状态；对于星型、接枝等特殊结构，则需要补充臂数、接枝率等特定参数。合理的特征选择不仅能够清楚地界定保护范围，也有助于在审查和无效程序中维护权利要求的稳定性。
